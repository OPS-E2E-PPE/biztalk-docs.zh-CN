# <a name="tutorial-3-hosting-the-echo-adapter-in-iis"></a>教程 3:承载在 IIS 中的 Echo 适配器
本教程提供了用于托管 Echo 适配器开发的分步说明[教程 1:开发 Echo 适配器](../../adapters-and-accelerators/wcf-lob-adapter-sdk/tutorial-1-develop-the-echo-adapter.md)。 更具体地说的步骤说明如何通过使用托管在 Internet 信息服务 (IIS) 的适配器[!INCLUDE[afsvcdevwizshort](../../includes/afsvcdevwizshort-md.md)]。 将还使用在 SharePoint 中的业务数据目录功能来调用 IIS 为宿主的适配器，此 EchoGreetings 操作，然后在 Web 部件中显示结果。  
  
> [!TIP]
>  Echo 适配器、 测试代码和安装脚本是在你 BizTalk 安装文件附带`\BizTalk Server\ASDK_x86\Program Files\WCF LOB Adapter SDK\Documents\Samples`或`\BizTalk Server\ASDK_x64\Program Files\WCF LOB Adapter SDK\Documents\Samples`。
