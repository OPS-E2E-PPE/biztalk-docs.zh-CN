# <a name="tutorials-to-learn-the-wcf-lob-adapter-sdk"></a>教程，了解 WCF LOB 适配器 SDK
[!INCLUDE[afproductnameshort](../../includes/afproductnameshort-md.md)]教程包含有关如何使用信息[!INCLUDE[afproductnameshort](../../includes/afproductnameshort-md.md)]开发功能丰富的业务适配器，以便于您的企业中企业应用程序集成的行。 通过使用[!INCLUDE[afproductnameshort](../../includes/afproductnameshort-md.md)]，操作和公开的数据可供任何应用程序都可以使用 WCF 绑定，其中包括[!INCLUDE[btsBizTalkServerNoVersion](../../includes/btsbiztalkservernoversion-md.md)]。  
  
 教程包含围绕一组预定义操作的简单示例。 不需要可用于完成这些教程的业务系统的实际行。 但是，可以从了解元数据与编写出站处理程序及其他适配器开发需求，应用在教程中提供详细信息。  

> [!TIP]
> 已完成的 Echo 适配器随 BizTalk 安装文件`\BizTalk Server\ASDK_x86\Program Files\WCF LOB Adapter SDK\Documents\Samples`或`\BizTalk Server\ASDK_x64\Program Files\WCF LOB Adapter SDK\Documents\Samples`。
  
 使用以下教程可以了解如何使用关键部分的[!INCLUDE[afproductnameshort](../../includes/afproductnameshort-md.md)]:  
  
- [教程 1:开发 Echo 适配器](../../adapters-and-accelerators/wcf-lob-adapter-sdk/tutorial-1-develop-the-echo-adapter.md)。 提供用于创建从一组预定义的方法，后者将作为业务系统的行的字符串操作公开的适配器的分步说明。 适配器提供的实现中常见的处理程序的许多[!INCLUDE[afproductnameshort](../../includes/afproductnameshort-md.md)]包括搜索、 浏览、 入站和出站操作。 在本教程的成功完成后，将拥有的功能的适配器。  
  
- [教程 2：使用 Echo 适配器通过.NET](../../adapters-and-accelerators/wcf-lob-adapter-sdk/tutorial-2-consume-the-echo-adapter-from-net.md)。 对于使用 Echo 适配器开发中提供的分步说明[教程 1:开发 Echo 适配器](../../adapters-and-accelerators/wcf-lob-adapter-sdk/tutorial-1-develop-the-echo-adapter.md)从.NET 项目。 将添加到新项目的适配器服务引用，并提供代码来测试 Echo 适配器的入站和出站操作。  
  
- [教程 3：承载在 IIS 中的 Echo 适配器](../../adapters-and-accelerators/wcf-lob-adapter-sdk/tutorial-3-hosting-the-echo-adapter-in-iis.md)。 提供了用于托管 Echo 适配器开发的分步说明[教程 1:开发 Echo 适配器](../../adapters-and-accelerators/wcf-lob-adapter-sdk/tutorial-1-develop-the-echo-adapter.md)Internet 信息服务 (IIS) 中处理。 您将使用 svcutil.exe （ServiceModel 元数据实用工具） 来创建简单的客户端应用程序以使用托管适配器 EchoString 操作。  
  
  这些教程提供了了解的一些关键功能的结构化的方法[!INCLUDE[afproductnameshort](../../includes/afproductnameshort-md.md)]。 它们可以已完成，但不知道[!INCLUDE[afproductnameshort](../../includes/afproductnameshort-md.md)]或[!INCLUDE[nextref_btsWinCommFoundation](../../includes/nextref-btswincommfoundation-md.md)]。 但是，你将了解如果您了解适配器设计背后的概念并了解如何在中得以适配器设计[!INCLUDE[afproductnameshort](../../includes/afproductnameshort-md.md)]。 有关详细信息，请参阅：
