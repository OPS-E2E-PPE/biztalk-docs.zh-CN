# <a name="operational-tasks"></a>操作任务
Microsoft[!INCLUDE[A4SWIFT_CurrentVersion_FirstRef](../../includes/a4swift-currentversion-firstref-md.md)]包含一个或多个服务器和一系列企业应用程序集成 (EAI)，自动化财务业务流程，并促进使用 SWIFT FIN 标准的消息交换的工具。  

 操作包括管理和管理 BizTalk Server 应用程序。 A4SWIFT 使用以下工具和控制台执行这些操作：  

- **BizTalk Server 管理控制台。** 使用此工具用于管理 BizTalk 组，包括所有 BizTalk 数据库，用于跟踪业务流程的性能。  

- **业务活动监视 (BAM) 框架。** 使用此工具来跟踪数据项和业务事务过程中收集的业务里程碑。  

  SWIFT 反汇编程序增强了 （修饰） 与 A4SWIFT 一起处理的消息的已提升属性。 在这种情况下，HAT 和 BAM 查询表达式 （可选） 包含用于筛选这些属性。 A4SWIFT 升级属性的列表，请参阅[A4SWIFT_ * 升级的属性](../../adapters-and-accelerators/accelerator-swift/a4swift-promoted-properties.md)。  

  有关 MicrosoftBizTalk 服务器操作的信息，请参阅 BizTalk Server 帮助中的操作部分。
