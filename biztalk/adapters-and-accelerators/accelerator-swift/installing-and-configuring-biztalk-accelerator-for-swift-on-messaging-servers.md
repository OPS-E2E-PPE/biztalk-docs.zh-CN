# <a name="installing-and-configuring-biztalk-accelerator-for-swift-on-messaging-servers"></a>上安装和配置 BizTalk Accelerator for SWIFT 消息传送服务器
本部分介绍如何安装和配置[!INCLUDE[A4SWIFT_CurrentVersion_FirstRef](../../includes/a4swift-currentversion-firstref-md.md)]消息传送的服务器上。 这些服务器主要用于与 SWIFT 网络进行通信。  
  
### <a name="to-install-and-configure-biztalk-accelerator-for-a4swift-on-the-messaging-server"></a>若要为安装和配置 BizTalk Accelerator A4SWIFT 消息传送服务器上  
  
1.  执行的自定义安装[!INCLUDE[A4SWIFT_CurrentVersion_abbrev](../../includes/a4swift-currentversion-abbrev-md.md)]。 安装**MRSR**和**SWIFT 消息**功能。  
  
    > [!NOTE]
    >  不需要安装 Web 组件消息修复和新提交功能，因为此服务器没有 MRSR 站点。  
  
    > [!NOTE]
    >  安装完成后，将启动配置向导。  
  
2.  在 Microsoft[!INCLUDE[A4SWIFT_CurrentVersion_abbrev](../../includes/a4swift-currentversion-abbrev-md.md)]配置控制台中，配置**MCRR**。
