# <a name="updating-swift-messaging-standards"></a>更新 SWIFT 消息传送标准
协会的全球 Interbank 金融电信 (SWIFT) 金融服务行业提出建议，并采用标准的更改在每年一次更新其消息传送标准。 SWIFT 更新其标准版本指南中，当[!INCLUDE[btaA4SWIFT2.3abbrevnonumber](../../includes/btaa4swift2-3abbrevnonumber-md.md)]必须更新架构和 BRE 策略以符合最新的 SWIFT 规范。  
  
 SWIFT 12 个月您都需要的 SWIFT 网络实施更改之前通常宣布了初步的更新。 更新通常最终完成之前 SWIFT 的九个月的"go live"日期。 任何计划时，应考虑此时间线[!INCLUDE[btaA4SWIFT2.3abbrevnonumber](../../includes/btaa4swift2-3abbrevnonumber-md.md)]实现。  
  
 有关最新详细信息[!INCLUDE[btaA4SWIFT2.3abbrevnonumber](../../includes/btaa4swift2-3abbrevnonumber-md.md)]更新和版本，请转到 BizTalk Accelerator for SWIFT 网站，网址[ http://go.microsoft.com/fwlink/?LinkId=28272 ](http://go.microsoft.com/fwlink/?LinkId=28272)。
