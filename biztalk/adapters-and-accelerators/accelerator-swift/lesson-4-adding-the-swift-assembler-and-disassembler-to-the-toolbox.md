# <a name="lesson-4-adding-the-swift-assembler-and-disassembler-to-the-toolbox"></a>第 4 课： 向工具箱添加 SWIFT 汇编程序和反汇编程序
在本课程中，你将添加对 SWIFT 汇编程序 (ASM) 和反汇编程序 (DASM) 的引用，以便使用 BizTalk 管道设计器时，这些组件是工具箱中的可用。  
  
### <a name="to-add-the-swift-asm-and-dasm-components-to-the-toolbox"></a>若要向工具箱添加 SWIFT ASM 和 DASM 组件  
  
1.  在 Visual Studio 中，在**工具**菜单上，单击**选择工具箱项**。  
  
2.  在选择工具箱项对话框中，在**BizTalk 管道组件**选项卡上，选择**SWIFT 汇编程序**和**SWIFT 反汇编程序**。  
  
3.  单击 **“确定”**。  
  
 除非你已有接收或发送管道打开，在这种情况下的 ASM 或 DASM 将不可用，汇编程序和反汇编程序组件现可从工具箱。
