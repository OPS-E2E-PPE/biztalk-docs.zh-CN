# <a name="getting-started-with-biztalk-accelerator-for-swift"></a>开始使用 BizTalk Accelerator for SWIFT
Microsoft[!INCLUDE[A4SWIFT_CurrentVersion_FirstRef](../../includes/a4swift-currentversion-firstref-md.md)]提供了一组架构、 规则和关联的工具来帮助你的内部应用程序的 SWIFT 消息传送集成。 A4SWIFT 要求 MicrosoftBizTalk 服务器，并使用[!INCLUDE[btsBizTalkServerNoVersion](../../includes/btsbiztalkservernoversion-md.md)]组件和功能来实现分析，验证，处理时，规则的架构和业务流程。 A4SWIFT 还实现了解除批处理包含多个财务 (FIN) 消息、 消息修复和新提交和 FIN 响应对帐的文件。  
  
 可以通过转到 A4SWIFT 端到端教程直接获取 A4SWIFT 立即亲身体验。 [模块 1：创建 SWIFT 解决方案](../../adapters-and-accelerators/accelerator-swift/module-1-creating-a-swift-solution.md)来帮助你构建企业应用程序集成 (EAI) 解决方案，可自动执行的内部业务流程。 [模块 2：添加新架构项目](../../adapters-and-accelerators/accelerator-swift/module-2-adding-a-new-schemas-project.md)通过帮助您开发自动化的解决方案来处理所选的 SWIFT FIN 消息将生成 EAI 项目。
