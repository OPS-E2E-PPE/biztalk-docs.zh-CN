# <a name="approval-stage"></a>审核阶段
第一次签名并提交以供审批条消息后，批准链中的每个审批者评审和副署消息，他或她的数字签名添加到堆栈。 审批者只能查看和副署消息。 他们无法更改它而不会破坏数字签名堆栈。
