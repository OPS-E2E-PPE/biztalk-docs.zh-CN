# <a name="approval-stage"></a>审批阶段
第一次签名并提交以供审核条消息后，批准链中的每个审批者评审和副署的消息，将其数字签名添加到堆栈。 审批者只能查看和消息进行副署。 它们不能更改它而不会中断的数字签名堆栈。
