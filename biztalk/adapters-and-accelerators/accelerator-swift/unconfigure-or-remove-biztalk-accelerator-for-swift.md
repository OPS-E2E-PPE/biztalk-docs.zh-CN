# <a name="unconfigure-or-remove-biztalk-accelerator-for-swift"></a>取消配置或删除 BizTalk Accelerator for SWIFT
可以更改或删除 Microsoft[!INCLUDE[A4SWIFT_CurrentVersion_FirstRef](../../includes/a4swift-currentversion-firstref-md.md)]从通过运行计算机**安装向导**或通过使用**控制面板**。  

## <a name="change-or-update-your-a4swift-installation"></a>更改或更新在 A4SWIFT 安装  

1.  打开**Control Panel**，然后双击**卸载程序**。  

2.  在程序列表中，右键单击 Microsoft **BizTalk Accelerator for SWIFT**，然后选择**更改**。  

3.  上**程序维护**页上，选择**修改**，然后选择**下一步**。  

4.  上**自定义安装**页上，选择你想要更改或删除，并选择的组件**下一步**。  


## <a name="remove-a4swift-from-your-computer"></a>从计算机中删除 A4SWIFT  

1. 打开**Control Panel**，然后双击**卸载程序**。  

2. 在程序列表中，右键单击 Microsoft **BizTalk Accelerator for SWIFT**，然后选择**卸载**。  

3. 在中**程序维护**页上，选择**删除**，然后选择**下一步**。  

4. 上**摘要**页上，选择**卸载**。  

   > [!NOTE]
   >  卸载程序 0}run，，然后删除[!INCLUDE[A4SWIFT_CurrentVersion_abbrev](../../includes/a4swift-currentversion-abbrev-md.md)]。  

5. 完成后，选择**完成**。
