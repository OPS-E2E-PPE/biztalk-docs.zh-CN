# <a name="lesson-1-submitting-a-sample-flat-file"></a>第 1 课：提交示例平面文件
您提交示例平面文件，以测试您的解决方案。  
  
### <a name="to-submit-a-sample-mt103-flat-file"></a>若要提交示例 MT103 平面文件  
  
1. 将从 MT103_Sample.txt 文件复制\<*驱动器*\>: \Program Files\Microsoft BizTalk Accelerator for SWIFT\SDK\Tutorial 到\<*驱动器*\>: \Labs\Inbound\FlatFile 文件夹。  
  
2. 验证在一段时间后，MT103_Sample.txt 文件将从文件夹中消失。  
  
3. 浏览到\<*驱动器*\>: \Labs\Outbound 文件夹。 验证该文件夹包含一个名为 {GUID}.xml 文件。  
  
4. 右键单击文件 {GUID}.xml，指向**打开**，然后单击**记事本**。 执行相同的文件称为从 MT103_Sample.txt \<*驱动器*\>: \Program Files\Microsoft BizTalk Accelerator for SWIFT\SDK\Tutorial。 验证 XML 文件的标记中的内容与平面文件中的内容相同。  
  
   > [!NOTE]
   >  不要删除生成的文件;将在后面的课程中使用它。  
  
   请继续执行[第 2 课：提交是无效的 MT103 消息](../../adapters-and-accelerators/accelerator-swift/lesson-2-submitting-an-mt103-message-that-is-not-valid.md)。
