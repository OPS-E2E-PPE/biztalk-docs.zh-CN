# <a name="swift-messages"></a>SWIFT 消息
协会的全球 Interbank 金融电信 (SWIFT) 是提供到 200 的国家/地区的超过 7000 个金融机构安全消息传送服务和接口软件行业拥有合作。 SWIFT 提供到银行、 经纪商和投资管理器，以及市场中付款、 财务、 证券和贸易的基础结构的消息传递服务。 这些服务帮助客户降低成本、 提高自动化，并管理风险。 SWIFT 也是 interbank 通信的主要标准正文。  
  
 您可以访问 SWIFT 网站下载，网址[ http://go.microsoft.com/fwlink/?LinkId=27885 ](http://go.microsoft.com/fwlink/?LinkId=27885)。  
  
 Microsoft[!INCLUDE[A4SWIFT_CurrentVersion_FirstRef](../../includes/a4swift-currentversion-firstref-md.md)]提供了架构和验证支持 SWIFT 财务 (FIN) 消息中列出[支持消息](../../adapters-and-accelerators/accelerator-swift/supported-messages.md)主题中，基于 SWIFT 标准版本指南\<版本\>(SRG\<版本\>)。  
  
 **邮件副本**  
  
 之前和转换之后，A4SWIFT 存储消息的副本。 这些副本是可通过在 BizTalk 管理控制台访问。
