# <a name="swift-receive-adapter-security-context"></a>SWIFT 接收适配器安全上下文
接收器适配器，发送适配器与启动 SWIFTNet 链接 (SNL/RA) 命令提示符下 （SWIFTNet 启动）。 远程协助 SNL/配置文件 (paramconfig) 中配置接收适配器可执行文件。 在启动适配器初始化 SNL 库基于命令行参数。 配置值被缓存以供将来使用。
