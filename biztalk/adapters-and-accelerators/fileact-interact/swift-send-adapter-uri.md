# <a name="swift-send-adapter-uri"></a>SWIFT 发送适配器 URI
发送适配器的 URI 是构造基于响应程序名称 (SWIFT: / /\<响应方名称\>)。 响应方名称是由 SWIFT 定义的可分辨的名称。 它指示需要向其发送消息。 SWIFT 使用此值将消息路由到接收方。
