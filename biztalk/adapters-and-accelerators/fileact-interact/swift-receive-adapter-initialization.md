# <a name="swift-receive-adapter-initialization"></a>SWIFT 接收适配器初始化
SWIFT 服务器应用程序的初始化过程是类似于 SWIFT 的客户端应用程序。 初始化参数作为命令行参数传递到接收方可执行文件。 在设计时，接收位置中的配置的应用程序名称作为命令行参数传递。 此应用程序名称用于构造接收位置来检索已配置的属性的 URI。  
  
 有关使用命令行参数的信息，请参阅 Microsoft 中的"如何配置接收适配器环境"主题[!INCLUDE[swift_adapter](../../includes/swift-adapter-md.md)]安装和配置指南。
