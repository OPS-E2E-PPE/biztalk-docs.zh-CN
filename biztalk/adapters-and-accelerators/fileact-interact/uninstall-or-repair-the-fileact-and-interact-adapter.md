# <a name="uninstall-or-repair-the-fileact-and-interact-adapter"></a>卸载或修复 FileAct 和交互适配器
若要卸载或修复 BizTalk FileAct 和交互适配器的步骤。  
  
## <a name="uninstall-the-fileact-and-interact-adapter"></a>卸载 FileAct 和交互适配器  
  
1.  在**控制面板**，选择**卸载程序**。  
  
2.  选择**BizTalk FileAct 和交互适配器**，然后选择**卸载**或**更改**。  
  
3.  选择**删除**选项，然后再选中**下一步**。  
  
4.  在摘要中，选择**卸载**。  
  
5.  选择**完成**时完成。

## <a name="repair-the-fileact-and-interact-adapter-installation"></a>修复 FileAct 和交互适配器安装  
  
1.  在**控制面板**，选择**卸载程序**。  
  
2.  选择**BizTalk FileAct 和交互适配器**，然后选择**更改**。  
3.  上**程序维护**页上，选择**修复**选项，然后再选中**下一步**。  
  
4.  在摘要中，选择**卸载**。  
  
5.  选择**完成**时完成。
