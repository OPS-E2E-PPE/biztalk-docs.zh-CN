# <a name="fileact-adapter-delivery-notification"></a>FileAct 适配器传递通知
验证后完成 （包括签名和摘要式验证） 的回执的消息请求中所述，FileAct 适配器传输到发件人相应传递通知。  
  
 同样，如果 FileAct 适配器接收收到传递通知，它将传输正确的响应。
