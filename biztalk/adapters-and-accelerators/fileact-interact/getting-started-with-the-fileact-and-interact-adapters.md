# <a name="getting-started-with-the-fileact-and-interact-adapters"></a>开始使用 FileAct 和 InterAct 适配器
协会的全球 Interbank 金融电信 (SWIFT) 提供了安全消息传送服务和接口的软件到批发财务实体。  
  
 SWIFTNet 是为参与金融服务行业的机构之间的安全连接的全局财务消息传送网络。  
  
 SWIFTNet FileAct 和 InterAct 适配器传输的文件提供 BizTalk Server 和 SWIFT Secure IP Network (SIPN) 之间的连接。  
  
 SIPN 互联金融机构、 金融业基础结构和客户的安全专用网络上传输消息和文件。  
  
 FileAct 和 InterAct 适配器使用 SWIFTNet 链接 (SNL) 应用程序编程接口 (Api) 连接到 SIPN。
