# <a name="interact-adapter-server-application"></a>InterAct 适配器服务器应用程序
本部分介绍服务器应用程序请求消息的组合的一般说明。 这些是从服务器端 SNL 传递到服务器应用程序的 XML 文档。 在这种情况下，这些消息是在服务器端执行的 SWIFTNet 基元的第一部分的特征。
