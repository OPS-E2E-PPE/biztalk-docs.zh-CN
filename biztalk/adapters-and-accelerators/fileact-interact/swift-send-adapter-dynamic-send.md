# <a name="swift-send-adapter-dynamic-send"></a>SWIFT 发送适配器动态发送
发送适配器动态发送模式拉取队列中的消息。 动态发送的 URI 为：  
  
```  
SWIFT://<queue name>/<Force Acquire>/<Session Mode>/<Order By>/<Recovery mode>  
```  
  
 可用于业务流程或管道构建 URI。
