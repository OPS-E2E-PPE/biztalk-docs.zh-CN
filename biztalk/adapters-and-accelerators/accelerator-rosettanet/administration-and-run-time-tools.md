# <a name="administration-and-run-time-tools"></a>管理和运行时工具
Microsoft BizTalk Server 和 Microsoft[!INCLUDE[BTARN_CurrentVersion_FirstRef](../../includes/btarn-currentversion-firstref-md.md)]提供一套工具，用于执行管理和运行任务，例如部署、 配置管理、 管理、 业务规则管理和监视。 有关详细信息[!INCLUDE[btsBizTalkServerNoVersion](../../includes/btsbiztalkservernoversion-md.md)]工具，请参阅 MicrosoftBizTalk Server 帮助。  
  
## <a name="biztalk-explorer"></a>BizTalk Explorer  
 使用 BizTalk 浏览器来配置和管理项目、 参与方和业务流程。 BizTalk 浏览器中显示配置数据库的内容显示程序集、 端口和参与方等项。  
  
## <a name="biztalk-accelerator-for-rosettanet-management-console"></a>BizTalk Accelerator for RosettaNet 管理控制台  
 您使用[!INCLUDE[btaBTARN3.3abbrevnonumber](../../includes/btabtarn3-3abbrevnonumber-md.md)]管理控制台来管理[!INCLUDE[btaBTARN3.3abbrevnonumber](../../includes/btabtarn3-3abbrevnonumber-md.md)]实现，其中包括：  
  
- 流程配置设置、 本组织、 合作伙伴和协议  
  
- 证书  
  
- BizTalk 和[!INCLUDE[btaBTARN3.3abbrevnonumber](../../includes/btabtarn3-3abbrevnonumber-md.md)][!INCLUDE[btsSQLServerNoVersion](../../includes/btssqlservernoversion-md.md)]数据库  
  
- 事件  
  
- 性能日志和警报  
  
## <a name="business-rule-engine"></a>业务规则引擎  
 BizTalk 业务规则引擎 (BRE) 是评估的一个运行时推理引擎，可用于根据事实评估规则和基于结果启动操作。  
  
## <a name="biztalk-administration-console"></a>BizTalk 管理控制台  
 可以使用 BizTalk 管理控制台进行报告、 分析和调试实时业务数据和针对您的业务流程进行优化。 此外可以使用 BizTalk 管理控制台中监视的运行状况[!INCLUDE[btsBizTalkServerNoVersion](../../includes/btsbiztalkservernoversion-md.md)]场中，还可以跟踪和查看特定事件和消息。  
  
## <a name="windows-performance-monitors"></a>Windows 性能监视器  
 [!INCLUDE[btaBTARN3.3abbrevnonumber](../../includes/btabtarn3-3abbrevnonumber-md.md)] 支持使用[!INCLUDE[btsWinNoVersion](../../includes/btswinnoversion-md.md)]连接状态和吞吐量的性能监视器。
