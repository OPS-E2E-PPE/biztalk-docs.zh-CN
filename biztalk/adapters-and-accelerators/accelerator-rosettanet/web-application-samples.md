# <a name="web-application-samples"></a>Web 应用程序示例
[!INCLUDE[btsCoName](../../includes/btsconame-md.md)]® [!INCLUDE[BTARN_CurrentVersion_FirstRef](../../includes/btarn-currentversion-firstref-md.md)] 软件开发工具包 (SDK) 包含一个 RNIFSend.aspx 示例和一个 RNIFReceive.aspx 示例。 这些 ASPX 页将 RNIF 消息从发起方发送到响应方，或者在响应方接收 RNIF 消息。  
  
 ASPX 页的功能的详细信息，请参阅[发送和接收 ASPX 页](../../adapters-and-accelerators/accelerator-rosettanet/send-and-receive-aspx-pages.md)。 本节提供了生成和运行这些示例的说明。
