# <a name="web-application-samples"></a>Web 应用程序示例
The Microsoft®[!INCLUDE[BTARN_CurrentVersion_FirstRef](../../includes/btarn-currentversion-firstref-md.md)]软件开发工具包 (SDK) 包含一个 RNIFSend.aspx 示例和一个 RNIFReceive.aspx 示例。 这些 ASPX 页将 RNIF 消息从发起方发送到响应方，或接收 RNIF 消息响应方。  
  
 ASPX 页的功能的详细信息，请参阅[发送和接收 ASPX 页](../../adapters-and-accelerators/accelerator-rosettanet/send-and-receive-aspx-pages.md)。 本部分提供有关生成和运行示例的说明。
