# <a name="double-action-tutorial"></a>双操作教程
本教程介绍如何使用 Microsoft® 的端到端解决方案[!INCLUDE[BTARN_CurrentVersion_FirstRef](../../includes/btarn-currentversion-firstref-md.md)]。 本教程详细介绍了必须为通过创建两个虚构的公司之间的贸易合作伙伴方案来实现 RosettaNet 兼容解决方案要遵循的步骤：Contoso (供应商组织） 和 Fabrikam，买方组织。  
  
 本教程使用四个不同的合作伙伴接口流程 (Pip):  
  
- 0c2-异步测试请求  
  
- 0c4-同步测试查询  
  
- 3A2-请求价格与可用性  
  
- 3A4-请求采购订单  
  
  本教程探讨 rosettanet 解决方案，包括使用的几个不同方面[!INCLUDE[btaBTARN3.3abbrevnonumber](../../includes/btabtarn3-3abbrevnonumber-md.md)]管理控制台中，创建的业务 (LOB) 应用程序，以及创建可用于将集成的自定义业务流程企业资源规划 (ERP) 系统。
