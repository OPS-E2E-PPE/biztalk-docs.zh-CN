# <a name="creating-the-contoso-solution"></a>创建 Contoso 解决方案
本部分详细介绍你必须按照为 Contoso 组织的步骤。 第一步是添加两个组织使用 Microsoft® 联系人信息和合作伙伴协议[!INCLUDE[btaBTARN3.3abbrevnonumber](../../includes/btabtarn3-3abbrevnonumber-md.md)]管理控制台。 然后，创建业务线 (LOB) 架构，并构造这些架构和其各自的基于 RosettaNet 合作伙伴接口流程 (PIP) 架构之间的映射。 接下来，你配置用于 SQL 适配器用于将信息发送到 Fabrikam 与 ERP 系统和 HTTP 适配器进行通信的端口信息。 最后一步是自定义专用业务流程要在 BizTalk Server 中使用业务规则引擎 (BRE)。
