# <a name="microsoft-biztalk-accelerator-for-rosettanet-documentation"></a>Microsoft BizTalk Accelerator for RosettaNet 文档

 作为一种解决方案在 Microsoft BizTalk Server 基础上构建[!INCLUDE[btaBTARN3.3abbrevnonumber](../../includes/btabtarn3-3abbrevnonumber-md.md)]提供必需的应用程序和平台来开发符合 RNIF 的解决方案。 这些解决方案，用户可以自动化其电子商务流程，例如订单管理和库存管理。 [!INCLUDE[btaBTARN3.3abbrevnonumber](../../includes/btabtarn3-3abbrevnonumber-md.md)] 充分利用现有[!INCLUDE[btsBizTalkServerNoVersion](../../includes/btsbiztalkservernoversion-md.md)]功能来处理 XML 和电子数据交换 (EDI) 各企业之间的通信。
