# <a name="testing-the-double-action-tutorial"></a>测试双操作教程
在本部分中，您可以使用在前面几节中创建的 Contoso 和 Fabrikam 解决方案来测试四种不同的合作伙伴接口流程 (Pip):0c2，0c4，3A2 和 3A4。 在 Fabrikam 计算机上使用 LOBWebApplication 来发出请求。 系统将使用增强的通信通道向 Contoso 计算机的请求。 双操作业务流程将生成适当的响应根据您使用的 PIP。
