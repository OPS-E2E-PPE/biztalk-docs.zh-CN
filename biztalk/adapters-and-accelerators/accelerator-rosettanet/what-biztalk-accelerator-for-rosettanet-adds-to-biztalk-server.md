# <a name="what-biztalk-accelerator-for-rosettanet-adds-to-biztalk-server"></a>BizTalk Accelerator for RosettaNet 向 BizTalk Server 的添加
Microsoft[!INCLUDE[BTARN_CurrentVersion_FirstRef](../../includes/btarn-currentversion-firstref-md.md)]构建到 RosettaNet 兼容的集成系统的 Microsoft BizTalk Server 的集成系统。 它会添加必须所要实现 RosettaNet 兼容解决方案的所有内容。  
  
在安装[!INCLUDE[btaBTARN3.3abbrevnonumber](../../includes/btabtarn3-3abbrevnonumber-md.md)]在 BizTalk Server 上和[!INCLUDE[btaBTARN3.3abbrevnonumber](../../includes/btabtarn3-3abbrevnonumber-md.md)]将功能添加到 BizTalk Server 引擎的核心。 它将添加到 BizTalk Server 提供的那些功能、 工具和实用程序。 它还添加到 BizTalk Server 软件开发工具包 (SDK) 提供的应用程序编程接口 (Api)。
