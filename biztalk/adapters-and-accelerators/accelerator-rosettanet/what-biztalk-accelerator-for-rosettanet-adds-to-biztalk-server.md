# <a name="what-biztalk-accelerator-for-rosettanet-adds-to-biztalk-server"></a>BizTalk Accelerator for RosettaNet 将添加到 BizTalk Server
[!INCLUDE[btsCoName](../../includes/btsconame-md.md)] [!INCLUDE[BTARN_CurrentVersion_FirstRef](../../includes/btarn-currentversion-firstref-md.md)] 生成[!INCLUDE[btsCoName](../../includes/btsconame-md.md)]BizTalk Server 集成到 RosettaNet 符合集成系统的系统。 它将实现 RosettaNet 兼容解决方案所需的一切都添加了进去。  
  
你安装[!INCLUDE[btaBTARN3.3abbrevnonumber](../../includes/btabtarn3-3abbrevnonumber-md.md)]在 BizTalk Server 上和[!INCLUDE[btaBTARN3.3abbrevnonumber](../../includes/btabtarn3-3abbrevnonumber-md.md)]将功能添加到 BizTalk Server 引擎的核心。 并扩展了 BizTalk Server 提供的功能、工具和实用工具。 它还扩展了 BizTalk Server 软件开发工具包 (SDK) 提供的应用程序编程接口 (API)。
