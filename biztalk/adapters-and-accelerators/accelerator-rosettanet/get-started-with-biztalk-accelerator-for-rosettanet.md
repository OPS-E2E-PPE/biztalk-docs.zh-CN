# <a name="get-started-with-biztalk-accelerator-for-rosettanet"></a>开始使用 BizTalk Accelerator for RosettaNet
Microsoft BizTalk Accelerator for RosettaNet (BTARN) 简化了开发和部署的 RosettaNet 基于标准的集成解决方案。 [!INCLUDE[btaBTARN3.3abbrevnonumber](../../includes/btabtarn3-3abbrevnonumber-md.md)] 支持 RosettaNet 实现框架 (RNIF) 版本 1.1 和 2.0.01。 RNIF 是一种开放式网络应用程序框架，使业务合作伙伴能够协同运行 RosettaNet 合作伙伴接口流程 (Pip)。 请参阅[GS1 美国](http://go.microsoft.com/fwlink/?LinkID=33859)有关 RosettaNet 标准的详细信息。
  
 [!INCLUDE[btaBTARN3.3abbrevnonumber](../../includes/btabtarn3-3abbrevnonumber-md.md)] 此外支持化工数据交换 (CIDX) Chem eStandards 的交换。  
  
本部分提供有关如何使用 BizTalk Server 与 RosettaNet 加速器在公司中的企业应用程序集成 (EAI) 和业务合作伙伴之间自动执行企业到企业采购特定于角色的信息解决方案。  

## <a name="loopback-tutorial"></a>Loopback 教程

包含介绍如何使用 RosettaNet 加速器模拟一台计算机上的家庭和合作伙伴组织之间的流程实施的详细的步骤。

在本教程中，您将创建本组织、 合作伙伴组织和贸易协议、 使用 Loopback 实用工具创建镜像协议，，然后运行示例流程来验证环回方案。

转到[Loopback 教程](loopback-tutorial.md)。 

## <a name="double-action-tutorial"></a>双操作教程

使用 RosettaNet 加速器的端到端解决方案。 本教程详述了通过创建两个虚构的公司之间的贸易合作伙伴方案来实现 RosettaNet 兼容解决方案的步骤：Contoso (供应商组织） 和 Fabrikam，买方组织。

它包括使用 BTARN 管理控制台、 创建业务线 (LOB) 应用程序，以及创建自定义业务流程。

转到[双操作教程](double-action-tutorial.md)。 


## <a name="private-process-tutorial"></a>专用流程教程
使用 RosettaNet 加速器的端到端解决方案。 本教程详述了通过创建两个虚构的公司之间的贸易方案来实现 RosettaNet 兼容解决方案的步骤：Contoso (供应商组织） 和 Fabrikam，买方组织。

它概述了创建基于 RosettaNet 的解决方案，包括 ERP 集成、 业务策略的实施、 专用流程的自定义和改进安全通信的各个方面。

转到[专用流程教程](private-process-tutorial.md)
