# <a name="get-started-with-biztalk-accelerator-for-rosettanet"></a>入门 RosettaNet BizTalk 快捷键
Microsoft BizTalk Accelerator RosettaNet (BTARN) 简化了开发和部署 RosettaNet 基于标准的集成解决方案。 [!INCLUDE[btaBTARN3.3abbrevnonumber](../../includes/btabtarn3-3abbrevnonumber-md.md)]支持 RosettaNet 实现框架 (RNIF) 版本 1.1 和 2.0.01。 RNIF 是一种开放式网络应用程序框架，允许业务合作伙伴以协作方式运行 RosettaNet 合作伙伴接口流程 (PIP)。 请参阅[GS1 美国](http://go.microsoft.com/fwlink/?LinkID=33859)有关 RosettaNet 标准的详细信息。
  
 [!INCLUDE[btaBTARN3.3abbrevnonumber](../../includes/btabtarn3-3abbrevnonumber-md.md)] 还支持化工数据交换 (CIDX) Chem eStandards 的交换。  
  
本部分提供有关如何使用 BizTalk Server 和 RosettaNet 快捷键在公司中的企业应用程序集成 (EAI)，以及在业务合作伙伴来自动执行企业到企业采购的特定于角色的信息解决方案。  

## <a name="loopback-tutorial"></a>环回教程

包含介绍如何使用 RosettaNet 快捷键来模拟单台计算机上的主页和合作伙伴组织之间的进程实现的详细的步骤。

在本教程中，您可以创建本组织，合作伙伴组织和贸易协议，并使用 Loopback 实用工具创建镜像协议，然后运行示例流程来验证环回方案。

转到[环回教程](loopback-tutorial.md)。 

## <a name="double-action-tutorial"></a>Double 操作教程

使用 RosettaNet 加速器端到端解决方案。 本教程详细介绍的步骤来通过创建两个虚构公司之间的贸易合作伙伴方案实现 RosettaNet 合规的解决方案： Contoso，供应商组织和 Fabrikam，buyer 组织。

它包括使用 BTARN 管理控制台、 创建业务线 (LOB) 应用程序，以及创建自定义业务流程。

转到[Double 操作教程](double-action-tutorial.md)。 


## <a name="private-process-tutorial"></a>专用流程教程
使用 RosettaNet 加速器端到端解决方案。 本教程详细介绍的步骤来通过创建两个虚构公司之间的贸易方案实现 RosettaNet 合规的解决方案： Contoso，供应商组织和 Fabrikam，buyer 组织。

本教程探讨了创建基于 RosettaNet 的解决方案的各个方面，包括 ERP 集成、业务策略的实施、专用流程的自定义以及改进安全通信。

转到[专用流程教程](private-process-tutorial.md)
