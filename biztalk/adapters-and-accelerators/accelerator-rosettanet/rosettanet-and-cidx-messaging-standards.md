# <a name="rosettanet-and-cidx-messaging-standards"></a>RosettaNet 和 CIDX 标准消息传送
主要目的[!INCLUDE[btsCoName](../../includes/btsconame-md.md)]®[!INCLUDE[BTARN_CurrentVersion_FirstRef](../../includes/btarn-currentversion-firstref-md.md)]旨在处理符合 RosettaNet 或筛选行业数据交换 (CIDX) 消息传递标准的消息。 本部分提供有关使用两种主要 RosettaNet 标准的信息： RosettaNet 实现框架 (RNIF) 和合作伙伴接口进程 (Pip)。 此外，本节还提供有关 CIDX Chem eStandards 的信息。  
  
 RosettaNet 标准是一套新兴的并被广泛接受的标准，用于在高科技方案中实现自动的消息交换。 RosettaNet 组织的目的是创建、实现并完善电子商务流程标准。 这些标准在高科技制造业和供应链应用程序中广泛使用。 管理 RosettaNet 组织的六个不同的行业协会对应用范围进行了如下阐释：  
  
-   电子元件  
  
-   半导体制造商  
  
-   信息技术  
  
-   电信业务  
  
-   物流  
  
-   解决方案提供商  
  
 CIDX Chem eStandards 是一套被广泛接受的标准，用于在化工行业方案中实现自动消息交换。  
  
 有关标准的详细信息，请参阅位于的 RosettaNet 网站[http://go.microsoft.com/FWLink/?LinkID=33859](http://go.microsoft.com/FWLink/?LinkID=33859)和位于的 CIDX 网站[http://go.microsoft.com/FWLink/?LinkID=34540](http://go.microsoft.com/FWLink/?LinkID=34540)。
