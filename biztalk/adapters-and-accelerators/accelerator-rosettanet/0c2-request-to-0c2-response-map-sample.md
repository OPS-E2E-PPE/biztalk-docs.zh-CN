# <a name="0c2-request-to-0c2-response-map-sample"></a>0 C 2 请求设置为 0 C 2 响应映射示例
_0C2RequestMessageTo0C2ResponseMessage.btm 示例演示如何将 0C2 请求消息映射到 0C2 响应消息。  
  
 默认情况下， [!INCLUDE[btsCoName](../../includes/btsconame-md.md)]®[!INCLUDE[btsBizTalkServer2006r3](../../includes/btsbiztalkserver2006r3-md.md)]安装程序将安装中的示例\<*驱动器*>: \Program Files\\ [!INCLUDE[btsCoName](../../includes/btsconame-md.md)] Microsoft BizTalk\<版本 >RosettaNet\SDK\PIPAutomation\DoubleAction 快捷键。  
  
## <a name="sample-contents"></a>示例内容  
 此示例演示如何将 0C2 请求消息的字段映射到 0C2 响应消息。 可以将此示例与双操作 PIPAutomation 业务流程示例一起使用。 该双操作 PIPAutomation 业务流程示例演示如何实现为双操作 0C2、0C4、3A2 和 3A4 PIP 自动生成响应的业务流程。
