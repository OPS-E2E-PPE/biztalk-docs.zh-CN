# <a name="creating-and-configuring-biztalk-ports-for-contoso"></a>创建和配置 Contoso 的 BizTalk 端口
在本部分中，您将您当前的解决方案集成到 Microsoft® BizTalk Server。 执行此操作之前，请将自己的程序集安装到全局程序集缓存 (GAC) 中，然后在配置数据库中配置它。 然后使用 SQL 适配器和 HTTP 适配器创建和配置发送端口。
