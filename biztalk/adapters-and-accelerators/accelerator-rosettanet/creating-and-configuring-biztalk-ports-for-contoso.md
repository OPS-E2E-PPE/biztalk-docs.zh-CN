# <a name="creating-and-configuring-biztalk-ports-for-contoso"></a>创建和配置 Contoso 的 BizTalk 端口
在本部分中，您将您当前的解决方案集成到 Microsoft® BizTalk Server。 执行此操作之前，安装您的程序集在全局程序集缓存 (GAC)，然后将其配置中配置数据库。 然后创建并配置发送端口使用 SQL 适配器和 HTTP 适配器。
