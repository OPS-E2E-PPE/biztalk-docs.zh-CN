# <a name="private-process-tutorial"></a>专用流程教程
本教程中包含的完整端到端解决方案中使用 Microsoft® [!INCLUDE[BTARN_CurrentVersion_FirstRef](../../includes/btarn-currentversion-firstref-md.md)]。 本教程详细介绍了必须为通过创建两个虚构的公司之间的贸易方案来实现 RosettaNet 兼容解决方案要遵循的步骤：Contoso (供应商组织） 和 Fabrikam，买方组织。  
  
 本教程中实现的方案使用 3A2-价格与可用性合作伙伴接口流程 (PIP)。 购买前买方 Fabrikam 向供应商 Contoso 发送 3A2-价格与可用性请求。 此 PIP 可使 Fabrikam 获取特定产品，它们就可以处理通过业务规则来确定要购买产品有关的信息。 下图显示了 3A2 PIP 交换过程在发起方和响应方组织之间的通信模式。  
  
 ![&#60;No Change&#62;](../../adapters-and-accelerators/accelerator-rosettanet/media/rn3-intro-eetut-3a2flow.gif "RN3_Intro_EETut_3A2Flow")  
  
 本教程的重点是 Contoso 解决方案。 它概述了创建基于 RosettaNet 的解决方案，包括 ERP 集成、 业务策略的实施、 专用流程的自定义和改进安全通信的各个方面。
