# <a name="private-process-tutorial"></a>私有过程教程
本教程包含使用 [!INCLUDE[btsCoName](../../includes/btsconame-md.md)]® [!INCLUDE[BTARN_CurrentVersion_FirstRef](../../includes/btarn-currentversion-firstref-md.md)] 的完整端对端解决方案。 本教程详述了通过在两个虚构的公司：Contoso（供应商组织）和 Fabrikam（买方组织）之间创建贸易方案来实现 RosettaNet 兼容解决方案时，应该遵循的步骤。  
  
 本教程实施的方案使用 3A2-价格与可用性合作伙伴接口流程 (PIP)。 买方 Fabrikam 在购买前会向供应商 Contoso 发送 3A2-价格与可用性请求。 此 PIP 可使 Fabrikam 获取特定产品的相关信息，然后可以通过业务规则对这些信息进行处理，从而确定是否要购买该产品。 下图显示了 3A2 PIP 交换过程中发起方组织和响应方组织之间的通信模式。  
  
 ![&#60;无更改 &#62;] (../../adapters-and-accelerators/accelerator-rosettanet/media/rn3-intro-eetut-3a2flow.gif "RN3_Intro_EETut_3A2Flow")  
  
 本教程的重点在于 Contoso 解决方案。 本教程探讨了创建基于 RosettaNet 的解决方案的各个方面，包括 ERP 集成、业务策略的实施、专用流程的自定义以及改进安全通信。
