# <a name="3a2-request-to-3a2-response-map-sample"></a>3A2 请求到 3A2 响应映射示例
_3A2RequestMessageTo3A2ResponseMessage.btm 示例演示如何将 3A2 请求消息映射到 3A2 响应消息。  
  
 默认情况下，Microsoft® BizTalk Server 安装程序将安装中的示例\<*驱动器*\>: \Program Files\Microsoft BizTalk\<版本\>Accelerator for RosettaNet\SDK\PIPAutomation\DoubleAction。  
  
## <a name="sample-contents"></a>示例内容  
 此示例演示如何将 3A2 请求消息的字段映射到 3A2 响应消息。 此示例可用于双操作 PIPAutomation Orchestration 示例。 双操作 PIPAutomation 业务流程示例演示如何实现自动生成双操作响应的业务流程 0c2、 0c4、 3A2 和 3A4 Pip。
