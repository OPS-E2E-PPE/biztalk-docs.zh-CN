# <a name="connect-to-siebel-using-the-adapter"></a>连接到 Siebel 使用适配器
[!INCLUDE[adaptersiebel_short](../../includes/adaptersiebel-short-md.md)]要求适配器客户端提供连接字符串，调用统一资源标识符 (URI)，若要连接到 Siebel 系统的连接。 使用一个连接 URI，适配器客户端可以指定连接参数，以连接到外部系统。 有关连接 URI 的详细信息，请参阅[创建与 Siebel 系统的连接](../../adapters-and-accelerators/adapter-siebel/create-a-connection-to-the-siebel-system.md)。  
  
 请确保符合安全指导原则建立与 Siebel 系统的连接时。 有关安全指导原则的详细信息，请参阅[保护 Siebel 应用程序](../../adapters-and-accelerators/adapter-siebel/secure-your-siebel-applications.md)。
