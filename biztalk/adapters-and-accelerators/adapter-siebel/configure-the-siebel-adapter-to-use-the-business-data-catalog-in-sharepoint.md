# <a name="configure-the-siebel-adapter-to-integrate-the-siebel-system-with-the-business-data-catalog-and-sharepoint"></a>配置 Siebel 适配器为 Siebel 系统与业务数据目录和 SharePoint
[!INCLUDE[afproductnamelong](../../includes/afproductnamelong-md.md)]包括[!INCLUDE[afsvcdevwizlong](../../includes/afsvcdevwizlong-md.md)]，分别用于执行特定的 LOB 项目的 WCF 服务。 此 WCF 服务承载在宿主环境如 Microsoft Internet 信息服务 (IIS) 中。 Business Data Catalog Definition Editor 使用 WCF 服务承载的 WCF 服务中获取 Web 服务描述语言 (WSDL) URL。 使用 WSDL，Business Data Catalog Definition Editor 中提取可用于 WCF 服务的方法。 这些方法可以用于建立实体和实体之间的关联。  
  
 Business Data Catalog Definition Editor 可帮助您创建可以使用 Microsoft Office SharePoint Server 应用程序定义文件 （XML 文件）。 一旦应用程序定义文件导入到 Microsoft Office SharePoint Server，可以创建 Web 部件，以提供从企业应用程序的信息。 详细信息，请参阅"创建"Web 部件中[步骤 3:创建 SharePoint 应用程序以从 Siebel 检索数据](../../adapters-and-accelerators/adapter-siebel/step-3-create-a-sharepoint-application-to-retrieve-data-from-siebel.md)在[教程 1:从 SharePoint 站点上的 Siebel 系统提供数据](../../adapters-and-accelerators/adapter-siebel/tutorial-1-presenting-data-from-a-siebel-system-on-a-sharepoint-site.md)。
