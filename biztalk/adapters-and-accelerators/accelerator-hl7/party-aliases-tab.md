# <a name="party-aliases-tab"></a>参与方别名选项卡
您使用**参与方别名**选项卡以查看有关所选的参与方的信息。  
  
> [!NOTE]
>  使用 BizTalk 浏览器中创建参与方。  
  
 有关创建参与方的详细信息，请参阅"管理方使用 BizTalk 浏览器"中 BizTalk Server 帮助。  
  
 在中**BTAHL7 配置资源管理器**对话框中，在**参与方别名**选项卡上，执行以下操作：  
  
|使用此选项|执行的操作|  
|--------------|----------------|  
|**名称**|查看参与方组织的名称。|  
|**ReplTest1**|查看参与方 ID 值。|
