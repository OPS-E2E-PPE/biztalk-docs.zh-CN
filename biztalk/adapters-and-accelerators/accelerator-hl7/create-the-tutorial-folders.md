# <a name="create-the-tutorial-folders"></a>创建教程文件夹
创建以下文件夹使用[!INCLUDE[btsWinNoVersion](../../includes/btswinnoversion-md.md)]资源管理器：  
  
-   \<*驱动器*:\>\Batching 教程  
  
-   \<*驱动器*:\>\Batching Tutorial\Instances  
  
-   \<*驱动器*:\>\Program Files\Microsoft BizTalk\<版本\>HL7\SDK\End 端到端 Tutorial\Tutorial_BatchACKDrop 的快捷键  
  
-   \<*驱动器*:\>\Program Files\Microsoft BizTalk\<版本\>HL7\SDK\End 端到端 Tutorial\Tutorial_BatchMsgDrop 的快捷键
