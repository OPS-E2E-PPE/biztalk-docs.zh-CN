# <a name="hl7-2x-common-schema-files"></a>HL7 2.X 通用架构文件
每个 HL7 消息架构要求提供的数据类型、 段和表的架构定义的通用架构文件。 这些常见的架构文件定义每个版本的 HL7 2.X。 Microsoft 中的已部署的项目[!INCLUDE[btsVStudioNoVersion](../../includes/btsvstudionoversion-md.md)]包含消息架构必须引用包含三个常见的架构的常见项目的已部署的实例。
