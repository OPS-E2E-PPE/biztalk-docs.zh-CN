# <a name="administration-and-run-time-tools"></a>管理和运行的时工具
[!INCLUDE[btsCoName](../../includes/btsconame-md.md)]BizTalk Server 和[!INCLUDE[btsCoName](../../includes/btsconame-md.md)] [!INCLUDE[btaBTAHL7NoNumber](../../includes/btabtahl7nonumber-md.md)] ([!INCLUDE[btaBTAHL71.3abbrevnonumber](../../includes/btabtahl71-3abbrevnonumber-md.md)]) 提供一套工具，用于执行管理和运行时间的任务，如部署、 配置管理、 管理、 业务规则管理和监视。 有关详细信息[!INCLUDE[btsBizTalkServerNoVersion](../../includes/btsbiztalkservernoversion-md.md)]工具，请参阅[!INCLUDE[btsCoName](../../includes/btsconame-md.md)]BizTalk Server 帮助。  
  
## <a name="biztalk-explorer"></a>BizTalk 浏览器  
 您可以使用 BizTalk 浏览器来配置和管理项目、参与方和业务流程。 BizTalk 资源管理器将显示配置数据库 （也称为 BizTalk 管理数据库），的内容将显示诸如程序集、 端口和方 （在分层树中）。  
  
## <a name="btahl7-configuration-explorer"></a>BTAHL7 配置资源管理器  
 与[!INCLUDE[btaBTAHL71.3abbrevnonumber](../../includes/btabtahl71-3abbrevnonumber-md.md)]配置资源管理器，你可以查看和配置批处理定义、 确认配置和日志记录数据存储设置。 使用[!INCLUDE[btaBTAHL71.3abbrevnonumber](../../includes/btabtahl71-3abbrevnonumber-md.md)]配置资源管理器，你可以计划激活、 请求和方级别的批处理终止。  
  
## <a name="business-rule-engine"></a>业务规则引擎 (Business Rule Engine)  
 BizTalk 业务规则引擎 (BRE) 是一种运行时推理引擎，可用于根据事实评估规则，并根据评估结果采取相应措施。  
  
## <a name="windows-performance-monitors"></a>Windows 性能监视器  
 [!INCLUDE[btaBTAHL71.3abbrevnonumber](../../includes/btabtahl71-3abbrevnonumber-md.md)]支持使用[!INCLUDE[btsWinNoVersion](../../includes/btswinnoversion-md.md)]性能监视连接状态和在整个 （处理和拒绝）。
