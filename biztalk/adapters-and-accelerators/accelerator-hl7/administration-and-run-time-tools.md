# <a name="administration-and-run-time-tools"></a>管理和运行的时工具
Microsoft BizTalk Server 和 Microsoft [!INCLUDE[btaBTAHL7NoNumber](../../includes/btabtahl7nonumber-md.md)] ([!INCLUDE[btaBTAHL71.3abbrevnonumber](../../includes/btabtahl71-3abbrevnonumber-md.md)]) 提供一套工具，用于执行管理和运行时任务，例如部署、 配置管理、 管理、 业务规则管理和监视。 有关详细信息[!INCLUDE[btsBizTalkServerNoVersion](../../includes/btsbiztalkservernoversion-md.md)]工具，请参阅 MicrosoftBizTalk Server 帮助。  
  
## <a name="biztalk-explorer"></a>BizTalk Explorer  
 使用 BizTalk 浏览器来配置和管理项目、 参与方和业务流程。 BizTalk 浏览器中显示配置数据库 （也称为 BizTalk 管理数据库），的内容 （在层次结构树中） 显示程序集、 端口和参与方等项。  
  
## <a name="btahl7-configuration-explorer"></a>BTAHL7 配置资源管理器  
 使用[!INCLUDE[btaBTAHL71.3abbrevnonumber](../../includes/btabtahl71-3abbrevnonumber-md.md)]配置资源管理器，您可以查看和配置批次定义、 确认配置和日志记录数据存储区设置。 使用[!INCLUDE[btaBTAHL71.3abbrevnonumber](../../includes/btabtahl71-3abbrevnonumber-md.md)]配置资源管理器，你可以计划激活、 请求和参与方级别批处理终止。  
  
## <a name="business-rule-engine"></a>业务规则引擎  
 BizTalk 业务规则引擎 (BRE) 是评估的一个运行时推理引擎，可用于根据事实评估规则和基于结果启动操作。  
  
## <a name="windows-performance-monitors"></a>Windows 性能监视器  
 [!INCLUDE[btaBTAHL71.3abbrevnonumber](../../includes/btabtahl71-3abbrevnonumber-md.md)] 支持使用[!INCLUDE[btsWinNoVersion](../../includes/btswinnoversion-md.md)]性能监视的连接状态和在整个过程中 （处理和拒绝）。
