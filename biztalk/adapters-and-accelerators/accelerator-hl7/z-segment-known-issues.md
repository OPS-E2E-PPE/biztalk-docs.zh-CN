# <a name="z-segment-known-issues"></a>Z 段已知的问题
本部分包含可帮助你避免 Z 段错误的有用信息。  
  
## <a name="characters-not-recommended-to-use-for-naming-z-segments"></a>不建议使用用于命名的 Z 段的字符  
 建议，不使用以下字符命名的 Z 段变 HL7 段名称，并可能会导致错误：  
  
-   FHS  
  
-   BHS  
  
-   BTS  
  
-   FTS
