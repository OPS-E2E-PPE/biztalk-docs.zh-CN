# <a name="configuring-logging"></a>配置日志记录
卫生保健应用程序需要消息的完整日志记录。 这些应用程序处理关键，因为时间敏感和货币数据，记录将成为应用程序的关键部分。 系统管理员需要能够执行以下操作：  
  
-   将消息存储在配置位置  
  
-   关联到实际的消息记录的消息  
  
-   列出每个合作伙伴或每个进程类型的消息  
  
-   在这些记录的消息在预定时间运行的报表
