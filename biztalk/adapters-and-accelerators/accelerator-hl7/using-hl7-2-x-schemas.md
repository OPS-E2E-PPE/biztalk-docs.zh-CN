# <a name="using-hl7-2x-schemas"></a>使用 HL7 2.X 架构
本部分讨论了 Microsoft 支持的运行状况级别 7 (HL7) 标准的 2.X 版本[!INCLUDE[HL7_CurrentVersion_FirstRef](../../includes/hl7-currentversion-firstref-md.md)]。  
  
> [!IMPORTANT]
>  您可能需要更新随 BTAHL7 以符合标准 HL7 架构。 若要执行此操作，请参阅[解决数据库错误](../../adapters-and-accelerators/accelerator-hl7/resolving-database-errors.md)。  
  
> [!NOTE]
>  BTAHL7 引擎无法处理消息实例符合 HL7 架构具有不明确的架构结构。 不明确的架构结构是指 HL7 标准尚未完全定义。 此类架构包括为 CSU 和 SUR.消息类型
