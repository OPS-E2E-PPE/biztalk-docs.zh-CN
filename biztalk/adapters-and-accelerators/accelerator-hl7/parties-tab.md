# <a name="parties-in-btahl7-configuration-explorer"></a>BTAHL7 配置资源管理器中的参与方
您使用**参与方**选项卡以查看可用的参与方和指定[!INCLUDE[btaBTAHL71.3abbrevnonumber](../../includes/btabtahl71-3abbrevnonumber-md.md)]配置选择，为特定参与方，并配置确认的属性。 

## <a name="parties-ui-explained"></a>UI 的参与方所述
**参与方**选项卡包含一个左侧和右侧窗格。 在左窗格中显示可用的参与方。 右窗格包含以下选项卡的所选的参与方：  
  
- **参与方别名**。 使用此选项卡以查看有关在左窗格中选择的参与方信息。  
  
  > [!NOTE]
  >  使用 BizTalk 浏览器中创建参与方。  
  
- **批处理定义**。 使用此选项卡来配置[!INCLUDE[btaBTAHL71.3abbrevnonumber](../../includes/btabtahl71-3abbrevnonumber-md.md)]批处理。  
  
- **批处理计划**。 使用此选项卡来激活出站批。  
  
- **确认**。 使用此选项卡来指定入站和生成确认的属性。  
  
- **验证**。 使用此选项卡选择入站和生成消息的消息验证选项。  
  
- **MSH 映射**。 使用此选项卡上启用入站消息的消息标头转换。  
  
  > [!NOTE]
  >  必须配置参与方别名、 批次定义、 批处理计划和为所有贸易参与方的确认信息。  
  > 
  >  您可以通过按 F5，或通过右键单击参与方列表中，刷新的参与方列表并选择**刷新**。
