# <a name="step-8-restart-biztalk-server"></a>步骤 8： 重新启动 BizTalk Server
在此步骤中，你重新启动[!INCLUDE[btsBizTalkServerNoVersion](../../includes/btsbiztalkservernoversion-md.md)]顺序你所做的更改在本教程中才会生效。  
  
### <a name="to-restart-biztalk-server"></a>重新启动 BizTalk Server  
  
1.  启动**BizTalk Server 管理**。  
  
2.  在[!INCLUDE[btsCoName](../../includes/btsconame-md.md)][!INCLUDE[btsBizTalkServerNoVersion](../../includes/btsbiztalkservernoversion-md.md)]管理控制台中，展开**BizTalk Server 管理**，展开**BizTalk 组**，然后展开**平台设置**. 单击**主机**，然后单击**BizTalkServerApplication**。  
  
3.  右键单击你的服务器，并依次**重新启动**。  
  
 继续执行[步骤 9： 验证零碎的入站的批处理方案](../../adapters-and-accelerators/accelerator-hl7/step-9-verify-the-fragmented-inbound-batch-scenario.md)。
