# <a name="execute-scalar-functions-in-sql-server-using-the-sql-adapter"></a>在使用 SQL 适配器的 SQL Server 中执行标量函数
SQL Server 中的 Transact SQL 和 CLR 标量函数作为中的操作[!INCLUDE[adaptersql](../../includes/adaptersql-md.md)]。 中的操作名称[!INCLUDE[adaptersqlshort](../../includes/adaptersqlshort-md.md)]是 SQL Server 中的标量函数的名称相同。  
  
 标量函数中的所有参数都都在相应的操作。 中的操作的返回值[!INCLUDE[adaptersqlshort](../../includes/adaptersqlshort-md.md)]中定义 SQL Server 中的标量函数的返回值相同。  
  
 有关执行标量函数的详细信息，请参阅[调用中使用 BizTalk Server 的 SQL Server 的标量函数](../../adapters-and-accelerators/adapter-sql/invoke-scalar-functions-in-sql-server-using-biztalk-server.md)。
