# <a name="lesson-5-deploy-the-solution"></a>第 5 课：部署解决方案
在本课程中，将部署 BizTalk 解决方案使用[!INCLUDE[btsBizTalkServerNoVersion](../../includes/btsbiztalkservernoversion-md.md)]管理控制台。 在 BizTalk 业务流程设计器中，创建逻辑端口。 现在创建物理端口并将逻辑端口映射到物理端口，然后配置 BizTalk 应用程序。  
  
 配置应用程序后，将启动并测试应用程序。
