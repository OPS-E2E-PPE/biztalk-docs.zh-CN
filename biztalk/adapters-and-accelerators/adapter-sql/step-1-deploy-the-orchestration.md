# <a name="step-1-deploy-the-orchestration"></a>第 1 步：部署业务流程
![步骤 1，共 4 步](../../adapters-and-accelerators/adapter-oracle-ebs/media/step-1of4.gif "Step_1of4")  
  
 **若要完成的时间：** 5 分钟  
  
 **目标：** 在此步骤中，将部署业务流程解决方案。  
  
## <a name="prerequisites"></a>先决条件  
 你必须完成中的步骤[第 4 课：执行插入操作上采购订单表](../../adapters-and-accelerators/adapter-sql/lesson-4-perform-an-insert-operation-on-the-purchase-order-table.md)。  
  
### <a name="to-deploy-the-solution"></a>若要部署解决方案  
  
1.  在解决方案资源管理器，右键单击解决方案名称，然后单击**属性**。  
  
2.  在属性页对话框中，在树控件中，展开**配置属性**，然后单击**配置**。  
  
3.  上**配置**页上，在**部署**列中，选择 BizTalk 项目中，对应的复选框，然后单击**确定**。  
  
4.  在解决方案资源管理器，右键单击解决方案名称，然后单击**部署解决方案**。  
  
     在屏幕底部的输出窗格如下所示：**部署：1 个成功，0 失败，0 已跳过**。  
  
## <a name="what-did-i-just-do"></a>我只需做了什么？  
 在此步骤中，你已部署到 BizTalk 业务流程[!INCLUDE[btsBizTalkServerNoVersion](../../includes/btsbiztalkservernoversion-md.md)]管理控制台。
