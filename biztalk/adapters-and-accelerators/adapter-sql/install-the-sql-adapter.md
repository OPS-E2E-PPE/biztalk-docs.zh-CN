# <a name="install-the-sql-adapter"></a>安装 SQL 适配器
## <a name="install-options"></a>安装选项
你可以安装[!INCLUDE[adaptersqlshort](../../includes/adaptersqlshort-md.md)]两种方式：

* 选项 1：使用[!INCLUDE[adapterpacknoversion](../../includes/adapterpacknoversion-md.md)]安装：请参阅[安装的 BizTalk 适配器包 2016年](../../adapters-and-accelerators/install-the-biztalk-adapter-pack-2016.md)，或[安装 BizTalk 适配器包 2013 R2 和 2013年](../../adapters-and-accelerators/install-biztalk-adapter-pack-2013-r2-and-2013.md)。 此外可以使用此选项来安装其他基于 WCF 的适配器。
  
* 选项 2：其本身而言，如果只想要安装[!INCLUDE[adaptersqlshort](../../includes/adaptersqlshort-md.md)]作为一个单独的适配器。 对于此选项，在本部分中使用的安装指南。
  
> [!IMPORTANT]
>  [!INCLUDE[adaptersqlshort](../../includes/adaptersqlshort-md.md)]附带[!INCLUDE[adapterpacknoversion](../../includes/adapterpacknoversion-md.md)]，并独立[!INCLUDE[adaptersqlshort](../../includes/adaptersqlshort-md.md)]无法在同一台计算机上共存。  如果只想[!INCLUDE[adaptersqlshort](../../includes/adaptersqlshort-md.md)]，然后安装独立适配器。 如果你认为你可能想[!INCLUDE[adaptersqlshort](../../includes/adaptersqlshort-md.md)]，任何其他基于 WCF 的适配器，然后使用[!INCLUDE[adapterpacknoversion](../../includes/adapterpacknoversion-md.md)]安装程序。
