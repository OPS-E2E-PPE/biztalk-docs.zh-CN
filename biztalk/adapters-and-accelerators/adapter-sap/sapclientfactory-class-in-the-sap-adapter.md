# <a name="sapclientfactory-class-in-the-sap-adapter"></a>SAP 适配器 SAPClientFactory 类
以下部分列出的方法和属性**SAPClientFactory**类。 这派生自**System.Data.Common.DbProviderFactory**。  
  
## <a name="supported-properties"></a>支持的属性  
  
|“属性”|获取/设置|Description|  
|----------|--------------|-----------------|  
|**实例**|-|SAPClientFactory 单一实例|  
  
## <a name="supported-methods"></a>受支持的方法  
  
|“属性”|Description|  
|----------|-----------------|  
|**CreateCommand()**|创建 SAPCommand 的实例|  
|**CreateConnection()**|创建 SAPConnection 的实例|  
|**CreateConnectionStringBuilder()**|创建 SAPConnectionStringBuilder 的实例|  
|**CreateParameter()**|创建 SAPParameter 的实例|
