# <a name="use-the-sap-adapter-with-sharepoint"></a>使用包含 SharePoint 的 SAP 适配器
大多数业务线 (LOB) 应用程序具有相对较少所需的行业专业人员可以访问的专有接口。 但是，由于组织中所有级别的信息工作者需要相关信息，行业专家必须将所需的数据导出到更熟悉的应用程序，如 Microsoft Excel 为信息工作者的信息。  
  
 Microsoft Office 客户端应用程序和功能的新集合的最常见的界面将后端业务数据的信息工作者，Microsoft Office 消除这种差距。 Microsoft Office 客户端应用程序无缝地通过接口向信息工作者熟悉呈现的 LOB 应用程序的数据。 一个示例是 Microsoft Office SharePoint Server 中的业务数据目录。
