# <a name="use-the-sap-adapter-with-sharepoint"></a>使用带有 SharePoint SAP 适配器
大多数的业务线 (LOB) 应用程序具有相对较少所需的行业专业人员可以访问的专有接口。 但是，因为组织中所有级别信息工作者需要相关信息，行业专家必须将所需的数据导出到更熟悉的应用程序，像 Microsoft Excel 这样的信息工作人员可以使用该信息。  
  
 Microsoft Office 客户端应用程序和功能的新集合通过将后端业务数据到最常见的界面，从而为信息工作者，Microsoft Office 跨越这种差别。 Microsoft Office 客户端应用程序无缝通过为信息工作者所熟悉的接口提供从 LOB 应用程序的数据。 一个示例是 Microsoft Office SharePoint Server 中的业务数据目录。
