# <a name="use-the-data-provider-for-sap-with-ssrs"></a>使用包含 SSRS 的 SAP 数据提供程序
可以使用[!INCLUDE[adoprovidersaplong](../../includes/adoprovidersaplong-md.md)]以及 SQL Server Reporting Services (SSRS) 来创建基于 SAP 数据的报表。 有关 SSRS 的详细信息，请参阅[ http://go.microsoft.com/fwlink/?LinkId=125922 ](http://go.microsoft.com/fwlink/?LinkId=125922)。  
  
 若要创建基于 SAP 数据的报表，必须使用[!INCLUDE[btsVStudioNoVersion](../../includes/btsvstudionoversion-md.md)]创建报表服务器项目并使用[!INCLUDE[adoprovidersapshort](../../includes/adoprovidersapshort-md.md)]若要从 SAP 系统中提取数据。  
  
 在本部分中的主题提供有关如何创建用于基于 SAP 数据生成报表的报表服务器项目的信息。
