# <a name="browse-search-and-get-metadata-for-trfc-operations-in-sap"></a>浏览、 搜索和获取 tRFC 操作在 SAP 中的元数据
Trfc 不是 SAP 系统中一个单独的项目。 这些归通过单独的节点在[!INCLUDE[adaptersap_short](../../includes/adaptersap-short-md.md)]因为它们的元数据的特征是不同的 Rfc。 但是的浏览体验，搜索和检索 Trfc 的元数据与完全相同的 Rfc。 请参阅[浏览、 搜索和获取 RFC 操作在 SAP 中的元数据](../../adapters-and-accelerators/adapter-sap/browse-search-and-get-metadata-for-rfc-operations-in-sap.md)有关使用信息[!INCLUDE[addadapterservrefshort](../../includes/addadapterservrefshort-md.md)]或[!INCLUDE[consumeadapterservshort](../../includes/consumeadapterservshort-md.md)]浏览、 搜索和检索 Trfc 的元数据从 SAP 系统。  
  
 请注意，[!INCLUDE[adaptersap_short](../../includes/adaptersap-short-md.md)]特殊的操作的图面**RfcConfirmTransID**下**TRFC**节点。 SAP 适配器使用此操作提交到 SAP 服务器进行的 tRFC 调用。 有关此操作的详细信息，请参阅[Trfc 在 SAP 中的操作](../../adapters-and-accelerators/adapter-sap/operations-on-trfcs-in-sap.md)。
