# <a name="enterprise-applications-adapters-in-biztalk-server"></a>BizTalk Server 中的企业应用程序适配器

## <a name="overview"></a>概述

Microsoft BizTalk Server 包括使企业应用程序能够与每个其他接口的适配器。 企业应用程序包括以下适配器：  

* JD Edwards EnterpriseOne
* 博士 Edwards OneWorld
* PeopleSoft Enterprise
* TIBCO Enterprise Message Service
* TIBCO Rendezvous 

使用这些适配器，可以连接到本地业务线 (LOB) 系统，以获取数据，然后将数据放。 

## <a name="install"></a>Install
BizTalk Server 中附带了企业应用程序适配器。 有关特定的安装步骤中，请参阅：

[安装和配置 Microsoft BizTalk 适配器为企业应用程序](../adapters-and-accelerators/install-configure-biztalk-adapters-enterprise-applications.md)
