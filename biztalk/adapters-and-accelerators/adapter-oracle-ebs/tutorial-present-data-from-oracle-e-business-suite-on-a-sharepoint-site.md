# <a name="tutorial-present-data-from-oracle-e-business-suite-on-a-sharepoint-site"></a>教程：从 SharePoint 站点上的 Oracle E-business Suite 的呈现数据
本教程提供了详细的说明[!INCLUDE[adapteroraclebusinessshort](../../includes/adapteroraclebusinessshort-md.md)]使用 Microsoft Office SharePoint Server 在 SharePoint 门户网站上，从 Oracle E-business Suite 中显示数据。 它还演示了如何在 Microsoft Office SharePoint Server，您可以从 SharePoint 门户网站上配置 Oracle E-business Suite 项目执行全文搜索中配置的搜索应用程序。  
  
 若要演示如何执行此操作，请考虑存储有关员工的信息的 Oracle E-business Suite 中的示例接口表。 在本教程中，在从基于搜索字符串使用 Oracle E-business Suite 中检索客户列表的 Microsoft Office SharePoint Server 中创建的应用程序：  
  
-   Microsoft Office SharePoint Server 中业务数据列表 Web 部件  
  
-   Microsoft Office SharePoint Server 中的搜索功能  
  
> [!NOTE]
>  本教程前，请确保已安装使用的所有先决条件[!INCLUDE[adapteroraclebusinessshort](../../includes/adapteroraclebusinessshort-md.md)]使用 Microsoft Office SharePoint Server。 有关先决条件的信息，请参阅[!INCLUDE[adapterpacknoversion](../../includes/adapterpacknoversion-md.md)]安装指南中，通常安装在 C:\Program Files\Microsoft BizTalk 适配器 Pack\Documents。
