# <a name="key-features-in-biztalk-adapter-for-oracle-e-business-suite"></a>用于 Oracle E-business Suite 的 BizTalk 适配器的主要功能
本部分列出了中的关键功能[!INCLUDE[adapteroracleebusinesslong](../../includes/adapteroracleebusinesslong-md.md)]。  
  
## <a name="technology-related-features"></a>与技术相关的功能  
  
|                                                                           功能                                                                            |                                                                                                                                                                                                                                                                                                                                                                       注释                                                                                                                                                                                                                                                                                                                                                                       |
|--------------------------------------------------------------------------------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
|                                                        Windows Communication Foundation (WCF) 的使用                                                         |                                                                                                    [!INCLUDE[adapteroraclebusinessshort](../../includes/adapteroraclebusinessshort-md.md)]之上生成[!INCLUDE[afproductnamelong](../../includes/afproductnamelong-md.md)] ([!INCLUDE[afproductnameshort](../../includes/afproductnameshort-md.md)])。 反过来，[!INCLUDE[afproductnameshort](../../includes/afproductnameshort-md.md)]在 WCF 顶层构建。 适配器公开为 WCF 通道向适配器客户端。 这样，连接、 元数据交换和与外部系统的业务数据交换。                                                                                                    |
|                                                 对 WCF 通道模型和 WCF 服务模型的支持                                                 |                                                                                                                                                在 WCF*通道*模型中，适配器客户端可以使用[!INCLUDE[adapteroraclebusinessshort](../../includes/adapteroraclebusinessshort-md.md)]通过直接发送和接收 XML 消息。<br /><br /> 在 WCF*服务*模型中，适配器客户端可以生成一个.NET 代理类从 Web 服务描述语言 (WSDL) 使用获取[!INCLUDE[adapteroraclebusinessshort](../../includes/adapteroraclebusinessshort-md.md)]。                                                                                                                                                |
|                                                                        ODP.NET 的使用                                                                        |                                                                                                                                                                                                                                                                               [!INCLUDE[adapteroraclebusinessshort](../../includes/adapteroraclebusinessshort-md.md)]使用 Oracle 数据提供程序的.NET (ODP.NET) 与 Oracle E-business Suite 的接口。                                                                                                                                                                                                                                                                               |
|                                                      连接到 Oracle E-business Suite 的两种方法                                                       | 适配器客户端可以连接到 Oracle E-business Suite 中，通过使用 net 服务名称 tnsnames.ora 文件中或者通过直接指定连接参数，并因此无需使用网络服务名称或 tnsnames.ora 文件。 Tnsnames.ora 文件以连接到 Oracle E-business Suite 中无需将保存您需要手动更新连接参数 （net 服务名称） 的麻烦 tnsnames.ora 文件中每个客户端计算机上时添加或更新中的 Oracle 服务器应用环境。 有关详细信息，请参阅[配置到 Oracle E-business Suite 的连接](../../adapters-and-accelerators/adapter-oracle-ebs/configure-the-connection-uri-for-oracle-e-business-suite.md)。 |
|                                                              支持 Windows 身份验证                                                              |                                                       适配器客户端可以使用 Windows 身份验证连接到 Oracle E-business Suite。 Windows 身份验证使您能够确定用户的标识基于 Windows 登录凭据，并因此可帮助你利用内置的 Windows 环境的安全性。 有关详细信息中的 Windows 身份验证[!INCLUDE[adapteroraclebusinessshort](../../includes/adapteroraclebusinessshort-md.md)]，请参阅[连接到 Oracle E-business Suite 使用 Windows 身份验证](../../adapters-and-accelerators/adapter-oracle-ebs/connect-to-oracle-e-business-suite-using-windows-authentication.md)。                                                        |
| 支持使用[!INCLUDE[adapteroraclebusinessshort](../../includes/adapteroraclebusinessshort-md.md)]与 Microsoft Office SharePoint Server (MOSS) |                                                                                                                                                                                                                                              适配器可用于从 Oracle E-business Suite 到 MOSS 门户上显示的数据。 有关详细信息，请参阅[Microsoft Office SharePoint Server 中使用 Oracle E-business 适配器](https://msdn.microsoft.com/library/dd788485.aspx)。                                                                                                                                                                                                                                               |
  
## <a name="metadata-related-features"></a>与元数据相关的功能  
  
|功能|注释|  
|-------------|-------------|  
|批量检索的元数据|适配器客户端可以浏览和搜索 Oracle E-business Suite 和基础数据库公开的所有项目的元数据。 项目会显示基于连接的用户的凭据，并将其：<br /><br /> -Oracle E-business Suite 项目的每个应用程序分组存放。<br />-按每个 Oracle E-business Suite 和基础数据库中的项目分组。<br />-按基础数据库项目的每个架构分组。|

## <a name="operations-related-features"></a>与操作相关的功能  
  
|                                      功能                                      |                                                                                                                                                                                                                                                                                注释                                                                                                                                                                                                                                                                                 |
|-----------------------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
|          对界面表和界面视图上的操作的支持           |                                                                                                                                                                          适配器客户端可以执行基本的 Insert、 Update、 Delete 和 Oracle E-business Suite 的接口表的 Select 操作。 仅可以对 Oracle E-business Suite 界面视图执行选择操作。                                                                                                                                                                           |
|                         对执行 PL/SQL Api 的支持                         |                                                                                                                                                                                                               适配器客户端可以执行在 Oracle E-business Suite 中的 PL/SQL Api。 PL/SQL Api 包含存储的过程和函数在包中。                                                                                                                                                                                                                |
|                     对正在执行的并发程序的支持                     |                                                                                                                                                                                                                                              适配器客户端可以在 Oracle E-business Suite 中执行并发程序。                                                                                                                                                                                                                                               |
|                        用于执行请求集的支持                         |                                                                                          适配器客户端可以在 Oracle E-business Suite 中执行请求集。 请求集是一组并发程序，并可用于与预定义的参数运行多个并发程序。 请求集分阶段执行。 有关请求集的详细信息，请参阅[ http://go.microsoft.com/fwlink/?LinkId=129539 ](http://go.microsoft.com/fwlink/?LinkId=129539)。                                                                                          |
|           在插入操作中指定内联值的支持            | 可以使用**InlineValue**插入操作将计算的值插入接口表和 Oracle 中的数据库表中的属性。 这是一个可选属性，可用于插入操作中的所有简单的数据记录。 如果指定此属性的值，它将替代指定的值的一条记录。 有关 InlineValue 属性的详细信息，请参阅[对界面表和界面视图操作](../../adapters-and-accelerators/adapter-oracle-ebs/operations-on-interface-tables-and-interface-views.md)。 |
|                      支持使用轮询的入站调用                      |                                                                                                                   [!INCLUDE[adapteroraclebusinessshort](../../includes/adapteroraclebusinessshort-md.md)]支持接收"基于轮询的"数据更改的消息即适配器执行指定的 SQL 语句、 存储的过程、 函数或在包中的过程中检索数据，并向客户端提供的结果。                                                                                                                   |
|                   对调用函数和过程的支持                   |                                                                                                                                      适配器客户端可以调用函数和基础的 Oracle 数据库中的过程。 [!INCLUDE[adapteroraclebusinessshort](../../includes/adapteroraclebusinessshort-md.md)]作为可以在基础的 Oracle 数据库执行的操作的图面函数和过程。                                                                                                                                       |
|                         支持复合操作                          |                                         [!INCLUDE[adapteroraclebusinessshort](../../includes/adapteroraclebusinessshort-md.md)]使适配器客户端能够在 Oracle E-business Suite 中执行复合操作。 复合操作可以包含任意数量的以下操作，并按任何顺序：<br /><br /> -对界面表、 界面视图、 表和视图的操作。<br />存储过程、 函数和过程中显示的包作为在适配器中的操作。                                          |
|         对执行任意 SQL 语句和 PL/SQL 块的支持          |                              [!INCLUDE[adapteroraclebusinessshort](../../includes/adapteroraclebusinessshort-md.md)]使适配器客户端能够执行任意 SQL 语句或 PL/SQL 块使用 ExecuteReader 和 ExecuteScalar 或 ExecuteNonQuery 操作。 有关这些操作的详细信息，请参阅[支持 ExecuteNonQuery、 ExecuteReader 和 ExecuteScalar 操作](../../adapters-and-accelerators/adapter-oracle-ebs/support-for-executenonquery-executereader-and-executescalar-operations.md)。                               |
| 对用户定义类型 (Udt)、 布尔参数和 PL/SQL 表类型的支持 |                                                                                                     适配器客户端可以支持包含 Udt，项目中的操作和操作中存储的过程和包含布尔参数和 PL/SQL 表类型的函数。 有关 UDT 的支持信息，请参阅[支持的 Oracle User-Defined 类型](../../adapters-and-accelerators/adapter-oracle-ebs/support-for-oracle-user-defined-types2.md)。                                                                                                      |
|                     支持的输入和输出的 REF CURSOR                      |                                                                                                                                                                                                                                       适配器客户端能够运行过程和函数的输入和输出 REF CURSOR。                                                                                                                                                                                                                                       |
|                       对 PL/SQL 记录的数据类型的支持                        |                                                                                                                                                                                                                                  适配器客户端能够使用记录和存储的过程和函数中的嵌套的记录类型进行操作。                                                                                                                                                                                                                                   |
|           对重载的函数和 PL/SQL API 中的过程的支持           |                                                                                   适配器客户端可以调用重载的函数和 PL/SQL Api 中的过程。 但是，[!INCLUDE[adapteroraclebusinessshort](../../includes/adapteroraclebusinessshort-md.md)]不支持使用重载过程使用强类型化和弱类型化 REF CURSOR。 在内部，适配器将这两个强类型化和弱类型化 REF CURSOR 视为只是 REF CURSOR。                                                                                    |
|                     对数据库更改通知的支持                     |      适配器客户端可以从基于触发的 SELECT 语句的 Oracle 数据库接收数据库更改通知。 作为适配器客户端和在结果集的 SELECT 语句更改时，将通过 Oracle 数据库发送通知。 有关数据库更改通知的详细信息，请参阅[接收数据库更改通知使用 ORacle 数据库适配器中的不考虑](../../adapters-and-accelerators/adapter-oracle-database/before-you-receive-database-change-notifications-using-the-oracle-db-adapter.md)。      |
|                               同义词支持                                |                                                                     适配器客户端可以执行对为表、 视图、 存储的过程、 函数和包创建的同义词的操作。 有关同义词，以及如何可以使用详细信息[!INCLUDE[adapteroraclebusinessshort](../../includes/adapteroraclebusinessshort-md.md)]若要执行对同义词的操作，请参阅[对同义词的操作](../../adapters-and-accelerators/adapter-oracle-ebs/operations-on-synonyms2.md)。                                                                      |
