# <a name="integrate-oracle-e-business-suite-with-the-business-data-catalog-and-sharepoint"></a>将 Oracle 电子商务套件与业务数据目录和 SharePoint 集成
[!INCLUDE[afproductnamelong](../../includes/afproductnamelong-md.md)]包括[!INCLUDE[afsvcdevwizlong](../../includes/afsvcdevwizlong-md.md)]，分别用于执行特定的 LOB 项目的 WCF 服务。 此 WCF 服务承载在宿主环境如 Microsoft Internet 信息服务 (IIS) 中。 Business Data Catalog Definition Editor 使用 WCF 服务承载的 WCF 服务中获取 Web 服务描述语言 (WSDL) URL。 使用 WSDL，Business Data Catalog Definition Editor 中提取可用于 WCF 服务的方法。 这些方法可以用于建立实体和实体之间的关联。
