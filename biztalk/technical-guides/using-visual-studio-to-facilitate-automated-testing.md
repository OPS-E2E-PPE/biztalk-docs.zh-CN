# <a name="using-visual-studio-to-facilitate-automated-testing"></a>使用 Visual Studio 以便于自动测试
Visual Studio 2010 提供功能强大的负载测试功能，能模拟数百个用户同时访问服务器应用程序的负载配置文件。 此负载测试功能提供了所选的关键性能指标，以及供将来分析数据库中存储这些指标的能力的实时度量值。 本部分介绍使用 Visual Studio 负载测试以评估 BizTalk 服务器应用程序的性能。
