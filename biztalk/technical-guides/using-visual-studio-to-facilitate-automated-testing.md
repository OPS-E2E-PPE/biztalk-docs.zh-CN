# <a name="using-visual-studio-to-facilitate-automated-testing"></a>使用 Visual Studio 优化自动测试
Visual Studio 2010 提供了功能强大的负载测试功能，能模拟多达数百个用户同时访问服务器应用程序的负载配置文件。 此负载测试功能提供了所选的关键性能指标，以及能够以供未来分析数据库中存储这些指标的实时度量值。 本部分介绍如何使用 Visual Studio 负载测试来评估 BizTalk Server 应用程序的性能。
