# <a name="how-to-add-a-reference-to-an-application"></a>如何添加对应用程序的引用
本主题提供有关使用 BizTalk Server 管理控制台从到另一个应用程序添加的引用信息。 当你想要在同一 BizTalk 组中的另一个应用程序中已存在的当前应用程序中使用项目时添加的引用。 如果你想要使用特定于 EDI 架构、 管道和业务流程 （BizTalk EDI 应用程序中已部署） 从另一个应用程序，如 BizTalk Application 1，则必须添加对 BizTalk EDI 应用程序的引用。 您不应将自定义项目添加到 BizTalk EDI 应用程序。  

 有关信息：  

- 将引用添加到另一个应用程序，请参阅[如何添加对另一个应用程序的引用](http://go.microsoft.com/fwlink/?LinkID=155011)(http://go.microsoft.com/fwlink/?LinkID=155011)。  

- 添加对的引用[!INCLUDE[btsBizTalkServerNoVersion](../includes/btsbiztalkservernoversion-md.md)]EDI 应用程序，请参阅[如何添加对 BizTalk Server EDI 应用程序的引用](http://go.microsoft.com/fwlink/?LinkId=155026)(<http://go.microsoft.com/fwlink/?LinkId=155026>)。
