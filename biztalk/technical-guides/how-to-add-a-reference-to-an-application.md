# <a name="how-to-add-a-reference-to-an-application"></a>如何添加对应用程序的引用
本主题提供有关使用 BizTalk Server 管理控制台来添加从到另一个应用程序的引用信息。 如果需要在当前应用程序中使用在同一 BizTalk 组中的另一应用程序中已存在的项目，就要添加引用。 如果你想要使用 EDI 特定架构、 管道和业务流程 （已部署在 BizTalk EDI 应用程序中） 从如 BizTalk 应用程序 1 的另一个应用程序，你必须添加到 BizTalk EDI 应用程序的引用。 不应将自定义项目添加到 BizTalk EDI 应用程序中。  
  
 有关的信息：  
  
-   引用添加到另一个应用程序，请参阅[如何添加对另一个应用程序的引用](http://go.microsoft.com/fwlink/?LinkID=155011)(http://go.microsoft.com/fwlink/?LinkID=155011)。  
  
-   添加对引用[!INCLUDE[btsBizTalkServerNoVersion](../includes/btsbiztalkservernoversion-md.md)]EDI 应用程序，请参阅[如何添加对 BizTalk Server EDI 应用程序的引用](http://go.microsoft.com/fwlink/?LinkId=155026)(http://go.microsoft.com/fwlink/?LinkId=155026)。
