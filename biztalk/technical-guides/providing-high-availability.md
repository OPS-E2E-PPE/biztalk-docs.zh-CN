# <a name="providing-high-availability"></a>提供高可用性
为提供高可用性[!INCLUDE[btsBizTalkServerNoVersion](../includes/btsbiztalkservernoversion-md.md)]通过在环境中实现冗余，以便每个功能的组件的环境。 BizTalk 主机的冗余，可以通过安装运行的多台计算机实现[!INCLUDE[btsBizTalkServerNoVersion](../includes/btsbiztalkservernoversion-md.md)]BizTalk 组，然后配置要在多个运行的计算机上运行的主机的实例中[!INCLUDE[btsBizTalkServerNoVersion](../includes/btsbiztalkservernoversion-md.md)]组中。 中的其他组件的冗余[!INCLUDE[btsBizTalkServerNoVersion](../includes/btsbiztalkservernoversion-md.md)]环境，可以通过为中的资源配置组件实现[!INCLUDE[btsWinSvrNoVersion](../includes/btswinsvrnoversion-md.md)]群集。 [!INCLUDE[btsBizTalkServerNoVersion](../includes/btsbiztalkservernoversion-md.md)]此外可以适应 BizTalk 主机配置中的资源为[!INCLUDE[btsWinSvrNoVersion](../includes/btswinsvrnoversion-md.md)]群集，建议以提供高可用性，对于某些 BizTalk 适配器。  
  
 本部分介绍如何为中的功能组件提供高可用性[!INCLUDE[btsBizTalkServerNoVersion](../includes/btsbiztalkservernoversion-md.md)]环境。
