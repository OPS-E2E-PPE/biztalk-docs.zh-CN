# <a name="installing-biztalk-server-2010-accelerator-for-swift"></a>安装 BizTalk Server 2010 Accelerator for SWIFT
![徽标](../technical-guides/media/bts-10-installaccelerator-logo.gif "BTS_10_InstallAccelerator_Logo")  
  
 BizTalk 技术文章  
  
 **安装 BizTalk Server 2010 Accelerator for SWIFT**  
  
 **编写器：** Maria Quian，Microsoft Corporation  
  
 **技术审阅人员：** Andres Naranjo，Mandi Ohlinger  
  
 **发布日期：** 2012 年 6 月  
  
 **适用于：** BizTalk Server 2010 Accelerator for SWIFT  
  
 **摘要：** 成功安装了 Microsoft BizTalk Server 2010 Accelerator for SWIFT (A4SWIFT) 的基础是了解 A4SWIFT 功能和每项功能所依赖的 BizTalk 组件。 每个 A4SWIFT 功能依赖于其他正确配置 BizTalk 组件。 影响构建 A4SWIFT 环境的其他注意事项包括现有的网络基础结构和已配置的 BizTalk Server 环境。 本文档将引用，并使用现有的文档生成 BizTalk Server 环境。 A4SWIFT 继承 BizTalk Server 2010 的所有软件和硬件要求。  
  
 安装指南是用于每个 A4SWIFT 相关的组件，但不是单个编译的文档的基础进行阐述的安装/配置步骤。 本文档将将它们组合在一起，并了解一些安装和配置过程中遇到的常见问题。
