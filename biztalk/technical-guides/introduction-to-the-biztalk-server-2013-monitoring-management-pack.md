# <a name="introduction-to-the-biztalk-server-2013-monitoring-management-pack"></a>BizTalk Server 2013 监视管理包简介
[!INCLUDE[btsBizTalkServerNoVersion](../includes/btsbiztalkservernoversion-md.md)]主动式和反应式监视 BizTalk Server 部署中，应用程序和其构成项目的管理包提供。 此管理包提供的监视包括运行状况监视、 可用性和配置监视、 性能数据收集以及默认阈值。 你可以集成到面向服务的监视方案监视 BizTalk Server 组件。 此管理包提供作为 Web 下载。  
  
 除了运行状况监视功能，此管理包包括仪表板视图，如启用实时诊断并解决检测到的问题的应用程序和部署视图。 [!INCLUDE[btsBizTalkServerNoVersion](../includes/btsbiztalkservernoversion-md.md)]管理包提供用于全面监视 BizTalk Server 的重要事件和性能计数器，以提供集中式的管理和监视 BizTalk Server 部署和应用程序体验。  
  
## <a name="getting-the-latest-management-pack"></a>获取最新的管理包  
 [BizTalk Server 管理包](http://www.microsoft.com/download/details.aspx?id=39617)(http://www.microsoft.com/download/details.aspx?id=39617)  
  
## <a name="where-do-i-start"></a>如何开始？  
 若要开始使用[!INCLUDE[btsBizTalkServerNoVersion](../includes/btsbiztalkservernoversion-md.md)]管理包，必须首先安装 System Center Operations Manager，然后下载并安装[!INCLUDE[btsBizTalkServerNoVersion](../includes/btsbiztalkservernoversion-md.md)]管理包。
