# <a name="how-to-view-management-pack-details"></a>如何查看管理包详细信息
有关监视器和关联的替代值的信息，请查看监视器的知识。  
  
### <a name="to-view-knowledge-for-a-monitor"></a>若要查看监视器的知识  
  
1.  在操作控制台中，单击**创作**按钮。  
  
2.  展开**管理包对象**，然后单击**监视器**。  
  
3.  在监视器窗格中，展开目标直到达到监视器级别。 或者，可以使用**搜索**框查找特定监视器。  
  
4.  单击监视器中，在监视器窗格中，单击**查看知识**。  
  
5.  单击**产品知识**选项卡。
