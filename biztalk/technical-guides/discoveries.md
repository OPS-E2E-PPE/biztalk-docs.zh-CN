# <a name="discoveries"></a>发现
[!INCLUDE[btsBizTalkServerNoVersion](../includes/btsbiztalkservernoversion-md.md)]管理包提供集中式的管理和监视体验，BizTalk Server 与相关的项目。  
  
## <a name="discovery-of-artifacts"></a>项目的发现  
 与[!INCLUDE[btsBizTalkServerNoVersion](../includes/btsbiztalkservernoversion-md.md)]管理包监视代理发现了一个运行时计算机中的项目。 默认情况下，管理包发现联接到 BizTalk 组的第一个计算机上的项目。 但是如果你想计划发现所选的计算机上的项目然后必须使用替代功能来更改自动发现的设置。 若要使用替代更改自动发现的设置，请参阅[可选配置](../technical-guides/optional-configurations.md)。  
  
 操作和 SCOM 操作控制台中执行的任务是唯一的将发现并监视所有运行时计算机上的项目。 若要查看 BizTalk Server 管理包提供的所有任务，请单击**任务**在 Operations Manager 2007 R2/2012年操作控制台的监视窗格中。
