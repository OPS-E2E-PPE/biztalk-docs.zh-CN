# <a name="performing-a-simple-update"></a>执行简单的更新
BizTalk 应用程序或项目的简单更新要求没有停机时间，要求不需要更改代码，以及可以由 IT 专业人员或业务分析师 singlehandedly 执行。 这些类型的更改是通常为将其他发送端口添加到发送端口组、 与相应订阅筛选器将添加其他发送端口、 更改 URI 发送端口，或更新 BRE 中的规则一样简单。 有关详细信息，请参阅[创建和使用业务规则](http://go.microsoft.com/fwlink/?LinkId=155136)(http://go.microsoft.com/fwlink/?LinkId=155136) 中[!INCLUDE[btsBizTalkServerNoVersion](../includes/btsbiztalkservernoversion-md.md)]帮助。
