# <a name="performing-a-simple-update"></a>执行简单更新
BizTalk 应用程序或项目的简单的更新要求没有停机时间、 不需要更改任何代码，并可由 IT 专业人员或业务分析师 singlehandedly 执行。 这些类型的更改是通常与将附加的发送端口添加到发送端口组、 使用相应的订阅筛选器添加其他发送端口、 更改 URI 的发送端口，或更新在 BRE 规则一样简单。 有关详细信息，请参阅[创建和使用业务规则](http://go.microsoft.com/fwlink/?LinkId=155136)(<http://go.microsoft.com/fwlink/?LinkId=155136>) 中[!INCLUDE[btsBizTalkServerNoVersion](../includes/btsbiztalkservernoversion-md.md)]帮助。
