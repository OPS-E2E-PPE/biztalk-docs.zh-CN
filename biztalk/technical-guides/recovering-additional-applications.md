# <a name="recovering-additional-applications"></a>恢复其他应用程序
可能必须还原整个应用程序解决方案的一部分的其他应用程序。 恢复其他应用程序的过程必须由应用程序团队定义并作为整体的灾难恢复计划的一部分。
