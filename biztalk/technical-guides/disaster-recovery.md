# <a name="disaster-recovery"></a>灾难恢复
灾难恢复过程通过采用的步骤可恢复失败的系统操作来提高系统可用性。 灾难恢复与不同的容错能力的灾难恢复通常需要手动干预来还原发生故障的系统，而容错的设计可以继续操作而无需手动干预，如果系统遇到故障条件。
