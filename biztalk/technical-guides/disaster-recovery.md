# <a name="disaster-recovery"></a>灾难恢复
灾难恢复过程通过采用的步骤，恢复失败的系统的操作来提高系统的可用性。 灾难恢复与不同的容错能力，在灾难恢复通常需要手动操作来还原发生故障的系统，而容错的设计可以继续运行而无需手动干预，如果系统在遇到失败条件。
