# <a name="monitoring-sql-servers"></a>监视 SQL Server

## <a name="use-sql-management-pack"></a>使用 SQL 管理包
Microsoft SQL Server 管理包提供主动式和反应式监视 SQL Server 在企业环境中。 可用性和配置监视，为企业级监视生成性能数据收集以及默认阈值。 本地和远程连接检查有助于确保数据库的可用性。  
  
 使用 SQL Server 管理包中嵌入的专业技术，可以主动管理 SQL Server，并变得严重之前先行确定问题。 此管理包可提高安全性、 可用性和 SQL Server 基础结构的性能。  
  
 附带了包的 Microsoft SQL Server 管理包指南描述管理包的内容，并介绍如何将其部署。 管理包的功能包括：  
  
-   监视 SQL Server、 SQL 代理、 报表服务器、 Notification Services 包含的服务的状态  
  
-   监视数据库的状态  
  
-   监视在数据库中，可配置按 %或 MB 的可用空间  
  
-   确保数据库已正确配置  
  
-   确保客户端可以连接到 SQL Server  
  
-   监视阻塞的进程  
  
-   观看有关失败的代理作业和过多的时间执行的作业  
  
-   监视复制和故障的警报的运行状况  
  
-   监视数据库镜像状态
