# <a name="implementing-automated-testing"></a>实现自动测试
BizTalk Server 解决方案通常部署在"任务关键型"的情况下，BizTalk 解决方案凭此是业务的核心组件。 在这些情况下，持续的性能和稳定性 BizTalk 解决方案会关键业务要求;如果 BizTalk 解决方案失败，则关联的停机时间成本是有意义。 在这种情况下，它是一项极其重要解决方案放入到生产环境之前进行全面测试 BizTalk 解决方案问题。 不测试或缺乏测试解决方案导致的停机时间成本相比，与正确的测试解决方案相关联的成本要小。 因此，测试 BizTalk Server 解决方案无疑是任何 BizTalk Server 解决方案部署的最重要阶段。  
  
 本部分介绍用于在生产环境中运行该解决方案之前测试 BizTalk 解决方案的正确方法。
