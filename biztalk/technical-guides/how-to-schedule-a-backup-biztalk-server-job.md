# <a name="how-to-schedule-a-backup-biztalk-server-job"></a>如何安排备份 BizTalk Server 作业
备份 BizTalk Server 作业按照 SQL Server 代理服务制定的计划运行。 如果要修改备份频率，可使用 SQL Server 企业管理器来更改备份 BizTalk Server 作业的计划。 有关如何安排备份 BizTalk Server 作业的详细信息，请参阅[如何安排备份 BizTalk Server 作业](http://go.microsoft.com/fwlink/?LinkId=154674)(<http://go.microsoft.com/fwlink/?LinkId=154674>) 中[!INCLUDE[btsBizTalkServerNoVersion](../includes/btsbiztalkservernoversion-md.md)]帮助。
