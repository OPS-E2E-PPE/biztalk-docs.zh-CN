 **摘要：** Microsoft® BizTalk® Server 2010 可帮助组织应对挑战，即创建连接不同系统的自动化的业务流程。 从 EAI 和 B2B 集成中其初始根，BizTalk Server 会继续添加新功能和引擎改进功能，让开发人员创建功能强大的面向服务的体系结构和业务流程集成和管理解决方案和使管理员和业务用户可以更有效地监视正在进行的业务流程。 BizTalk Server 2010 表示 Microsoft 的长期策略，以启用连接的企业中的下一个版本。
