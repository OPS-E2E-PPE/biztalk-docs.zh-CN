# <a name="maintaining-biztalk-server2"></a>维护 BizTalk Server2
维护活动是服务监视和控制 (SMC) 系统管理函数定义由 Microsoft Operations Framework (MOF) 的一部分。 SMC 的主要目标是观察的运行状况你[!INCLUDE[btsBizTalkServerNoVersion](../includes/btsbiztalkservernoversion-md.md)]系统。 SMC 检查可能启动更正操作来避免潜在的服务事件以及发生时，服务事件的影响降至最低。  
  
 对于本指南此部分，SMC 活动划分为四个三大类： 可靠性、 管理、 安全性和性能。 每日、 每周和每月的维护清单组织 SMC 检查根据应执行每个检查的频率。 如果 SMC 检查指示不需要执行更正操作，清单将定向到的其他信息来解决该问题的源。  
  
 与解决可靠性相关的各节，管理、 安全性和性能问题包含有关识别和解决过程中遇到的许多常见问题信息[!INCLUDE[btsBizTalkServerNoVersion](../includes/btsbiztalkservernoversion-md.md)]操作。  
  
 有关 Microsoft Operations Framework 的详细信息，请参阅[http://go.microsoft.com/fwlink/?linkid=88047](http://go.microsoft.com/fwlink/?linkid=88047)。 有关服务监视和控制函数的详细信息，请参阅[http://go.microsoft.com/fwlink/?LinkId=155341](http://go.microsoft.com/fwlink/?LinkId=155341)。
