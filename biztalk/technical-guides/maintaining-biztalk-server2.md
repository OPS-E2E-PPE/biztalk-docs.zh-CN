# <a name="maintaining-biztalk-server2"></a>维护 BizTalk Server2
维护活动是在服务监视和控制 (SMC) 系统管理函数定义由 Microsoft Operations Framework (MOF) 的一部分。 SMC 的主要目标是要观察的运行状况在[!INCLUDE[btsBizTalkServerNoVersion](../includes/btsbiztalkservernoversion-md.md)]系统。 SMC 检查可能会启动补救措施以避免潜在的服务事件并在出现时，服务事件的影响降至最低。  
  
 用于指南的本部分的目的，SMC 活动分为四个三大类： 可靠性、 管理、 安全性和性能。 每日、 每周和每月维护清单组织 SMC 检查根据每个检查应执行的频率。 如果 SMC 检查指示需要更正操作，清单将定向到的附加信息来解决该问题的源。  
  
 解决可靠性相关的各节，管理、 安全性和性能问题包含有关识别和解决过程中遇到的许多常见问题的信息[!INCLUDE[btsBizTalkServerNoVersion](../includes/btsbiztalkservernoversion-md.md)]操作。  
  
 有关 Microsoft Operations Framework 的详细信息，请参阅[ http://go.microsoft.com/fwlink/?linkid=88047 ](http://go.microsoft.com/fwlink/?linkid=88047)。 有关服务监视和控制函数的详细信息，请参阅[ http://go.microsoft.com/fwlink/?LinkId=155341 ](http://go.microsoft.com/fwlink/?LinkId=155341)。
