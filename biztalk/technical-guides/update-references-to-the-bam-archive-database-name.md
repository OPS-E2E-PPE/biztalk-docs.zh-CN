# <a name="update-references-to-the-bam-archive-database-name"></a>更新对 BAM 存档数据库名称的引用
如果备份了 BAM 存档数据库，在系统或数据发生故障时您可以将该备份还原到其他计算机，并可以重命名该备份。  
  
 若要还原 BAM 存档数据库，请执行中的步骤[如何还原数据库备份 BizTalk Server 作业中](../technical-guides/how-to-restore-databases-in-the-backup-biztalk-server-job.md)。 此外，您必须使用新的服务器名称和数据库名称更新 BAM SSIS 包。  
  
## <a name="how-to-update-references-to-bam-archive-database"></a>如何更新对 BAM 存档数据库的引用  
 有关如何更新对 BAM 存档数据库的引用的说明，请参阅[更新到新的 BAM 存档数据库的引用](../technical-guides/how-to-move-the-bam-archive-database1.md#BKMK_UpdateArch)。
