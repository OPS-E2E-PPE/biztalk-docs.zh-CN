# <a name="appendix-a-silent-installation"></a>附录 A：无提示安装
本主题列出了创建以静默方式安装的步骤[!INCLUDE[btsBizTalkServerNoVersion](../includes/btsbiztalkservernoversion-md.md)]。  
  
## <a name="create-a-silent-installation"></a>创建无提示安装  
  
1.  单击**启动**，单击**所有程序**，单击**附件**，然后单击**命令提示符下**。  
  
2.  转到安装位置。 在命令提示符下键入`setup.exe /``<command name> <options>`，然后按**ENTER**。 日志文件会显示安装状态。

 **其他**  
  
- [!INCLUDE[btsBizTalkServerNoVersion](../includes/btsbiztalkservernoversion-md.md)] 支持自动的电子软件分发，也称为无提示安装。 无提示安装执行以下任务：  
  
  - 安装[!INCLUDE[btsBizTalkServerNoVersion](../includes/btsbiztalkservernoversion-md.md)]具有相同的配置的计算机上。  
  
  - 允许系统管理员安装[!INCLUDE[btsBizTalkServerNoVersion](../includes/btsbiztalkservernoversion-md.md)]而无需用户干预的远程计算机上。  
  
  - 用户没有需监视安装和提供输入。  
  
- 若要创建无提示安装，请使用命令行选项来禁止所有交互。  
  
- 完成无提示安装，在命令窗口中不显示任何消息。 使用安装日志文件查看状态。
