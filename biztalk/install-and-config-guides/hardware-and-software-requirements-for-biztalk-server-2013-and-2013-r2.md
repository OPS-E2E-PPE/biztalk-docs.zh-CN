# <a name="hardware-and-software-requirements-for-biztalk-server-2013-and-2013-r2"></a>硬件和软件要求适用于 BizTalk Server 2013 和 2013 R2
本主题列出了 [!INCLUDE[btsBizTalkServerNoVersion](../includes/btsbiztalkservernoversion-md.md)] 2013 和 [!INCLUDE[bts2013r2](../includes/bts2013r2-md.md)] 要求。  

##  <a name="BKMK_HardRequirements"></a> 硬件要求  
 下表显示了对 [!INCLUDE[btsBizTalkServerNoVersion](../includes/btsbiztalkservernoversion-md.md)] 计算机的最低硬件要求。 在生产环境中，通信量可能会对服务器的硬件提出更高要求。  


|       组件        |                                                                                                                                                                                                                                                                                                                                                                                       最低要求                                                                                                                                                                                                                                                                                                                                                                                       |
|------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| 计算机和处理器 | 具有与 Intel Pentium 兼容的 CPU 的计算机：<br /><br /> -1 GHz 或更高的单处理器<br />-900 MHz 或更高的双处理器<br />-700 MHz 或更高的四核处理器<br /><br /> 支持超线程和双核处理器<br /><br /> [!INCLUDE[btsBizTalkServerNoVersion](../includes/btsbiztalkservernoversion-md.md)] 的 64 位版本需要在基于 x64 的系统上运行的 64 位操作系统。 基于与 AMD64 (x86-64) 和扩展内存 64 位技术 (EM64T) 处理器结构兼容的 CPU 的计算机被视作基于 x64 的系统。<br /><br /> [!INCLUDE[btsBizTalkServerNoVersion](../includes/btsbiztalkservernoversion-md.md)] 不支持基于 Itanium 的系统上。 |
|         内存         |                                                                                                                                                                                                                                                                                                                                                                                           2 GB 的 RAM                                                                                                                                                                                                                                                                                                                                                                                           |
|       硬盘        |                                                                                                                                                                                                                                                                                                               10 GB 的可用硬盘空间，用于包括操作系统和所有必备软件的完全安装。 硬盘必须是 NTFS 格式。                                                                                                                                                                                                                                                                                                                |
|         驱动器          |                                                                                                                                                                                                                                                                                                                                                                          CD-ROM 或 DVD-ROM 驱动器（共享或连接）                                                                                                                                                                                                                                                                                                                                                                           |
