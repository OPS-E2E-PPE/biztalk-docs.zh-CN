# <a name="hardware-and-software-requirements-for-biztalk-server-2016"></a>硬件和软件要求适用于 BizTalk Server 2016

## <a name="hardware-requirements"></a>硬件要求
下表列出了在 BizTalk Server 计算机的最低硬件要求。 在生产环境中的流量可能需要更高的硬件需求的服务器。 

| Resource  |最低要求 |
| --- | --- |
|计算机和处理器 | 具有与 Intel Pentium 兼容的 CPU 的计算机： <ul><li>1 GHz 或更高的单处理器</li><li>900 MHz 或更高的双处理器</li><li>700 MHz 或更高的四核处理器</li></ul> **注意**：支持超线程和双核处理器<br/><br/>64 位版本的 BizTalk Server 需要基于 x64 的系统上运行的 64 位操作系统。 基于与 AMD64 (x86-64) 和扩展内存 64 位技术 (EM64T) 处理器结构兼容的 CPU 的计算机被视作基于 x64 的系统。<br/><br/>BizTalk Server 不支持基于 Itanium 的系统上。 |
|内存 | 2 GB 或更多|
|硬盘  |10 GB 的可用硬盘空间用于包括操作系统和所有必备软件的完整安装。 硬盘必须是 NTFS 格式。|

> [!TIP] 
> 列出的硬件要求是最小值。 每个环境都不同，并且没有你的环境可能需要更多的机率相当高。 请参阅[有关安装、 大小调整、 部署和维护 BizTalk Server 解决方案的建议](http://social.technet.microsoft.com/wiki/contents/articles/666.recommendations-for-installing-sizing-deploying-and-maintaining-a-biztalk-server-solution.aspx)。

## <a name="service-pack-and-cumulative-update-support"></a>Service Pack 和累积更新支持

BizTalk Server 支持所有 service pack、 累积更新、 安全更新和修补程序。 强烈建议安装最新的 Windows、 SQL Server、 Visual Studio 和任何已安装程序更新它。 根据 Microsoft 产品的基准支持提供对该产品 Service Pack 的支持。 请参阅[支持生命周期索引](http://go.microsoft.com/fwlink/p/?LinkID=151890)为 BizTalk Server、 SQL Server、 Visual Studio 和其他 Microsoft 程序。

[BizTalk Server 的 service Pack 和累积更新列表](https://support.microsoft.com/help/2555976)
