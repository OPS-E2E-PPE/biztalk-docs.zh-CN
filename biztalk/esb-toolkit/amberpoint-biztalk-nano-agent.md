# <a name="amberpoint-biztalk-nano-agent"></a>AmberPoint BizTalk Nano 代理
AmberPoint SOA 管理系统 (SMS) 提供了一套全面的可视化效果和分析工具来解决运行时管理的问题。 [!INCLUDE[esbToolkit](../includes/esbtoolkit-md.md)]和 Microsoft BizTalk Server 轻松集成 SOA 业务应用程序和服务，但生成的复合应用程序和业务流程的分析是更具挑战性。 通过其 BizTalk Nano 代理 AmberPoint 的 BizTalk Server 集成扩展了现有的运行时管理功能，包括 BizTalk Server 接收位置和发送端口，而不考虑其基础的适配器绑定。
