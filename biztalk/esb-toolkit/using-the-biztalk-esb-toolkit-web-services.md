# <a name="using-the-biztalk-esb-toolkit-web-services"></a>使用 BizTalk ESB 工具包 Web 服务
[!INCLUDE[esbToolkit](../includes/esbtoolkit-md.md)]公开其所有核心都功能作为 Web 服务; 这使 ESB，可能正在运行非 Microsoft 平台上参与 ESB 交换并充分利用 ESB 功能之外的实体。 所有 Web 服务都位于单独的 ESB 安装目录的 \Web 文件夹的子文件夹。 [!INCLUDE[esbToolkit](../includes/esbtoolkit-md.md)]包括以下 ESB Web 服务：  
  
-   [路线接入点 Web 服务](../esb-toolkit/the-itinerary-on-ramp-web-services.md)  
  
-   [解析程序 Web 服务](../esb-toolkit/the-resolver-web-service.md)  
  
-   [转换 Web 服务](../esb-toolkit/the-transformation-web-service.md)  
  
-   [异常处理 Web 服务](../esb-toolkit/the-exception-handling-web-service.md)  
  
-   [BizTalk 操作 Web 服务](../esb-toolkit/the-biztalk-operations-web-service.md)
