# <a name="alerts-by-application-or-service"></a>应用程序或服务的警报
此选项显示在指定时间段为一组指定的应用程序生成的所有警报的聚合视图。 选择特定的服务将显示一个报表，列出该特定服务的警报。 可以将警报的列表导出为 Microsoft Excel 电子表格。  
  
### <a name="to-view-charts-and-lists-of-alerts-by-application-or-service"></a>若要查看应用程序或服务的图表和警报列表  
  
1.  打开[门户报告页](../esb-toolkit/portal-reports-page.md)。  
  
2.  单击**选择应用程序**链接上方的第一个图表，然后选择或清除复选框将显示已安装 Microsoft BizTalk Server 应用程序列表中。 此外可以使用显示来选择所有或无应用程序的链接。 单击**保存**若要显示的信息对于所选应用程序，或单击**取消**放弃所做的更改。  
  
3.  若要更改的期限为其图表显示的信息，请选择**小时、 天、 周、 月、 季度、 年**或**所有**的第一个图表上方的下拉列表中。  
  
4.  选择一个应用程序;可为每个服务该应用程序中显示的警报图表会发生更改。  
  
5.  在图表; 中选择服务页将显示一个表，其中列出了该服务的警报。  
  
6.  单击**导出到 Excel**若要下载可以在 Microsoft Excel 中查看的格式中的警报列表。
