# <a name="alerts-by-application-or-service"></a>由应用程序或服务的警报
此选项显示指定时间段内针对一组指定的应用程序生成的所有警报的聚合视图。 选择特定的服务将显示一个报表，列出针对该特定服务的警报。 可以将警报的列表导出为 Microsoft Excel 电子表格。  
  
### <a name="to-view-charts-and-lists-of-alerts-by-application-or-service"></a>若要查看由应用程序或服务的图表和警报列表  
  
1.  打开[门户报表页](../esb-toolkit/portal-reports-page.md)。  
  
2.  单击**选择应用程序**链接上方的第一个图表，然后选择或清除复选框中显示的已安装 Microsoft BizTalk Server 应用程序的列表。 你也可以使用显示选择所有或应用程序的任何链接。 单击**保存**以显示所选应用程序的信息或单击**取消**放弃所做的更改。  
  
3.  若要更改时间的图表显示的信息，请选择**小时、 天、 周、 月、 季度、 每年，** 或**所有**第二个图表上方的下拉列表中。  
  
4.  选择一个应用程序;若要显示警报，该应用程序中服务的每个图表会发生更改。  
  
5.  在图表; 中选择一个服务该页面显示表，其中列出为该服务的警报。  
  
6.  单击**导出到 Excel**下载可以在 Microsoft Excel 中查看的格式中的警报列表。
