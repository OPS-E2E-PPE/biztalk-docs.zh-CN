# <a name="installing-the-sql-adapter-sample"></a>安装 SQL 适配器示例
本部分介绍安装 SQL 适配器示例的过程。 必须安装中的示例[安装和运行路线接入点示例](../esb-toolkit/installing-and-running-the-itinerary-on-ramp-sample.md)并[安装和运行动态解析示例](../esb-toolkit/installing-and-running-the-dynamic-resolution-sample.md)安装和使用此示例之前。  
  
 **若要安装 SQL 适配器示例**  
  
1.  在 Windows 资源管理器中打开文件夹 \Source\Samples\SqlAdapter\Install\Scripts。  
  
2.  双击名为 Setup_bin.cmd 的脚本文件。
