# <a name="using-the-helper-classes"></a>通过帮助程序类
[!INCLUDE[esbToolkit](../includes/esbtoolkit-md.md)] 包括一组由其他进程、 服务和工具包中的组件的帮助器类。 因为这些帮助器类也是公共的您可以自己所需的代码中使用它们。  
  
 在某些情况下，您必须使用它们的 ESB 中的其他类的方法提供的相应类型的参数。 例如，解析程序 Web 服务方法返回的实例的泛型集合**冲突解决程序**类。  
  
 以下各节介绍类中包含的帮助器[!INCLUDE[esbToolkit](../includes/esbtoolkit-md.md)]:

-   [Resolver 类](../esb-toolkit/the-resolver-class.md)  
  
-   [ResolverDictionary 类](../esb-toolkit/the-resolverdictionary-class.md)  
  
-   [Resolver Manager (ResolverMgr) 类](../esb-toolkit/the-resolver-manager-resolvermgr-class.md)
