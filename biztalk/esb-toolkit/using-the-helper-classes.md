# <a name="using-the-helper-classes"></a>使用的帮助程序类
[!INCLUDE[esbToolkit](../includes/esbtoolkit-md.md)]包括一组由其他进程、 服务和工具包中的组件的帮助器类。 因为这些帮助器类也是公共的你可以根据需要代码中使用它们。  
  
 在某些情况下，你必须使用它们的 ESB 中的其他类的方法提供的适当类型的参数。 例如，冲突解决程序 Web 服务方法返回的实例的泛型集合**冲突解决程序**类。
