# <a name="installing-the-namespace-component-sample"></a>安装 Namespace 组件示例
Namespace 组件示例要求 Namespace 组件 （在程序集 Microsoft.Practices.ESB.Namespace.PipelineComponents.dll） 驻留在你的 Microsoft BizTalk Server 安装文件夹的 PipelineComponents 文件夹中。 安装[!INCLUDE[esbToolkit](../includes/esbtoolkit-md.md)]核心会自动将程序集复制到正确的位置。  
  
 本部分介绍将 Namespace 组件示例安装到 GlobalBank.ESB BizTalk 应用程序的过程。 可以从解决方案项目安装 Namespace 组件示例，也可以使用随附的 Windows Installer 文件[!INCLUDE[esbToolkit](../includes/esbtoolkit-md.md)]。 本节包含下列主题：  
  
-   [安装使用安装脚本 Namespace 组件示例](../esb-toolkit/install-the-namespace-component-sample-using-the-install-scripts.md)  
  
-   [程序集和项目安装 Namespace 组件示例](../esb-toolkit/assemblies-and-artifacts-installed-by-the-namespace-component-sample.md)
