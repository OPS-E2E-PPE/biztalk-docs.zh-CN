# <a name="portal-admin-pages"></a>门户管理员页
将鼠标移**管理员**选项卡上，弹出菜单提供了一系列门户管理任务。 **管理员**选项卡时才会显示为具有管理权限 （Microsoft BizTalk Server Administrators 帐户组的成员） 的用户。 以下选项将在弹出菜单上：  
  
-   **管理审核日志**。 此菜单项将打开[审核日志页](../esb-toolkit/audit-log-page.md)包含审核事件的列表。 从此页中，可以查看与审核事件相关联的消息详细信息。  
  
-   **错误设置**。 此菜单项将打开[错误设置页](../esb-toolkit/fault-settings-page.md)，可以指定一系列的管理设置的审核、 警报服务队列和警报的电子邮件服务。  
  
-   **管理警报**（管理视图）。 此菜单项将打开[门户警报页](../esb-toolkit/portal-alerts-page.md)管理视图中。 管理员可以使用此页可以查看和编辑所有警报和订阅系统中。 他们还可以查看和编辑与特定警报相关联的订阅。  
  
-   **注册表设置**。 此菜单项将打开[注册表设置页](../esb-toolkit/registry-settings-page.md)可以在其中指定一系列用于注册表集成、 电子邮件通知和模板的管理设置。
