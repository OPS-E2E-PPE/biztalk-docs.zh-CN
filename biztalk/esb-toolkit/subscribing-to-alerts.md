# <a name="subscribing-to-alerts"></a>订阅警报
### <a name="to-subscribe-to-an-alert"></a>若要订阅警报  
  
1.  打开[门户警报页](../esb-toolkit/portal-alerts-page.md)若要查看现有警报的列表，以及对这些警报在当前订阅。  
  
2.  在操作列中，单击**AddSubscriber**现有警报的图标。 这将打开[添加警报订阅和编辑订阅页](../esb-toolkit/add-alert-subscription-and-edit-subscription-pages.md)。  
  
3.  在门户将警报通知发送到与用于访问门户的帐户关联的电子邮件地址。 如果你想要使用不同的电子邮件地址，选中的复选框旁边**自定义电子邮件**文本框中，并键入在文本框中的首选电子邮件地址。  
  
4.  选择中的复选框**计划**部分中如果你想要指定在段时在门户应将警报发送到你，，然后指定中时间段的开始和结束时间**开始时间**并**EndTime**下拉列表。  
  
5.  如果当你将不能接收和操作警报一段，则输入中的开始和结束日期**买卖期**框。  
  
6.  选择**间隔**中**分钟**和多个实例的同一个警报发生时发送只有一个的比较引发的警报在门户将过程。 这可以防止出现一次快速故障创建大量的相同警报消息。  
  
7.  单击**保存**按钮以创建新的订阅。  
  
### <a name="to-edit-an-existing-alert-subscription"></a>若要编辑现有的警报订阅  
  
1.  打开[门户警报页](../esb-toolkit/portal-alerts-page.md)若要查看现有警报的列表，以及对这些警报在当前订阅。  
  
2.  单击**编辑**想要修改的列表中的订阅旁边的链接**我的订阅**部分中的页后，可以打开[添加警报订阅和编辑订阅页](../esb-toolkit/add-alert-subscription-and-edit-subscription-pages.md).  
  
3.  在门户将警报通知发送到与用于访问门户的帐户关联的电子邮件地址。 如果你想要使用不同的电子邮件地址，选中的复选框旁边**自定义电子邮件**文本框中，并键入在文本框中的首选电子邮件地址。  
  
4.  选择中的复选框**计划**部分中如果你想要指定在段时在门户应将警报发送到你，，然后指定中时间段的开始和结束时间**开始时间**并**EndTime**下拉列表。  
  
5.  键入的开始和结束日期**买卖期**是否存在时将不能接收和操作警报一段。  
  
6.  选择**间隔**中**分钟**期间的门户将比较引发的警报和多个实例的同一个警报发生时发送只有一个。 这可以防止出现一次快速故障创建大量的相同警报消息。  
  
7.  单击**保存**按钮可更新订阅。  
  
### <a name="to-delete-an-existing-alert-subscription"></a>若要删除现有的警报订阅  
  
1.  打开[门户警报页](../esb-toolkit/portal-alerts-page.md)若要查看现有警报的列表，以及对这些警报在当前订阅。  
  
2.  单击**删除**想要删除的列表中的订阅旁边的链接**我的订阅**页部分。
