# <a name="installing-the-exception-management-samples-using-install-scripts"></a>安装使用安装脚本异常管理示例
本部分介绍如何从提供的安装脚本安装异常管理示例[!INCLUDE[esbToolkit](../includes/esbtoolkit-md.md)]。  
  
 **若要从安装脚本安装 ESB 异常管理示例**  
  
1.  在 **“开始”** 菜单上，单击 **“运行”**。  
  
2.  在**运行**对话框中，键入**cmd**，然后按 ENTER 以打开命令提示符。  
  
3.  运行以下命令，替换*\<路径 >*参数替换为你想要安装的.cmd 文件的完整路径 （默认路径和名称在此版本中是 \Source\Samples\Exception Handling\Install\Scripts\\):  
  
    ```  
    <path>\Setup_bin.cmd  
    ```
