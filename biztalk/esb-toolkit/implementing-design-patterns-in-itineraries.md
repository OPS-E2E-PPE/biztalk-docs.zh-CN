# <a name="implementing-design-patterns-in-itineraries"></a>在路线中实现设计模式
本主题介绍常见的面向服务的体系结构 (SOA) 模式和类别，可使用实现[!INCLUDE[esbToolkit](../includes/esbtoolkit-md.md)]。 每个模式类别包括每个相关的模式和更多详细信息的链接的说明。 每个单独模式说明内容包括对相关的引用[!INCLUDE[esbToolkit](../includes/esbtoolkit-md.md)]示例应用程序实现的模式和开发活动，旨在向开发人员介绍如何创建 ESB 的解决方案来实现这些模式的链接。 本部分包括以下主题部分：  
  
-   [消息路由模式](../esb-toolkit/message-routing-patterns.md)  
  
-   [消息转换模式](../esb-toolkit/message-transformation-patterns.md)  
  
-   [服务介质模式](../esb-toolkit/service-mediation-patterns.md)  
  
-   [服务管理模式](../esb-toolkit/service-management-patterns.md)
