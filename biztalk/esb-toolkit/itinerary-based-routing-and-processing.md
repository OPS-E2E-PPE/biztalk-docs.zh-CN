# <a name="itinerary-based-routing-and-processing"></a>基于路线的路由和处理
[!INCLUDE[esbToolkit](../includes/esbtoolkit-md.md)]实现通过自定义管道组件使用的路由滑动模式。 消息元数据和其他因素用于确定适当的路由滑动也称为路线，要用于每个消息。 此路由滑动即可执行消息转换、 调用业务流程服务，并定义条消息，路由 BizTalk Server 将执行的步骤，有效地分离核心 BizTalk Server 引擎中的消息处理指令。  
  
 有关如何处理路线的详细信息，请参阅[主要方案和开发任务](../esb-toolkit/key-scenarios-and-development-tasks.md)。
