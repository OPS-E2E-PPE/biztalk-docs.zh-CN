# <a name="installing-and-running-the-biztalk-operations-sample"></a>安装和运行 BizTalk 操作示例
Microsoft BizTalk 操作示例演示如何查询上使用 ESB BizTalk 操作服务的 BizTalk 应用程序的操作数据。 ESB 管理门户使用此服务返回或更新 BizTalk 管理和跟踪数据库中的信息。 可用的信息，包括：  

- 已部署 BizTalk 应用程序和主机的状态  

- 活动或挂起服务实例  

- 消息和它们的属性  

- 已部署的 BizTalk 应用程序  

- 接收端口属性和接收位置  

- 发送端口属性  

  本节包含下列主题：  

- [安装 BizTalk 操作示例](../esb-toolkit/installing-the-biztalk-operations-sample.md)  

- [运行 BizTalk 操作示例](../esb-toolkit/running-the-biztalk-operations-sample.md)
