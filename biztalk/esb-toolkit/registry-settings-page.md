# <a name="registry-settings-page"></a>注册表设置页
图 1 显示了注册表设置页上，可以在其中修改的设置在门户时，使用它与通用描述、 发现和集成 (UDDI) 服务进行交互。  
  
 ![注册表设置页](../esb-toolkit/media/ch8-registrysettingspage.gif "Ch8-RegistrySettingsPage")  
  
 **图 1**  
  
 **ESB 管理门户注册表设置页**  
  
 以下列表说明了如何使用 ESB 管理门户注册表设置页的功能：  
  
-   **UDDI 的详细信息**在页面顶部的部分包含的 URI 或 UDDI 服务器的 URL 的文本框中，则默认为本地 Microsoft UDDI 服务。 有两个复选框下本部分中，您可以启用自动发布 UDDI 注册请求的请求并指定是否访问 UDDI 服务器将通过匿名帐户，而不是当前用户的 Microsoft Windows 帐户。  
  
-   **电子邮件通知**页部分包含一个复选框，如果您希望在门户中向发送自动电子邮件通知的注册请求的选择。 在下，此复选框，是一系列的文本框中提供的电子邮件服务器，若要使用，详细信息"to"和"from"电子邮件地址、 邮件主题和消息正文。  
  
-   **默认设置**页部分包含的文本框，输入默认联系人和电子邮件地址门户注册过程中使用。  
  
-   若要编辑的页中的值后更新注册表设置，请单击**更新**按钮。  
  
-   若要返回到以前的页面，而无需更改的注册表设置，请单击**取消**按钮。
