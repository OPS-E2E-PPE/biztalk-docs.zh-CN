# <a name="registry-settings-page"></a>“注册表设置”页
图 1 显示了注册表设置页中，你可以在其中修改设置此门户使用时它与通用、 描述、 发现和集成 (UDDI) 服务进行交互。  
  
 ![注册表设置页](../esb-toolkit/media/ch8-registrysettingspage.gif "Ch8 RegistrySettingsPage")  
  
 **图 1**  
  
 **ESB 管理门户注册表设置页**  
  
 以下列表说明如何使用 ESB 管理门户注册表设置页的功能：  
  
-   **UDDI 详细信息**页顶部的部分的 URI 或 UDDI 服务器的 URL 包含一个文本框，则默认为本地 Microsoft UDDI 服务。 有此部分下，可以启用自动发布 UDDI 登记的请求并指定是否到 UDDI 服务器的访问权限将通过匿名帐户，而不是当前用户的 Microsoft Windows 帐户的两个复选框。  
  
-   **电子邮件通知**页部分包含一个复选框，如果你希望在门户中向发送自动电子邮件通知的注册请求，则选择。 在下，此复选框，没有文本框中提供的电子邮件服务器，若要使用，详细信息的其中一系列"to"和"发件人' 电子邮件地址、 邮件主题和消息正文。  
  
-   **默认设置**页部分包含文本框中输入的默认联系人和电子邮件地址在门户的注册过程中使用的位置。  
  
-   若要编辑的页中的值后更新注册表设置，请单击**更新**按钮。  
  
-   若要返回到以前的页面，而无需更改的注册表设置，请单击**取消**按钮。
