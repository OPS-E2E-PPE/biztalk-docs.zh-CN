# <a name="viewing-history-information"></a>查看历史记录信息
ESB 管理门户将维护对通用描述、 发现和集成 (UDDI) 注册的所有请求的历史记录。 管理员可以查看此列表在门户中。  
  
### <a name="to-view-uddi-request-history"></a>若要查看 UDDI 请求历史记录  
  
1.  请确保您登录到在门户中使用该帐户是 （门户管理员的自动成员） 的 Microsoft BizTalk Server Administrators 帐户组的成员。  
  
2.  指向**注册表**在门户的主菜单上，选项卡，然后单击**管理挂起的请求**以打开门户[管理挂起的请求页](../esb-toolkit/manage-pending-requests-page.md)。  
  
3.  单击**查看请求历史记录**链接。
