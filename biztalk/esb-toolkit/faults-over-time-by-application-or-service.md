# <a name="faults-over-time-by-application-or-service"></a>一段时间，由应用程序或服务的错误
此选项对于一组指定的应用程序指定期间内显示的错误的计数。 你可以选择应用程序以显示趋势图，显示一段时间的应用程序中的特定服务的错误数。 选择特定的服务将显示一个报表，列出的该服务的错误。 你还可以为 Microsoft Excel 电子表格中导出的错误的列表。  
  
### <a name="to-view-charts-and-lists-of-faults-over-time-by-application-or-service"></a>随着时间的推移查看图表和错误的列表，由应用程序或服务  
  
1.  打开[门户报表页](../esb-toolkit/portal-reports-page.md)。  
  
2.  单击**选择应用程序**链接上方的第一个图表，然后选择或清除复选框中显示的已安装 Microsoft BizTalk Server 应用程序的列表。 你也可以使用显示选择所有或应用程序的任何链接。 单击**保存**以显示所选应用程序的信息或单击**取消**放弃所做的更改。  
  
3.  若要更改时间的图表显示的信息，请选择**小时、 天、 周、 月、 季度、 每年，** 或**所有**第二个图表上方的下拉列表中。  
  
4.  单击图表密钥部分中的应用程序，以显示该应用程序中服务的每个错误的趋势图表的名称。  
  
5.  单击此图表以显示列出为所选服务的所有错误的表中的服务的名称。  
  
6.  单击**导出到 Excel**下载可以在 Microsoft Excel 中查看的格式错误的列表。
