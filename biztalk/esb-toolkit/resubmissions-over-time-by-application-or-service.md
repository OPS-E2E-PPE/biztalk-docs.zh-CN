# <a name="resubmissions-over-time-by-application-or-service"></a>超时由应用程序或服务重新提交
此选项显示的一组指定的应用程序在指定期间内的失败的消息重新提交的计数。 您可以选择要显示趋势图表显示一段时间让应用程序中的特定服务重新提交数的应用程序。 选择特定的服务将显示一个报表，列出该服务重新提交。 此外可以为 Microsoft Excel 电子表格导出重新提交的列表。  
  
### <a name="to-view-charts-and-lists-of-resubmissions-over-time-by-application-or-service"></a>若要通过应用程序或服务随着时间的推移查看图表和列表的重新提交  
  
1.  打开[门户报告页](../esb-toolkit/portal-reports-page.md)。  
  
2.  单击**选择应用程序**链接上方的第一个图表，然后选择或清除复选框将显示已安装 Microsoft BizTalk Server 应用程序列表中。 此外可以使用显示来选择所有或无应用程序的链接。 单击**保存**若要显示的信息对于所选应用程序，或单击**取消**放弃所做的更改。  
  
3.  若要更改的期限为其图表显示的信息，请选择**小时、 天、 周、 月、 季度、 年**或**所有**的第一个图表上方的下拉列表中。  
  
4.  单击图表密钥部分中的应用程序，以显示该应用程序中的服务的每个重新提交的趋势图表的名称。  
  
5.  单击以显示一个表，列出为所选服务的所有重新提交此图表中的服务名称。  
  
6.  单击**导出到 Excel**以下载可以在 Microsoft Excel 中查看的格式重新提交的列表。
