# <a name="manage-alerts-administration-view"></a>管理警报 （管理视图）
图 1 显示管理视图，管理员可以使用来查看所有警报和及其活动的摘要的列表中的通知页。 表显示每个警报的行。 每行显示警报的名称、 创建者的姓名、 警报 （例如，在最后一个小时、 天或月警报激活的数量） 的常规统计信息的链接警报的订阅服务器上，若要查看列表，并包含一系列 cli 的操作列的计数若要对警报执行操作的 ckable 图标。  
  
 ![管理警报页](../esb-toolkit/media/ch8-managealertspage.jpg "Ch8-ManageAlertsPage")  
  
 **图 1**  
  
 **ESB 管理门户管理警报 （管理视图） 页**  
  
 以下列表说明了如何使用 ESB 管理门户管理警报页的功能：  
  
-   单击**创建警报**顶部的创建新的警报页的链接。  
  
-   单击**导出到 Excel**链接以将警报的列表导出为 Microsoft Excel 电子表格。  
  
-   单击 +**查看订户**行以显示用于警报的订阅服务器列表中的链接。 在行中，展开该列表并允许管理员从警报中删除订阅服务器上每个操作列中显示一个删除图标。 同时，该链接将更改到-**隐藏订户**，允许您折叠的订阅服务器列表。  
  
-   单击该行中的警报执行的任务的操作列中的图标。 （按它们出现的顺序） 图标和任务如下所示：  
  
    -   **查看警报详细信息。** 此图标显示所选警报的详细信息。  
  
    -   **查看历史记录**。 此图标显示包含所选警报的所有激活的表。  
  
    -   **添加订阅者**。 此图标允许管理员将订阅服务器添加到所选警报。  
  
    -   **编辑警报**。 此链接允许管理员编辑警报的详细信息。  
  
    -   **删除警报**。 此链接中删除的警报和所有关联的订阅。
