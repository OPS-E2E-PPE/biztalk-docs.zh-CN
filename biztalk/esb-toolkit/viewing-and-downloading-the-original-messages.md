# <a name="viewing-and-downloading-the-original-messages"></a>查看和下载原始消息
若要更详细地查看错误消息，请使用**故障查看器**ESB 管理门户的页。  
  
### <a name="to-view-the-properties-and-status-of-a-fault-message"></a>若要查看的属性和状态的错误消息  
  
1.  在上找到的错误消息[门户错误页](../esb-toolkit/portal-faults-page.md)ESB 管理门户。  
  
2.  单击要打开 ESB 故障消息查看器中的行中的任意位置[错误详细信息视图](../esb-toolkit/fault-details-view.md)。  
  
3.  滚动到底部的页上，消息的列表，然后单击要打开该文件的所选的故障消息的消息标识符[消息详细信息视图](../esb-toolkit/message-details-view.md)。  
  
### <a name="to-view-and-download-individual-messages-from-a-fault-message"></a>若要查看和下载单个消息中的错误消息  
  
1.  在上找到的错误消息[门户错误页](../esb-toolkit/portal-faults-page.md)ESB 管理门户。  
  
2.  单击要打开 ESB 故障消息查看器中的行中的任意位置[错误详细信息视图](../esb-toolkit/fault-details-view.md)视图。  
  
3.  向下滚动到页面底部的消息的列表。  
  
4.  单击要打开该文件的所选的故障消息的消息标识符[消息详细信息视图](../esb-toolkit/message-details-view.md)查看，或单击**下载**下载邮件正文。  
  
5.  若要从消息的详细信息视图中下载该消息，请单击**下载**内容的消息的文本框下方的链接。
