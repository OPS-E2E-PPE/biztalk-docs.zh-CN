# <a name="viewing-and-downloading-the-original-messages"></a>查看和下载原始的消息
若要查看更多详细信息中的错误消息，请使用**错误查看器**ESB 管理门户页。  
  
### <a name="to-view-the-properties-and-status-of-a-fault-message"></a>若要查看的属性和错误消息的状态  
  
1.  在上找到的错误消息[门户错误页](../esb-toolkit/portal-faults-page.md)ESB 管理门户。  
  
2.  要打开 ESB 错误消息查看器中的行中单击任一处[错误详细信息视图](../esb-toolkit/fault-details-view.md)。  
  
3.  滚动到底部的页上，消息的列表，然后单击以打开该文件的选择的故障消息的消息标识符[消息详细信息视图](../esb-toolkit/message-details-view.md)。  
  
### <a name="to-view-and-download-individual-messages-from-a-fault-message"></a>若要查看和下载各个消息从错误消息  
  
1.  在上找到的错误消息[门户错误页](../esb-toolkit/portal-faults-page.md)ESB 管理门户。  
  
2.  要打开 ESB 错误消息查看器中的行中单击任一处[错误详细信息视图](../esb-toolkit/fault-details-view.md)视图。  
  
3.  向下滚动到页面底部的消息列表。  
  
4.  单击以打开该文件的选择的故障消息的消息标识符[消息详细信息视图](../esb-toolkit/message-details-view.md)查看，或单击**下载**下载消息正文。  
  
5.  若要下载消息详细信息视图中的消息，请单击**下载**下消息内容的文本框中的链接。
