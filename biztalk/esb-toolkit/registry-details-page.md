# <a name="registry-details-page"></a>“注册表详细信息”页
图 1 显示注册表的详细信息页上，可以在其中发布现有的通用、 描述、 发现和集成 (UDDI) 注册表请求、 创建新的请求，或编辑现有请求。  
  
 ![注册表详细信息页面](../esb-toolkit/media/ch8-registrydetailspage.gif "Ch8 RegistryDetailsPage")  
  
 **图 1**  
  
 **注册表 ESB 管理门户的详细信息页**  
  
 以下列表说明如何使用注册表 ESB 管理门户的详细信息页的功能：  
  
-   **注册表的详细信息**页顶部的部分显示注册表信息。 若要更改这些信息，可以使用[注册表设置页](../esb-toolkit/registry-settings-page.md)位于主**注册表**选项卡。  
  
-   **注册表服务**页部分包含下拉列表和文本框中，您可在其中输入要向其注册服务的详细信息。 这些详细信息包括业务提供程序;服务名称和说明;和有关该类别和终结点的信息。  
  
-   **注册表联系**页部分包含你在其中输入请求联系人详细信息的文本框; 包括名称、 电子邮件地址和用户的类型。  
  
-   如果打开从注册表的详细信息页面[管理挂起的请求页](../esb-toolkit/manage-pending-requests-page.md)，注册表的详细信息页包含可用于发布挂起的注册删除挂起的注册，或更新它，请返回到的按钮前一页。  
  
-   如果打开从注册表的详细信息页面[新注册表条目页](../esb-toolkit/new-registry-entry-page.md)，注册表的详细信息页包含可用于发布挂起的注册或其更新，请返回到上一页的按钮。
