# <a name="esb-management-portal-feature-reference"></a>ESB 管理门户功能参考
ESB 管理门户提供了轻松地监视、 管理和调试 ESB 应用程序故障信息的视图。 它还提供可用于管理警报、 将 UDDI 信息发布和管理门户的功能。 若要打开门户，请转到 http://localhost/ESB.Portal/。  
  
 在门户中提供了以下视图：  
  
-   [门户主页上](../esb-toolkit/portal-home-page.md)。 此页显示的应用程序的总体状态、 错误的摘要、 UDDI 注册和显示的错误类型和应用程序细分的图表的最近的请求。  
  
-   [门户故障页](../esb-toolkit/portal-faults-page.md)。 此页显示被截断为每个错误; 属性的列表它由一系列条件支持的错误进行的分析。  
  
-   [门户故障消息查看器](../esb-toolkit/portal-fault-message-viewer.md)。 此页才可访问但门户错误页。 它显示的单个错误消息的详细信息，并允许您修复并重新提交消息。  
  
-   [门户警报页](../esb-toolkit/portal-alerts-page.md)。 此页面允许用户创建和修改警报，并配置警报的电子邮件设置。 在门户可以发送警报电子邮件消息特定类型的错误或任何错误时，会发生。  
  
-   [门户报告页](../esb-toolkit/portal-reports-page.md)。 此页显示图表，该值指示错误、 警报和消息重新提交活动从应用程序的角度看，并支持的应用程序中的各个服务的运行状况分析。  
  
-   [门户注册表页](../esb-toolkit/portal-registry-pages.md)。 此页面到 UDDI 服务，用户可以发布服务终结点 （例如 Microsoft BizTalk Server 接收位置）。 UDDI 服务充当组织内的服务终结点的中央注册表。  
  
-   [门户管理员页](../esb-toolkit/portal-admin-pages.md)。 此页面允许管理员配置门户设置，查看审核日志的已编辑和重新提交消息，并管理用户和组的警报。  
  
-   [门户我的设置页面](../esb-toolkit/portal-my-settings-page.md)。 此页面，用户可以选择默认应用程序和门户的时间段筛选器。
