# <a name="esb-management-portal-feature-reference"></a>ESB 管理门户功能参考
ESB 管理门户提供了方便地监视、 管理和调试 ESB 应用程序的故障信息的视图。 它还提供可用于管理警报、 将 UDDI 信息发布和管理门户的功能。 若要打开门户网站，请转到 http://localhost/ESB.Portal/。  
  
 以下视图是在门户中可用：  
  
-   [门户主页上](../esb-toolkit/portal-home-page.md)。 此页显示整个应用程序状态、 错误的摘要、 UDDI 注册和显示的错误以及按应用程序类型和细分的图表的最近的请求。  
  
-   [门户错误页](../esb-toolkit/portal-faults-page.md)。 此页显示截断的每个错误; 的属性的列表它由一系列条件支持的错误的分析。  
  
-   [门户的错误消息查看器](../esb-toolkit/portal-fault-message-viewer.md)。 此页才可访问但门户错误页。 它显示的单独错误消息的详细信息，并允许你修复并重新提交消息。  
  
-   [门户警报页](../esb-toolkit/portal-alerts-page.md)。 此页面允许用户创建和修改警报，并配置警报的电子邮件设置。 门户可以发送警报的电子邮件时特定类型的错误或任何错误，发生该事件。  
  
-   [门户报表页](../esb-toolkit/portal-reports-page.md)。 此页显示图表，该值指示错误，警报，以及从应用程序的角度的消息重新提交活动，并支持的应用程序中的各个服务的运行状况分析。  
  
-   [门户注册表页](../esb-toolkit/portal-registry-pages.md)。 此页到 UDDI 服务允许用户 （如 Microsoft BizTalk Server 接收位置） 发布服务终结点。 UDDI 服务充当在组织内的服务终结点的中央注册表。  
  
-   [门户管理员页](../esb-toolkit/portal-admin-pages.md)。 此页面允许管理员以配置门户设置，查看审核日志的编辑和重新提交消息，并管理用户和组的警报。  
  
-   [门户我设置页](../esb-toolkit/portal-my-settings-page.md)。 此页面允许用户选择的默认应用程序和为该门户的时间段筛选器。
