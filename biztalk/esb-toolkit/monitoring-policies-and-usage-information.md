# <a name="monitoring-policies-and-usage-information"></a>监视策略和使用情况信息
SOA 服务管理器提供一种机制，允许你创建和编辑监视策略。 例如，图 1 显示应用的策略模板并激活应用程序监视的屏幕。  
  
 ![监视功能](../esb-toolkit/media/ch9-monitoringfeatures.jpg "Ch9 MonitoringFeatures")  
  
 **图 1**  
  
 **SOA 服务管理器在 Workbench 页上显示可用的监视功能**  
  
 SOA 服务管理器还提供了有关应用程序、 服务，使用情况信息和个人发送端口和接收位置。 此信息包括显示随时间、 服务使用情况和响应时间信息的每个操作的相对使用情况和使用情况的图表。 图 2 显示了一些示例。  
  
 ![监视图表](../esb-toolkit/media/ch9-monitoringcharts.jpg "Ch9 MonitoringCharts")  
  
 **图 2**  
  
 **选定的 SOA 服务管理器可以生成以帮助监视服务器和应用程序行为和性能的图表**  
  
 此外，SOA 服务管理器包括允许你查看各条消息，包括大小、 响应时间、 操作名称和相关联的 SOAP 消息，如图 3 中所示的详细信息的功能。  
  
 ![Soap 消息详细信息](../esb-toolkit/media/ch9-soapmessagedetails.jpg "Ch9 SoapMessageDetails")

 **SOA 服务管理器显示各个邮件的详细信息**
