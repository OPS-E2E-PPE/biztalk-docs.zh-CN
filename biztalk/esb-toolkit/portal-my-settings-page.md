# <a name="portal-my-settings-page"></a>门户我设置页
ESB 管理门户将会保留每个用户的设置。 这些设置包括应用程序中的图表上显示的列表[门户主页上](../esb-toolkit/portal-home-page.md)，和图表的默认期限。 图 1 显示了我的设置页上，您可以在其中编辑这些设置。  
  
 ![门户 Mysetting 页](../esb-toolkit/media/ch8-portalmysettingspage.jpg "Ch8-PortalMySettingsPage")  
  
 **图 1**  
  
 **ESB 管理门户我的设置页**  
  
 以下列表说明了如何使用 ESB 管理门户我的设置页的功能：  
  
-   你想要在门户主页中的图表中包括错误数据的当前已安装 Microsoft BizTalk Server 应用程序的每个选中的复选框。 门户将使用此列表作为默认值时打开主页上，但您可以更改其在主页上基于每个会话中使用的链接。  
  
-   选择您要在主页页面上的图表中显示数据的默认间隔。 您可以选择包含数据的上一个**小时、 天、 周、 月、 季度、 年**或**所有**。 在门户时打开主页上，但可以在主页上基于每个会话中使用下拉列表更改为默认使用此值。  
  
-   单击**保存**按钮可更新默认设置。  
  
-   单击**取消**按钮以返回到以前的页面而不更新你的设置。
