# <a name="portal-my-settings-page"></a>门户我设置页
ESB 管理门户将保留为每个用户的设置。 这些设置包括应用程序上显示的图表列表[门户主页上](../esb-toolkit/portal-home-page.md)，和图表的默认期限。 图 1 显示我的设置页上，你可以在其中编辑这些设置。  
  
 ![门户 MySettings 页](../esb-toolkit/media/ch8-portalmysettingspage.jpg "Ch8 PortalMySettingsPage")  
  
 **图 1**  
  
 **ESB 管理门户我设置页**  
  
 以下列表说明如何使用 ESB 管理门户我的设置页的功能：  
  
-   你想要在门户主页中的图表中包括错误数据的当前安装的 Microsoft BizTalk Server 应用程序的每个选中的复选框。 门户打开主页上，但你可以更改它基于每个会话的主页中使用的链接，会为默认使用此列表。  
  
-   选择您要在主页上的图表中显示数据的默认时间间隔。 你可以选择包括上一个数据**小时、 天、 周、 月、 季度、 每年，**或**所有**。 门户打开主页上，但你可以更改它在主页上基于每个会话中使用下拉列表，会为默认使用此值。  
  
-   单击**保存**按钮来更新你的默认设置。  
  
-   单击**取消**按钮以返回到以前的页面而不更新你的设置。
