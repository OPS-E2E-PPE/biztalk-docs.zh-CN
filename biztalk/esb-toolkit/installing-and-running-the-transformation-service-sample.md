# <a name="installing-and-running-the-transformation-service-sample"></a>安装和运行转换服务示例
转换 Web 服务直接公开部署在 Microsoft BizTalk 通过 Web 服务，从而消除到 BizTalk Messagebox 数据库的行程的消息的映射。 转换 Web 服务允许外部应用程序提交到 Microsoft.Practices.ESB 应用程序文档，并将其转换使用部署在 BizTalk 映射。
