# <a name="faults-by-application-or-service"></a>应用程序或服务故障
此选项显示在指定时间段为一组指定的应用程序生成的所有错误的聚合视图。 选择特定的服务将显示一个报表，列出该特定服务的错误。 您可以为 Microsoft Excel 电子表格导出错误的列表。  
  
### <a name="to-view-charts-and-lists-of-faults-by-application-or-service"></a>若要查看应用程序或服务的图表和列表的错误  
  
1.  打开[门户报告页](../esb-toolkit/portal-reports-page.md)。  
  
2.  单击**选择应用程序**链接上方的第一个图表，然后选择或清除复选框将显示已安装 Microsoft BizTalk Server 应用程序列表中。 此外可以使用显示来选择所有或无应用程序的链接。 然后单击**保存**若要显示的信息对于所选应用程序，或单击**取消**放弃所做的更改。  
  
3.  若要更改的期限为其图表显示的信息，请选择**小时、 天、 周、 月、 季度、 年**或**所有**的第一个图表上方的下拉列表中。  
  
4.  选择一个应用程序;此图表更改为每个服务该应用程序中显示错误。  
  
5.  单击**导出到 Excel**以下载可以在 Microsoft Excel 中查看的格式错误的列表。
