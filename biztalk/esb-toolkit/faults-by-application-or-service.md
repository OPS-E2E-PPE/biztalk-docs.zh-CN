# <a name="faults-by-application-or-service"></a>由应用程序或服务的错误
此选项显示指定时间段内为一组指定的应用程序生成的所有错误的聚合视图。 选择特定的服务将显示一个报表，列出的该特定服务的错误。 可以将错误的列表导出为 Microsoft Excel 电子表格。  
  
### <a name="to-view-charts-and-lists-of-faults-by-application-or-service"></a>若要查看图表和列表的错误由应用程序或服务  
  
1.  打开[门户报表页](../esb-toolkit/portal-reports-page.md)。  
  
2.  单击**选择应用程序**链接上方的第一个图表，然后选择或清除复选框中显示的已安装 Microsoft BizTalk Server 应用程序的列表。 你也可以使用显示选择所有或应用程序的任何链接。 然后单击**保存**以显示所选应用程序的信息或单击**取消**放弃所做的更改。  
  
3.  若要更改时间的图表显示的信息，请选择**小时、 天、 周、 月、 季度、 每年，** 或**所有**第二个图表上方的下拉列表中。  
  
4.  选择一个应用程序;要为每个服务该应用程序中显示错误图表会发生更改。  
  
5.  单击**导出到 Excel**下载可以在 Microsoft Excel 中查看的格式错误的列表。
