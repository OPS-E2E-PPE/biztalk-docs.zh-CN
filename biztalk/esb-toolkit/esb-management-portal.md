# <a name="esb-management-portal"></a>ESB 管理门户
[!INCLUDE[esbToolkit](../includes/esbtoolkit-md.md)]的全面的基于 Web 的异常和错误消息管理框架，促进一致的系统和业务异常的处理方式。 包含的门户作为示例的门户应用程序可以如何提供度量值和可见性 ESB 活动。  
  
 ESB 管理门户的审核和跟踪机制通过提供异常中介功能。 它还会捕获正在处理"已在异常发生时的消息 （包括上下文属性）。 这样，功能强大的异常恢复和"修复并重新提交"模式。  
  
 若要允许管理员和用户可以轻松地管理 ESB 应用程序，此门户还提供可生成基于自定义条件的警报并向订户发送通知的警报和通知机制。  
  
 若要监视应用程序性能并分析故障趋势，在门户可以生成和显示的图表和报表，通过指定时间段内，可配置的范围并将包含从指定的 BizTalk 应用程序的数据。  
  
 另外，在门户集成，以使用 Microsoft UDDI 3.0 服务器 BizTalk 终结点发布到 UDDI，提供图形用户界面，可自动或由管理员控制审批过程。  
  
 ESB 管理门户的详细信息，请参阅[的异常管理框架组件](../esb-toolkit/the-components-of-the-exception-management-framework.md)。
