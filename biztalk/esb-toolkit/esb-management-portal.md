# <a name="esb-management-portal"></a>ESB 管理门户
[!INCLUDE[esbToolkit](../includes/esbtoolkit-md.md)]包括一个全面的基于 Web 的异常和错误消息管理框架，以促进一致的系统和业务异常的处理。 包含的门户用作了度量值和可见性 ESB 活动如何提供门户应用程序的示例。  
  
 通过审核和跟踪一种机制，ESB 管理门户提供异常中介功能。 它还会捕获"处于飞行状态"已在异常发生时的消息 （包括上下文属性）。 这使强大异常恢复和"修复并重新提交"模式。  
  
 若要允许管理员和用户可以轻松地管理 ESB 应用程序，该门户还提供一种警报和通知的机制，可生成基于自定义条件的警报并向订户发送通知。  
  
 若要监视应用程序性能并分析错误趋势，门户可以生成和显示的图表和报表，通过指定时间段内，可配置的范围以及包含从指定的 BizTalk 应用程序的数据。  
  
 此外，门户与 Microsoft UDDI 3.0 服务器发布 BizTalk 终结点向 UDDI，提供一个图形用户界面，可以自动或由管理员控制审批过程集成。  
  
 有关 ESB 管理门户的详细信息，请参阅[异常 Management Framework 的组件](../esb-toolkit/the-components-of-the-exception-management-framework.md)。
