# <a name="single-sign-on-event-10827"></a>单一登录： 事件 10827
## <a name="details"></a>详细信息  
  
|                 |                                                                                                      |
|-----------------|------------------------------------------------------------------------------------------------------|
|  产品名称   |                                      企业单一登录                                       |
| 产品版本 |                      [!INCLUDE[btsSSOVersion](../includes/btsssoversion-md.md)]                      |
|    事件 ID     |                                                10827                                                 |
|  事件源   |                                                ENTSSO                                                |
|    组件    |                                                 N/A                                                  |
|  符号名称  |                             ENTSSO_E_PSADMIN_INVALID_GROUP_ADAPTER_TYPE                              |
|  消息正文   | 组适配器的应用程序类型无效。 有效的应用程序类型是“密码同步组适配器”。 |
  
## <a name="explanation"></a>解释  
 组适配器的应用程序类型无效。 有效的应用程序类型是“密码同步组适配器”。  
  
## <a name="user-action"></a>用户操作  
 使用一个有效的应用程序类型。
