# <a name="error---xsl-transformation-failure"></a>错误-XSL 转换失败
**错误代码**  
  
 btm1050  
  
 **说明**  
  
 在转换过程中发生所指示的 XSLT 错误。  
  
 **用户执行任何操作**  
  
 根据所指示的 XSLT 错误，修改映射和/或源架构和目标架构以清除错误源。
