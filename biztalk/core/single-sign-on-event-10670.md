|                 |                                                                                                                                      |
|-----------------|--------------------------------------------------------------------------------------------------------------------------------------|
|  产品名称   |                                                      企业单一登录                                                       |
| 产品版本 |                                      [!INCLUDE[btsSSOVersion](../includes/btsssoversion-md.md)]                                      |
|    事件 ID     |                                                                10670                                                                 |
|  事件源   |                                                                ENTSSO                                                                |
|    组件    |                                                                 N\A                                                                  |
|  符号名称  |                                                  SSO_INFO_CHANGED_WINDOWS_PASSWORD                                                   |
