# <a name="single-sign-on-event-10670"></a>单一登录：事件 10670
## <a name="details"></a>详细信息  

|                 |                                                                                                                                      |
|-----------------|--------------------------------------------------------------------------------------------------------------------------------------|
|  产品名称   |                                                      企业单一登录                                                       |
| 产品版本 |                                      [!INCLUDE[btsSSOVersion](../includes/btsssoversion-md.md)]                                      |
|    事件 ID     |                                                                10670                                                                 |
|  事件源   |                                                                ENTSSO                                                                |
|    组件    |                                                                 N\A                                                                  |
|  符号名称  |                                                  SSO_INFO_CHANGED_WINDOWS_PASSWORD                                                   |
|  消息正文   | 已成功更改 Windows password.%r<br /><br /> 跟踪 ID: %1 %r<br /><br /> 适配器: %2 %r<br /><br /> Windows 帐户： %3 |

## <a name="explanation"></a>解释  
 此信息事件表明 SSO 已成功更改 Windows 密码。

- 不不需要任何用户操作。
