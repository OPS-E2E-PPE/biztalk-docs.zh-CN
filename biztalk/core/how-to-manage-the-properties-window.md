# <a name="how-to-manage-the-properties-window"></a>如何管理“属性”窗口
BizTalk 编辑器使用 Microsoft [!INCLUDE[btsVStudioNoVersion](../includes/btsvstudionoversion-md.md)] 的“属性”窗口来显示与架构树视图中的节点关联的属性的值，并提供用于更改这些值的界面。 本主题为管理“属性”窗口的特定特性提供了分步说明。  
  
### <a name="to-turn-property-descriptions-on-and-off"></a>打开和关闭属性说明  
  
-   在属性窗口中，右键单击任何属性名称，，然后单击**说明**。  
  
    > [!NOTE]
    >  使用此过程可以打开或关闭属性的显示。  
  
### <a name="to-arrange-properties-alphabetically-within-their-categories"></a>在各类别中按字母顺序排列属性  
  
-   单击**按分类顺序**属性窗口中的属性列表上方的工具栏按钮。  
  
### <a name="to-arrange-properties-alphabetically-without-regard-to-their-categories"></a>不考虑类别按字母顺序排列属性  
  
-   单击**按字母顺序**属性窗口中的属性列表上方的工具栏按钮。
