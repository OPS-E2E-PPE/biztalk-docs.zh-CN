# <a name="host-instances"></a>主机实例
A*主机实例*是 BizTalk Server 中的主机的物理安装位置。 管理员使用 Windows 管理规范 (WMI) 或 BizTalk 管理控制台来安装主机实例。 BizTalk Server 可以支持多个主机实例。  
  
 下图显示了服务器、主机和主机实例之间的关系：

 有关主机的详细信息，请参阅[主机](../core/hosts.md)。
