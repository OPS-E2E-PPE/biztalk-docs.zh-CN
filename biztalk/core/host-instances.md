# <a name="host-instances"></a>主机实例
一个*主机实例*是在 BizTalk Server 中的主机物理安装。 管理员可使用 Windows Management Instrumentation (WMI) 或 BizTalk 管理控制台来安装主机实例。 BizTalk Server 可以支持多个主机实例。  
  
 下图显示了服务器、 主机和主机实例之间的关系。  
  
 ![主机、 主机实例和服务器之间的关系](../core/media/ebiz-ops-adm01.gif "ebiz_ops_adm01")

 有关主机的详细信息，请参阅[主机](../core/hosts.md)。
