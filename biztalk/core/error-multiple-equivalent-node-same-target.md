# <a name="error---multiple-equivalent-node-same-target"></a>错误-多个等效节点相同的目标
**错误代码**  
  
 btm1025  
  
 **说明**  
  
 源架构中的指定的节点相冲突的子节点的**等效**组节点，节点都链接到目标架构中指定的节点。 在源架构中只能有一个节点可以出现在给定的实例消息中。  
  
 **用户执行任何操作**  
  
 确保的节点的子确定只有一个**等效**组节点都连接到目标架构中给定的节点。
