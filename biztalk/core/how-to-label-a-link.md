# <a name="how-to-label-a-link"></a>如何标记链接
标签有助于记录 functoid 的用途或映射中的链接。 你可以使用**标签**属性可以命名链接。 当您的映射包含大型架构结构，并且标识到 functoid 的输入输出链接变得困难时，此功能很有用。  
  
> [!NOTE]
>  有关如何标签和注释 functoid 的信息，请参阅[如何标记和注释 Functoid](../core/how-to-label-and-comment-a-functoid.md)。  
  
 下图显示了一个链接标签。  
  
 ![标签粘贴链接](../core/media/new-labelling-link.gif "New_Labelling_link")  
  
## <a name="prerequisites"></a>先决条件  
 这些说明需要 BizTalk 映射器处于运行状态。  
  
### <a name="to-add-a-label-to-a-link"></a>若要向链接添加标签  
  
1.  选择要添加标签的链接。  
  
2.  在**属性**窗口中，提供新名称出现在**标签**字段。  
  
    > [!IMPORTANT]
    >  最大允许的字符数为 256。 如果指定超过 256 个字符的字符串，则接受前 256 个字符和其余部分将被丢弃。  
  
     ![链接标签属性](../core/media/new-to-label-link.gif "New_To_Label_Link")
