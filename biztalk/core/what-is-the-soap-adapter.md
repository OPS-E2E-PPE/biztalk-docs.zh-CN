# <a name="what-is-the-soap-adapter"></a>SOAP 适配器概述
Web Services 是指接口符合 Web Services 描述语言 (WSDL) 所述标准的程序。 您可以使用 Microsoft BizTalk Server 业务流程创建和使用 Web Services，以便通过易于管理的直观方式将独立的相关功能组合在一起。 有关 Web 服务和业务流程的详细信息，请参阅[使用 Web 服务](../core/using-web-services.md)。  
  
 BizTalk Web 服务发布向导是附带[!INCLUDE[btsBizTalkServerNoVersion](../includes/btsbiztalkservernoversion-md.md)]以加快开发过程中用于发布 Web 服务。 在创建要以 Web Services 方式公开的业务流程之后，BizTalk Web Services 发布向导将为您生成一个 Web Services 项目。 有关 BizTalk Web 服务发布向导的详细信息，请参阅[发布作为 Web 服务业务流程](../core/publishing-an-orchestration-as-a-web-service.md)。 请注意，在运行 BizTalk Web Services 发布向导之前，必须启用 BizTalk Web Services。 有关启用 BizTalk Web 服务的详细信息，请参阅[启用 Web 服务](../core/enabling-web-services.md)。  
  
 SOAP 适配器由两个适配器组成：一个发送适配器和一个接收适配器。
