# <a name="what-is-the-soap-adapter"></a>SOAP 适配器是什么？
Web 服务是具有遵守规定 Web 服务描述语言 (WSDL) 中的标准的界面的程序。 Microsoft BizTalk Server 业务流程可用于创建和使用 Web 服务进行独立但相关的业务功能结合起来以易于管理且直观的方式。 有关 Web 服务和业务流程的详细信息，请参阅[使用 Web 服务的](../core/using-web-services.md)。  
  
 BizTalk Web Services 发布向导随[!INCLUDE[btsBizTalkServerNoVersion](../includes/btsbiztalkservernoversion-md.md)]以加快发布 Web 服务的开发过程。 创建你想要将作为 Web 服务公开的业务流程后，BizTalk Web Services 发布向导为您生成一个 Web 服务项目。 有关 BizTalk Web Services 发布向导的详细信息，请参阅[业务流程发布为 Web 服务](../core/publishing-an-orchestration-as-a-web-service.md)。 请注意在运行 BizTalk Web Services 发布向导之前, 必须启用 BizTalk Web services。 有关启用 BizTalk Web services 的详细信息，请参阅[启用 Web 服务](../core/enabling-web-services.md)。  
  
 SOAP 适配器由两个适配器组成： 一个发送适配器和接收适配器。
