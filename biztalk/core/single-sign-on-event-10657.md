# <a name="single-sign-on-event-10657"></a>单一登录： 事件 10657
## <a name="details"></a>详细信息  

|                 |                                                            |
|-----------------|------------------------------------------------------------|
|  产品名称   |                 企业单一登录                  |
| 产品版本 | [!INCLUDE[btsSSOVersion](../includes/btsssoversion-md.md)] |
|    事件 ID     |                           10657                            |
|  事件源   |                           ENTSSO                           |
|    组件    |                            N\A                             |
|  符号名称  |          SSO_INFO_PASSWORD_SYNC_ADAPTERS_STARTED           |
|  消息正文   |     已针对外部适配器启动密码同步。%r     |

## <a name="explanation"></a>解释  
 此信息事件表示已针对外部适配器启动密码同步。

- 不需要用户进行任何操作。
