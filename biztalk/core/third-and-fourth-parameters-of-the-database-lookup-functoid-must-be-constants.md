# <a name="error---third-and-fourth-parameters-of-the-database-lookup-functoid-must-be-constants"></a>错误-的第三个和第四个参数数据库查找 Functoid 必须为常量。
**错误代码**  
  
 btm1078  
  
 **说明**  
  
 第三个和第四个参数**数据库查找**functoid 是表和列名称。 这些参数必须是常数值。  
  
 **用户执行任何操作**  
  
 编辑 functoid 输入的参数中**配置 Functoid 输入**对话框。 单击网格上的 functoid。 在属性窗口中，选择**输入参数**属性单击省略号 (**...**) 按钮。 **配置 Functoid 输入**此时将显示对话框。
