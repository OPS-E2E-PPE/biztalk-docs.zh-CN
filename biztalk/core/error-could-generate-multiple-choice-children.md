# <a name="error---could-generate-multiple-choice-children"></a>错误-无法生成多个选择的子节点
**错误代码**  
  
 btm1027  
  
 **说明**  
  
 指向目标架构不正确，导致多个节点对**选择**组节点以便生成。  
  
 **用户执行任何操作**  
  
 因此重新链接到目标架构，只有单个节点内的**选择**可以生成组节点。
