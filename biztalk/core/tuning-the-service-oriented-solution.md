# <a name="tuning-the-service-oriented-solution"></a>优化服务面向解决方案
业务流程通过业务活动监视 (BAM) API 来收集有关多个应用程序部分的性能信息。 这些数据对于优化该应用程序可能十分有用。 当然，您也可以使用内置性能计数器。  
  
 BAM 有关的详细信息，请参阅[使用业务活动监视](../core/using-business-activity-monitoring.md)。 有关如何解决方案使用 BAM 的详细信息，请参阅[监视服务面向解决方案与 BAM](../core/monitoring-the-service-oriented-solution-with-bam.md)。 有关 BizTalk 性能计数器的详细信息，请参阅[业务流程引擎性能计数器](../core/orchestration-engine-performance-counters.md)。  
  
 您也可以通过调整 BizTalk 主机的值来优化该解决方案。 有关详细信息，请参阅[如何修改主机属性](../core/how-to-modify-host-properties.md)。
