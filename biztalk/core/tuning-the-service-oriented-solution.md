# <a name="tuning-the-service-oriented-solution"></a>优化面向服务的解决方案
业务流程收集有关应用程序通过业务活动监视 (BAM) API 中的不同部分的性能信息。 您可能会发现数据优化应用程序中很有用。 此外，当然，使用内置的性能计数器。  
  
 有关 BAM 的详细信息，请参阅[使用业务活动监视](../core/using-business-activity-monitoring.md)。 有关该解决方案如何使用 BAM 的详细信息，请参阅[监视面向服务的解决方案使用 BAM](../core/monitoring-the-service-oriented-solution-with-bam.md)。 有关 BizTalk 性能计数器的详细信息，请参阅[业务流程引擎性能计数器](../core/orchestration-engine-performance-counters.md)。  
  
 此外可以通过调整 BizTalk 主机的值来优化该解决方案。 有关详细信息，请参阅[如何修改主机属性](../core/how-to-modify-host-properties.md)。
