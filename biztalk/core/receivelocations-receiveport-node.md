# <a name="receivelocations-receiveport-node"></a>ReceiveLocations （ReceivePort 节点）
绑定文件的 ReceivePort 节点的 ReceiveLocations 节点是所有 ReceiveLocation 节点包含有关与绑定文件一起导出的接收位置的特定信息的父节点。  
  
## <a name="nodes-in-the-receivelocations-node"></a>节点中的 ReceiveLocations 节点  
 下表列出了可为绑定文件的此节点设置的属性：  
  
|**名称**|**节点类型**|**数据类型**|**说明**|**限制**|**注释**|  
|--------------|-------------------|-------------------|---------------------|----------------------|------------------|  
|[ReceiveLocation](../core/receivelocation-receivelocations-node.md)|录制|接收位置 (ComplexType)|指定与绑定文件一起导出的接收位置有关的信息。|可选|默认值：无|
