# <a name="how-to-construct-a-web-message-with-no-message-parts"></a>如何构造没有消息部分的 Web 消息
没有消息部分的 Web 消息是不具有输入的参数的方法。  
  
### <a name="to-construct-a-web-message-that-does-not-have-message-parts"></a>若要构造没有消息部分的 Web 消息  
  
1.  与打开业务流程中，打开**工具箱**然后单击**BizTalk 业务流程**选项卡。  
  
2.  添加**构造消息**形状。  
  
3.  选择的消息**构造消息**形状。  
  
> [!NOTE]
>  不需要具有**消息赋值**形状内**构造消息**形状。
