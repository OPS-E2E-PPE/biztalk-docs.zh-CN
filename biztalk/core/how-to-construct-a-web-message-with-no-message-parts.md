# <a name="how-to-construct-a-web-message-with-no-message-parts"></a>如何构造没有消息部分的 Web 消息
Web 消息如果没有消息部分，则属于不带输入参数的方法。  
  
### <a name="to-construct-a-web-message-that-does-not-have-message-parts"></a>构造没有消息部分的 Web 消息  
  
1.  与打开的业务流程，打开**工具箱**单击**BizTalk 业务流程**选项卡。  
  
2.  添加**构造消息**形状。  
  
3.  选择的消息**构造消息**形状。  
  
> [!NOTE]
>  不要求你具有**消息分配**形状内**构造消息**形状。
