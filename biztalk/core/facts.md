# <a name="facts"></a>事实
事实是有关现实世界的离散的信息片段。 事实可以来源于多个源 （事件系统、 业务应用程序、 数据库表等中的对象），并且必须将其填充到业务规则引擎中[!INCLUDE[btsBizTalkServerNoVersion](../includes/btsbiztalkservernoversion-md.md)]通过使用以下元素之一：  
  
- .NET 对象（方法、属性和字段）  
  
- XML 文档（元素、属性和文档子节）  
  
- 数据库行集（表列中的值）  
  
  在业务规则编辑器中，您可以使用事实浏览器来浏览数据，并将数据从各个源引入规则。
