# <a name="using-the-mqseries-adapter"></a>使用 MQSeries 适配器

## <a name="overvew"></a>概述
MQSeries 适配器需要与 BizTalk Server 的集成，并将 MQSeries 服务器用作主要消息传送系统的组织提供服务。  
  
 网络管理员部署和配置适配器。 软件开发人员可以通过创建适配器组件来扩展功能。  
  
> [!NOTE]
>  对于多台计算机，可以使用 MQSeries 适配器与 BizTalk Server 配置。
