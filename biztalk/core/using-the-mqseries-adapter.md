# <a name="using-the-mqseries-adapter"></a>使用 MQSeries 适配器

## <a name="overvew"></a>概述
MQSeries 适配器服务组织需要与 BizTalk Server 的集成，并使用 MQSeries 服务器作为主消息传送系统。  
  
 网络管理员可部署和配置该适配器。 软件开发人员可通过创建适配器组件来扩展功能。  
  
> [!NOTE]
>  与 BizTalk Server 的适配器配置 MQSeries 可用于多台计算机。
