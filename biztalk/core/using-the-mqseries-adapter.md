# <a name="using-the-mqseries-adapter"></a>使用 MQSeries 适配器
MQSeries 适配器提供服务的组织需要与集成[!INCLUDE[btsBizTalkServerNoVersion](../includes/btsbiztalkservernoversion-md.md)]和使用 MQSeries 服务器作为主消息传送系统。  
  
 网络管理员可部署和配置该适配器。 软件开发人员可通过创建适配器组件来扩展功能。  
  
> [!NOTE]
>  你可以使用与该 MQSeries 适配器[!INCLUDE[btsBizTalkServerNoVersion](../includes/btsbiztalkservernoversion-md.md)]为多台计算机配置。
