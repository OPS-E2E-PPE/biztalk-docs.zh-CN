# <a name="about-the-biztalk-orchestration-engine"></a>关于 BizTalk 业务流程引擎
在运行时，BizTalk 业务流程引擎执行 BizTalk 业务流程设计器生成的 XLANG/s 文件。 业务流程设计器是用于以可视方式设计业务流程的丰富图形工具。 它可以生成 XLANG/s 文件，这些文件具有 .odx 扩展名，在其标头中包含其他直观信息，同时在其主体中包含自定义的属性信息。  
  
> [!NOTE]
>  作为消息传递使用的某些表达式功能的 C# 语言，可以查看 XLANG/s。  
  
 业务流程引擎的主要功能如下：  
  
-   持久化  
  
-   作为 .NET 组件的宿主  
  
-   中的  
  
-   大消息支持  
  
-   运行时验证  
  
-   加载阻止
