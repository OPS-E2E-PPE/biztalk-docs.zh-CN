# <a name="btadinstalldir"></a>BTAD_InstallDir
BTAD_InstallDir 包含 BizTalk 应用程序的安装路径。 操作创建或更新，并且 BTAD_HostClass 是 BizTalkHostInstance 时，此窗口才可用。  
  
## <a name="remarks"></a>备注  
 BTAD_InstallDir 设置为 Windows Installer 包的 TARGETDIR 属性。 默认值为 %ProgramFiles%\Generated by BizTalk\\<*应用程序名称*\>。
