# <a name="business-rules-biztalk-server-samples-folder"></a>业务规则 （BizTalk Server 示例文件夹中）
Microsoft [!INCLUDE[btsBizTalkServerNoVersion](../includes/btsbiztalkservernoversion-md.md)] 在其软件开发工具包 (SDK) 中包括了几个业务规则示例。 本部分详细介绍每个业务规则示例所演示的功能、生成和运行每个示例的说明以及可以预期得到的结果。  
  
## <a name="in-this-section"></a>本节内容  
  
-   [业务规则 Hello World1 （BizTalk Server 示例）](../core/business-rules-hello-world1-biztalk-server-sample.md)。 演示如何使用示例事实集创建、保存、加载和运行 BizTalk 规则集。  
  
-   [业务规则 Hello World2 （BizTalk Server 示例）](../core/business-rules-hello-world2-biztalk-server-sample.md)。 演示如何为版本，发布和部署设置为共享 SQL 规则存储，使用可扩展标记语言 (XML) 规则**策略**对象，并执行动态策略更新。  
  
-   [借处理使用业务规则 （BizTalk Server 示例）](../core/loans-processing-using-business-rules-biztalk-server-sample.md)。 演示如何在业务流程中使用 BizTalk 规则集和事实计算实例消息中的字段的值。  
  
-   [医疗声明处理和测试策略 （BizTalk Server 示例）](../core/medical-claims-processing-and-testing-policies-biztalk-server-sample.md)。 演示如何创建对派生自数据库表和入站文档的事实进行检查的多规则规则集。
