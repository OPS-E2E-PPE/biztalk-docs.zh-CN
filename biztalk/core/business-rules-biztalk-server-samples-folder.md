# <a name="business-rules-biztalk-server-samples-folder"></a>业务规则 （BizTalk Server 示例文件夹）
Microsoft[!INCLUDE[btsBizTalkServerNoVersion](../includes/btsbiztalkservernoversion-md.md)]在其软件开发工具包 (SDK) 包含几个业务规则示例。 本部分提供有关每个业务规则示例，说明用于构建和运行示例，以及可以预期的结果所演示的功能的详细的信息。  
  
## <a name="in-this-section"></a>本节内容  
  
-   [业务规则 Hello World1 （BizTalk Server 示例）](../core/business-rules-hello-world1-biztalk-server-sample.md)。 演示如何创建、 保存、 加载和运行 BizTalk 规则集使用示例事实集。  
  
-   [业务规则 Hello World2 （BizTalk Server 示例）](../core/business-rules-hello-world2-biztalk-server-sample.md)。 演示如何为版本，发布和部署可扩展标记语言 (XML) 规则设置为共享 SQL 规则存储，使用**策略**对象，并执行动态策略更新。  
  
-   [加载使用业务规则 （BizTalk Server 示例） 处理](../core/loans-processing-using-business-rules-biztalk-server-sample.md)。 演示如何使用 BizTalk 规则集和业务流程中的事实计算实例消息中的字段的值。  
  
-   [医疗索赔处理和测试保单 （BizTalk Server 示例）](../core/medical-claims-processing-and-testing-policies-biztalk-server-sample.md)。 演示如何创建派生自数据库表和入站的文档的事实进行检查的多规则规则集。
