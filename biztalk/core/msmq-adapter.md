# <a name="msmq-adapter"></a>MSMQ 适配器
MSMQ 适配器，可以使用从 Message Queuing 4.0 [!INCLUDE[btsBizTalkServerNoVersion](../includes/btsbiztalkservernoversion-md.md)]。 将消息队列技术与 BizTalk Server 相集成，在能够跨异类网络进行通信的不同时间运行的应用程序和可能临时脱机的系统。  
  
 MSMQ 适配器功能包括：  
  
-   改进的性能  
  
-   支持多线程操作  
  
-   支持 Message Queuing 4.0 功能，包括支持远程事务性读取和接收来自子队列 （适用于 Message Queuing 4.0）。  
  
-   支持处理大于 4 MB 的消息
