# <a name="how-to-delete-schemas"></a>如何删除架构
有时，您可能希望将架构从其 BizTalk 项目中排除，同时还要将该架构从硬盘中完全删除。 此操作称为删除架构，而不是排除架构。 本主题将介绍删除架构所需的步骤。  
  
### <a name="to-delete-a-schema"></a>删除架构  
  
1.  在解决方案资源管理器，右键单击你想要删除，然后单击架构**删除**。  
  
2.  在确认对话框中，单击 **“确定”**。  
  
> [!NOTE]
>  所删除的架构将从文件系统中永久删除，因此请谨慎进行操作并定期备份文件。
