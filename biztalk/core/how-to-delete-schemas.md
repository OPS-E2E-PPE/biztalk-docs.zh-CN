# <a name="how-to-delete-schemas"></a>如何删除架构
有时可能想要从其 BizTalk 项目中排除架构不仅架构从硬盘中完全删除。 此即为删除，而不排除架构。 本主题介绍删除架构所需的步骤。  
  
### <a name="to-delete-a-schema"></a>删除架构  
  
1.  在解决方案资源管理器，右键单击你想要删除，然后单击架构**删除**。  
  
2.  在确认对话框中，单击 **“确定”**。  
  
> [!NOTE]
>  从文件系统被永久删除已删除的架构，因此请谨慎进行操作并定期备份你的文件。
