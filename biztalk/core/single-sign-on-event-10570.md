# <a name="single-sign-on-event-10570"></a>单一登录： 事件 10570
## <a name="details"></a>详细信息  
  
|||  
|-|-|  
|产品名称|企业单一登录|  
|产品版本|[!INCLUDE[btsSSOVersion](../includes/btsssoversion-md.md)]|  
|事件 ID|10570|  
|事件源|ENTSSO|  
|组件|N/A|  
|符号名称|SSO_WARN_SSO_DISABLED_FOR_CHANGE_SSO_ADMIN|  
|消息正文|要更新 SSO Administrators 帐户名称，必须首先禁用 SSO.%r|  
  
## <a name="explanation"></a>解释  
 若要更新 SSO 管理员帐户名，必须首先禁用 SSO。  
  
## <a name="user-action"></a>用户操作  
 有关禁用 SSO 的信息，请参阅[如何禁用 SSO](../core/how-to-disable-sso.md)。
