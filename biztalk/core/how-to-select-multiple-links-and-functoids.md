# <a name="how-to-select-multiple-links-and-functoids"></a>如何选择多个链接和 Functoid
当你想要执行映射中 functoid 和/或链接的一组类似的操作时，可以全部在同一时间选择该组 functoid 和/或链接。 本主题提供有关如何执行此操作的信息。  
  
## <a name="prerequisites"></a>先决条件  
 这些说明需要 BizTalk 映射器处于运行状态。  
  
## <a name="why-do-you-need-to-bulk-select-linksfunctoids"></a>为什么需要批量选择链接 /functoid 到？  
 您可以批量选择链接和/或 functoid 可以执行以下任一操作：  
  
-   复制/剪切并粘贴在同一网格页或同一映射中的另一个网格页中所选内容。  
  
    > [!NOTE]
    >  有关如何复制、 剪切和粘贴链接和 functoid 的信息，请参阅[如何复制、 剪切和粘贴链接和 Functoid](../core/how-to-copy-cut-and-paste-links-and-functoids.md)。  
  
-   网格页之间移动所选内容  
  
    > [!NOTE]
    >  有关如何在一个网格页之间移动关系的信息，请参阅[如何移动网格页面之间的关系](../core/how-to-move-a-relationship-between-grid-pages.md)。  
  
-   编辑所选内容中 functoid 和链接的常见属性  
  
    > [!NOTE]
    >  有关如何设置注释和标签为 functoid 和链接的信息，请参阅[如何标记链接](../core/how-to-label-a-link.md)或[如何标签和注释 Functoid](../core/how-to-label-and-comment-a-functoid.md)。  
  
-   删除多个链接和/或在一个步骤中的 functoid  
  
    > [!NOTE]
    >  有关如何删除 functoid 的信息，请参阅[如何删除 Functoid](../core/how-to-delete-functoids.md)。  
  
## <a name="to-select-multiple-links-and-functoids"></a>若要选择多个链接和 functoid  
 您可以批量选择 functoid 和/或链接任何以下方法：  
  
-   单击该链接或 functoid。 按住 CTRL 键，然后单击其他链接和/或你想要选择的 functoid。  
  
     下图显示了所选的 functoid 和链接被突出显示。  
  
     ![批量选择 functoid 和链接](../core/media/bulkselect-functois-links.gif "BulkSelect_Functois 和链接")  
  
-   在图面上拖动鼠标。 选择矩形下的 functoid 将突出显示。  
  
     ![Functoid 的矩形选择](../core/media/bulkselect-selectionrectangle.gif "BulkSelect_SelectionRectangle")  
  
-   您可以使用拖动的组合和 ctrl 键并单击。 在映射图面上拖动鼠标和选择 functoid 和/或链接该范围内。 按住 CTRL 键，然后单击 functoid 和/或选择范围外的链接。  
  
## <a name="to-select-all-links-and-functoids-on-the-grid-page"></a>若要选择所有链接和 functoid 的网格页  
 可以在任何通过以下方式选择的所有 functoid 和链接在映射器网格页上：  
  
-   右键单击网格页上的任意位置，然后单击**全**。  
  
-   在网格页上的任意位置单击，然后单击在键盘中按 CTRL + A。  
  
-   单击网格页上的任意位置。 从 Visual Studio 菜单中，单击**编辑**，然后单击**全**。
