# <a name="managing-bam"></a>管理 BAM
使用业务活动监视 (BAM)，业务用户可以实时查看关键业务绩效指标，这使他们可以更有效地工作。 业务用户可以使用此信息提高业务操作的速度和效率。 BAM 以一种各种交易一致的实时方式收集业务活动信息，而不论各种不同类型的环境的复杂性如何。
