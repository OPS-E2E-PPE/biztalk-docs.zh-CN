# <a name="managing-bam"></a>管理 BAM
业务活动监视 (BAM)，为业务用户提供对关键业务绩效指标的实时访问权限。 业务用户可以使用此信息以提高速度和业务运营的有效性。 BAM 收集实时的事务一致方式无论复杂性如何，异类环境的业务活动信息。
