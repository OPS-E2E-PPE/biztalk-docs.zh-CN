# <a name="managing-servers"></a>管理服务器
BizTalk Server 管理控制台中的“服务器”节点列出了 BizTalk Server 组所属的所有服务器。 使用本部分中的过程，您可以向组中添加服务器或从中删除服务器，或者将服务器从一个组移动到另一个组。

-   [如何从组中删除服务器](../core/how-to-remove-a-server-from-a-group.md)
