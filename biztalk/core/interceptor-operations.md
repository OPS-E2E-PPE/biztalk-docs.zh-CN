# <a name="interceptor-operations"></a>侦听器运算
本部分包含有关常见的侦听器配置文件操作的信息。 这些操作可用于 Windows Workflow Foundation (WF) 和 Windows Communication Framework (WCF) 侦听器并且构建在通用侦听器框架之上的任何其他侦听器。 WF 和 WCF 侦听器将扩展这些运算符通过定义用于检索和操作侦听器数据的附加的特定于域的操作提供的常见功能。
