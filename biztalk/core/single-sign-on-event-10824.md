|-----------------|------------------------------------------------------------------------------------------|
|  产品名称   |                                企业单一登录                                 |

|    事件 ID     |                                          10824                                           |
|  事件源   |                                          ENTSSO                                          |
|    组件    |                                           N/A                                            |
