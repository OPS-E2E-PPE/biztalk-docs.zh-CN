# <a name="single-sign-on-event-10824"></a>单一登录：事件 10824
## <a name="details"></a>详细信息  
  
|                 |                                                                                          |
|-----------------|------------------------------------------------------------------------------------------|
|  产品名称   |                                企业单一登录                                 |
| 产品版本 |                [!INCLUDE[btsSSOVersion](../includes/btsssoversion-md.md)]                |
|    事件 ID     |                                          10824                                           |
|  事件源   |                                          ENTSSO                                          |
|    组件    |                                           不可用                                            |
|  符号名称  |                          ENTSSO_E_PSADMIN_INVALID_ADAPTER_TYPE                           |
|  消息正文   | 无效的应用程序适配器的类型。 有效的应用程序类型为密码同步适配器。 |
  
## <a name="explanation"></a>解释  
 无效的应用程序适配器的类型。 有效的应用程序类型为密码同步适配器。  
  
## <a name="user-action"></a>用户操作  
 使用一种有效的应用程序类型。
