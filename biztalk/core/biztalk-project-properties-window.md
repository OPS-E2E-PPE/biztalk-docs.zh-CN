# <a name="biztalk-project-properties-window"></a>BizTalk 项目属性窗口
BizTalk 项目中的项目属性窗口[!INCLUDE[btsVStudioNoVersion](../includes/btsvstudionoversion-md.md)]包含以下属性：  
  
## <a name="bpel-compliance"></a>符合 BPEL 规范  
 指定是否生成符合业务处理执行语言 (BPEL) 的输出。  
  
## <a name="project-file"></a>项目文件  
 包含有关项目的生成、配置和其他信息的文件的名称。  
  
## <a name="project-folder"></a>项目文件夹  
 项目文件的位置。
