# <a name="using-biztalk-mapper-commands"></a>使用 BizTalk 映射器命令
当 BizTalkMapper 变为活动状态时，它将添加名为菜单**BizTalk** microsoft [!INCLUDE[btsVStudioNoVersion](../includes/btsvstudionoversion-md.md)] shell。 此菜单可以访问 BizTalk 映射器命令和它们的功能。 BizTalk 映射器处于活动状态时**BizTalk**菜单将提供专用于编辑 BizTalk 映射的命令。  

 如果适用，BizTalk 映射器使用的现有[!INCLUDE[btsVStudioNoVersion](../includes/btsvstudionoversion-md.md)]菜单项用于明显等效于标准应用程序功能的命令。 例如，BizTalk 映射器处于活动状态时，**保存**命令**文件**菜单将更改保存到当前正在编辑的映射。  

 下表介绍可以在开发映射过程中使用的 BizTalk 映射器中的命令。  


|                                            Command<br /><br /> （菜单位置，如果有）                                             |                                                                                                                                                                                                                                                                                                                                                            Description                                                                                                                                                                                                                                                                                                                                                            |
|-------------------------------------------------------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
|                                     **打开映射**<br /><br /> (文件&#124;打开&#124;文件...)                                      |                                                                                                                                                                                                                                                                打开 BizTalk 映射在 BizTalk 映射器中进行编辑。<br /><br /> 在快捷菜单中选择映射时，解决方案资源管理器并通过双击解决方案资源管理器中的地图上也可用。                                                                                                                                                                                                                                                                 |
|                                            **关闭映射**<br /><br /> (文件&#124;关闭)                                            |                                                                                                                                                                                                                     对并提示您保存任何未保存的更改的当前映射关闭 BizTalk 映射器。<br /><br /> 也可通过使用标准关闭 ([**X**]) 中主编辑窗口右上角的按钮[!INCLUDE[btsVStudioNoVersion](../includes/btsvstudionoversion-md.md)]。                                                                                                                                                                                                                      |
|                             **新的映射**<br /><br /> (项目&#124;添加&#124;新建项...&#124;映射)                              |                                                                                                                                                                                                                              创建新的 BizTalk 映射以进行编辑，并打开 BizTalk 映射器。<br /><br /> 选择映射模板中的**添加新项**对话框。<br /><br /> 也可在解决方案资源管理器中 BizTalk 项目的快捷菜单 (添加&#124;新项&#124;映射)。                                                                                                                                                                                                                               |
|                          **打开源架构**<br /><br /> (*源仅在新映射的架构的树视图中*)                          |                                                                                                                                                                                                                                                                                                                  允许选择源架构中使用的新映射**BizTalk 类型选取器**对话框。                                                                                                                                                                                                                                                                                                                  |
|                     **打开目标架构**<br /><br /> (*目标的新的映射架构树视图中*)                     |                                                                                                                                                                                                                                                                                                               在使用新映射中允许的目标架构选择**BizTalk 类型选取器**对话框。                                                                                                                                                                                                                                                                                                                |
| **Save Map**<br /><br /> (文件&#124;保存 map.btm)<br /><br /> (文件&#124;保存 map.btm 另存为...)<br /><br /> (文件&#124;全部保存) |                                                                                                                                                                                                                                                                                            将保存其自己的名称，以新名称，或作为保存所有未保存的更改，分别的一部分当前编辑 BizTalk 映射。                                                                                                                                                                                                                                                                                             |
|                                         **创建链接**<br /><br /> *（拖放仅）*                                          |  创建源和目标架构树视图中的架构节点之间或网格页中的 functoid 之间的链接。<br /><br /> 可以创建的链接：<br /><br /> -将架构节点拖到另一个架构树中的另一个 schema 节点或 functoid。<br />拖动 functoid 到架构节点或移到另一个 functoid。<br /><br /> 可通过在任一方向; 中拖动来创建链接换而言之，你可以开始在源 （左端） 或链接的目标 （右端）。<br /><br /> 您可以通过将链接的一个终结点从一个节点或 functoid 拖动到另一个节点或 functoid 来替换链接。 有关详细信息，请参阅[如何创建链接](../core/how-to-create-links.md)。  |
|                                         **添加 Functoid**<br /><br /> *（拖放仅）*                                         |                                                                                                                                                                                                                            将 functoid 添加到显示的网格页。<br /><br /> 可以通过将提取 functoid 从选项卡中拖动到网格页中添加 functoid[!INCLUDE[btsVStudioNoVersion](../includes/btsvstudionoversion-md.md)]对应于该 functoid 的类别的工具箱。                                                                                                                                                                                                                            |
|            **删除链接**和**删除的 Functoid**<br /><br /> (编辑&#124;删除)<br /><br /> (BizTalk&#124;删除)             |                                                                                                                                                                                                                                                 删除当前选定的链接或 functoid，在显示的网格页中，并确认提示您进行。<br /><br /> 在适用时，此命令为还可在选定的链接或 functoid 的快捷菜单上。                                                                                                                                                                                                                                                  |
|                             **查找 Schema 节点**<br /><br /> (编辑&#124;查找和替换&#124;查找)                             |                                                                                                                                                                                                                                                                                                          用于在架构树视图中，对于大型架构很有用的节点名称提供搜索功能。                                                                                                                                                                                                                                                                                                           |
|                 **属性**<br /><br /> (视图&#124;属性窗口)<br /><br /> (BizTalk&#124;属性)                  |                                                                                                                                                                                              提供的链接、 functoid、 源和目标架构树视图节点的属性和更高级别对象，如映射本身的访问。<br /><br /> 也可为所选的链接或 functoid，而不是在架构树中节点的快捷菜单。 按 F4 还会显示属性窗口。                                                                                                                                                                                               |
|                                         **添加页面**<br /><br /> (BizTalk &#124; Add Page)                                          |                                                                                                                                                                                                                                                                                              将新的网格页，也称为层添加到网格视图。<br /><br /> 在网格页选项卡的快捷菜单上也可用。                                                                                                                                                                                                                                                                                               |
|                                      **删除页**<br /><br /> (BizTalk &#124; Delete Page)                                       |                                                                                                                                                                                                                                                                                   删除显示的网格页上，也称为层，并确认提示您进行。<br /><br /> 在网格页选项卡的快捷菜单上也可用。                                                                                                                                                                                                                                                                                    |
|                                    *重排页顺序*\*<br /><br /> (BizTalk&#124;重排页顺序)                                     |                                                                                                                                                                                                                                                                            允许的网格页，也称为层，使用重新排序**重排页顺序**对话框。<br /><br /> 在网格页选项卡的快捷菜单上也可用。                                                                                                                                                                                                                                                                             |
|                                      **重命名页**<br /><br /> (BizTalk &#124; Rename Page)                                       |                                                                                                                                                                                                                                                                                            允许重命名的显示的网格页上，也称为层。<br /><br /> 在网格页选项卡的快捷菜单上也可用。                                                                                                                                                                                                                                                                                             |
|                                 **展开树节点**<br /><br /> (BizTalk&#124;展开树节点)                                  | 完全展开当前选定 （并且至少部分折叠）**架构**，**组**，或**记录**源或目标架构树视图中的节点。<br /><br /> 在适用时，此命令为还可在所选节点的快捷菜单上。 **注意：** 在大型和复杂架构中，单击时**展开树节点**上包含复杂类型的节点，该架构中的某些节点可能仍会保持折叠状态。 这是因为递归过程扩展仅一个给定复杂类型的第一个匹配项。 您必须手动展开之后出现相同的类型。 此行为用于大型和复杂的架构中展开节点时优化性能。 |
|                               **折叠树节点**<br /><br /> (BizTalk&#124;折叠树节点)                                |                                                                                                                                                                                                                                折叠当前选定 （并且至少部分展开）**架构**，**组**，或**记录**源或目标架构树视图中的节点。<br /><br /> 在适用时，此命令为还可在所选节点的快捷菜单上。                                                                                                                                                                                                                                |
|                                     **网格预览**<br /><br /> (BizTalk&#124;网格预览)                                      |                                                                                                                                                                                                                                                                     允许快速而准确地内显示的网格页中使用滚动浏览**网格预览**对话框。<br /><br /> 此命令也是可在网格页的快捷菜单上。                                                                                                                                                                                                                                                                     |
|                                   **替换架构**<br /><br /> (BizTalk&#124;替换架构)                                    |                                                                                                                                                                                                                                                              允许的源或目标架构使用的替换**BizTalk 类型选取器**对话框。<br /><br /> 在源和目标架构树视图的快捷菜单上也可用。                                                                                                                                                                                                                                                               |
|                         **BizTalk 映射器选项**<br /><br /> (工具&#124;选项&#124;BizTalk 映射器)                         |                                                                                                                                                                                                                                                                    允许在网格视图中，包括所选内容、 链接、 网格和背景的各种颜色的配置。 此外可以使用此命令更改 schema 节点显示中使用的字体。                                                                                                                                                                                                                                                                     |

 下表介绍可在解决方案资源管理器中选定映射的快捷菜单上的其他映射命令。  


|      架构命令      |                                                                                                                                                                                                                                                             Description                                                                                                                                                                                                                                                             |
|--------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
|         **打开**         |                                                                                                                                                                                                            在 BizTalk 映射器中打开所选的映射。<br /><br /> 双击某个映射也会它在 BizTalk 映射器中。                                                                                                                                                                                                             |
|      **打开方式**       |                                                                                                                                                                                                                     允许在不同的 XML 编辑器，包括 BizTalk 映射器中打开所选的映射。                                                                                                                                                                                                                     |
|       **测试映射**       |                                                                                                                                                                                                                                                       测试选定的映射。                                                                                                                                                                                                                                                       |
|     **验证映射**     |                                                                                                                                                                                                                                                         验证映射。                                                                                                                                                                                                                                                          |
|      **调试映射**       |                                                                                                                                                                               如果成功编译映射时，调试映射将启动 XSLT 调试程序。 它允许您逐步 XLST 的生成，就像任何其他 Visual Studio 调试器一样。                                                                                                                                                                                |
| **从项目中排除** |                                                                                                                                                            从当前 BizTalk 项目中移除当前选定的映射。<br /><br /> 使用**添加&#124;现有项**命令以重新添加以前从当前 BizTalk 项目中排除的映射。                                                                                                                                                             |
|   **剪切、 复制、 粘贴**   | 使用这些命令来执行标准[!INCLUDE[btsVStudioNoVersion](../includes/btsvstudionoversion-md.md)]剪切、 复制和粘贴整个映射在 BizTalk 项目内的行为。<br /><br /> 此外可以剪切、 复制和粘贴映射项目，如链接和/或 functoid。 有关详细信息，请参阅[如何复制、 剪切和粘贴 Functoid](../core/how-to-copy-cut-and-paste-a-functoid.md)并[如何复制、 剪切和粘贴链接和 Functoid](../core/how-to-copy-cut-and-paste-links-and-functoids.md)。 |
|        **删除**        |                                                                                                                                                                                                                           永久删除当前选定的映射，并确认提示您进行。                                                                                                                                                                                                                            |
|        **重命名**        |                                                                                                                                                                                                                                     允许当前所选的映射就地重命名。                                                                                                                                                                                                                                      |
|      **属性**      |                                                                                                                                                                                                                           在属性窗口中显示当前选定的映射的属性。                                                                                                                                                                                                                           |
