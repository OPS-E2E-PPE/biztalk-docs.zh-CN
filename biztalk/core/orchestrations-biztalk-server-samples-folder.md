# <a name="orchestrations-biztalk-server-samples-folder"></a>业务流程 （BizTalk Server 示例文件夹）
Microsoft[!INCLUDE[btsBizTalkServerNoVersion](../includes/btsbiztalkservernoversion-md.md)]在其软件开发工具包 (SDK) 包含几个业务流程示例。 本部分提供有关这些业务流程示例，说明用于构建和运行示例，以及可以预期的结果所演示的功能的详细的信息。 下面列出的示例按其复杂程度进行排序，即 HelloWorld 是列表中最基本的示例。 在列表后面的示例可能基于列表中前面的示例所示的功能。  
  
## <a name="in-this-section"></a>本节内容  
  
-   [HelloWorld （BizTalk Server 示例）](../core/helloworld-biztalk-server-sample.md)。 演示如何使用 BizTalk 业务流程 XML 消息处理成相关但不同类型的 XML 消息和映射。  
  
-   [MethodCall （BizTalk Server 示例）](../core/methodcall-biztalk-server-sample.md)。 演示如何调用。BizTalk 业务流程中的基于网络的方法。  
  
-   [CallOrchestration （BizTalk Server 示例）](../core/callorchestration-biztalk-server-sample.md)。 演示如何从另一个业务流程调用 BizTalk 业务流程。  
  
-   [补偿 （BizTalk Server 示例）](../core/compensation-biztalk-server-sample.md)。 演示如何使用**补偿**形状在业务流程中的。  
  
-   [PartyResolution （BizTalk Server 示例）](../core/partyresolution-biztalk-server-sample.md)。 演示如何使用 BizTalk 业务流程与参与方解析路由实例消息。  
  
-   [BPEL 导入 （BizTalk Server 示例）](../core/bpel-import-biztalk-server-sample.md)。 演示如何从进程和相关的项目的业务流程执行语言 (BPEL) 说明创建业务流程。
