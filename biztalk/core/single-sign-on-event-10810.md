# <a name="single-sign-on-event-10810"></a>单一登录： 事件 10810
## <a name="details"></a>详细信息  
  
|||  
|-|-|  
|产品名称|企业单一登录|  
|产品版本|[!INCLUDE[btsSSOVersion](../includes/btsssoversion-md.md)]|  
|事件 ID|10810|  
|事件源|ENTSSO|  
|组件|N/A|  
|符号名称|ENTSSO_E_HISSO_INVALID_CREDENTIALS|  
|消息正文|指定的凭据无效或与 SSO 数据库中的凭据不匹配。|  
  
## <a name="explanation"></a>解释  
 指定的凭据无效或与 SSO 数据库中的凭据不匹配。  
  
## <a name="user-action"></a>用户操作  
 将 SSO 数据库中的信息设置为与外部系统的信息匹配。
