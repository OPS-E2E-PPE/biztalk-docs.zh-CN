# <a name="single-sign-on-event-10578"></a>单一登录： 事件 10578
## <a name="details"></a>详细信息  
  
|||  
|-|-|  
|产品名称|企业单一登录|  
|产品版本|[!INCLUDE[btsSSOVersion](../includes/btsssoversion-md.md)]|  
|事件 ID|10578|  
|事件源|ENTSSO|  
|组件|N/A|  
|符号名称|SSO_INFO_CHANGED_APP_USER_GROUP|  
|消息正文|已更新应用程序用户帐户。%r<br /><br /> 新的 Application Users: %1 %r<br /><br /> 旧的 Application Users: %2 %r<br /><br /> 跟踪 ID: %3 %r<br /><br /> 客户端用户: %4 %r<br /><br /> 应用程序名称： %5|  
  
## <a name="explanation"></a>解释  
 这是信息性消息，可以用于跟踪 SSO 系统中发生的与安全有关的重要事件。 此消息表明已更新应用程序用户帐户已经被更新。  
  
## <a name="user-action"></a>用户操作  
 不需要任何用户操作。
