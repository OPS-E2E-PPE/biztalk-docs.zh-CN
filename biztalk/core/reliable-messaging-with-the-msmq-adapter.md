# <a name="reliable-messaging-with-the-msmq-adapter"></a>使用 MSMQ 适配器可靠消息传递
通过使用特定的配置设置以及使用事务，可以提高使用 MSMQ 适配器发送和接收消息的可靠性。  
  
> [!NOTE]
>  如果不使用事务，则在计算机发生故障时将丢失群集配置中的消息。
