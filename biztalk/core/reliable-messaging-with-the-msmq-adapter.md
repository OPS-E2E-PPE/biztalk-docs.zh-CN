# <a name="reliable-messaging-with-the-msmq-adapter"></a>使用 MSMQ 适配器可靠消息传送
您可以提高可靠性的发送和接收消息使用 MSMQ 适配器，通过使用特定的配置设置并通过使用事务。  
  
> [!NOTE]
>  如果不使用事务，如果计算机出现故障可能会丢失群集配置中的消息。
