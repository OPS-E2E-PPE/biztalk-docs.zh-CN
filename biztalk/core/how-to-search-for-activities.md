# <a name="how-to-search-for-activities"></a>如何搜索活动
使用 BAM Web 门户上的业务活动搜索来搜索跟踪数据，也称为 BAM 数据库中存储的实例。  
  
 针对当前正在执行的进程，以及存储的数据的这两个实时数据的搜索查询。  
  
### <a name="to-search-the-bam-database"></a>若要在 BAM 数据库中搜索  
  
1.  在 Internet Explorer 中，在**地址**框中，键入 BAM Web 门户的 URL。  
  
2.  从**我的视图**导航窗格中，单击某个视图展开视图以显示任务列表。  
  
3.  单击**活动搜索**任务以显示活动的列表...  
  
4.  单击用于加载活动搜索页内容框架中的活动。  
  
5.  在中**查询**生成器中，选择搜索的查询参数。  
  
6.  在中**列选择器**选择表示要在搜索中返回的数据项目的列。  
  
7.  单击**执行查询**在顶部的按钮的内容帧。  
  
     结果显示在**结果**活动搜索内容页区域。  
  
> [!NOTE]
>  可以选择最多 20 个列。  
  
> [!NOTE]
>  此版本不支持逻辑运算符的分组。
