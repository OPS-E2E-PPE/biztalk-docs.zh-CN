# <a name="how-to-search-for-activities"></a>如何搜索活动
使用 BAM Web 门户中的“业务活动搜索”，可以搜索存储在 BAM 数据库中的跟踪数据，这些数据又称为实例。  
  
 搜索既针对当前执行的流程的实时数据进行查询，也可以针对存储的数据进行查询。  
  
### <a name="to-search-the-bam-database"></a>搜索 BAM 数据库  
  
1.  在 Internet Explorer 中，在**地址**框中，键入 BAM Web 门户的 URL。  
  
2.  从**MyViews**导航窗格中，单击一个视图，以展开的视图以显示任务列表。  
  
3.  单击**活动搜索**任务以显示活动的列表...  
  
4.  单击用于加载活动搜索页内容的帧中的活动。  
  
5.  在**查询**生成器选择搜索查询参数。  
  
6.  在**列选择器**选择表示要在搜索中返回的数据项的列。  
  
7.  单击**执行查询**顶部的按钮右侧的内容的帧。  
  
     结果显示在**结果**活动搜索内容页区域。  
  
> [!NOTE]
>  您可以选择最多 20 个列。  
  
> [!NOTE]
>  此版本不支持对逻辑运算符进行分组。
