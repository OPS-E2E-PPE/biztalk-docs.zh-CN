# <a name="use-biztalk-editor-commands"></a>使用 BizTalk 编辑器命令

## <a name="overview"></a>概述
当 BizTalk 编辑器变为活动状态时，它将添加名为菜单**BizTalk**到[!INCLUDE[btsVStudioNoVersion](../includes/btsvstudionoversion-md.md)]shell。 此菜单可以访问 BizTalk 编辑器命令和它们的功能。 BizTalk 编辑器处于活动状态时**BizTalk**菜单将提供专用于编辑 BizTalk 架构的命令。  

 此外，如果适用，BizTalk 编辑器中使用的现有[!INCLUDE[btsVStudioNoVersion](../includes/btsvstudionoversion-md.md)]菜单项用于明显等效于标准应用程序功能的命令。 例如，BizTalk 编辑器处于活动状态时，**保存**命令**文件**菜单将更改保存到当前正在编辑的架构。
