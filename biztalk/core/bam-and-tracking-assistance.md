# <a name="bam-and-tracking-assistance"></a>BAM 和跟踪协助
你可以通过单击来请求协助通过 BAM 门户**协助**活动状态页上的按钮。  
  
 协助请求被置于应用程序事件日志。 请求的源是 BAM Web 服务和事件 ID 为 1000年。  
  
 消息正文位于“说明”字段中，它包含请求的主题、问题说明文本、BizTalk 服务实例 ID、消息 ID、BizTalk 管理数据库服务器以及 BizTalk 管理数据库名称。
