# <a name="bam-and-tracking-assistance"></a>BAM 和跟踪协助
可通过单击来请求通过 BAM 门户的协助**协助**活动状态页上的按钮。  
  
 协助请求位于应用程序事件日志中。 请求的源是 BAM Web 服务和事件 ID 为 1000年。  
  
 消息正文是在说明字段中，包含请求的主题、 问题说明、 BizTalk 服务实例 ID、 消息 ID、 BizTalk 管理数据库服务器和 BizTalk 管理数据库名称的文本。
