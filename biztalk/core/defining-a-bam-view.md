# <a name="defining-a-bam-view"></a>定义 BAM 视图
有时，您不希望向所有业务用户公开您的所有业务数据。 您可以创建不同的 BAM 视图，用于公开不同类型的信息。  
  
> [!IMPORTANT]
>  所有别名、持续时间、组、维度和度量的名称必须是唯一的，因为它们将是同一视图的组成部分。  
  
> [!NOTE]
>  要定义 BAM 视图，您必须至少已创建了一个 BAM 活动。 有关创建活动的详细信息，请参阅[如何定义业务活动](../core/how-to-define-a-business-activity.md)。  
  
> [!NOTE]
>  在 Excel 工作簿中，BAM 视图显示为工作表。 您可以为一个 BAM 活动创建多个视图。 使用 Excel 应用程序窗口底部的工作表选项卡，可以选择一个 BAM 视图进行编辑。
