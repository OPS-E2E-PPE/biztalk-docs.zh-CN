# <a name="defining-a-bam-view"></a>定义 BAM 视图
有时你不想要公开的业务数据对所有业务用户所有。 可以创建不同的 BAM 视图，用于公开不同类型的信息。  
  
> [!IMPORTANT]
>  所有别名、 持续时间、 组、 维度和度量值名称必须是视图的唯一的因为它们将是视图的相同的一部分。  
  
> [!NOTE]
>  若要定义 BAM 视图，您必须已创建至少一个 BAM 活动。 有关创建活动的详细信息，请参阅[如何定义业务活动](../core/how-to-define-a-business-activity.md)。  
  
> [!NOTE]
>  在 Excel 工作簿中显示为工作表的 BAM 视图。 可以创建 BAM 活动的多个视图。 在 Excel 应用程序窗口的底部使用工作表选项卡选择一个 BAM 视图进行编辑。
