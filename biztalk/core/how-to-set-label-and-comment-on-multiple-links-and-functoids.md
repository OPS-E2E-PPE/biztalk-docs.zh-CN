# <a name="how-to-set-label-and-comment-on-multiple-links-and-functoids"></a>如何在多个链接和 Functoid 上设置标签和注释
您可以设置通用标签和/或多个 functoid 和/或链接的注释。 本主题提供有关如何执行此操作的详细信息。  
  
 有关如何选择多个 functoid 和/或链接的信息，请参阅[如何选择多个链接和 Functoid](../core/how-to-select-multiple-links-and-functoids.md)。  
  
 选择多个 functoid 和/或链接之后, 可以执行以下任一操作：  
  
-   设置通用标签为所选链接。  
  
-   一个常见的标签和/或注释为所选 functoid 设置。  
  
-   设置通用标签为所选的 functoid 和链接。  
  
## <a name="prerequisites"></a>先决条件  
 这些说明需要 BizTalk 映射器处于运行状态。  
  
## <a name="to-set-a-common-label-for-selected-links"></a>若要设置通用标签为所选链接  
 在多个链接设置通用标签是一个原子操作。 您可以撤消或重做此操作。  
  
 若要设置通用标签为所选链接：  
  
1.  在映射器网格页上，选择所需的链接。  
  
2.  在中**属性**窗口中，输入在合适的标签**标签**属性。  
  
## <a name="to-set-a-common-label-andor-comment-for-selected-functoids"></a>若要设置通用标签和/或所选 functoid 的注释  
 设置通用标签和设置为所选 functoid 通用注释是两个不同的原子操作。 您可以撤消或重做这些操作。  
  
 若要设置通用标签/注释为所选 functoid:  
  
1.  在映射器网格页上，选择所需的 functoid。  
  
2.  在中**属性**窗口中，输入合适的标签和/或注释**标签**并**注释**属性。  
  
## <a name="to-set-a-common-label-for-selected-functoids-and-links"></a>若要设置通用标签为所选的 functoid 和链接  
 可以设置通用标签为所选的 functoid 和链接。 **标签**是普遍适用于 functoid 和链接的唯一属性。 您可以撤消或重做此操作。  
  
 若要设置通用标签为所选的 functoid 和链接：  
  
1.  在映射器网格页上，选择所需的 functoid 和链接。  
  
2.  在中**属性**窗口中，输入在合适的标签**标签**属性。
