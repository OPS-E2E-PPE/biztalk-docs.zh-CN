# <a name="replaying-actions"></a>重播操作
业务流程调试器，您可以重播的业务流程步骤的操作。 当您打开业务流程时，光标放置在第一个操作，例如，初始化。 使用向上和向下箭头向前和向后通过业务流程的步骤。 跟踪进度，如呈现的业务流程焦点移动的形状。
