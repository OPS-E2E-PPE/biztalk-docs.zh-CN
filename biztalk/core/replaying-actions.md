# <a name="replaying-actions"></a>重播操作
使用业务流程调试器，您可以逐步重播业务流程操作。 在打开业务流程时，光标将放在第一个操作（例如，初始化）上。 使用向上键和向下键可以在业务流程中逐步向前和向后移动。 随着呈现的业务流程焦点在形状间移动，对进度进行跟踪。
