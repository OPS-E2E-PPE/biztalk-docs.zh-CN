# <a name="single-sign-on-event-10600"></a>单一登录： 事件 10600
## <a name="details"></a>详细信息  
  
|||  
|-|-|  
|产品名称|企业单一登录|  
|产品版本|[!INCLUDE[btsSSOVersion](../includes/btsssoversion-md.md)]|  
|事件 ID|10600|  
|事件源|ENTSSO|  
|组件|N/A|  
|符号名称|SSO_ERROR_CS_CALLBACK_ACCESS_DENIED|  
|消息正文|配置存储服务器访问被拒绝。%r|  
  
## <a name="explanation"></a>解释  
 消息发送到配置存储服务器但未被接受。 这可能是由许多不同原因引起的，例如协议不正确或没有足够的安全权限。  
  
## <a name="user-action"></a>用户操作  
 请注意此消息中的信息和事件日志中的任何相关信息并与 Microsoft 产品支持服务联系。
