# <a name="viewing-live-bam-data"></a>查看实时 BAM 数据
下面是查看实时 BAM 数据的不同方法。  
  
-   当活动实例实时形成或从历史（归档）记录形成后，查看单个活动实例（例如，采购订单或贷款流程）。 此视图仅显示业务级别的相关数据，掩盖了各种不同实现的复杂性。 有关查看的单个活动实例的详细信息，请参阅[如何查看单个活动](../core/how-to-view-an-individual-activity.md)。  
  
-   导航到相关活动实例，例如，与给定采购订单相关联的发货活动，或者包括该采购订单金额的发票。 有关导航到相关的活动实例的详细信息，请参阅[如何查看相关的活动和相关文档](../core/how-to-view-related-activities-and-related-documents.md)。  
  
-   浏览关于当前正在处理的或已发生的所有业务活动的聚合（关键性能指标）。 聚合可以实时完成，也可以基于特定时间制作的活动快照来生成。 关于浏览聚合的详细信息，请参阅[如何视图 BAM 聚合](../core/how-to-view-bam-aggregations.md)。  
  
-   使用 BAM 中的一个新功能：进度维度。 使用这些维度可以基于给定的完成阶段对活动进行筛选或分组。 关于进度的维度的详细信息，请参阅[进度维度](../core/progress-dimension.md)。  
  
-   基于活动实例的进度或业务数据搜索活动实例。 例如，您可以搜索等待客户签名并且金额大于给定美元数的贷款。 有关在 BAM 中搜索活动的详细信息，请参阅[如何搜索活动](../core/how-to-search-for-activities.md)。
