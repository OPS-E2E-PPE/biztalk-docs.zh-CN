# <a name="viewing-live-bam-data"></a>查看实时 BAM 数据
以下是可以查看实时 BAM 数据的不同方法。  
  
-   随着在实时或从历史 （归档） 记录，请查看单个活动实例 （例如，采购订单或贷款流程）。 此视图显示仅将数据在业务级别相关，隐藏了各种不同实现的复杂性。 有关查看单个活动实例的详细信息，请参阅[如何查看单个活动](../core/how-to-view-an-individual-activity.md)。  
  
-   导航到相关的活动实例，例如，给定采购订单或在其中将包括发票发货与相关联。 有关导航到相关的活动实例的详细信息，请参阅[如何查看相关活动和相关文档](../core/how-to-view-related-activities-and-related-documents.md)。  
  
-   当前正在处理的或已发生的所有业务活动围绕浏览聚合 （关键绩效指标）。 可以进行聚合实时也可以基于在特定时间执行的活动的快照。 有关浏览聚合的详细信息，请参阅[如何查看 BAM 聚合](../core/how-to-view-bam-aggregations.md)。  
  
-   使用进度维度，在 BAM 中的新功能之一。 这些维度允许筛选或分组在完成了指定阶段的活动。 有关进度维度的详细信息，请参阅[进度维度](../core/progress-dimension.md)。  
  
-   搜索活动实例根据其进度或业务数据。 例如，您可以搜索等待客户签名的贷款和美元金额大于给定的值。 有关在 BAM 中搜索活动的详细信息，请参阅[如何搜索活动](../core/how-to-search-for-activities.md)。
