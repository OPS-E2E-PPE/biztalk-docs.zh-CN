# <a name="how-to-open-the-biztalk-server-administration-console"></a>如何打开 BizTalk Server 管理控制台
可以使用以下过程以打开 BizTalk Server 管理控制台，并进行排序或刷新的详细信息窗格中显示的数据。  
  
## <a name="prerequisites"></a>先决条件  
 若要执行此过程，必须为 BizTalk Server Operators 组的成员或 BizTalk Server Administrators 组的成员身份登录。  
  
### <a name="to-open-the-biztalk-server-administration-console"></a>若要打开 BizTalk Server 管理控制台  
  
- 单击**启动**，单击**程序**，单击[!INCLUDE[btsBizTalkServerStartMenuItemui](../includes/btsbiztalkserverstartmenuitemui-md.md)]，然后单击**BizTalk Server 管理**。  
  
   或  
  
- 单击**启动**，单击**运行**，类型**BTSmmcLauncher.exe**，然后单击**确定**。  
  
### <a name="to-refresh-the-biztalk-server-administration-console"></a>若要刷新 BizTalk Server 管理控制台  
  
-   右键单击你想要刷新，然后单击控制台树中的节点**刷新**。  
  
     这将刷新该节点内的所有对象。  
  
### <a name="to-sort-columns-in-the-details-pane-of-the-biztalk-server-administration-console"></a>若要对 BizTalk Server 管理控制台的详细信息窗格中的列进行排序  
  
-   在控制台树中，单击要查看其信息的详细信息窗格中的项。  
  
-   在详细信息窗格中，单击你想要进行排序所依据的列标题。 这将按升序排序标题下的项进行排序。  
  
-   单击列标题以按降序 （反序） 顺序进行排序。  
  
### <a name="to-reorder-the-columns-in-the-details-pane-of-the-biztalk-server-administration-console"></a>在 BizTalk Server 管理控制台的详细信息窗格中的列重新排序  
  
-   在详细信息窗格中，单击并拖动列将列移到新位置，若要更改的详细信息窗格中显示列的顺序。
