# <a name="how-to-save-a-pipeline"></a>如何保存管道
保存管道存储到.btp 文件，它指定的管道配置有关的信息：  
  
-   管道的名称。  
  
-   模板和它所基于的策略 （发送或接收）。  
  
-   所包括的顺序和组的每个阶段中使用的组件在管道组件的序列。  
  
-   为每个管道组件的配置信息。  
  
### <a name="to-save-a-pipeline"></a>若要保存管道  
  
1.  在“文件”菜单上，单击“保存”。  
  
     -或-  
  
2.  通过单击屏幕顶部附近其选项卡设计图面上显示相应的管道。  
  
3.  上**文件**菜单上，单击**另存为**。  
  
4.  在中**将文件另存为**对话框中**文件**名称字段中，键入管道的名称。 将为您提供一个默认名称。  
  
5.  单击“确定” 。
