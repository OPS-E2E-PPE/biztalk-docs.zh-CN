# <a name="how-to-save-a-pipeline"></a>如何保存管道
保存管道可将有关管道配置的信息存储到 .btp 文件中，其中指定了以下信息：  
  
-   管道的名称。  
  
-   管道基于的模板和策略（发送或接收）。  
  
-   管道包含的组件的序列，包括在每个阶段使用的组件顺序和组件集。  
  
-   每个管道组件的配置信息。  
  
### <a name="to-save-a-pipeline"></a>保存管道  
  
1.  在“文件”菜单上，单击“保存”。  
  
     -或者-  
  
2.  通过单击屏幕顶部附近的管道选项卡，在设计图面上显示相应的管道。  
  
3.  上**文件**菜单上，单击**另存为**。  
  
4.  在**文件另存为**对话框中，在**文件**名称字段，键入管道的名称。 系统将为您提供默认名称。  
  
5.  单击 **“确定”**。
