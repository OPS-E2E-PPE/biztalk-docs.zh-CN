# <a name="biztalk-orchestrations-tab-toolbox"></a>BizTalk 业务流程选项卡工具箱
当您打开使用业务流程设计器的项目 ([!INCLUDE[btsBizTalkServerNoVersion](../includes/btsbiztalkservernoversion-md.md)]业务流程)，工具箱还会显示**BizTalk 业务流程**选项卡。此选项卡上的工具称为*形状*，可见，并且可供使用时**BizTalk 业务流程**展开选项卡。  
  
 若要在 BizTalk 上使用业务流程**业务流程**选项卡上，将其拖到活动的 BizTalk 业务流程的设计图面上。 可以拖动**端口**形状和**角色链接**形状拖到左侧的端口图面或右侧端口图面。 到中心流程区域上，您可以将所有其他形状-描述业务流程的处理流程的形状。  
  
 有关使用工具箱的详细信息和**工具箱**选项卡上，请参阅中的"管理选项卡和项工具箱中的"[!INCLUDE[btsVStudioNoVersion](../includes/btsvstudionoversion-md.md)]文档。
