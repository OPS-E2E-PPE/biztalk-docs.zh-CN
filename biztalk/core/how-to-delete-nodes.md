# <a name="how-to-delete-nodes"></a>如何删除节点
有时，您可能希望删除架构树中的节点。 本主题介绍了执行此任务的逐步说明。  
  
### <a name="to-delete-a-node"></a>删除节点  
  
1.  右键单击你想要删除，然后单击该节点**删除**。  
  
2.  单击**是**在确认对话框中。  
  
> [!CAUTION]
>  当你删除**记录**节点，其所有子节点也被删除。  
  
> [!CAUTION]
>  如果在架构树中的其他位置存在对所删除节点的引用，则还将删除引用节点。  
  
> [!NOTE]
>  如果节点呈灰色，则表示该节点为只读节点，无法删除。
