# <a name="how-to-delete-nodes"></a>如何删除节点
将您想要删除架构树中的节点。 本主题提供执行此任务的分步说明。  
  
### <a name="to-delete-a-node"></a>若要删除节点  
  
1.  右键单击你想要删除，然后单击的节点**删除**。  
  
2.  单击**是**在确认对话框中。  
  
> [!CAUTION]
>  当你删除**记录**节点，所有子节点也会被删除。  
  
> [!CAUTION]
>  如果您删除具有架构树中的其他位置对它的引用的节点，也将删除引用节点。  
  
> [!NOTE]
>  如果一个节点将呈灰色，它是只读的并且不能删除它。
