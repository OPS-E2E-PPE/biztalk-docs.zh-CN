# <a name="debugging-orchestrations"></a>调试业务流程
使用 BizTalk Server 管理控制台中的组中心数据库页上，BizTalk 使您能够跟踪、 视图和高级进度或业务流程和实现它的组件的低级别的详细信息的报表。 除此之外，你还可以通过将信息写入事件日志以跟踪输出，或使用自定义代码来跟踪您的业务流程来调试你的业务流程。
