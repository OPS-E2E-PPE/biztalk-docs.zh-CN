# <a name="information---display-extension-xml-file"></a>信息-显示扩展 XML 文件
**错误代码**  
  
 btm1057  
  
 **说明**  
  
 与扩展对象关联的 XML 存储在所指示的文件中。  
  
 **用户执行任何操作**  
  
 可以使用提供的链接访问生成的扩展 XML，该 XML 中包含程序集名称映射的前缀。
