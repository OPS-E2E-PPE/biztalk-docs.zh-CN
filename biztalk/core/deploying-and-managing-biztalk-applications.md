# <a name="deploying-and-managing-biztalk-applications"></a>部署和管理 BizTalk 应用程序
本部分提供有关管理 BizTalk 应用程序，包括如何部署、 修改、 更新和取消部署的信息。 BizTalk 应用程序提供一种查看和管理的项，称为"项目"组成 BizTalk 业务解决方案的方法。 项目的示例包括 BizTalk 程序集、.NET 程序集、 架构、 映射、 绑定和证书。  
  
 您可以创建 BizTalk 应用程序并向其添加项目。 你可以然后查看、 打包、 部署和管理应用程序和其项目作为单个实体内的 BizTalk 管理控制台或通过使用 BTSTask 命令行工具。  
  
 有关 BizTalk 应用程序部署和维护的背景信息，请参阅[了解 BizTalk 应用程序部署和管理](../core/understanding-biztalk-application-deployment-and-management.md)。 有关部署简单的应用程序的分步说明，这将使您熟悉的应用程序部署功能[!INCLUDE[btsBizTalkServerNoVersion](../includes/btsbiztalkservernoversion-md.md)]，请参阅[演练：部署基本 BizTalk 应用程序](../core/walkthrough-deploying-a-basic-biztalk-application.md)。
