# <a name="using-map-file-properties-for-testing"></a>使用映射文件属性进行测试

## <a name="overview"></a>概述
映射的属性，可通过在解决方案资源管理器，在地图上右击控制着测试期间的许多重要行为：  
  
- 对源和目标架构的输入和输出实例的验证  
  
- 是否要创建输入的实例或使用现有的选项  
  
- 输入和输出测试实例的格式  
  
  有关映射属性的信息，请参阅**映射属性参考** [!INCLUDE[ui-guidance-developers-reference](../includes/ui-guidance-developers-reference.md)]。
