# <a name="soap-adapter"></a>SOAP 适配器
Microsoft BizTalk Server Web 服务使用 SOAP 适配器时接收和发送 Web 服务请求。 SOAP 适配器，可将业务流程作为 Web 服务发布和使用外部 Web 服务。
