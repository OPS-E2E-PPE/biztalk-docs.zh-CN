# <a name="soap-adapter"></a>SOAP 适配器
Microsoft BizTalk Server Web Services 在接收和发送 Web Services 请求时使用 SOAP 适配器。 使用 SOAP 适配器，您可以将业务流程发布为 Web Services 并使用外部 Web Services。
