# <a name="pipeline-designer-keyboard-shortcuts"></a>管道设计器键盘快捷键
通过使用键盘（而不是鼠标），您可以在设计图面中进行导航。 下表显示了可供您使用的键盘快捷键：  
  
|键盘快捷键|效果|  
|-----------------------|------------|  
|向下键|将光标移到下一步的连接线条或形状，包括**启动**和**结束**形状。|  
|向上键|将光标移到上一条连线或形状更高版本，包括**启动**和**结束**形状。|  
|Page Up|将光标焦点移至阶段的上一个开始形状。 这样将会向上滚动页面，以便显示管道的前面部分。|  
|Page Down|将光标焦点移至阶段的下一个开始形状。 这样将会向下滚动页面，以便显示管道的后面部分。|  
|Home|更改焦点移到**启动**的管道的形状。|  
|END|更改焦点移到**结束**的管道的形状。|
