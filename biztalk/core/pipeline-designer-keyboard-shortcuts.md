# <a name="pipeline-designer-keyboard-shortcuts"></a>管道设计器键盘快捷方式
您可以通过而不鼠标使用键盘导航设计图面。 下表显示了可以使用键盘快捷方式。  
  
|键盘快捷键|效果|  
|-----------------------|------------|  
|向下键|将光标移到一条连接线或形状，包括**启动**并**最终**形状。|  
|向上键|将光标移到上一个连接的线条或形状更高版本，包括**启动**并**最终**形状。|  
|Page Up|将光标焦点更改到某一阶段的上一个开始形状。 向上滚动页面，以便显示管道的前面部分效果。|  
|Page Down|将光标焦点更改到某一阶段的下一步开始形状。 此操作将页面向下滚动，以便显示管道的下一个部分。|  
|Home|将焦点移到**启动**管道形状。|  
|END|将焦点移到**最终**管道形状。|
