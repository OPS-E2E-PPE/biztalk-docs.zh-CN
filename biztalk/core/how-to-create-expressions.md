# <a name="how-to-create-expressions"></a>如何创建表达式
BizTalk 表达式编辑器，可以创建 XLANG/s 表达式以扩展各种业务流程设计器形状的功能。 可以创建 XLANG/s 表达式来做出的决策、 设置延迟、 进行.NET 调用和测试 while 循环条件。  
  
 这样，您可以将值分配给消息或消息部分在消息赋值形状中，并可进行.NET 调用和操作中的变量的值**表达式**形状。 您还可以使用它来构造复杂的布尔表达式中**循环**并**判定**形状，并设置延迟**延迟**形状。  
  
### <a name="to-open-biztalk-expression-editor"></a>若要打开 BizTalk 表达式编辑器  
  
1.  右键单击**表达式**形状，**循环**形状，**消息赋值**形状**延迟**形状或一个分支**判定**形状。  
  
2.  单击**编辑表达式**。  
  
3.  将显示 BizTalk 表达式编辑器。  
  
### <a name="to-create-or-edit-an-expression"></a>若要创建或编辑表达式  
  
1.  键入或编辑的 BizTalk 表达式编辑器中的文本字段的表达式。  
  
2.  单击“确定”。 这将保存表达式并将其应用于所选形状。
