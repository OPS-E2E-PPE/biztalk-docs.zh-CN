# <a name="how-to-use-the-bam-add-in-for-excel"></a>如何使用用于 Excel 的 BAM 外接程序
要使用用于 Excel 的 BAM 外接程序模板定义 BAM 视图，请按以下过程中的步骤操作。  
  
 此过程假定您已打开 Excel。  
  
### <a name="to-define-a-bam-view"></a>定义 BAM 视图  
  
1.  在工具栏上单击**BAM**菜单项，然后单击**BAM 活动**并定义你的活动。  
  
     有关创建活动的详细信息，请参阅[定义业务活动](../core/how-to-define-a-business-activity.md)。  
  
2.  上**BAM**菜单上，单击**BAM 视图**。  
  
3.  在**业务活动监视视图向导**，单击**下一步**以继续。  
  
4.  上**BAM 视图**页上，确认**创建新视图**已选中，然后单击**下一步**。  
  
5.  上**新 BAM 视图： 名称以及活动**页上，在**视图名称**框中，键入一个描述性名称。  
  
    > [!IMPORTANT]
    >  不要使用**BizTalkOrchestration**作为视图名称。 此名称保留在 BAM 的其他部分中。  
  
6.  在活动列表中，选择你想要在你当前定义，然后单击 BAM 视图中包含每个活动对应的复选框**下一步**。  
  
7.  上**新 BAM 视图： 查看项**页上，选择你想要包含在 BAM 视图当前定义，而单击每个活动项对应的复选框**下一步**。
