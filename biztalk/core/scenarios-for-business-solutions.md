# <a name="scenarios-for-business-solutions"></a>业务解决方案的方案
本部分介绍两个完整的模型 BizTalk 应用程序。 每个应用程序表示特定的业务模式，并演示其他成员集成模式。  
  
 每个部分概述了应用程序开发人员指南描述的模式和设计决策过程中开发应用程序和用于生成和运行应用程序的部署信息。  
  
> [!NOTE]
>  模式和这些部分中介绍的指南旨在阐释一种特定类型的业务解决方案的方法。 模式是简单的机制，旨在应用于特定的问题，通常与其他模式相结合。 它们不应插入到应用程序。 "按原样"提供的示例代码，不适合生产环境中使用。 它仅用于阐释模式，并因此不包含额外的代码，如异常处理、 日志记录、 安全性和验证。 Microsoft 不支持"按原样"的示例代码部署到生产环境。
