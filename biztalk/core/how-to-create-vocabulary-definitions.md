# <a name="how-to-create-vocabulary-definitions"></a>如何创建词汇定义
可以使用词汇定义向导来创建词汇定义。 您可以定义为常数值、 值的范围、 一组值或.NET 程序集、 XML 文档或数据库表中的元素的词汇定义。 如果您选择的公共变量，将有**获取**并**设置**选项就像在数据库和 XML 定义向导中一样。  
  
 或者，可以通过从一个三个选项卡中选择一个事实创建新的词汇定义 — 例如，数据库列、 XML 节点或.NET 类的成员，该事实拖到**词汇**选项卡上，并将其放入未发布的词汇版本。  
  
> [!NOTE]
>  无法将词汇定义添加到已发布的词汇版本。  
  
### <a name="to-define-a-vocabulary-definition-as-a-constant-value"></a>若要将词汇定义定义为常量值  
  
1.  右键单击该词汇版本，然后依次**添加新定义**。  
  
    > [!NOTE]
    >  您还可以拖动并将项放从事实浏览器的其他选项卡：XML 架构、 数据库和.NET 类  
  
2.  在词汇定义向导中，选择**常数值、 值的范围或值的集合**，然后单击**下一步**。  
  
3.  编辑定义名称和定义说明。  
  
    > [!NOTE]
    >  定义显示名称的最大长度为 512 个字符。  
  
4.  选择**常数值**，然后单击**下一步**。  
  
5.  从可用的系统类型的下拉列表中选择一个定义类型。  
  
6.  编辑显示名称和值，然后依次**完成**。  
  
### <a name="to-define-a-vocabulary-definition-as-a-range-of-values"></a>若要将词汇定义定义为一系列值  
  
1.  右键单击该词汇版本，然后依次**添加新定义**。  
  
2.  在词汇定义向导中，选择**常数值、 值的范围或值的集合**，然后单击**下一步**。  
  
3.  编辑定义名称和定义说明。  
  
4.  选择**值的范围**，然后单击**下一步**。  
  
5.  从下拉列表中选择一个定义类型。  
  
6.  单击**范围下限**，然后单击**编辑**指定范围下限的值。 此时会打开参数定义对话框。  
  
7.  选择**使用常数值**并输入常量值，或选择**已发布的词汇中使用定义**并浏览到某个词汇定义，，然后单击**确定**。  
  
8.  重复步骤 6 和 7**范围上限**。  
  
    > [!NOTE]
    >  **范围上限**值必须超过**范围下限**值。  
  
9. 键入显示格式字符串，或单击**默认**以还原为默认的显示格式字符串，然后单击**完成**。  
  
    > [!NOTE]
    >  格式字符串参数索引应包含在大括号中 — 例如，"{0}"和"{1}"，以用作和范围下限参数的占位符。

与带格式的字符串值的范围  
  
### <a name="to-define-a-vocabulary-definition-as-a-set-of-values"></a>若要将词汇定义定义为一组值  
  
1.  右键单击该词汇版本，然后依次**添加新定义**。  
  
2.  在词汇定义向导中，选择**常数值、 值的范围或值的集合**，然后单击**下一步**。  
  
3.  编辑定义名称和定义说明。  
  
4.  选择**值的集合**，然后单击**下一步**。  
  
5.  输入定义类型和显示名称。  
  
6.  若要将成员添加到组中，选择**使用常数值**并输入常量值，或选择**已发布的词汇中使用定义**并浏览到某个词汇定义，，然后单击**添加**。  
  
7.  为想要包括在集中的所有项重复步骤 6，以便使用常数或词汇定义的任意组合。  
  
8.  若要移动的相对顺序的一组内的成员，请选择它，然后单击**向上**或**向下**。  
  
9. 若要从集中删除的成员，请选择它，然后单击**删除**。  
  
10. 完成你的集后，单击**完成**。  
  
     ![词汇定义](../core/media/461d0d70-52ed-4f43-927d-3efb1fbfb934.gif "461d0d70-52ed-4f43-927d-3efb1fbfb934")  
值集  
  
### <a name="to-define-a-vocabulary-definition-as-a-net-class-or-class-member"></a>将词汇定义定义为.NET 类或类成员  
  
1.  右键单击该词汇版本，然后依次**添加新定义**。  
  
2.  在词汇定义向导中，选择 **.NET 类或类成员**，然后单击**下一步**。  
  
3.  编辑**定义名称**并**说明**字段。  
  
4.  单击 **“浏览”**。  
  
5.  在中 **.NET 程序集**对话框中，选择程序集，然后单击**确定**。  
  
    > [!NOTE]
    >  可用程序集必须位于全局程序集缓存 (GAC) 中。 业务规则编辑器加载.NET 程序集的.NET 程序集在浏览时**事实数据资源管理器**窗口中或在 **.NET 类或类成员定义**页**词汇定义**窗口。  如果您在 GAC 中更新该程序集，则关闭业务规则编辑器然后重新启动它，以便加载已更新的 .NET 程序集。 业务规则编辑器并不自动刷新该程序集。  
  
6.  展开程序集节点。  
  
7.  选择一个类，或展开某个类，并选择一个类成员，然后单击**确定**。  
  
8.  单击**下一步**，并指定显示名称。  
  
     有关详细信息，请参阅"使用参数指定词汇定义的显示格式"本主题中的更高版本。  
  
9. 单击 **“完成”**。  
  
     ![词汇定义](../core/media/4259101f-26dd-488f-81c0-f8d225e4016e.gif "4259101f-26dd-488f-81c0-f8d225e4016e")  
Net 对象词汇定义  
  
### <a name="to-define-a-vocabulary-definition-as-an-xml-document-element-or-attribute"></a>将词汇定义定义为 XML 文档元素或属性  
  
1.  右键单击该词汇版本，然后依次**添加新定义**。  
  
2.  在词汇定义向导中，选择**XML 文档元素或属性**，然后单击**下一步**。  
  
3.  键入定义名称和定义说明。  
  
4.  单击**浏览**以查找架构文件并指定文档元素或属性。  
  
5.  从下拉列表中，选择与元素或属性架构中的类型兼容的类型。  
  
    > [!NOTE]
    >  如果到或从文档元素或属性的类型指定的类型不能执行此操作有效的转换，您将会在运行时出错。  
  
6.  选择操作类型，以指示你计划要获取值的元素或属性，或将其值设置。  
  
7.  如果您选择将值设置，请单击**下一步**，然后指定显示格式。  
  
8.  单击 **“完成”**。  
  
     ![词汇定义](../core/media/fd81b534-ec41-4147-b716-1e10ffc01d6f.gif "fd81b534-ec41-4147-b716-1e10ffc01d6f")  
XML 词汇定义  
  
> [!NOTE]
>  永远不会验证存在定义的元素和文档类型。 如果所添加的文档不具有元素，则会发生运行时错误。 如果你断言具有未知的文档类型的文档，它只需将被忽略。  
  
#### <a name="to-define-a-vocabulary-definition-as-a-database-table-or-database-table-column"></a>将词汇定义定义为数据库表或数据库表列  
  
1.  右键单击词汇版本，然后单击**添加新定义**。  
  
2.  在词汇定义向导中，选择**数据库表或数据库表列定义**，然后单击**下一步**。  
  
3.  键入定义名称和定义说明。  
  
4.  单击 **“浏览”**。  
  
5.  选择要使用连接的 SQL Server 计算机。 如果当前未启动 SQL Server 计算机，请选择**启动 SQL Server 停止，则此为**复选框。  
  
6.  选择身份验证类型。 如果选择 SQL Server 身份验证，请键入你的登录名和密码，然后单击**确定**。  
  
7.  选择的表或表列，其中你想要将绑定到，然后依次**确定**。  
  
8.  如果选择了表列，选择是否想要获取其值或将其值设置。 如果您选择获取它的值，键入显示名称。 如果您选择将其值设置，请单击**下一步**指定显示格式。  
  
     有关详细信息，请参阅"使用参数指定词汇定义的显示格式"本主题中的更高版本。  
  
9. 如果选择了表，键入显示名称。  
  
10. 单击 **“完成”**。  
  
    > [!NOTE]
    >  默认情况下**DataConnection**使用绑定

#### <a name="to-specify-the-display-format-of-a-vocabulary-definition-by-using-parameters"></a>若要使用参数指定词汇定义的显示格式  
  
1.  从列表中选择一个参数**参数**在词汇定义向导中，然后单击**编辑**。  
  
2.  选择**使用常数值**并输入常量值，或选择**已发布的词汇中使用定义**并浏览到某个词汇定义，，然后单击**确定**。  
  
3.  键入显示格式字符串，或单击**默认**以还原为默认的显示格式字符串，然后单击**完成**。  
  
    > [!NOTE]
    >  格式字符串参数索引应包含在大括号中 — 例如，"{0}"和"{1}"，以用作占位符的参数。
