# <a name="how-to-create-vocabulary-definitions"></a>如何创建词汇定义
您可以使用词汇定义向导来创建词汇定义。 词汇定义可以定义为常数值、值的范围或值的集合，或定义为 .NET 程序集、XML 文档或数据库表的元素。 如果你选择的公共变量，将有**获取**和**设置**选项就像在数据库和 XML 定义向导。  
  
 或者，可以通过从三个选项卡之一选择事实创建新的词汇定义 — 例如，数据库列、 XML 节点或一个.NET 类的成员-事实拖动到**词汇**选项卡上，并将其放置到词汇未发布版本。  
  
> [!NOTE]
>  您不能将词汇定义添加到已发布的词汇版本中。  
  
### <a name="to-define-a-vocabulary-definition-as-a-constant-value"></a>将词汇定义定义为常数值  
  
1.  右击词汇版本，然后单击**添加新定义**。  
  
    > [!NOTE]
    >  您还可以从事实浏览器的其他选项卡（“XML 架构”、“数据库”和“.NET 类”）中拖放项  
  
2.  在词汇定义向导中，选择**常量值、 值的范围或设置的值**，然后单击**下一步**。  
  
3.  编辑定义名称和定义说明。  
  
    > [!NOTE]
    >  定义显示名称的最大长度为 512 个字符。  
  
4.  选择**常量值**，然后单击**下一步**。  
  
5.  从可用系统类型的下拉列表中选择一个定义类型。  
  
6.  编辑的显示名称和值，并依次**完成**。  
  
### <a name="to-define-a-vocabulary-definition-as-a-range-of-values"></a>将词汇定义定义为值的范围  
  
1.  右击词汇版本，然后单击**添加新定义**。  
  
2.  在词汇定义向导中，选择**常量值、 值的范围或设置的值**，然后单击**下一步**。  
  
3.  编辑定义名称和定义说明。  
  
4.  选择**值的范围**，然后单击**下一步**。  
  
5.  从下拉列表中选择一个定义类型。  
  
6.  单击**范围低**，然后单击**编辑**指定较低范围的值。 此时，将打开“参数定义”对话框。  
  
7.  选择**使用常量值**并输入一个常量值，或选择**发布的词汇中使用定义**并浏览到词汇定义，，然后单击**确定**。  
  
8.  重复步骤 6 和 7 适用于**范围上限**。  
  
    > [!NOTE]
    >  **范围上限**值必须超过**范围低**值。  
  
9. 键入的显示格式字符串或单击**默认**以还原到默认的显示格式字符串，然后单击**完成**。  
  
    > [!NOTE]
    >  格式字符串应包括在大括号中的参数索引 — 例如，"{0}"和"{1}"—to 充当高和低范围参数的占位符的不同而不同。

带有格式字符串的值的范围  
  
### <a name="to-define-a-vocabulary-definition-as-a-set-of-values"></a>将词汇定义定义为值的集合  
  
1.  右击词汇版本，然后单击**添加新定义**。  
  
2.  在词汇定义向导中，选择**常量值、 值的范围或设置的值**，然后单击**下一步**。  
  
3.  编辑定义名称和定义说明。  
  
4.  选择**设置的值**，然后单击**下一步**。  
  
5.  输入定义类型和显示名称。  
  
6.  若要将成员添加到组中，选择**使用常量值**并输入一个常量值，或选择**发布的词汇中使用定义**并浏览到词汇定义，，然后单击**添加**。  
  
7.  重复步骤 6，以便使用常数或词汇定义的任意组合，向集合添加所需数目的项。  
  
8.  若要移动的相对顺序的集内的成员，请选择它，然后单击**向上**或**下**。  
  
9. 若要从集中删除成员，请选择它，然后单击**删除**。  
  
10. 完成你的集后，单击**完成**。  
  
     ![词汇定义](../core/media/461d0d70-52ed-4f43-927d-3efb1fbfb934.gif "461d0d70-52ed-4f43-927d-3efb1fbfb934")  
值的集合  
  
### <a name="to-define-a-vocabulary-definition-as-a-net-class-or-class-member"></a>将词汇定义定义为 .NET 类或类成员  
  
1.  右击词汇版本，然后单击**添加新定义**。  
  
2.  在词汇定义向导中，选择**.NET 类或类成员**，然后单击**下一步**。  
  
3.  编辑**定义名称**和**说明**字段。  
  
4.  单击 **“浏览”**。  
  
5.  在**.NET 程序集**对话框中，选择一个程序集，，然后单击**确定**。  
  
    > [!NOTE]
    >  可用程序集必须位于全局程序集缓存 (GAC) 中。 业务规则编辑器加载.NET 程序集的.NET 程序集在浏览时**事实数据资源管理器**窗口中或在**.NET 类或类成员定义**页**词汇定义**窗口。  如果您在 GAC 中更新该程序集，则关闭业务规则编辑器然后重新启动它，以便加载已更新的 .NET 程序集。 业务规则编辑器并不自动刷新该程序集。  
  
6.  展开该程序集节点。  
  
7.  选择类，或展开某个类，并选择一个类成员，然后单击**确定**。  
  
8.  单击**下一步**，并指定显示名称。  
  
     有关详细信息，请参阅本主题中后面部分的“使用参数指定词汇定义的显示格式”。  
  
9. 单击 **“完成”**。

Net 对象词汇定义  
  
### <a name="to-define-a-vocabulary-definition-as-an-xml-document-element-or-attribute"></a>将词汇定义定义为 XML 文档元素或属性  
  
1.  右击词汇版本，然后单击**添加新定义**。  
  
2.  在词汇定义向导中，选择**XML 文档元素或属性**，然后单击**下一步**。  
  
3.  键入定义名称和定义说明。  
  
4.  单击**浏览**以查找架构文件并指定文档元素或属性。  
  
5.  从下拉列表中，选择一个与架构中元素或属性的类型相兼容的类型。  
  
    > [!NOTE]
    >  如果在指定的类型和文档元素或属性的类型之间无法实现有效的转换，则会在运行时出现错误。  
  
6.  选择一种操作类型，以指示是要获取该元素或属性的值，还是要设置它的值。  
  
7.  如果你选择的值设置，请单击**下一步**，然后指定的显示格式。  
  
8.  单击 **“完成”**。  
  
     ![词汇定义](../core/media/fd81b534-ec41-4147-b716-1e10ffc01d6f.gif "fd81b534-ec41-4147-b716-1e10ffc01d6f")  
XML 词汇定义  
  
> [!NOTE]
>  系统从不验证所定义元素和文档类型是否存在。 如果所添加的文档中不存在该元素，则会出现运行时错误。 如果添加未知类型的文档，则只会将其忽略。  
  
#### <a name="to-define-a-vocabulary-definition-as-a-database-table-or-database-table-column"></a>将词汇定义定义为数据库表或数据库表列  
  
1.  右击词汇版本，然后单击**添加新定义**。  
  
2.  在词汇定义向导中，选择**数据库表或数据库表列定义**，然后单击**下一步**。  
  
3.  键入定义名称和定义说明。  
  
4.  单击 **“浏览”**。  
  
5.  选择要连接的 SQL Server 计算机。 如果当前未启动 SQL Server 计算机，选择**启动 SQL Server 如果已停止**复选框。  
  
6.  选择身份验证类型。 如果选择 SQL Server 身份验证，键入你的登录名和密码，，然后单击**确定**。  
  
7.  选择的表或你想要将绑定到，然后单击的表列**确定**。  
  
8.  如果选择了表列，请选择是要获取它的值还是设置它的值。 如果选择获取它的值，请键入显示名称。 如果你选择将其值设置，请单击**下一步**若要指定显示格式。  
  
     有关详细信息，请参阅本主题中后面部分的“使用参数指定词汇定义的显示格式”。  
  
9. 如果选择了表，请键入显示名称。  
  
10. 单击 **“完成”**。  
  
    > [!NOTE]
    >  默认情况下，**该组**使用绑定

#### <a name="to-specify-the-display-format-of-a-vocabulary-definition-by-using-parameters"></a>使用参数指定词汇定义的显示格式  
  
1.  从列表中选择一个参数**参数**在词汇定义向导，然后单击**编辑**。  
  
2.  选择**使用常量值**并输入一个常量值，或选择**发布的词汇中使用定义**并浏览到词汇定义，，然后单击**确定**。  
  
3.  键入的显示格式字符串或单击**默认**以还原到默认的显示格式字符串，然后单击**完成**。  
  
    > [!NOTE]
    >  格式字符串应包括在大括号中的参数索引 — 例如，"{0}"和"{1}"—to 用作占位符的参数的不同而不同。
