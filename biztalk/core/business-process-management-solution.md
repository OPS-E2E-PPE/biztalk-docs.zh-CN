# <a name="business-process-management-solution"></a>业务流程管理解决方案
业务流程管理解决方案演示了如何设计 BizTalk 应用程序来管理业务流程服务订单处理等。 该解决方案演示了如何构造流程管理器，并提供有关将流程划分成不同的阶段的指导。 解决方案还介绍了如何构造可中断业务流程以及大量复杂异常处理。  
  
 各节概述了解决方案的模式和设计选项和有关生成和运行解决方案的信息的详细说明。  
  
> [!NOTE]
>  模式和这些部分中介绍的指南旨在阐释一种特定类型的业务解决方案的方法。 模式是简单的机制，旨在应用于特定的问题，通常与其他模式相结合。 它们不应插入到应用程序。 "按原样"提供的示例代码，不适合生产环境中使用。 它仅用于阐释模式，并因此不包含额外的代码，如异常处理、 日志记录、 安全性和验证。 Microsoft 不支持"按原样"的示例代码部署到生产环境。
