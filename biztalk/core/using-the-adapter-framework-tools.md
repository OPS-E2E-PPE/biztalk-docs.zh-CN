# <a name="using-the-adapter-framework-tools"></a>使用适配器 Framework 工具
BizTalk 适配器框架会使用以下工具和视图管理和开发自定义适配器：  
  
-   **BizTalk Server 管理控制台。** 这是一个独立的应用程序可从**启动**菜单。 它用于管理几乎所有操作都有关你[!INCLUDE[btsBizTalkServerNoVersion](../includes/btsbiztalkservernoversion-md.md)]安装。 它允许你安装新适配器;管理其关联的接收和发送处理程序;和管理端口、 程序集和业务流程。  
  
-   **添加适配器元数据向导。** 此向导都可在 Microsoft 中的 BizTalk 项目[!INCLUDE[btsVStudioNoVersion](../includes/btsvstudionoversion-md.md)]。 便可通过右键单击中的解决方案资源管理器窗口中的项目名称[!INCLUDE[btsVStudioNoVersion](../includes/btsvstudionoversion-md.md)]，然后单击**添加生成的项**。 开发人员可使用“添加适配器元数据”向导与应用程序适配器进行通信。 使用此向导为请求架构与这些应用程序适配器对应到你的项目。  
  
-   **适配器属性页。** 这些是可通过[!INCLUDE[btsBizTalkServerNoVersion](../includes/btsbiztalkservernoversion-md.md)]管理控制台。 使用这些页面来配置接收处理程序、 接收位置、 发送处理程序和发送适配器的端口。
