# <a name="creating-a-net-application-to-test-a-published-web-service"></a>创建.NET 应用程序以测试已发布的 Web 服务
若要测试已发布的 Web 服务，可以创建使用已发布的 Web 服务的 ASP.NET Web 客户端应用程序。 [!INCLUDE[btsVStudioNoVersion](../includes/btsvstudionoversion-md.md)]帮助集合包含创建 ASP.NET Web 客户端应用程序的重要演练。 可以使用本演练测试已发布的 Web 服务。  
  
 本演练创建 ASP.NET Web 客户端应用程序、 添加 Web 引用，并且提供了示例代码演示如何访问已发布的 Web 服务。 本演练将通过在调试模式下运行应用程序。  
  
 有关信息和有关创建 XML Web 服务客户端项目的过程，请参阅"演练：访问 XML Web 服务使用 Visual Basic 或 Visual C#"中[!INCLUDE[btsVStudioNoVersion](../includes/btsvstudionoversion-md.md)]帮助集合中处[ http://go.microsoft.com/fwlink/?LinkId=62263 ](http://go.microsoft.com/fwlink/?LinkId=62263)。 您可以替换在此示例中，即 TempConvert1 所使用的 Web 服务已发布 BizTalk Web 服务。
