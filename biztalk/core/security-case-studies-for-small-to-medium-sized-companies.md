# <a name="security-case-studies-for-small-to-medium-sized-companies"></a>对于小型到中型公司的安全性案例研究
安全是所有公司重视确保仅选择的一组人员或应用程序可以访问其数据和资源的一个问题。 公司处理和在不同的方式实现安全性。  
  
 此部分提供了部署 Microsoft 的指导原则[!INCLUDE[btsBizTalkServerNoVersion](../includes/btsbiztalkservernoversion-md.md)]在安全环境中。 它提供信息来帮助您评估您的 BizTalk Server 实现，用于小型和中型公司的示例体系结构的潜在威胁。  
  
 很难运行业务今天而不必考虑安全性。 如果您提供在线交易，您想要保护你的客户的信用卡信息。 如果您为财富 500 强公司工作，你想要让恶意用户远离您的网络。 如果你是桌面的用户，你想要防止损坏您的数据和妨碍来完成工作的病毒和蠕虫。 几乎每天您都会听到或阅读有关新的软件漏洞;有关新蠕虫和病毒的速度提高分布，更难彻底，和更多具有破坏力比前面的;和有关公司的网站： 被恶意用户。 无论是您的业务-大或小，几个客户还是数百万客户、 一台计算机或数百台计算机 — 您需要阻止恶意用户及程序 （病毒、 蠕虫） 破坏数据、 计算机和您的工作的能力。
