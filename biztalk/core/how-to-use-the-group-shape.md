# <a name="how-to-use-the-group-shape"></a>如何使用组形状
![](../core/media/ebiz-orch-group.gif "ebiz_orch_group")  
组形状  
  
## <a name="annotations-and-placeholders"></a>批注和占位符  
 可以使用**组**形状作为功能的占位符，需要添加，也可以使用它以进行有关执行的操作中它的批注。  
  
 上的 name 属性**组**形状可以最多 512 个字符。 您可以键入到批注**名称**属性，当折叠形状时，将显示整个名称。 展开该形状时，该名称被截断。
