# <a name="local-groups"></a>本地组
Configuration Manager 创建必要的 Windows 组和用户帐户默认情况下如果一台计算机上安装 BizTalk Server 和所有必备软件。 BizTalk Server 支持只在单台计算机配置中的本地组和用户帐户。
