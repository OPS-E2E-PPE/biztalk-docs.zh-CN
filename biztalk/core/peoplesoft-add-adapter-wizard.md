# <a name="peoplesoft-add-adapter-wizard"></a>PeopleSoft 添加适配器向导
使用从 PeopleSoft 系统访问元数据信息添加适配器向导。  
  
|使用此选项|执行的操作|  
|--------------|----------------|  
|选择的适配器列表中|从列表中选择适配器。|  
|SQL Server|从列表中选择 SQL Server，然后单击**下一步**。|  
|“数据库”|连接到 SQL server 所在的同一个数据库。|  
|Port|选择已配置的发送端口并单击**下一步**。|
