## <a name="explanation"></a>解释  
 此错误表明未找到架构。  
  
## <a name="user-action"></a>用户操作  
 若要解决此错误，请部署该文档架构：  
  
1.  启动**Microsoft Visual Studio**。  
  
2.  创建或打开现有项目  
  
3.  将架构添加到项目中。  
  
4.  在 Visual Studio 中部署项目。
