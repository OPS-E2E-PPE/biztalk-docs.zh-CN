# <a name="error---unreferenced-source-schema"></a>错误-未引用的源架构
**错误代码**  
  
 btm1051  
  
 **说明**  
  
 指示的源架构不包含的映射，根据需要 Microsoft BizTalk 服务器项目中。  
  
 **用户执行任何操作**  
  
 将源架构添加到 BizTalk 项目通过使用包含映射**添加现有项**可用的命令**文件**菜单在解决方案中的 BizTalk 项目的快捷菜单资源管理器。
