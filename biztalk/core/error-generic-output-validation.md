# <a name="error---generic-output-validation"></a>错误-泛型输出验证
**错误代码**  
  
 btm1047  
  
 **说明**  
  
 无法根据目标架构来验证测试映射操作创建的输出实例消息文件，具体原因并未指明。  
  
 **用户执行任何操作**  
  
 在 BizTalk 编辑器中打开目标架构，并使用**验证架构**，**验证实例**，和**生成实例**命令中，右键单击时可用在解决方案资源管理器，以找出问题的架构。
