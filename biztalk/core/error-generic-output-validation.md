# <a name="error---generic-output-validation"></a>错误-一般性输出验证
**错误代码**  
  
 btm1047  
  
 **说明**  
  
 测试映射操作创建输出实例消息文件不是根据目标架构验证原因不明。  
  
 **用户执行任何操作**  
  
 在 BizTalk 编辑器中打开目标架构并使用**验证架构**，**验证实例**，并**生成实例**命令中，右键单击时可用在解决方案资源管理器，以找出问题的架构。
