# <a name="tutorials-using-the-microsoft-biztalk-adapter-for-tibco-rendezvous"></a>教程：使用用于 TIBCO Rendezvous 的 Microsoft BizTalk 适配器

## <a name="overview"></a>概述
用于 TIBCO Rendezvous 的 Microsoft BizTalk 适配器是开发和运行时环境，用于业务线流程管理和自动化。 本部分介绍设置和部署该适配器，这样，您可以获得快速入门中存在的端到端练习的两个 SDK 示例。  
  
 这些教程包含有关将适配器添加到 BizTalk Server 的详细分步信息，创建相应的发送和接收端口、 构建和部署示例，以及处理这些示例的测试数据。
