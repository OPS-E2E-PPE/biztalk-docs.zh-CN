# <a name="single-sign-on-event-10833"></a>单一登录： 事件 10833
## <a name="details"></a>详细信息  
  
|                 |                                                            |
|-----------------|------------------------------------------------------------|
|  产品名称   |                 企业单一登录                  |
| 产品版本 | [!INCLUDE[btsSSOVersion](../includes/btsssoversion-md.md)] |
|    事件 ID     |                           10833                            |
|  事件源   |                           ENTSSO                           |
|    组件    |                            N/A                             |
|  符号名称  |            ENTSSO_E_SSO_DATABASE_NOT_CONFIGURED            |
|  消息正文   |       指定的 SSO 数据库未配置。        |
  
## <a name="explanation"></a>解释  
 系统找不到您指定的 SSO 数据库。  
  
## <a name="user-action"></a>用户操作  
 确认您指定的 SSO 数据库和 SQL Server 名称正确无误。
