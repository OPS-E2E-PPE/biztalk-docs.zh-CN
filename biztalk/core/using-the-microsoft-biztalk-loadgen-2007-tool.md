# <a name="using-the-microsoft-biztalk-loadgen-2007-tool"></a>使用 Microsoft BizTalk LoadGen 2007 工具
Microsoft BizTalk LoadGen 2007 工具适用于开发人员和 IT 专家模拟 BizTalk Server 上的负载。 使用此工具，可以模拟检测性能和压力根据 BizTalk 部署到的负载。 此外，此工具还可以扩展由开发人员来模拟自定义传输的负载。  
  
> [!NOTE]
>  下载[LoadGen](https://www.microsoft.com/download/details.aspx?id=14925)。 
  
 此工具应仅在测试环境中使用并不应在生产环境中使用。 此工具提供"作为-是"不受支持。  
  
> [!NOTE]
>  可以使用 MSMQ 适配器使用 BizTalk Server 负载生成工具，但必须执行一些附加步骤。 有关说明，请参阅[与 MSMQ 将 LoadGen 2007](../core/using-loadgen-2007-with-msmq.md)。  
  
> [!NOTE]
>  LoadGen 可能不支持在 64 位计算机上。 您可以使用 LoadGen 远程运行在 32 位计算机上生成对 64 位服务器的负载。
