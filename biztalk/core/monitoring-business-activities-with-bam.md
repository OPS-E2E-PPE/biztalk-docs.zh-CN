# <a name="monitoring-business-activities-with-bam"></a>使用 BAM 监视业务活动
业务分析员可以使用业务活动监视 (BAM) 来监视有关业务流程的数据。 业务分析员可以使用 Microsoft Office Excel 中的 BAM 工作簿来定义要从业务流程中收集哪些数据，以及业务用户查看所收集数据的方式。  
  
 业务分析员可以使用 BAM 来监视涉及多个应用程序的流程。 例如，采购订单流程中有一些部分可能是通过不同应用程序完成的（其中包括 BizTalk Server，数据库活动，从内部网站、供应商网站和发运方网站发送出的消息，或发送到内部网站、供应商网站和发运方网站的消息）。  
  
 业务用户可以使用 BAM 来查看有关业务流程的实时数据和归档（历史）数据。 例如，业务用户可能需要知道何时到达大于 $100 的采购订单（实时数据），或者本月到达了多少个大于 $100 的采购订单（归档数据）。 BAM 可以聚合实时数据和归档数据，以提高显示数据的速度。  
  
 有关安装 BAM 软件和硬件要求信息，请参阅，BizTalk Server 安装指南，网址[BizTalk Server 什么的新建、 安装、 配置和升级](../install-and-config-guides/biztalk-server-what-s-new-installation-configuration-and-upgrade.md)。
