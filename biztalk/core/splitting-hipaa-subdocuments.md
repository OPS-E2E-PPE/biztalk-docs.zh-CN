  
 <span data-ttu-id="84a84-118">第三个批注项位于 HIPAA 架构内的相应记录级别。</span><span class="sxs-lookup"><span data-stu-id="84a84-118">The third annotation entry is located at the appropriate record levels within the HIPAA schema.</span></span> <span data-ttu-id="84a84-119">此属性也必须设置为 **是**。</span><span class="sxs-lookup"><span data-stu-id="84a84-119">This property must also be set to **yes**.</span></span>

```  
subdocument_creation_break = "yes"
```  
