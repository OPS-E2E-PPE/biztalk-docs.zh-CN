# <a name="testing-published-web-services"></a>测试已发布的 Web Services
在开发 Web Services 时，你可能想通过重新发布你的 Web Services 来对其进行测试。 每次重新发布 Web Services 时，必须重置 Internet 信息服务 (IIS)。  
  
 你可以通过创建 Web Services 客户端应用程序或使用浏览器（如 Internet Explorer）来测试发布的 Web Services。
