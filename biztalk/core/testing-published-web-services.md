# <a name="testing-published-web-services"></a>测试已发布的 Web 服务
开发您的 Web 服务时，你可能想要对其进行重新发布你的 Web 服务测试。 每次重新发布 Web 服务，则必须重置 Internet 信息服务 (IIS)。  
  
 创建 Web 服务客户端应用程序或使用 Internet Explorer 等浏览器，可以测试已发布的 Web 服务。
