# <a name="error---schema-file-format-not-valid"></a>错误-架构文件格式无效
**说明**  
  
 架构文件未包含有效的 XML 架构定义语言 (XSD) 语法。  
  
 **用户执行任何操作**  
  
 查看源架构文件以确定问题原因。
