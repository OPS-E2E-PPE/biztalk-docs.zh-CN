# <a name="single-sign-on-event-10656"></a>单一登录： 事件 10656
## <a name="details"></a>详细信息  

|                 |                                                            |
|-----------------|------------------------------------------------------------|
|  产品名称   |                 企业单一登录                  |
| 产品版本 | [!INCLUDE[btsSSOVersion](../includes/btsssoversion-md.md)] |
|    事件 ID     |                           10656                            |
|  事件源   |                           ENTSSO                           |
|    组件    |                            N\A                             |
|  符号名称  |          SSO_INFO_PASSWORD_SYNC_ADAPTERS_STARTING          |
|  消息正文   |     外部适配器的密码同步正在启动。%r     |

## <a name="explanation"></a>解释  
 此信息事件表明外部适配器的密码同步正在启动。  

## <a name="user-action"></a>用户操作  

- 不需要用户进行任何操作。
