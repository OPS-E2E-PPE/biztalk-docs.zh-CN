# <a name="edi-and-as2-biztalk-server-samples-folder"></a>EDI 和 AS2（BizTalk Server 示例文件夹）
Microsoft BizTalk Server 在其软件开发工具包 (SDK) 中包含的以下的 EDI 和 AS2 示例。 本部分提供了有关这些 EDI 示例所演示的功能、用于生成和运行这些示例的说明以及预期结果的详细信息。  
  
## <a name="in-this-section"></a>本节内容  
  
-   [通过文件 （BizTalk Server 示例） 的 AS2](../core/as2-over-file-biztalk-server-sample.md)。 演示如何通过 FILE 接收位置接收 AS2 消息。  
  
-   [消息扩充示例 （BizTalk Server 示例）](../core/message-enrichment-sample-biztalk-server-sample.md)。 演示如何将 EDI 文档的交换标头附加到事务集消息中。  
  
## <a name="additional-sample-code"></a>其他示例代码  
  
-   [通过文件发送端口发送 AS2 消息](../core/sending-an-as2-message-over-a-file-send-port.md)。 本主题介绍如何将标头写入 AS2 消息，以及如何升级 AS2 标头上下文属性。  
  
-   [为出站的参与方解析编写 AS2 上下文属性](../core/writing-as2-context-properties-for-outbound-party-resolution.md)。 本主题介绍如何将 AS2To 属性或 UserHttpHeaders 属性（与 AS2 属性一起）写入上下文，以便您可以执行出站 AS2 参与方解析。
