# <a name="edi-and-as2-biztalk-server-samples-folder"></a>EDI 和 AS2 （BizTalk Server 示例文件夹）
Microsoft BizTalk Server 在其软件开发工具包 (SDK) 包含了以下 EDI 和 AS2 示例。 本部分提供有关这些 EDI 示例，说明用于构建和运行示例，以及可以预期的结果所演示的功能的详细的信息。  
  
## <a name="in-this-section"></a>本节内容  
  
-   [AS2 over File （BizTalk Server 示例）](../core/as2-over-file-biztalk-server-sample.md)。 演示如何通过 FILE 接收 AS2 消息接收位置。  
  
-   [消息充实示例 （BizTalk Server 示例）](../core/message-enrichment-sample-biztalk-server-sample.md)。 演示如何将交换标头附加到 EDI 文档的事务集消息。  
  
## <a name="additional-sample-code"></a>其他示例代码  
  
-   [通过 FILE 发送端口发送 AS2 消息](../core/sending-an-as2-message-over-a-file-send-port.md)。 本主题介绍如何将标头写入 AS2 消息以及如何升级 AS2 标头上下文属性。  
  
-   [写入 AS2 上下文属性以进行出站参与方解析](../core/writing-as2-context-properties-for-outbound-party-resolution.md)。 本主题介绍如何将 AS2To 属性或 UserHttpHeaders 属性 （与 AS2 属性） 写入到上下文，以便你可以执行出站 AS2 参与方解析。
