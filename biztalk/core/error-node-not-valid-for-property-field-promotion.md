# <a name="error---node-not-valid-for-property-field-promotion"></a>错误-节点不适于属性字段提升
**错误代码**  
  
 BEC2001  
  
 **说明**  
  
 所指示的节点不能作为属性字段升级，因为它无法包含必需的文本内容。 仅**Field 元素**节点，**字段特性**节点，或**记录**具有混合或简单内容的节点可以提升作为属性字段。  
  
 **用户执行任何操作**  
  
 使用**属性字段**选项卡中**升级属性**对话框删除不允许的属性提升。
