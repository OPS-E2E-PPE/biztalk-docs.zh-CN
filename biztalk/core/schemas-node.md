# <a name="schemas-node"></a>架构节点
使用**架构**节点以显示当前 BizTalk 中可用的应用程序和系统架构中部署的架构。 右键单击**架构**节点可显示以下命令：  
  
-   **隐藏属性架构**。 选中时，筛选出架构用于将实例消息中的特定值升级到消息上下文。 当选择此菜单项时，它将替换上**架构**由节点菜单**显示属性架构**。  
  
-   **显示属性架构**。 选中时，筛选出属性架构。 当选择此菜单项时，它将替换上**架构**由节点菜单**隐藏属性架构**。
