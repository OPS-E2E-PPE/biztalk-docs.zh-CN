# <a name="using-the-bam-wcf-and-wf-interceptors"></a>使用 BAM WCF 和 WF 侦听器
BAM 侦听器将扩展的 BAM 侦听器功能[!INCLUDE[btsBizTalkServerNoVersion](../includes/btsbiztalkservernoversion-md.md)]到 Windows Workflow Foundation (WF)、 Windows Communication Framework (WCF) 和其他运行时环境。 通过使用 BAM 侦听器，可以跟踪业务流程，无需重新编译 WF 或 WCF 解决方案 — 通过配置文件使用 XML 和一系列元素，将应用程序事件映射到 BAM 活动定义的数据进行集成相关 ID、 继续标记和其他必需和可选项目。
