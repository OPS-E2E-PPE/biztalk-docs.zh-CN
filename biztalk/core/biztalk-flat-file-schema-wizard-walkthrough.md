# <a name="biztalk-flat-file-schema-wizard-walkthrough"></a>BizTalk 平面文件架构向导演练
下面的演练演示了如何从使用 BizTalk 平面文件架构向导的平面文件文档实例创建平面文件架构。  
  
 若要从交换实例创建平面文件架构，请首先按照演练中说明的步骤创建文档架构，但是应仅从“选择文档数据”屏幕中交换实例的说明单个文档结构的部分中选择，而不是选择整个交换实例。 然后，你定义信封架构。 有关定义信封架构和使用文档架构和信封架构配置使用的管道组件的详细信息，请参阅[信封架构](../core/envelope-schemas.md)和[如何配置 XML 汇编程序管道组件](../core/how-to-configure-the-xml-assembler-pipeline-component.md)。
