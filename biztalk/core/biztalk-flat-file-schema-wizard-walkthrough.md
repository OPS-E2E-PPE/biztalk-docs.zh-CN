# <a name="biztalk-flat-file-schema-wizard-walkthrough"></a>BizTalk 平面文件架构向导演练
下面的演练演示如何从使用 BizTalk 平面文件架构向导的平面文件文档实例创建平面文件架构。  
  
 若要从交换实例创建平面文件架构，您首先创建文档架构在本演练中所述的步骤，但仅在其中描述了从交换实例中的单个文档结构的部分中选择选择文档数据屏幕而不是选择整个交换实例。 然后，您定义信封架构。 有关定义信封架构和使用文档架构和信封架构配置用于管道组件的详细信息，请参阅[信封架构](../core/envelope-schemas.md)和[如何配置 XML 组装器管道组件](../core/how-to-configure-the-xml-assembler-pipeline-component.md)。
