# <a name="tutorials-using-the-microsoft-biztalk-adapter-for-tibco-enterprise-message-service"></a>教程： 使用 Microsoft BizTalk Adapter for TIBCO 企业消息服务
用于 TIBCO Enterprise Message Service 的 Microsoft BizTalk 适配器是用于行业过程管理和自动化的开发和运行时环境。 本部分介绍两个设置和部署适配器，以便你可以快速入门中存在一个端到端过程的 SDK 示例。  
  
 这些教程包括有关将适配器添加到 BizTalk Server 的详细分步信息，创建相应发送和接收端口，生成和部署这些示例中，以及处理示例的测试数据。
