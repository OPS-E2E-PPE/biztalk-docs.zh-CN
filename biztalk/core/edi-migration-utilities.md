# <a name="edi-migration-utilities"></a>EDI 迁移实用工具
EDI 迁移实用程序随以前版本的[!INCLUDE[btsBizTalkServerNoVersion](../includes/btsbiztalkservernoversion-md.md)]用于将基本 EDI 应用程序使用提供的 BizTalk Server 2006 R2 或 BizTalk Server 2009 一部分的 EDI 解决方案迁移。 将基本 EDI 应用程序直接迁移[!INCLUDE[prague](../includes/prague-md.md)]不支持。 您必须首先使用与这些版本的 [!INCLUDE[btsBizTalkServerNoVersion](../includes/btsbiztalkservernoversion-md.md)] 一同发售的 EDI 迁移实用工具将 Base EDI 应用程序迁移到 BizTalk Sever 2006 R2 或 BizTalk Server 2009，然后再使用与 [!INCLUDE[prague](../includes/prague-md.md)] 一同发售的参与方迁移工具，将参与方定义从 BizTalk Server 2006 R2 或 BizTalk Server 2009 迁移到 [!INCLUDE[prague](../includes/prague-md.md)]。  
  
 有关方迁移工具的详细信息，请参阅[迁移 BizTalk Server 以前版本中的 EDI 项目](http://msdn.microsoft.com/library/b956a97e-03d0-47ea-a2ce-c07a339c0f2c)。
