# <a name="request-assistance"></a>请求协助
如果需要从您的系统管理员联系或与其他技术支持人员的协助你[!INCLUDE[btsBizTalkServerNoVersion](../includes/btsbiztalkservernoversion-md.md)]实现，你可以使用**协助**底部的按钮**结果**详细信息将协助请求的页。  
  
## <a name="how-to-request-assistance"></a>如何请求帮助  
 单击**协助**按钮打开**请求技术协助 Web**页对话框。 若要请求协助，则输入一个描述你的问题中的使用者**主题**文本框。 在**问题描述**描述与系统管理员将需要解决问题的详细信息时遇到的问题的文本框。 当问题描述满意后时，单击**发送报告**按钮将发送协助请求。 您的请求将被放置在服务器应用程序事件日志中，您的系统管理员可以使用“组概述”页的消息与服务实例跟踪功能来检索该请求。
