# <a name="request-assistance"></a>请求协助
当你需要从您的系统管理员或其他技术支持人员的协助你[!INCLUDE[btsBizTalkServerNoVersion](../includes/btsbiztalkservernoversion-md.md)]实现，可以使用**协助**底部的按钮**结果**详细信息若要提出一个帮助请求的页。  
  
## <a name="how-to-request-assistance"></a>如何请求协助  
 单击**协助**按钮将打开**请求技术协助网页**页对话框。 若要请求协助，则输入描述您的问题中的使用者**使用者**文本框。 在中**问题说明**文本框中，描述您的系统管理员将需要解决的问题的详细信息时遇到的问题。 问题说明后，单击**发送报告**按钮发送帮助请求。 服务器应用程序事件日志中放置你的请求，您的系统管理员可以使用检索该消息和从组概述页的服务实例跟踪...
