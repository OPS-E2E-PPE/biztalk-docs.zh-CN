# <a name="single-sign-on-event-10834"></a>单一登录： 事件 10834
## <a name="details"></a>详细信息  
  
|                 |                                                                               |
|-----------------|-------------------------------------------------------------------------------|
|  产品名称   |                           企业单一登录                           |
| 产品版本 |          [!INCLUDE[btsSSOVersion](../includes/btsssoversion-md.md)]           |
|    事件 ID     |                                     10834                                     |
|  事件源   |                                    ENTSSO                                     |
|    组件    |                                      N/A                                      |
|  符号名称  |                          ENTSSO_E_SSO_NOT_SSO_ADMIN                           |
|  消息正文   | 当前用户不是指定的 SSO 管理员帐户的成员。 |
  
## <a name="explanation"></a>解释  
 要指定此操作，用户必须是指定的 SSO 管理员帐户的成员。

 有关帐户的详细信息，请参阅[如何指定 SSO Administrators 和 Affiliate Administrators 帐户](../core/how-to-specify-sso-administrators-and-affiliate-administrators-accounts.md)。
