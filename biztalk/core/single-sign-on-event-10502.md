# <a name="single-sign-on-event-10502"></a>单一登录： 事件 10502
## <a name="details"></a>详细信息  

|                 |                                                            |
|-----------------|------------------------------------------------------------|
|  产品名称   |                 企业单一登录                  |
| 产品版本 | [!INCLUDE[btsSSOVersion](../includes/btsssoversion-md.md)] |
|    事件 ID     |                           10502                            |
|  事件源   |                           ENTSSO                           |
|    组件    |                            N\A                             |
|  符号名称  |                  SSO_INFO_SERVICE_STARTED                  |
|  消息正文   |                SSO 服务已启动。                |

## <a name="explanation"></a>解释  
 此信息事件表示 ENTSSO 服务已启动。
