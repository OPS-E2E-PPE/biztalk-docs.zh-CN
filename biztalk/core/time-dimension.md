# <a name="time-dimension"></a>时间维度
时间维度，您可以根据时间创建的聚合。 例如时间维度可以用于创建下表：  
  
|Year|Month|Count|  
|----------|-----------|-----------|  
|2003|January|120|  
||February|230|  
||March|350|  
||April|280|  
  
 可以与任何其他维度结合使用时间维度。 例如，可以在行和列的数据维度使用时间维度创建以下表：  
  
|||网球拍|足球|  
|------|------|---------------------|------------------|  
||January|50|70|  
||February|120|110|  
||March|300|50|  
||April|220|60|
