# <a name="how-to-define-durations"></a>如何定义持续时间
持续时间指定的时间，定义的开始和结束里程碑的特定活动项。 例如，PO 的有效的天的数。  
  
 使用 BAM 视图向导来设置或修改持续时间。 您可以设置持续时间之前，使用 BAM 活动向导创建业务活动和项。 有关使用向导的详细信息，请参阅[定义业务活动和在 Excel 中的视图](../core/defining-business-activities-and-views-in-excel.md)。  
  
### <a name="to-define-durations"></a>若要定义持续时间  
  
1.  在 BAM 视图向导中，单击**下一步**直到看到第二个**编辑 BAM 视图：查看项**页上，然后依次**新的持续时间**。  
  
    > [!NOTE]
    >  可以创建一个持续时间之前，必须具有至少两个里程碑组。  
  
2.  键入的名称**持续时间名**。  
  
3.  从下拉列表中，选择**开始业务里程碑**，**结束业务里程碑**，**时间解析度**，然后单击**确定**。
