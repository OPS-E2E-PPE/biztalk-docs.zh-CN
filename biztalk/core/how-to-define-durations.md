# <a name="how-to-define-durations"></a>如何定义持续时间
持续时间指定一个时间长度，由特定活动项的开始里程碑和结束里程碑来定义。 例如，PO 的有效天数。  
  
 使用 BAM 视图向导可以设置或修改持续时间。 在设置持续时间之前，必须先使用 BAM 活动向导创建业务活动和项。 有关使用向导的详细信息，请参阅[定义业务活动和在 Excel 中的视图](../core/defining-business-activities-and-views-in-excel.md)。  
  
### <a name="to-define-durations"></a>定义持续时间  
  
1.  在 BAM 视图向导中，单击**下一步**直到看到第二个**编辑 BAM 视图： 查看项**页，，然后单击**新持续时间**。  
  
    > [!NOTE]
    >  在创建持续时间之前，必须至少已具有两个里程碑或里程碑组。  
  
2.  键入的名称**持续时间名称**。  
  
3.  从下拉列表中选择**开始业务里程碑**，**结束业务里程碑**，**时解析**，然后单击**确定**。
