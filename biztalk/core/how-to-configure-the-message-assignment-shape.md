# <a name="how-to-configure-the-message-assignment-shape"></a>如何配置消息赋值形状
![](../core/media/ebiz-orch-assign.gif "ebiz_orch_assign")  
消息赋值形状  
  
> [!NOTE]
>  一个**消息赋值**形状可以仅在存在**构造消息**形状。 如果在设计图面上，拖动一个消息分配的其他任何位置的新**构造消息**将创建形状。  
  
### <a name="to-configure-a-message-assignment-shape"></a>若要配置消息赋值形状  
  
1.  如果 BizTalk 表达式编辑器不可见，请右键单击**消息赋值**形状，然后单击**编辑表达式**或在属性窗口中，单击**省略号**(**...**) 按钮**表达式**属性。  
  
2.  在 BizTalk 表达式编辑器中，创建要分配值的表达式。 有关详细信息，请参阅[表达式的要求和限制](../core/requirements-and-limitations-for-expressions.md)。
