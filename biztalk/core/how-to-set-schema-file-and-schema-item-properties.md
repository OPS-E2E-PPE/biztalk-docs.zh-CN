# <a name="how-to-set-schema-file-and-schema-item-properties"></a>如何设置架构文件和架构项属性
在中设置架构文件属性和项属性[!INCLUDE[btsVStudioNoVersion](../includes/btsvstudionoversion-md.md)]属性窗口。 本主题介绍了检查和/或将架构文件/项属性设置所需的步骤。  
  
## <a name="set-schema-file-properties-in-the-properties-window"></a>在属性窗口中设置架构文件属性  
  
1. 在解决方案资源管理器，右键单击你想要检查和/或设置其相关属性，再单击的架构**属性**。  
  
    您可以检查和/或设置感兴趣的属性。 有关可用于 BizTalk 架构的属性的信息，请参阅**架构文件属性** [!INCLUDE[ui-guidance-developers-reference](../includes/ui-guidance-developers-reference.md)]。
  
2. 在 Visual Studio 属性窗口中，设置所需的架构文件属性，然后按 ENTER。  
  
## <a name="set-schema-item-properties-in-the-properties-window"></a>在属性窗口中设置架构项属性  
  
1. 在解决方案资源管理器，选择你想要检查和/或设置相关属性的架构。  
  
2. 如有必要，请按 F4 打开[!INCLUDE[btsVStudioNoVersion](../includes/btsvstudionoversion-md.md)]属性窗口。  
  
3. 在属性窗口中检查和/或设置感兴趣的属性。 有关 BizTalk 架构属性窗口中的可用属性的信息，请参阅 **[节点属性-按节点类型** [!INCLUDE[ui-guidance-developers-reference](../includes/ui-guidance-developers-reference.md)]。
  
    您设置的任何新属性值是根本不需要批准或取消所做的更改立即生效。
