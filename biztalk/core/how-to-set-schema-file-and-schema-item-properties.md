# <a name="how-to-set-schema-file-and-schema-item-properties"></a>如何设置架构文件和架构项属性
架构文件属性和项属性中设置[!INCLUDE[btsVStudioNoVersion](../includes/btsvstudionoversion-md.md)]属性窗口。 本主题将介绍对架构文件/项属性进行检查和/或设置所需的步骤。  
  
## <a name="set-schema-file-properties-in-the-properties-window"></a>在属性窗口中设置架构文件属性  
  
1.  在解决方案资源管理器，右键单击你想要检查和/或设置相关的属性，再单击的架构**属性**。  
  
     可以检查和/或设置感兴趣的属性。 有关可用于 BizTalk 架构的属性的信息，请参阅**架构文件属性** [!INCLUDE[ui-guidance-developers-reference](../includes/ui-guidance-developers-reference.md)]。
  
2.  在 Visual Studio 属性窗口中，设置文件属性的所需的架构，然后按 ENTER。  
  
## <a name="set-schema-item-properties-in-the-properties-window"></a>在属性窗口中设置架构项属性  
  
1.  在解决方案资源管理器中，选择要检查和/或设置其相关属性的架构。  
  
2.  如有必要，可按 F4 打开 [!INCLUDE[btsVStudioNoVersion](../includes/btsvstudionoversion-md.md)] 的“属性”窗口。  
  
3.  在“属性”窗口中，对目标属性进行检查和/或设置。 有关 BizTalk 架构属性窗口中的可用属性的信息，请参阅**[节点属性-通过节点类型** [!INCLUDE[ui-guidance-developers-reference](../includes/ui-guidance-developers-reference.md)]。
  
     无需对所做的更改进行批准或取消，所设置的任何新属性值都会立即生效。
