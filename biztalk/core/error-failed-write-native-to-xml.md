# <a name="error---failed-write-native-to-xml"></a>错误-无法在将本地写入到 XML
**错误代码**  
  
 btm1043  
  
 **说明**  
  
 对所指示的 XML 文件进行写入时出现文件写入错误。  
  
 **用户执行任何操作**  
  
 检查您是否有足够的磁盘空间和权限来创建所指示的文件。
