# <a name="how-to-move-and-copy-nodes"></a>如何移动和复制节点
您可能会遇到你想要将现有节点移动到架构树中的不同位置的情况。 您还可以通过复制现有节点，然后将其粘贴到架构树中的其他位置来节省时间。 本主题提供执行这些任务的分步说明。  
  
> [!NOTE]
>  BizTalk 编辑器支持复制和粘贴节点仅在架构，架构不之间。  
  
### <a name="to-move-a-node-within-a-schema"></a>若要将节点在架构内移动  
  
1.  如有必要，展开架构树以显示要移动的节点和想要将其移动到的位置。 有关展开和折叠架构树视图的详细信息，请参阅[管理架构树视图](../core/how-to-manage-the-schema-tree-view.md)。  
  
2.  单击并按住你想要移动并将其拖到树中的另一个位置的节点。  
  
> [!NOTE]
>  当开始向下拖动向上和向下架构树中，鼠标指针将变为箭头、 节点或子箭头。 此箭头指示释放鼠标按钮，或者节点之前、 节点之后或作为节点的子时节点将放置的位置。  
  
#### <a name="to-copy-a-node-within-a-schema"></a>复制架构中节点  
  
1.  右键单击你想要复制，，然后单击的节点**复制**。  
  
2.  右键单击**记录**要在其中插入所复制的节点，然后单击的节点或组节点**粘贴**。  
  
     复制节点的副本放在末尾的子节点**记录**节点或在步骤 2 中选择的组节点。  
  
> [!NOTE]
>  您只能使用在单个架构中的剪切/复制/粘贴功能。 换而言之，你不能将节点从一个架构复制到另一个架构。
