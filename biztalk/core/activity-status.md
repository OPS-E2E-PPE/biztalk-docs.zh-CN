# <a name="activity-status"></a>活动状态
结果详细信息的活动状态区域，可在单个页面上查看所有结果项的详细信息。  
  
 活动状态区域包含两个子区域，即;里程碑和数据。  
  
## <a name="milestones"></a>里程碑  
 里程碑区域显示为所选活动定义的所有里程碑。 里程碑数据显示的日期和时间已达到里程碑。 有些里程碑是互斥的;如批准和拒绝里程碑的订单。 在互斥里程碑的情况下一个将显示为里程碑的日期和时间数据和其他将为空白，该值指示不存在该里程碑的任何数据。  
  
## <a name="data"></a>数据  
 数据区域显示为相应活动定义的所有数据项。 按字母顺序显示的数据项目。
