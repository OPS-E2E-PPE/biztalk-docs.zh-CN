# <a name="activity-status"></a>活动状态
使用结果详细信息的“活动状态”区域，可以在一页中查看结果项的所有详细信息。  
  
 “活动状态”区域包含两个子区域，即“里程碑”和“数据”。  
  
## <a name="milestones"></a>里程碑  
 “里程碑”区域显示为所选活动定义的所有里程碑。 里程碑数据显示的日期和时间已达到里程碑。 某些里程碑是互相排斥;如批准和拒绝里程碑的顺序。 对于互斥里程碑，一个将显示对于里程碑的日期和时间数据和另一个将成为为空，表示没有为该里程碑的数据。  
  
## <a name="data"></a>data  
 “数据”区域显示为相应活动定义的所有数据项。 数据项目按字母顺序显示。
