# <a name="error---cannot-resolve-schemalocation-attribute"></a>错误-无法解析 schemaLocation 属性
**错误代码**  
  
 BEC8001  
  
 **说明**  
  
 BizTalk 编辑器找不到要打开该架构导入的架构。  
  
 **用户执行任何操作**  
  
 单击**架构**节点。 在属性窗口中，单击**导入**属性。 旁边的省略号 (**...**) 按钮将变为活动状态。 单击的按钮。 **导入**此时将显示对话框。 编辑对话框中导入的架构的路径。
