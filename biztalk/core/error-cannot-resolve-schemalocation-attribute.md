# <a name="error---cannot-resolve-schemalocation-attribute"></a>错误-无法解析 schemaLocation 属性
**错误代码**  
  
 BEC8001  
  
 **说明**  
  
 BizTalk 编辑器找不到由要打开的架构所导入的架构。  
  
 **用户执行任何操作**  
  
 单击**架构**节点。 在属性窗口中，单击**导入**属性。 省略号 (**...**) 按钮将变为活动状态。 单击该按钮。 **导入**此时将显示对话框。 在该对话框中进行编辑以将路径指定为所导入的架构。
