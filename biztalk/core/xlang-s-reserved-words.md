# <a name="xlang-s-reserved-words"></a>XLANG-s 保留字
XLANG/s 保留各种关键字、类型名称和标识符名称供自己使用。

## <a name="reserved-types"></a>保留类型  
 SchemaBase  
  
 PropertyBase  
  
## <a name="reserved-identifiers"></a>保留标识符  
 所有以 __（双下划线）开头的名称在 XLANG/s 中都属于保留名称。  
  
> [!NOTE]
>  XLANG/s 标识符可以是不能超过 511 个字符。  
  
> [!NOTE]
>  XLANG/s 代码行的长度不能超过 2040 个字符。
