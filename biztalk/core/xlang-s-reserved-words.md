# <a name="xlang-s-reserved-words"></a>Xlang-s 保留字
XLANG/s 保留各种关键字、 类型名称和标识符名称供自己使用。

## <a name="reserved-types"></a>保留的类型  
 SchemaBase  
  
 PropertyBase  
  
## <a name="reserved-identifiers"></a>保留的标识符  
 __ （双精度不足分数） 开头的所有名称被都保留在 XLANG/s。  
  
> [!NOTE]
>  XLANG/s 标识符可能不能超过 511 个字符。  
  
> [!NOTE]
>  XLANG/s 代码行可能不能超过 2040年个字符。
