# <a name="how-to-validate-schemas-in-visual-studio"></a>如何在 Visual Studio 中的架构验证
构造架构之后，您可以通过验证架构本身中隔离来检查您的工作。  
  
 本主题提供执行此验证任务的分步说明。  
  
### <a name="to-validate-a-schema-in-isolation"></a>若要验证的架构中隔离  
  
1.  在中**解决方案资源管理器**，右键单击你想要验证，然后单击该架构**验证架构**。  
  
2.  在输出窗口中查看结果。 在此窗口中显示成功和错误消息。  
  
> [!NOTE]
>  如果输出窗口并未打开与显示的架构验证是成功还是失败有关的信息，则可以手动打开它。 有关管理此窗口的详细信息，请参阅[管理其他 Visual Studio Windows](../core/how-to-manage-other-visual-studio-windows.md)。
