# <a name="single-sign-on-event-10752"></a>单一登录：事件 10752
## <a name="details"></a>详细信息  
  
|                 |                                                            |
|-----------------|------------------------------------------------------------|
|  产品名称   |                 企业单一登录                  |
| 产品版本 | [!INCLUDE[btsSSOVersion](../includes/btsssoversion-md.md)] |
|    事件 ID     |                           10752                            |
|  事件源   |                           ENTSSO                           |
|    组件    |                            不可用                             |
|  符号名称  |                    ENTSSO_E_APP_EXISTS                     |
|  消息正文   |              应用程序已存在。               |
  
## <a name="explanation"></a>解释  
 具有此名称的应用程序已存在。  
  
## <a name="user-action"></a>用户操作  
 为新应用程序指定不同的名称。
