# <a name="password-sync-adapters"></a>密码同步适配器
一个*密码同步适配器*是传播密码更改与非 Windows 系统的组件。 尽管密码同步适配器类似于传统的单一登录应用程序，但它们具有几个区别：  
  
-   它们是使用专门的接口进行管理。  
  
-   它们介绍在配置存储区中使用专用的 XML 格式。  
  
-   在 host Integration Server 使用的专用的功能来组织中配置存储的适配器。
