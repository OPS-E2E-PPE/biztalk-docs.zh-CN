# <a name="warning---body-xpath-not-valid"></a>警告-正文 XPath 无效
**错误代码**  
  
 BEC1009  
  
 **说明**  
  
 **正文 XPath**此信封架构中指示的根节点的属性引用了不是一个节点**记录**节点，根据需要。  
  
 **用户执行任何操作**  
  
 选择指示的根节点，然后选择的值**正文 XPath**属性，用于正确标识顶级**记录**节点相关联的消息正文。
