# <a name="warning---body-xpath-not-valid"></a>警告-正文 XPath 无效

 BEC1009  
  
 **说明**  
  
 **正文 XPath**此信封架构中指定的根节点的属性引用不是节点**记录**节点，根据需要。  
  
 **用户执行任何操作**  
  
 选择指定的根节点，选择的值**正文 XPath**正确标识顶级属性**记录**关联的消息正文的节点。
