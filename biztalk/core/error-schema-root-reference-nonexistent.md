# <a name="error---schema-root-reference-nonexistent"></a>错误-架构根引用不存在
**错误代码**  
  
 BEC2006  
  
 **说明**  
  
 **根引用**的属性**架构**节点正在引用不存在根节点。 所引用的根节点可能已删除或重命名后被设置为**根引用**属性值。  
  
 **用户执行任何操作**  
  
 设置**根引用**的属性**架构**再次节点。 该属性的下拉列表中包含根节点的所有当前有效的选项。
