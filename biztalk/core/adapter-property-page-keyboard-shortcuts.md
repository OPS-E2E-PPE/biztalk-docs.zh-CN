# <a name="adapter-property-page-keyboard-shortcuts"></a>适配器属性页键盘快捷方式
与所有标准 Microsoft Windows 应用程序，您可以使用适配器属性页中的键盘快捷方式访问适用的使用菜单命令。 下表描述了可用的键盘快捷方式。  
  
> [!NOTE]
>  您必须键入以逗号分隔的组合键。 例如，"CTRL + I，CTRL + R"意味着您必须先按 CTRL + I，然后您必须键入 CTRL + R。 如"ALT + Z，I，F"组合键意味着您必须先按 ALT + Z，但然后可以键入"I"和"F"需松开 ALT 键。  
  
|操作|键盘快捷键|  
|------------|-----------------------|  
|导航到属性网格中，并通过对话框按钮。 -或者-在属性网格中，并且在使用箭头按钮以导航到相应的属性值后，使用 TAB 键编辑属性值。 如果属性值具有省略号 （...） 按钮或下拉列表菜单，请改用 F4。|Tab|  
|在属性网格中的属性字段中导航。|箭头键|  
|置空属性名称字段中的可选字符串值。|Fn，N-或者-SHIFT + F10，N|  
|编辑具有省略号 （...） 按钮或下拉列表框; 的属性值所有其他人使用 TAB 键。|F4|  
|将所做的更改保存到您编辑的属性值。|Enter|  
|关闭弹出菜单。|Esc|
