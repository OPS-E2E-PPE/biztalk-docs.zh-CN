**错误代码**  
  
 btm1060  
  
 **说明**  
  
 映射器编译器不能在由生成任务指定的位置中创建 csharp 文件时发生错误。  
  
 **用户执行任何操作**  
  
 检查错误消息包含在消息 (错误 {1})。 如果它是未经授权的异常，则您可能不具有对项目输出文件夹的写权限。
