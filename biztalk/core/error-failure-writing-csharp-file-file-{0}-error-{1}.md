# <a name="error---failure-writing-csharp-file-ltfile---0gt-error-1"></a>错误-写入 CSharp 文件时出错 (&lt;file:---{0}&gt;)。 错误： {1}
**错误代码**  
  
 btm1060  
  
 **说明**  
  
 映射器编译器不能在生成任务指定的位置中创建 c# 文件时，会出现错误。  
  
 **用户执行任何操作**  
  
 检查包含在消息中的错误消息 (错误{1})。 如果它是未经授权的异常，则可能不需要的项目输出文件夹的写入权限。
