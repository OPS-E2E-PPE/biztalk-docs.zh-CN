# <a name="edifact-contrl-acknowledgment-error-codes"></a>EDIFACT CONTRL 确认错误代码
本主题列出了各个段的 EDIFACT CONTRL 确认中使用的错误代码。 有关这些段的详细信息，请参阅[EDIFACT CONTRL 确认](../core/edifact-contrl-acknowledgment.md)。  
  
 这些错误适用于交换、 组、 消息和数据级别。 在遇到支持的错误、 整个交换、 组或事务集将被拒绝。 没有"接受存在错误"条件对于 EDIFACT 编码的交换。  
  
 **标准 EDIFACT 错误代码**
