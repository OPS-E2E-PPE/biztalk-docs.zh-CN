# <a name="using-enhanced-features-in-biztalk-mapper"></a>使用 BizTalk 映射器中的增强功能
BizTalk 映射器引入了一组可用性增强功能可帮助您处理复杂的映射。  
  
 虽然现有 BizTalk 映射器可以更好地与小架构以及中型架构一起使用，但是对于大型和复杂的架构却无法有效地使用。 此处，新的增强功能扮演着重要角色。
