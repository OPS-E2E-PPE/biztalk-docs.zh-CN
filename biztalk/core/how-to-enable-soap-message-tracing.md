# <a name="how-to-enable-soap-message-tracing"></a>如何启用 SOAP 消息跟踪
你可以启用 SOAP 消息跟踪，以帮助你调试应用程序发布的 Web 服务。 已发布的 Web 服务，以返回到 Web 客户端的详细的异常中设置断点。 若要跟踪的 SOAP 消息，应使用的 SOAP 扩展。 有关 SOAP 扩展的详细信息，请参阅"如何为:: 实现 SOAP 扩展"网址[http://go.microsoft.com/fwlink/?LinkId=62314](http://go.microsoft.com/fwlink/?LinkId=62314)。
