# <a name="how-to-enable-soap-message-tracing"></a>如何启用 SOAP 消息跟踪
可以启用 SOAP 消息跟踪，以帮助您调试 Web services 发布应用程序。 已发布的 Web 服务返回到 Web 客户端的详细的异常中设置断点。 若要跟踪的 SOAP 消息，应使用 SOAP 扩展。 有关 SOAP 扩展的详细信息，请参阅"如何：实现 SOAP 扩展"处[ http://go.microsoft.com/fwlink/?LinkId=62314 ](http://go.microsoft.com/fwlink/?LinkId=62314)。
