# <a name="single-sign-on-event-10846"></a>单一登录：事件 10846
## <a name="details"></a>详细信息  
  
|                 |                                                                                          |
|-----------------|------------------------------------------------------------------------------------------|
|  产品名称   |                                企业单一登录                                 |
| 产品版本 |                [!INCLUDE[btsSSOVersion](../includes/btsssoversion-md.md)]                |
|    事件 ID     |                                          10846                                           |
|  事件源   |                                          ENTSSO                                          |
|    组件    |                                           不可用                                            |
|  符号名称  |                           ENTSSO_E_HOST_GROUP_ONE_ACCOUNT_ONLY                           |
|  消息正文   | 主机组应用程序不能具有多个帐户指定 Windows 帐户。 |
  
## <a name="explanation"></a>解释  
 主机组应用程序不能具有多个帐户指定 Windows 帐户。  
  
## <a name="user-action"></a>用户操作  
 有关主机组应用程序的详细信息，请参阅[SSO 关联应用程序](../core/sso-affiliate-applications.md)。
