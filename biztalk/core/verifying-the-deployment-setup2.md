# <a name="verifying-the-deployment-setup"></a>验证部署安装
使用 BizTalk Server 导入绑定文件之前，请验证以下：  
  
-   用于响应文件夹必须存在并在新计算机上完全相同或编辑绑定文件。  
  
-   TIBCO Enterprise Message Service 系统密码，如果存在在配置中，必须将另存为 * * * 绑定文件中。
