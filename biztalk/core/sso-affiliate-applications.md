 <span data-ttu-id="809c1-111">这些应用程序类型包括：</span><span class="sxs-lookup"><span data-stu-id="809c1-111">The application types are:</span></span>

 <span data-ttu-id="809c1-120">**主机组**主机组应用程序从概念上讲是组应用程序的相反值。</span><span class="sxs-lookup"><span data-stu-id="809c1-120">**Host Group** Host Group applications are conceptually the reverse of Group applications.</span></span> <span data-ttu-id="809c1-121">“主机组”应用程序支持定义的一组非 Windows 帐户到单个 Windows 帐户之间的映射。</span><span class="sxs-lookup"><span data-stu-id="809c1-121">They support mappings between a defined group of non-Windows accounts to a single Windows account.</span></span> <span data-ttu-id="809c1-122">这些非 Windows 帐户使用的单个 Windows 帐户是由该应用程序的 Application Users 帐户定义的。</span><span class="sxs-lookup"><span data-stu-id="809c1-122">The single Windows account that will be used by the non-Windows accounts is defined by the Application Users account for the application.</span></span> <span data-ttu-id="809c1-123">通过为每个非 Windows 帐户创建映射可定义允许访问此应用程序的非 Windows 帐户组。</span><span class="sxs-lookup"><span data-stu-id="809c1-123">The group of non-Windows accounts that is allowed to access this application is defined by creating a mapping for each non-Windows account.</span></span> <span data-ttu-id="809c1-124">“主机组”应用程序只能用于主机启动的 SSO。</span><span class="sxs-lookup"><span data-stu-id="809c1-124">Host Group applications may only be used for Host initiated SSO.</span></span>
