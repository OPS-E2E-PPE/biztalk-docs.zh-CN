# <a name="creating-tracking-profiles"></a>创建跟踪配置文件
创建新的跟踪配置文件或修改现有更好地管理和监视你的组织的特定业务流程。 跟踪配置文件编辑器 (TPE)，可定义要收集来满足业务分析人员的要求的数据。 创建或修改的配置文件可以是简单或根据您的业务要求的那样复杂。  
  
 作为开发人员，您可以创建基于 BAM 活动定义的新配置文件。 已部署的活动定义可能已为其定义的配置文件。 如果没有，通过执行这些任务来创建跟踪配置文件：  
  
-   选择部署服务器和数据库  
  
-   从 BizTalk 管理数据库中选择已部署的 BAM 活动定义  
  
-   定义从业务流程提取的数据  
  
-   连接活动，如果您的业务流程的实际实现跨多个业务流程
