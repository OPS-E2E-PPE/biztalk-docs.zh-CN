# <a name="application-deployment-biztalk-server-samples-folder"></a>应用程序部署 （BizTalk Server 示例文件夹）
Microsoft[!INCLUDE[btsBizTalkServerNoVersion](../includes/btsbiztalkservernoversion-md.md)]在其软件开发工具包 (SDK) 包含几个应用程序部署示例。 本部分提供有关每个示例，说明用于构建和运行示例，以及可以预期的结果所演示的功能的详细的信息。  
  
## <a name="in-this-section"></a>本节内容  
  
-   [模板 （应用程序部署示例）](../core/template-application-deployment-sample.md)。 此示例演示如何创建预处理和后处理脚本在应用程序部署的一个或多个阶段期间运行的执行自定义任务。  
  
-   [CreateApp （应用程序部署示例）](../core/createapp-application-deployment-sample.md)。 此示例说明了如何使用 BTSTask 命令行工具来自动执行应用程序部署任务。  
  
-   [SelectiveBindingImport （应用程序部署示例）](../core/selectivebindingimport-application-deployment-sample.md)。 此示例演示了如何使用预处理和后处理脚本来绑定文件导入到适用于目标环境的应用程序。  
  
-   [VisualStudioHostRestart （应用程序部署示例）](../core/visualstudiohostrestart-application-deployment-sample.md)。 此示例提供了演示如何使用脚本重新启动本地计算机上的所有主机实例。
