# <a name="application-deployment-biztalk-server-samples-folder"></a>应用程序部署（BizTalk Server 示例文件夹）
Microsoft [!INCLUDE[btsBizTalkServerNoVersion](../includes/btsbiztalkservernoversion-md.md)] 在其软件开发工具包 (SDK) 中包括了几个应用程序部署示例。 本部分详细介绍每个示例所演示的功能、生成和运行每个示例的说明以及可以预期得到的结果。  
  
## <a name="in-this-section"></a>本节内容  
  
-   [模板 （应用程序部署示例）](../core/template-application-deployment-sample.md)。 此示例演示如何创建前期和后期处理运行的应用程序部署的一个或多个阶段以及执行自定义任务的脚本。  
  
-   [CreateApp （应用程序部署示例）](../core/createapp-application-deployment-sample.md)。 此示例演示了如何使用 BTSTask 命令行工具自动执行应用程序部署任务。  
  
-   [SelectiveBindingImport （应用程序部署示例）](../core/selectivebindingimport-application-deployment-sample.md)。 此示例演示了如何使用前期和后期处理脚本绑定文件导入的应用程序适用于目标环境。  
  
-   [VisualStudioHostRestart （应用程序部署示例）](../core/visualstudiohostrestart-application-deployment-sample.md)。 此示例提供了演示如何通过使用脚本重新启动本地计算机上的所有主机实例。
