# <a name="orchestration-designer-keyboard-shortcuts"></a>业务流程设计器键盘快捷键
业务流程设计器设计图面提供了使用键盘的完整辅助功能。 此可访问性开始时打开设计图面上，则你可以在 Visual Studio 中通过选择菜单选项开头**新建**选项**文件**菜单。  
  
 在设计图面中也支持辅助功能，例如，通过按向上键和向下键，可以在其中选择其他形状和形状部分。 在设计图面上选择一个形状时, 突出设计图面上，并在业务流程视图窗口 （如果适用），同时显示**属性**网格还显示有关所选形状的信息。  
  
 业务流程设计器通过提供一组键盘快捷键来支持辅助功能。 根据当前使用的设计图面部分的不同，可用的键盘快捷键也有所不同。  
  
 您还可以使用 Visual Studio 提供的许多现有快捷键，就像在 BizTalk 表达式编辑器中使用 Microsoft IntelliSense 那样。
