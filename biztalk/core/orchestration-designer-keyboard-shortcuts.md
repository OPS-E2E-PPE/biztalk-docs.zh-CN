# <a name="orchestration-designer-keyboard-shortcuts"></a>业务流程设计器键盘快捷方式
业务流程设计器设计图面提供了使用键盘的完整辅助功能。 此辅助功能开始时打开设计图面上，你可以执行此操作在 Visual Studio 中通过选择菜单选项开头**新建**选项卡上**文件**菜单。  
  
 其中，例如，你可以选择不同的形状和形状部分通过按向上键和向下箭头在设计图面中也支持可访问性。 在设计图面上选择形状时，它被突出显示设计图面上和在业务流程视图窗口中 （如果适用），同时**属性**网格还显示有关所选形状的信息。  
  
 业务流程设计器通过提供大量的键盘快捷方式来支持可访问性。 具体取决于你当前正在使用设计图面上的哪个部分提供不同的键盘快捷方式。  
  
 此外可以使用许多 Visual Studio 中那样使用 Microsoft IntelliSense 在 BizTalk 表达式编辑器中可用的现有快捷方式。
