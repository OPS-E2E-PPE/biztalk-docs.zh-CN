# <a name="communication-direction"></a>通信方向
每个*端口*都有其自己的通信方向。 通信方向结合使用的端口类型的通信模式与用于完成如何可以使用一个端口的定义。 通信方向或极性，确定通过该端口将哪个方向传输消息。  
  
 如果端口的类型具有一种单向通信模式，其通信方向可以发送或接收。 如果端口的类型具有双向 （请求-响应） 通信模式，其通信方向可以发送接收，在其中将请求发送和接收的响应或接收的发送，在其中收到的请求和发送响应。
