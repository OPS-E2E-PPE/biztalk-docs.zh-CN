# <a name="engine-performance-and-capacity"></a>引擎性能和容量
引擎性能和容量规划 BizTalk Server 解决方案时是重要注意事项。 引擎性能是随着时间的推移来处理文档的功能和引擎容量是 BizTalk Server 系统能够维护的已接收但尚未处理的文档积压工作。
