# <a name="engine-performance-and-capacity"></a>引擎性能和容量
规划你的 BizTalk Server 解决方案时，引擎性能和容量是重要注意事项。 引擎性能随着时间的推移非常来处理文档的能力，而且引擎容量是你的 BizTalk Server 系统能够维护积压工作的已经接收但尚未处理的文档。
