# <a name="error---property-schema-structure-not-valid"></a>错误-不是有效的属性架构结构
**错误代码**  
  
 BEC2007  
  
 **说明**  
  
 此属性架构发现具有无效的结构。 属性架构的结构仅限于一个或多个**Field 元素**节点的子级**架构**节点。  
  
 **用户执行任何操作**  
  
 更正此属性架构的结构，以便**架构**节点只有**Field 元素**节点作为子节点。
