# <a name="error---property-schema-structure-not-valid"></a>错误-属性架构结构无效
**错误代码**  
  
 BEC2007  
  
 **说明**  
  
 此属性架构中发现无效的结构。 属性架构的结构被限制到一个或多个**Field 元素**节点的子级作为节点**架构**节点。  
  
 **用户执行任何操作**  
  
 更正此属性架构的结构，以便**架构**节点仅有**Field 元素**节点作为子节点。
