# <a name="how-to-use-expression-shape"></a>如何使用表达式形状
**表达式**形状使您能够输入您的业务流程中选择任何 XLANG/s 表达式。 例如，您可以借助 .NET 调用来运行外部程序，或仅处理业务流程变量的值。  
  
 虽然**表达式**形状是非常灵活，则不会使用它来执行高级业务流程逻辑，最好是将是在绘制自身的业务流程中可见的好办法。 一般情况下，很容易地理解和维护你的业务流程，如果你**表达式**形状包含简单和模块化表达式。  
  
### <a name="to-configure-an-expression-shape"></a>配置表达式形状  
  
1.  如果 BizTalk 表达式编辑器不可见，请右键单击**表达式**的形状，然后单击**编辑表达式**或在属性窗口中，单击省略号 (**...**) 按钮**表达式**属性。  
  
2.  在 BizTalk 表达式编辑器中，创建要赋值的表达式。 有关详细信息，请参阅[表达式的要求和限制](../core/requirements-and-limitations-for-expressions.md)。
