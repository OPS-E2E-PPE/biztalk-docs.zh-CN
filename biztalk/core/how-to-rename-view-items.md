# <a name="how-to-rename-view-items"></a>如何重命名视图项
通常为业务活动收集的数据包含敏感信息，例如信用卡号或社会安全号码。 这意味着某些类型的业务用户应看到已筛选的子集的数据。  
  
 重命名在 BAM 视图向导中的视图项。 使用此向导创建业务活动、 项和视图。 有关使用向导的详细信息，请参阅[定义业务活动和在 Excel 中的视图](../core/defining-business-activities-and-views-in-excel.md)。  
  
### <a name="to-rename-activity-view-items"></a>若要重命名活动的查看项  
  
1.  在 Excel 菜单上，单击**BAM**，单击**BAM 视图**，然后单击**下一步**直到您看到**编辑 BAM 视图：视图项页**页。  
  
2.  在**编辑 BAM 视图：视图项页**，在列表中选择的项，然后单击**下一步**。  
  
3.  在第二个**编辑 BAM 视图：查看项**页上，选择你想要重命名，并单击查看项**编辑**。  
  
4.  在中**重命名数据项**框中，键入新名称，然后单击**确定**。
