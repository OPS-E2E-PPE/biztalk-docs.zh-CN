# <a name="create-business-rules-using-the-business-rule-composer"></a>创建业务规则使用业务规则编辑器

## <a name="overview"></a>概述
业务规则编辑器，可创建业务策略包含一个或多个业务规则，并可部署这些策略。 它还允许您可以浏览事实 （XML、 数据库和.NET），并使用业务规则中的事实数据。 此外，它允许您创建基于事实的业务词汇，并在业务规则中使用这些词汇。  
  
 本部分提供有关使用业务规则编辑器创建业务规则的特定于任务的信息。
