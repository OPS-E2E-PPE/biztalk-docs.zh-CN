# <a name="warning---empty-target-namespace"></a>警告-目标为空 Namespace
**错误代码**  
  
 BEC1007  
  
 **说明**  
  
 **目标 Namespace**的属性**架构**尚未设置节点。 尽管没有要求，建议为每个架构定义唯一目标命名空间。  
  
 **用户执行任何操作**  
  
 如果没有充分的理由不到，选择**架构**节点的此架构，然后在 Microsoft®[!INCLUDE[btsVStudioNoVersion](../includes/btsvstudionoversion-md.md)]属性窗口中，提供的值**Target Namespace**是唯一的属性对此架构。
