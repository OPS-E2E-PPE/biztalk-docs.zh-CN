# <a name="parties"></a>方
参与方（或组织单位）表示位于 BizTalk Server 之外与业务流程进行交互的实体。 在 BizTalk Server 中，您与之交换消息的每个组织都以参与方表示。 您可通过在角色中登记参与方来定义该参与方的交互方式。
