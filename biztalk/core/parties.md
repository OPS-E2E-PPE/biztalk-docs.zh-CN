# <a name="parties"></a>参与方
参与方或组织单位，表示位于与业务流程交互的 BizTalk Server 之外的实体。 在 BizTalk Server 中，与之交换消息每个组织都以参与方表示。 您可以定义通过在角色中登记参与方的交互方式。
