# <a name="error---native-parsing"></a>错误-本地解析
**错误代码**  
  
 btm1041  
  
 **说明**  
  
 无法解析为测试映射操作指定的本地输入实例消息文件，因此出现以上解析错误消息。  
  
 **用户执行任何操作**  
  
 根据需要参考所指示的解析错误，更正与映射相关联的源架构或所指定文件中的本地输入实例消息，或者同时对这两者进行更正。
