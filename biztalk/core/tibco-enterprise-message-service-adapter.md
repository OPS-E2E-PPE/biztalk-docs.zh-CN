# <a name="tibco-enterprise-message-service-adapter"></a>TIBCO Enterprise Message Service 适配器
用于 TIBCO Enterprise Message Service 的 Microsoft BizTalk 适配器可以发布和订阅队列和主题由 TIBCO EMS 使用 btsBizTalkServerNoVersion 管理。  以下部分介绍如何开始、 创建应用程序，和的详细信息。  
   
## <a name="get-started"></a>入门
[开始](../core/getting-started-with-biztalk-adapter-for-tibco-enterprise-message-service.md)描述适配器功能，安装适配器，并包含一些教程。

## <a name="architecture"></a>体系结构
[体系结构](../core/planning-and-architecture16.md)描述适配器工作原理，提供了一些其他的要求，以及列出了任何限制。

## <a name="security"></a>安全性
[安全](../core/security-in-biztalk-adapter-for-tibco-ems.md)中 BizTalk 服务器以保护应用程序使用此适配器，需使用企业单一登录 (SSO)。

## <a name="create-the-artifacts"></a>创建的项目
[创建应用程序项目](../core/developing-applications5.md)列出了生成的架构、 创建发送和接收使用此适配器，以及如何使用业务流程中的消息上下文属性的项目的步骤。

## <a name="import-apps"></a>导入应用
[导入绑定](../core/deploying-biztalk-adapter-for-tibco-enterprise-message-service.md)讨论如何将应用程序的绑定文件导入到 BizTalk 管理和超过任何限制。 

## <a name="handle-exceptions"></a>处理异常
[使用 BizTalk Server 异常处理](../core/using-biztalk-server-exception-handling5.md)将不同的形状添加到您的业务流程来处理异常提供指导。
