# <a name="system-reference"></a>系统引用
**系统**引用命名空间包含基本类和基类定义常用值和引用数据类型、 事件和事件处理程序、 接口、 属性和处理异常。 此默认引用是必需的所有 BizTalk 项目才能正常工作。 不要删除**系统**引用。  
  
 有关详细信息**系统**引用，请参阅在 Visual Studio 结合使用集合中的"System Namespace" [ http://go.microsoft.com/fwlink/?LinkId=193568 ](http://go.microsoft.com/fwlink/?LinkId=193568)。
