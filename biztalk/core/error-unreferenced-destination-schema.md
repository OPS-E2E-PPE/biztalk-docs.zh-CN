# <a name="error---unreferenced-destination-schema"></a>错误-未引用的目标架构
**错误代码**  
  
 btm1052  
  
 **说明**  
  
 指示的目标架构不是 Microsoft BizTalk Server 项目。 **用户执行任何操作**  
  
 将目标架构添加到 BizTalk 项目通过使用包含映射**添加现有项**可用的命令**文件**菜单和中的 BizTalk 项目的快捷菜单上解决方案资源管理器。
