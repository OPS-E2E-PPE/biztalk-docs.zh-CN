# <a name="error---validate-instance-failure"></a>错误-验证实例失败
**说明**  
  
 **验证实例**命令无法验证指定的实例消息文件符合所指示的架构。  
  
 **用户执行任何操作**  
  
 查看显示的错误消息，以确定实例验证失败的原因。
