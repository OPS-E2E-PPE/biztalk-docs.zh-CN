# <a name="wcf-netnamedpipe-adapter"></a>Wcf-netnamedpipe 适配器
[!INCLUDE[btsCoName](../includes/btsconame-md.md)] [!INCLUDE[btsBizTalkServerNoVersion](../includes/btsbiztalkservernoversion-md.md)] 接收和发送 WCF 服务通过请求时，会使用 Wcf-netnamedpipe 适配器[NetNamedPipeBinding](http://go.microsoft.com/fwlink/?LinkId=82588)。 Wcf-netnamedpipe 适配器，您可以将业务流程和架构作为 WCF 服务发布并使用外部 WCF 服务与**NetNamedPipeBinding**。
