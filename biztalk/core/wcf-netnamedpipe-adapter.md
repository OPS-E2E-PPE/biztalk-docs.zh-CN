# <a name="wcf-netnamedpipe-adapter"></a>WCF-NetNamedPipe 适配器
[!INCLUDE[btsCoName](../includes/btsconame-md.md)][!INCLUDE[btsBizTalkServerNoVersion](../includes/btsbiztalkservernoversion-md.md)]时接收和发送 WCF 服务通过请求使用 WCF NetNamedPipe 适配器[NetNamedPipeBinding](http://go.microsoft.com/fwlink/?LinkId=82588)。 WCF NetNamedPipe 适配器使您能够发布业务流程和作为 WCF 服务的架构，并使用与外部 WCF 服务**NetNamedPipeBinding**。
