- Microsoft[!INCLUDE[btsPlatformsComApis](../includes/btsplatformscomapis-md.md)]操作系统。 您应该熟悉尤其是 Windows 安全功能。  
  
- 管理功能的企业单一登录，尤其是如何执行管理操作使用的用户界面。
