# <a name="single-sign-on-event-10769"></a>单一登录： 事件 10769
## <a name="details"></a>详细信息  
  
|||  
|-|-|  
|产品名称|企业单一登录|  
|产品版本|[!INCLUDE[btsSSOVersion](../includes/btsssoversion-md.md)]|  
|事件 ID|10769|  
|事件源|ENTSSO|  
|组件|N/A|  
|符号名称|ENTSSO_E_APP_INCOMPLETE_FIELDS|  
|消息正文|无法启用应用程序，因为有关此应用程序的字段不完整。|  
  
## <a name="explanation"></a>解释  
 无法启用应用程序，因为有关此应用程序的字段不完整。  
  
## <a name="user-action"></a>用户操作  
 在为给定的应用程序指定字段个数之后，您必须首先创建并填充这些字段，然后才能启用该应用程序。
