# <a name="error---functoid-has-no-output"></a>错误-Functoid 有任何输出
**错误代码**  
  
 btm1013  
  
 **说明**  
  
 所指示的 functoid 没有输出链接，因此无法使用。  
  
 **用户执行任何操作**  
  
 为所指示的 functoid 创建相应的输出链接。
