# <a name="single-sign-on-event-10537"></a>单一登录： 事件 10537
## <a name="details"></a>详细信息  
  
|||  
|-|-|  
|产品名称|企业单一登录|  
|产品版本|[!INCLUDE[btsSSOVersion](../includes/btsssoversion-md.md)]|  
|事件 ID|10537|  
|事件源|ENTSSO|  
|组件|CO|  
|符号名称|SSO_WARN_SSO_DISABLED|  
|消息正文|当前已禁用 SSO 系统。|  
  
## <a name="explanation"></a>解释  
 此警告事件表明 SSO 系统已由 SSO 管理员禁用。  
  
## <a name="user-action"></a>用户操作  
 若要解决此警告问题，请执行以下操作：  
  
-   请与您的 SSO 管理员联系以启用 SSO 系统。 使用 SSO 管理工具（GUI 或命令行）可以启用 SSO。  
  
 有关详细信息，请参阅 [!INCLUDE[btsBizTalkServerNoVersion](../includes/btsbiztalkservernoversion-md.md)] 帮助中的以下资源：  
  
-   [如何启用 SSO](../core/how-to-enable-sso.md)  
  
-   [如何禁用 SSO](../core/how-to-disable-sso.md)
