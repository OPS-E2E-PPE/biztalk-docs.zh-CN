# <a name="how-to-open-the-promote-properties-dialog-box"></a>如何打开升级属性对话框
在中管理属性升级**升级属性**对话框。 本主题提供分步说明来打开此对话框中，用于管理已升级的属性。  
  
> [!NOTE]
>  当你使用**快速升级**功能的 BizTalk 编辑器中，您添加一个新**属性字段**而无需打开升级**升级属性**对话框中，尽管提升将时，会看到在该对话框中下一步被打开它。  
  
 有两种方法打开升级属性对话框。  
  
### <a name="to-open-the-promote-properties-dialog-box-from-the-schema-node"></a>若要从 Schema 节点打开升级属性对话框  
  
1.  在架构树中，选择**\<架构\>** 节点。  
  
2.  在属性窗口中，单击省略号 (**...**) 中的值字段按钮**升级属性**属性。  
  
     **升级属性**对话框随即打开。  
  
### <a name="to-open-the-promote-properties-dialog-box-from-any-node"></a>若要打开升级属性对话框中的任何节点  
  
1.  在架构树中，选择任何节点。  
  
2.  右键单击所选的节点，单击**Promote**，然后单击**显示升级**  
  
     **升级属性**对话框随即打开。
