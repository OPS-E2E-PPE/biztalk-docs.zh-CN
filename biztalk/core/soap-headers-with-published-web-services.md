# <a name="soap-headers-with-published-web-services"></a>SOAP 标头与已发布 Web Services
可以将 SOAP 标头添加到已发布 Web Services。 在 BizTalk Web 服务发布向导在**Web Services 属性**页上，你可以检查**添加 SOAP 标头**框以将 SOAP 标头添加到已发布的 Web 服务。
