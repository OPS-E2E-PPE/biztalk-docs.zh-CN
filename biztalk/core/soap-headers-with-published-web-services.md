# <a name="soap-headers-with-published-web-services"></a>SOAP 标头与已发布的 Web 服务
您可以向已发布 Web services 添加 SOAP 标头。 在 BizTalk Web Services 发布向导在**Web 服务属性**页上，可以检查**添加 SOAP 标头**框可将 SOAP 标头添加到已发布 Web services。
