# <a name="error---failed-write-xslt-file"></a>错误-失败的写入 XSLT 文件
**错误代码**  
  
 btm1056  
  
 **说明**  
  
 尝试将此映射生成的 XSLT 写入到所指示的文件时出现此错误。  
  
 **用户执行任何操作**  
  
 根据所指示的错误，采取相应的措施清除错误源。 例如，释放更多的硬盘空间。
