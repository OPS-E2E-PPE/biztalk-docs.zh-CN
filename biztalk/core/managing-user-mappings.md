# <a name="managing-user-mappings"></a>管理用户映射
本部分提供有关如何创建和配置企业单一登录映射的信息。  
  
 管理员使用 ssomanage 实用程序来管理映射，而应用程序用户使用 ssoclient 实用程序来管理它们的映射。  
  
> [!NOTE]
>  应用程序管理员可以管理他们是管理员的关联应用程序与关联的映射，而 SSO 关联管理员和 SSO 管理员可以管理关联应用程序与所有关联的所有映射。
