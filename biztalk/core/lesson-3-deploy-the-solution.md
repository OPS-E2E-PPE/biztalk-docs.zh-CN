# <a name="lesson-3-deploy-the-solution"></a>第 3 课：部署解决方案
部署 EAISolution 的第一步是将程序集添加到 BizTalk 管理数据库和全局程序集缓存。  
  
 第二步是配置并启动应用程序。  
  
 在 [第 2 课：定义业务流程](../core/lesson-2-define-the-business-process.md)，您向 EAIProcess 业务流程添加了逻辑端口。 在本课程中，定义物理端口，并将物理端口绑定到逻辑端口。 此外必须分配业务流程、 接收端口和发送端口到主机。  主机是 BizTalk 项目的虚拟容器。  每个主机都已指定主机实例来处理项目。  
  
 在本课程中，您学习如何管理[!INCLUDE[btsBizTalkServerNoVersion](../includes/btsbiztalkservernoversion-md.md)]项目通过使用 BizTalk Server 管理控制台。 有关使用 BizTalk Server 管理控制台的信息，请参阅[使用 BizTalk Server 管理控制台](../core/using-the-biztalk-server-administration-console.md)。
