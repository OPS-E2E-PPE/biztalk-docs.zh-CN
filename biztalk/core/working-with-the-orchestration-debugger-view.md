# <a name="working-with-the-orchestration-debugger-view"></a>使用业务流程调试器视图
通过右键单击结果列表中的单元格，然后选择“业务流程调试器”，可以访问“业务流程调试器”视图。  
  
 业务流程调试器可显示单个实例的技术性处理活动。 该视图可显示业务流程的呈现图像。  
  
 从“业务流程调试器”视图，您可以转到“消息流”视图。
