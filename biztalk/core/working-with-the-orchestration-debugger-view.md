# <a name="working-with-the-orchestration-debugger-view"></a>使用业务流程调试器视图
可以通过右键单击结果列表中的单元格并选择业务流程调试器来访问业务流程调试器视图。  
  
 业务流程调试器显示单个实例的技术性处理活动。 该视图显示业务流程的呈现的图像。  
  
 从业务流程调试器视图中，你可以转到消息流视图。
