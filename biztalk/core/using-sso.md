# <a name="using-sso"></a>使用 SSO
可以使用 MMC 管理单元或命令行管理实用工具 (ssomanage) 来管理 SSO 系统。 这包括活动，例如更新 SSO 数据库，添加、 删除和管理应用程序，以及管理用户映射。  
  
> [!NOTE]
>  在系统上支持用户帐户控制 (UAC)，可能需要使用管理权限运行 SSO 命令行工具。  
  
 只有单一登录管理员才可以执行这些任务。
