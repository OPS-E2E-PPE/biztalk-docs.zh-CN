# <a name="nesting-orchestrations"></a>嵌套业务流程
你可以使用**调用 Orchestration**形状和**启动 Orchestration**形状来调用从另一个业务流程。 而且，可以对业务流程进行任意深度的嵌套：例如，被调用的业务流程可以调用第三个业务流程，而第三个业务流程还可以调用第四个业务流程，以此类推。
