# <a name="nesting-orchestrations"></a>嵌套业务流程
可以使用**调用业务流程**形状和**启动业务流程**形状来调用从另一个业务流程。 可以将业务流程进行任意深度的嵌套： 例如，调用的业务流程可以调用第三个业务流程，其可以调用第四个，依次类推。
