# <a name="nodes-that-control-instance-message-structure-and-variations"></a>控制实例消息结构和变体的节点
一些用于创建在 BizTalk 编辑器中的架构的节点类型控制结构和实例消息中的变体。 您使用**序列组**节点指定一系列元素必须出现在实例消息中的相应位置特定的顺序。 您使用**选择组**节点指定集合中的元素的一个元素可以出现在实例消息中的相应位置。 您使用**全部组**节点指定所有指定的元素可以在实例消息中的相应位置出现在任何顺序，但仅出现一次。 **\<等效\>** 节点及其子节点显示在架构树中以指示其中基于派生的多态性是起作用，允许一种许多相关复杂数据类型出现在实例消息中的位置实例消息中的相应位置。  
  
 本部分的其余部分提供有关此类节点的其他信息。
