# <a name="create-your-application"></a>创建你的应用程序
本部分包含的开发人员想要创建 BizTalk 服务器应用程序并对博士 Edwards OneWorld 使用 BizTalk 适配器的信息。 一旦创建了这些应用程序，然后你[更新 jdearglist 文件并添加异常处理](using-biztalk-server-exception-handling1.md)对你的业务流程。
  
> [!NOTE]
>  对于 Microsoft Windows 平台上安装的 JD Edward 系统，必须使用 BizTalk Server 文件适配器，对于 IBM AS400 系统上安装的系统，则使用 JD Edwards 接收。
