# <a name="create-your-application"></a>创建你的应用程序
本部分包含的开发人员想要创建 BizTalk Server 应用程序和使用适用于 JD Edwards OneWorld 的 BizTalk 适配器的信息。 一旦创建了这些应用程序，然后您[更新 jdearglist 文件并添加异常处理](using-biztalk-server-exception-handling1.md)到您的业务流程。
  
> [!NOTE]
>  必须将 BizTalk Server 文件适配器用于在 Microsoft Windows 平台上安装的 JD Edward 系统和 JD Edwards 接收对于 IBM AS400 系统安装的系统。
