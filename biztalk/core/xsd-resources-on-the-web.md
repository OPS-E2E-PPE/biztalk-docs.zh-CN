# <a name="xsd-resources-on-the-web"></a>Web 上的 XSD 资源
提供的 XML 架构定义 (XSD) 的详细信息语言，或甚至总结 XSD，是超出了本文的讨论范围。 有关更多详细信息，直接转到 XSD 规范和 World Wide Web 联合会 (W3C) 网站上维护的基础知识。 具体而言，以下教程建议，若要了解基础知识，而后面两个 XSD 的相关信息的权威来源的快速方法是第一个。 有关 XSD 的教程，请参阅以下 W3C 页：  
  
- XML 架构第 0 部分：入门指导， [ http://go.microsoft.com/fwlink/?LinkId=16787 ](http://go.microsoft.com/fwlink/?LinkId=16787)。  
  
- XML 架构第 1 部分：结构， [ http://go.microsoft.com/fwlink/?LinkId=16788 ](http://go.microsoft.com/fwlink/?LinkId=16788)。  
  
- XML 架构第 2 部分：数据类型， [ http://go.microsoft.com/fwlink/?LinkId=16789 ](http://go.microsoft.com/fwlink/?LinkId=16789)。  
  
  建议，必须使用 BizTalk 编辑器构造复杂架构之前深入了解的 XSD。
