# <a name="using-expressions-in-orchestrations"></a>在业务流程中使用表达式
BizTalk 表达式编辑器可用于输入要添加逻辑来处理和测试你的业务流程变量和消息的值的 XLANG/s 表达式。 但是，它不是使用它来执行高级业务流程逻辑，最好是在业务流程设计图本身中可见的好办法。 为业务过程透明度和重新配置的轻松，我们建议使用简单的模块化表达式。
