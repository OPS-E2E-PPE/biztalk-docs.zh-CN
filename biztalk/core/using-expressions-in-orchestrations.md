# <a name="using-expressions-in-orchestrations"></a>在业务流程中使用表达式
BizTalk 表达式编辑器可用于输入 XLANG/s 表达式，以添加逻辑来操作和测试你的业务流程变量和消息的值。 但是，它不是作为最佳做法，使用它来执行高级业务流程逻辑，最好是将是在绘制自身的业务流程中可见。 对业务进程透明度便捷的重新配置，我们建议你使用简单和模块化表达式。
