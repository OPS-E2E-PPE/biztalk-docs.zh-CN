# <a name="sendportref-port-node"></a>SendPortRef（端口节点）
绑定文件“端口”节点的 SendPortRef 节点包含有关服务引用的发送端口的信息。  
  
## <a name="nodes-in-the-sendportref-node"></a>SendPortRef 节点中的节点  
 下表列出了可为绑定文件的此节点设置的属性：  
  
|**名称**|**节点类型**|**数据类型**|**Description**|**限制**|**注释**|  
|--------------|-------------------|-------------------|---------------------|----------------------|------------------|  
|Name|Attribute|xs:string|指定服务引用的发送端口的名称。|可选|默认值：空|
