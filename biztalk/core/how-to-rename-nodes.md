# <a name="how-to-rename-nodes"></a>如何重命名节点
可能将您想要重命名现有节点。 本主题介绍在此任务上的两种变体。  
  
### <a name="to-rename-a-single-node"></a>若要重命名单个节点  
  
1.  选择**记录**节点， **Field 元素**节点，或**字段属性**你想要重命名的节点。  
  
2.  上**BizTalk**菜单上，单击**重命名**。  
  
3.  键入新名称的节点，然后按 ENTER。  
  
    > [!NOTE]
    >  也可以通过更改的值来更改节点及其**节点名称**属性。  
  
### <a name="to-rename-multiple-nodes-to-have-the-same-name"></a>若要重命名多个节点具有相同的名称  
  
1.  选择要赋予相同名称的节点集中的第一个节点。  
  
2.  按按住 ctrl 键，，然后选择要赋予相同名称的剩余节点。  
  
3.  在属性窗口中的值更改**节点名称**属性，并通过按 ENTER 完成。  
  
     所有所选的节点在架构树视图中使用新名称更新。
