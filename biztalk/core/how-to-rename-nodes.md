# <a name="how-to-rename-nodes"></a>如何重命名节点
有时，您可能想要重命名现有节点。 本主题将介绍此任务的两种不同情况。  
  
### <a name="to-rename-a-single-node"></a>重命名单个节点  
  
1.  选择**记录**节点， **Field 元素**节点，或**字段特性**你想要重命名的节点。  
  
2.  上**BizTalk**菜单上，单击**重命名**。  
  
3.  为该节点键入新名称，然后按 Enter。  
  
    > [!NOTE]
    >  也可以通过更改的值来更改节点其**节点名称**属性。  
  
### <a name="to-rename-multiple-nodes-to-have-the-same-name"></a>重命名多个节点以使其具有相同名称  
  
1.  选择要赋予相同名称的这组节点中的第一个节点。  
  
2.  按住 Ctrl，然后选择要赋予相同名称的其余节点。  
  
3.  在属性窗口中，更改值**节点名称**属性，并按 enter 键完成。  
  
     架构树视图中的所有选定节点都将用新名称更新。
