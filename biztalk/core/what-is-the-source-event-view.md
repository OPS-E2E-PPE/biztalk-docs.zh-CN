# <a name="what-is-the-source-event-view"></a>什么是源事件视图？
事件源视图是跟踪配置文件编辑器 (TPE) 提供的业务流程或消息架构的程序集或映射到活动定义的上下文属性从选择的显示。  
  
 源事件视图显示在用户界面的右窗格中。 所选数据源根据有所不同窗格的内容。  
  
## <a name="event-source-options"></a>事件源选项  
 有四个选项可用于事件源： 业务流程调度、 消息传送负载、 上下文属性和消息传送属性。  
  
 业务流程和消息传送负载要求选择要从中映射数据项的程序集。 然后从该程序集选择特定的业务流程或感兴趣的消息负载架构。 对于业务流程计划您的程序集中提供的业务流程的列表。 消息传送负载可用于从消息传送负载架构的列表中选择和上下文属性中的程序集和可公开访问的系统架构中显示的架构列表。  
  
 当你选择上下文属性时，系统将首先提供上下文属性名称的列表。 时选择上下文属性，在右窗格中显示的上下文属性相关的架构。 通过拖放到活动节点上的属性，然后可以将上下文属性映射到您的活动。  
  
 选择消息传送属性时系统将提供则可以映射到活动的已知消息传送属性的列表。
