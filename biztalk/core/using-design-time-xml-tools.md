# <a name="using-design-time-xml-tools"></a>使用设计时 XML 工具
使用 EDI XML 工具，您可以对 EDI 架构和映射执行设计时操作。 在 Visual Studio 的解决方案资源管理器窗格中右键单击架构或映射时，便会从显示的菜单中执行这些工具。
