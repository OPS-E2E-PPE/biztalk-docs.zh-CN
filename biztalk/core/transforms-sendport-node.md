# <a name="transforms-sendport-node"></a>Transforms （SendPort 节点）
绑定文件的发送端口节点的 Transforms 节点包含随绑定文件一起导出的单向发送端口的出站转换的集合。  
  
## <a name="nodes-in-the-transforms-node"></a>Transforms 节点中的节点  
 下表列出了可为绑定文件的此节点设置的属性：  
  
|**名称**|**节点类型**|**数据类型**|**说明**|**限制**|**注释**|  
|--------------|-------------------|-------------------|---------------------|----------------------|------------------|  
|[Transform（“SendPort-Transforms”节点）](../core/transform-sendport-transforms-node.md)|录制|转换 (ComplexType)|指定 BizTalk Server 映射或转换，这是一个表示源架构和目标架构之间的映射项。|可选|默认值：无|
