# <a name="success---test-map"></a>成功-测试映射
**错误代码**  
  
 btm1053  
  
 **说明**  
  
 对所指示映射的测试映射操作已成功，生成了所指示的输出实例消息文件。  
  
 **用户执行任何操作**  
  
 根据需要，检查所指示的由测试映射操作生成的输出实例消息文件。
