# <a name="simple-type-derivation"></a>简单类型派生
XML 架构定义 (XSD) 语言提供了多种机制，用于定义基于的现有简单类型派生，如下所示的简单类型的变体：  
  
- **限制**。 通过使用限制机制进行简单类型派生涉及到针对该类型的可能值的限制简介。 示例包括，数值类型的最小和/或最大值或字符串类型的最小值和最大长度。  
  
- **列表**。 通过使用列表机制进行简单类型派生允许单个属性或元素值定义为包含实例消息中的特定类型的空格分隔的值的列表。  
  
- **联合**。 通过使用联合机制进行简单类型派生将允许定义为单个值的多个指定类型之一的实例消息中的单个属性或元素值。  
  
  本部分介绍更多详细信息，包括如何通过在属性窗口中设置相应的节点属性定义这些派生以及如何构造相应 XSD 这些简单类型派生。
