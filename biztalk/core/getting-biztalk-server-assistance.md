# <a name="getting-biztalk-server-assistance"></a>获取 BizTalk Server 帮助
有关有来自 Microsoft 的信息的三个主要来源[!INCLUDE[btsBizTalkServerNoVersion](../includes/btsbiztalkservernoversion-md.md)]:  
  
- 文档、 教程和示例随[!INCLUDE[btsBizTalkServerNoVersion](../includes/btsbiztalkservernoversion-md.md)]。  
  
- [!INCLUDE[btsBizTalkServerNoVersion](../includes/btsbiztalkservernoversion-md.md)] Microsoft Developer Network (MSDN) 和 TechNet 上的站点。

## <a name="biztalk-server-documentation-tutorials-and-samples"></a>BizTalk Server 文档、 教程和示例  
 在安装 BizTalk Server 时，可以安装 SDK、 示例应用程序和教程。 核心文档、 示例和教程涵盖的概念和有效地使用 BizTalk Server 所需的过程。 BizTalk Server 帮助还提供使用 BizTalk Server 生成应用程序的编程接口和语言的参考资料。  
  
 如果在安装 SDK 和教程是使用[!INCLUDE[btsBizTalkServerNoVersion](../includes/btsbiztalkservernoversion-md.md)]安装程序，您可以访问它们通过两种方式：  
  
- 通过[!INCLUDE[btsBizTalkServerNoVersion_md](../includes/btsbiztalkservernoversion-md.md)]开始菜单上的程序组。  
  
- 通过 BizTalk Server 工具和实用程序介质中你[!INCLUDE[btsBizTalkServerNoVersion_md](../includes/btsbiztalkservernoversion-md.md)]安装文件夹。  
  
  [!INCLUDE[btsBizTalkServerNoVersion](../includes/btsbiztalkservernoversion-md.md)] 在 web 上提供了文档。
