# <a name="how-to-use-the-role-link-shape"></a>如何使用角色链接形状
A**角色链接**形状包含实现角色或提供程序角色的使用角色或使用者角色的占位符。 它既可以包含其中一种角色的占位符，也可以各含一个。  
  
 你可以直接添加端口类型**角色链接**形状，使用现有角色或新角色，并且现有或新的端口类型。  
  
> [!NOTE]
>  建议您在创建角色链接前，先创建角色链接类型和相关端口类型，以便可以使用现有角色链接类型来定义角色链接。 有关详细信息，请参阅[如何创建用户角色在业务流程中的链接](../core/how-to-create-role-links-in-orchestrations.md)。  
  
### <a name="to-add-a-role-link"></a>添加角色链接  
  
-   右击端口图面，然后单击**新角色链接**。  
  
     -或者-  
  
     从**BizTalk 业务流程**选项卡的工具箱拖**角色链接**到端口图面上的形状。  
  
### <a name="to-remove-a-role-link"></a>删除角色链接  
  
-   右键单击**角色链接**以删除，然后单击**删除**。
