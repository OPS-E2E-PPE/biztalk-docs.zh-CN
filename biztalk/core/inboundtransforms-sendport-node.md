# <a name="inboundtransforms-sendport-node"></a>InboundTransforms（“发送端口”节点）
绑定文件的发送端口节点的 InboundTransforms 节点包含的与绑定文件导出双向发送端口的入站转换的集合。  
  
## <a name="nodes-in-the-inboundtransforms-node"></a>InboundTransforms 节点中的节点  
 下表列出了可为绑定文件的此节点设置的属性：  
  
|**名称**|**节点类型**|**数据类型**|**Description**|**限制**|**注释**|  
|--------------|-------------------|-------------------|---------------------|----------------------|------------------|  
|[转换 （InboundTransforms 节点）](../core/transform-inboundtransforms-node.md)|录制|转换 (ComplexType)|指定 BizTalk Server 映射或转换，这是一个表示源架构和目标架构之间的映射项。|可选|默认值：无|
