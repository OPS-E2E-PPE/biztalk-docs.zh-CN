# <a name="how-to-set-jd-edwards-oneworld-pipelines"></a>如何设置 JD Edwards OneWorld 管道
用于 JD Edwards OneWorld 的 Microsoft BizTalk 适配器要求你选择 XMLTransmit 和 XMLReceive 的发送和接收管道，分别。  
  
### <a name="to-set-pipelines"></a>若要设置管道  
  
1.  上**启动**菜单，依次指向**Program Files**，指向**Microsoft BizTalk Server** ，然后单击**BizTalk Server 管理**若要启动 BizTalk Server 管理控制台。  
  
2.  展开 BizTalk Server 管理，展开**BizTalk 组**，展开**应用程序**，然后展开所需的应用程序。  
  
3.  选择**发送端口**。 在详细信息窗格中，右键单击所选的发送端口，单击**属性**。  
  
4.  在中**发送端口属性**对话框框中，执行以下操作：  
  
    1.  选择发送管道从**发送管道**下拉列表。  
  
    2.  选择从接收管道**接收管道**下拉列表。  
  
5.  单击“确定” 。
