# <a name="how-to-add-or-remove-messages-in-orchestrations"></a>如何在添加或删除消息业务流程
一条消息是通信的一个业务流程中的基本单位。 它包含一个或多个部件以及描述消息和其内容的属性的上下文数据。  
  
### <a name="to-add-a-message"></a>添加消息  
  
1.  在业务流程视图窗口中，右键单击**消息**，然后单击**新消息**。 **消息**文件夹扩展，如果折叠状态，并添加一个新消息。  
  
2.  名称的消息。  
  
3.  指定消息类型。  
  
### <a name="to-remove-a-message"></a>若要删除一条消息  
  
-   在业务流程视图窗口中，右键单击你想要删除，然后单击的邮件**删除**。
