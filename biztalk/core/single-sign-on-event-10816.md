# <a name="single-sign-on-event-10816"></a>单一登录：事件 10816
## <a name="details"></a>详细信息  
  
|                 |                                                               |
|-----------------|---------------------------------------------------------------|
|  产品名称   |                   企业单一登录                   |
| 产品版本 |  [!INCLUDE[btsSSOVersion](../includes/btsssoversion-md.md)]   |
|    事件 ID     |                             10816                             |
|  事件源   |                            ENTSSO                             |
|    组件    |                              不可用                              |
|  符号名称  |          ENTSSO_E_EXTERNAL_UPDATE_FAILED_MORE_RECENT          |
|  消息正文   | 更新 SSO 数据库中的外部凭据。 |
  
## <a name="explanation"></a>解释  
 过期的密码已收到来自外部系统，并且将被忽略。  
  
## <a name="user-action"></a>用户操作  
 不不需要任何用户操作。
