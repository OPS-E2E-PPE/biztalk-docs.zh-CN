# <a name="how-to-configure-a-pop3-receive-handler"></a>如何配置 POP3 接收处理程序
使用以下过程来更改与 POP3 相关联的主机接收处理程序。  
  
### <a name="to-configure-the-general-properties-for-a-pop3-receive-handler"></a>若要配置的常规属性 pop3 接收处理程序  
  
1.  在 BizTalk Server 管理控制台中，展开**BizTalk Server 管理**，展开**BizTalk 组**，展开**平台设置**，然后展开**适配器**。  
  
2.  在展开的适配器列表中，单击**POP3**，在右窗格中，右键单击你想要配置，接收处理程序，再单击**属性**。  
  
3.  在中**适配器处理程序属性**对话框中，在**常规**选项卡上，在**主机名**列表中，选择接收处理程序将关联的主机。  
  
4.  单击“确定” 。
