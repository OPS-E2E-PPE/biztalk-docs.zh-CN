# <a name="error---failed-write-extension-xml-file"></a>错误-失败的写入扩展 XML 文件
**错误代码**  
  
 btm1058  
  
 **说明**  
  
 尝试将此映射生成的扩展对象 XML 写入所指示的文件时出现此错误。  
  
 **用户执行任何操作**  
  
 根据所指示的错误，采取相应的措施清除错误源。 例如，释放更多的硬盘空间。
