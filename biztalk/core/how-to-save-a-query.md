# <a name="how-to-save-a-query"></a>如何保存查询
您可保存查询以备将来重用。 默认情况下，已保存的查询存储在**我的文档**创建并保存它们的用户的文件夹。  
  
## <a name="prerequisites"></a>必要條件  
 若要执行此过程，则必须以 BizTalk Server Operators 组成员的身份登录。  
  
### <a name="to-save-a-query"></a>保存查询  
  
1. 单击**启动**，单击**所有程序**，单击[!INCLUDE[btsBizTalkServerStartMenuItemui](../includes/btsbiztalkserverstartmenuitemui-md.md)]，然后单击**BizTalk Server 管理**。  
  
2. 在控制台树中，展开“[!INCLUDE[btsBizTalkServerAdminConsoleui](../includes/btsbiztalkserveradminconsoleui-md.md)]”，然后单击“BizTalk 组”。  
  
3. 在详细信息窗格中，单击**新查询**选项卡。  
  
4. 创建根据需要，一个查询，然后单击**另存为**。  
  
5. 在中**另存为**对话框框中，浏览到或创建你想要保存查询的文件夹。  
  
6. 在中**文件名**，为该查询中，键入一个名称，然后单击**保存**。
