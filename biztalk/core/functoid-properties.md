# <a name="functoid-properties"></a>Functoid 属性

## <a name="overview"></a>概述
Functoid 也具有属性可用于检查和修改在[!INCLUDE[btsVStudioNoVersion](../includes/btsvstudionoversion-md.md)]属性窗口。 某些 functoid 属性仅对特定 functoid 可用，并且对于其他 functoid 禁用。 一个 functoid 属性**输入参数**、 对几乎所有 functoid 尤其重要，可能将最常使用。 有关详细信息**输入参数**属性，请参阅[Functoid 输入参数](../core/functoid-input-parameters.md)。  
  
 所有 functoid 属性都分为**常规**属性。 有关 functoid 属性的完整列表，请参阅**常规 Functoid 属性** [!INCLUDE[ui-guidance-developers-reference](../includes/ui-guidance-developers-reference.md)]。
