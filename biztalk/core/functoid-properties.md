# <a name="functoid-properties"></a>Functoid 属性

## <a name="overview"></a>概述
Functoid 具有可在 [!INCLUDE[btsVStudioNoVersion](../includes/btsvstudionoversion-md.md)] 的“属性”窗口中进行检查和修改的属性。 某些 functoid 属性仅对特定 functoid 可用，而对于其他 functoid 则为禁用。 一个 functoid 属性，**输入参数**的几乎所有 functoid 尤为重要，可能将最常使用。 有关详细信息**输入参数**属性，请参阅[Functoid 的输入参数](../core/functoid-input-parameters.md)。  
  
 所有 functoid 属性都归类为**常规**属性。 Functoid 属性的完整列表，请参阅**常规 Functoid 属性** [!INCLUDE[ui-guidance-developers-reference](../includes/ui-guidance-developers-reference.md)]。
