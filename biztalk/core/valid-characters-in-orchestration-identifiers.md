# <a name="valid-characters-in-orchestration-identifiers"></a>业务流程标识符中的有效字符
业务流程标识符（如项名称、变量名称和类型名称）具有一些限制：  
  
-   任何标识符的第一个字符必须是字母或下划线字符 (_)。  
  
-   所有后续字符都必须是字母、数字、连接字符、组合字符或者格式化字符。  
  
-   大多数标点字符都不能用于业务流程标识符中。
