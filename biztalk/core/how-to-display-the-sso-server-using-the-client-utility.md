# <a name="how-to-display-the-sso-server-using-the-client-utility"></a>如何显示 SSO 服务器使用客户端实用工具
使用此命令显示用户当前指向的单一登录服务器。  
  
### <a name="to-display-the-sso-server-using-the-client-utility"></a>若要显示 SSO 服务器使用客户端实用工具  
  
1.  上**启动**菜单上，单击**运行**，然后键入**cmd**。  
  
2.  在命令行中，转至企业单一登录安装目录。 默认安装目录*\<驱动器\>*: \Program Files\Common Files\Enterprise Single Sign-on。  
  
3.  类型**ssoclient – showserver**。  
  
    > [!NOTE]
    >  在支持用户帐户控制 (UAC) 的系统上，可能需要具有管理权限才能运行该工具。
