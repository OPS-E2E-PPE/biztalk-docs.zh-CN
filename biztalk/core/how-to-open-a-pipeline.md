# <a name="how-to-open-a-pipeline"></a>如何打开管道
打开管道文件会导致管道设计器在设计区域中具有定义的管道和其阶段出现。 此外，如果它们尚不存在打开管道设计器工具箱中，将显示属性窗口。  
  
### <a name="to-open-a-pipeline"></a>若要打开管道  
  
1.  在解决方案资源管理器中双击你想要打开的管道。  
  
     -或-  
  
2.  在解决方案资源管理器中选择管道中，然后在**视图**菜单上，单击**打开**。
