# <a name="how-to-open-a-pipeline"></a>如何打开管道
打开管道文件将显示管道设计器，并在设计区域中显示已定义的管道及其阶段。 此外，还将显示“管道设计器工具箱”和“属性”窗口（如果这些窗口尚未打开）。  
  
### <a name="to-open-a-pipeline"></a>打开管道  
  
1.  在解决方案资源管理器中，双击要打开的管道。  
  
     -或者-  
  
2.  在解决方案资源管理器，选择管道，然后在**视图**菜单上，单击**打开**。
