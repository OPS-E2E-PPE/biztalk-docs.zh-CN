# <a name="understanding-tracked-data"></a>了解跟踪数据
运行时跟踪查询，被跟踪的信息将显示在查询结果窗口底部的结果列表中。  
  
## <a name="viewing-message-details"></a>查看消息详细信息  
 你可以跟踪消息属性。 此外可以将跟踪的消息正文保存到文件，并查看它们使用非 Microsoft 工具。  
  
-   您可以右键单击服务实例引用的任何消息，并选择消息详细信息。  
  
-   如果消息已处理但被跟踪，因为启用了跟踪功能，可以将其保存到硬盘并对其进行检查。  
  
-   可以将附加到业务流程实例，并使用业务流程调试器。  
  
## <a name="viewing-service-events"></a>查看服务事件  
 当挂起的服务出现在事件日志时，可以通过查看服务**消息流**，**业务流程调试器**，**显示跟踪的消息事件**，**显示实时服务实例**，从选项**上下文**菜单。  
  
## <a name="viewing-orchestration-events"></a>查看业务流程事件  
 使用业务流程调试器可以查看消息实例的路径已通过业务流程。 单步执行，该业务流程的呈现的图像显示进度消息，并允许您将断点放置在业务流程中以便进行调试。
