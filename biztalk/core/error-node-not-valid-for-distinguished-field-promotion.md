# <a name="error---node-not-valid-for-distinguished-field-promotion"></a>错误-节点不适于可分辨的字段升级
**错误代码**  
  
 BEC2014  
  
 **说明**  
  
 所指示的节点不能用作可分辨字段升级，因为它不能包含必需的文本内容。 仅**Field 元素**节点，**字段特性**节点，或**记录**具有混合或简单内容的节点可以提升作为可分辨的字段。  
  
 **用户执行任何操作**  
  
 使用**可分辨字段**选项卡**升级属性**对话框删除不允许的属性提升。
