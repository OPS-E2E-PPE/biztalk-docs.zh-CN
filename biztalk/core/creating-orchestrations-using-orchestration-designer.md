# <a name="creating-orchestrations-using-orchestration-designer"></a>创建业务流程使用业务流程设计器
业务流程中的[!INCLUDE[btsBizTalkServerNoVersion](../includes/btsbiztalkservernoversion-md.md)]工作流  
  
 ![在 BizTalk Server 中的业务流程](../core/media/ebiz-dev-busprcsf.gif "ebiz_dev_busprcsf")  
  
 BizTalk 业务流程设计器是一个工具用于方便地创建您的业务流程，会自动反映在基础代码和生成到可执行模块的可视表示形式。 它提供了各种对应于你可能想要执行的不同操作的形状。  
  
 本部分提供的概念性信息、 参考信息和任务相关的信息可帮助你了解和使用 BizTalk 业务流程和 BizTalk 业务流程设计器。  
  
 有关使用键盘快捷键的业务流程设计器的信息，请参阅[业务流程设计器键盘快捷方式](../core/orchestration-designer-keyboard-shortcuts.md)。
