# <a name="how-to-set-pipeline-component-properties"></a>如何设置管道组件属性
每个组件都具有应在使用前进行配置的属性。  
  
### <a name="to-set-pipeline-component-properties"></a>设置管道组件属性  
  
-   在属性窗口中，在**管道组件属性**部分中，单击**值**中的单元格**属性**行，然后输入新值。 如果**值**单元格包含箭头按钮，单击选项列表的按钮。 如果**值**单元格包含省略号 (**...**) 按钮，单击使用架构集合属性编辑器来配置属性按钮。 有关详细信息，请参阅[如何使用架构集合属性编辑器](../core/how-to-use-the-schema-collection-property-editor.md)。
