# <a name="how-to-set-pipeline-component-properties"></a>如何设置管道组件属性
每个组件的应在配置的之前，若要使用的属性。  
  
### <a name="to-set-pipeline-component-properties"></a>若要设置管道组件属性  
  
-   在属性窗口中**管道组件属性**部分中，单击**值**中的单元格**属性**行，并输入新值。 如果**值**单元格包含一个箭头按钮，单击选择列表的按钮。 如果**值**单元格包含一个省略号 (**...**) 按钮，再单击该按钮以使用架构集合属性编辑器来配置属性。 有关详细信息，请参阅[如何使用架构集合属性编辑器](../core/how-to-use-the-schema-collection-property-editor.md)。
