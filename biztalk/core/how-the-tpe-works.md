# <a name="how-the-tpe-works"></a>TPE 的工作原理
若要解决特定客户的订单或分析特定的业务事件和传送过程中的里程碑的趋势，例如，业务分析员可以指定他想要查看的数据的视图。 若要将数据提取业务分析员所需，开发人员使用 TPE 来执行特定的业务事件数据和实际的业务流程之间的映射。 创建、 保存、 测试和部署跟踪配置文件后，业务分析员可以检索收集要分析的业务流程特定方面的数据。
