# <a name="how-the-tpe-works"></a>键入的工作原理
若要解决特定的客户订单或分析特定业务事件和在传送过程中的里程碑的趋势，例如，业务分析师可以指定他想要查看数据的视图。 提取数据的业务分析师的需要，请开发人员使用键入要执行的特定业务事件数据和实际的业务流程之间的映射。 创建、 保存、 测试和部署的跟踪配置文件后，业务分析师可以检索收集分析的业务流程的特定方面的数据。
