# <a name="how-to-add-and-remove-correlation-sets"></a>如何添加和删除相关集
可以使用业务流程视图窗口添加到相关集或从您的业务流程中删除它们。 相关集是相关类型的实例。  
  
### <a name="to-add-and-configure-a-correlation-set"></a>若要添加和配置相关集  
  
-   在业务流程视图窗口中，右键单击**相关集**，然后单击**新建相关集**。  
  
     **相关性属性**对话框随即出现。 添加想要包括在相关集中的属性。 相关集，将创建相应的相关类型。  
  
### <a name="to-remove-a-correlation-set"></a>若要删除相关集  
  
-   在业务流程视图窗口中，右键单击你想要删除，然后单击相关集**删除**。
