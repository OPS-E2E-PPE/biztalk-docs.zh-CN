# <a name="how-to-add-and-remove-correlation-sets"></a>如何添加和删除关联集
可以使用“业务流程视图”窗口向业务流程添加相关集或从中删除相关集。 相关集是相关类型的实例。  
  
### <a name="to-add-and-configure-a-correlation-set"></a>添加和配置相关集  
  
-   在业务流程视图窗口中，右键单击**关联集**，然后单击**新关联集**。  
  
     **相关性属性**对话框随即出现。 添加要包含在相关集中的属性。 将创建相关集和相应的相关类型。  
  
### <a name="to-remove-a-correlation-set"></a>删除相关集  
  
-   在业务流程视图窗口中，右键单击你想要删除，然后单击相关集**删除**。
