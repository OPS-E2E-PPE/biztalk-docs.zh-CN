# <a name="how-to-delete-an-adapter-handler"></a>如何删除一个适配器处理程序
您可以使用 BizTalk Server 管理控制台来删除发送或接收适配器处理程序。  
  
 在删除适配器处理程序之前，您必须删除与该程序相关联的所有接收位置或发送端口。  
  
> [!NOTE]
>  您必须是 SSO Administration 组的成员才能删除适配器处理程序。  
  
### <a name="to-delete-an-adapter-handler"></a>删除适配器处理程序  
  
1.  在 BizTalk Server 管理控制台中，展开**BizTalk Server 管理**，展开**BizTalk 组**，展开**平台设置**，然后展开**适配器**。  
  
2.  在展开的适配器列表中，选择要删除其处理程序的适配器。  
  
3.  右键单击你想要删除，然后单击此适配器处理程序**删除**。  
  
4.  单击**是**以确认你想要删除此适配器处理程序。  
  
5.  单击 **“确定”**。
