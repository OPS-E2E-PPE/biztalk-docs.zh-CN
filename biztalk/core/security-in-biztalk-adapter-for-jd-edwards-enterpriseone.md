# <a name="security-in-jd-edwards-enterpriseone"></a>JD Edwards EnterpriseOne 中的安全性

## <a name="overview"></a>概述

用于 JD Edwards EnterpriseOne 的 Microsoft BizTalk 适配器提供单一登录 (SSO) 支持。 由企业单一登录工具创建的关联应用程序代表诸如 JD Edwards EnterpriseOne 之类的服务器系统。 

本部分提供有关在安全环境中部署用于 JD Edwards EnterpriseOne 的 Microsoft BizTalk Server 适配器指导原则。  
  
> [!CAUTION]
>  我们建议将用于 JD Edwards EnterpriseOne 的 BizTalk 适配器使用限制给已授权用户，如客户端文件直接连接到业务线应用程序。
