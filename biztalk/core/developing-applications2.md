# <a name="create-your-application"></a>创建你的应用程序
本部分包含开发人员希望创建 BizTalk Server 应用程序和与 JD Edwards EnterpriseOne 的 BizTalk 适配器一起使用的信息。 一旦创建了这些应用程序，然后就可以使用[添加异常处理](../core/using-biztalk-server-exception-handling3.md)到您的业务流程。 
  
> [!NOTE]
>  对于 Microsoft Windows 平台上安装的 JD Edward 系统，必须使用 BizTalk Server 文件适配器，对于 IBM AS400 系统上安装的系统，则使用 JD Edwards 接收。
