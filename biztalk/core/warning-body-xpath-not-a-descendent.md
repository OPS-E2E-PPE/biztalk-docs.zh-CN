# <a name="warning---body-xpath-not-a-descendent"></a>警告-正文 XPath 不是节点后代
**错误代码**  
  
 BEC1004  
  
 **说明**  
  
 **正文 XPath**此信封架构中指示的根节点的属性引用不是该根节点，所需的子代的节点。 此错误不应发生，除非**正文 XPath**属性 BizTalk 编辑器之外修改。  
  
 **用户执行任何操作**  
  
 选择指示的根节点，然后选择的值**正文 XPath**正确地标识相关联的消息正文的顶级节点的属性。
