# <a name="biztalk-mapper-keyboard-shortcuts"></a>BizTalk 映射器键盘快捷方式
与所有标准 Microsoft Windows 应用程序，您可以使用键盘快捷方式在 BizTalk 映射器来访问命令可使用菜单。 下表描述了可用的键盘快捷方式。  
  
> [!NOTE]
>  以逗号分隔的组合键表示了必须键入的序列。 例如，"CTRL + I，CTRL + R"表示必须先按 CTRL + I，然后必须键入 CTRL + R。 如"ALT + B，I，F"组合键意味着您必须先按 ALT + B，但然后可以键入"I"和"F"需松开 ALT 键。  
  
> [!IMPORTANT]
>  键盘快捷键正常工作，必须选择所需的项和焦点。
