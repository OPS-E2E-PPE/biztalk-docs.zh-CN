# <a name="biztalk-mapper-keyboard-shortcuts"></a>BizTalk 映射器键盘快捷键
与在所有标准 Microsoft Windows 应用程序中一样，您可以在 BizTalk 映射器中使用键盘快捷键来访问可通过菜单访问的命令。 下表对可用的键盘快捷键进行了说明：  
  
> [!NOTE]
>  以逗号分隔的组合键表示了必须遵循的按键顺序。 例如，"CTRL + I，CTRL + R"意味着你必须首先键入 CTRL + I，然后你必须键入 CTRL + R。 如"ALT + B，I，F"键组合意味着你必须首先键入 ALT + B，然后可以键入"I"和"F"没有按住 ALT 键。  
  
> [!IMPORTANT]
>  若要使键盘快捷键正常工作，必须选择并聚焦所需的项。
