# <a name="how-to-delete-functoids"></a>如何删除 Functoid
建立映射时，有时需要删除现有的 functoid。 删除 functoid 也会删除关联的输入和输出链接，以及任何其他输入的参数、 脚本规范和表网格配置。  
  
### <a name="to-delete-an-existing-functoid"></a>若要删除现有的 functoid  
  
1.  显示包含你想要删除的 functoid 的网格页。 有关使用网格页，包括网格页之间切换的详细信息请参阅[使用网格页](../core/working-with-grid-pages.md)。  
  
2.  选择你想要通过单击其上删除的 functoid。  
  
     可以选择多个 functoid，然后将其删除一次。 有关批量选择 functoid 和链接的详细信息，请参阅[如何选择多个链接和 Functoid](../core/how-to-select-multiple-links-and-functoids.md)。  
  
3.  上**编辑**菜单上，单击**删除**。  
  
    > [!NOTE]
    >  此外可以按 DELETE 键，或右键单击相关网格页中选定的 functoid，单击**删除**从上下文菜单。  
  
    > [!NOTE]
    >  您不能撤消 functoid 或链接删除。 有关如何撤消或重做用户操作的详细信息，请参阅[如何撤消或重做用户操作](../core/how-to-undo-or-redo-user-operations.md)。
