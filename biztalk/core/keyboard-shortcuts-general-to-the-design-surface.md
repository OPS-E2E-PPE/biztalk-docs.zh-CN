# <a name="keyboard-shortcuts-general-to-the-design-surface"></a>键盘快捷方式设计面通用的
下表描述的流程区域和端口图面区域的设计图面通用的键盘导航。  
  
|Key|效果|  
|---------|------------|  
|Tab|选择内容和焦点将移到与下一步的设计图面区域中的非活动选择形状在右侧 （即，过程区域或端口图面）。<br /><br /> 例如：选择内容和焦点目前在左侧端口图面中的端口上。 按下 TAB 键时，所选内容和焦点将移到在流程区域中具有非活动状态的焦点的形状中。|  
|按 SHIFT + TAB|与相同选项卡上，但方向相反。|  
|Page Down|向下滚动页面，以便显示更高版本的业务流程的部分。 所选内容和焦点不会更改。|  
|Page Up|相同 PAGE DOWN 键，但在相反的方向。 所选内容和焦点不会更改。|  
|SHIFT + 向下翻页|水平滚动整个设计图面的左侧的集为增量。 所选内容和焦点不会更改。|  
|SHIFT + PAGE UP|SHIFT + PAGE 相同不同之处在于向下设计面滚动到右侧。 所选内容和焦点不会更改。|  
|Enter|所选内容，这通常意味着转换到属性网格焦点执行默认操作。|  
|SPACE|选择当前具有焦点的形状。 如果焦点位于形状连接线，则不起作用。|  
|CTRL + SHIFT + F1|展开和折叠所选形状的智能标记之间切换。 向上和向下箭头可以然后使用智能标记的项之间移动。 展开后，Enter 或 Space 键执行相关的操作的智能标记项 （以及折叠智能标记）。|  
|DELETE|删除当前所选内容。|  
|CTRL + 向上 MWHEEL|显示出视图的设计图面上的缩放。|  
|CTRL + 向下 MWHEEL|显示缩放设计图面鉴定。|
