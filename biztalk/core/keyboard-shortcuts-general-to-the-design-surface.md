# <a name="keyboard-shortcuts-general-to-the-design-surface"></a>键盘快捷方式常规到设计图面
下表介绍一般过程区域和端口图面区域的设计图面键盘导航。  
  
|Key|效果|  
|---------|------------|  
|Tab|选择和焦点将移动到下一步的设计图面区域中的非活动选择形状右侧 （即，过程区域或端口图面）。<br /><br /> 示例： 选择和焦点位于当前左端口图面中的端口。 按下 TAB 键时，选择与焦点将移动到过程区域中有非活动状态的焦点的形状中。|  
|按 SHIFT + TAB|与相同选项卡上，但从相反的方向。|  
|Page Down|向下滚动页面，以便显示更高版本的业务流程的部分。 选择和焦点不会更改。|  
|Page Up|相同 PAGE DOWN 键，但在相反方向。 选择和焦点不会更改。|  
|SHIFT + PAGE DOWN|水平滚动到按集增量左整个设计图面。 选择和焦点不会更改。|  
|SHIFT + PAGE UP|SHIFT + PAGE 相同向下只不过设计曲面滚动到右侧。 选择和焦点不会更改。|  
|Enter|针对所选内容，这通常意味着转换到属性网格焦点执行默认操作。|  
|SPACE|选择当前具有焦点的形状。 如果焦点在形状连接器上，没有任何影响。|  
|CTRL + SHIFT + F1|展开和折叠的智能标记所选形状之间切换。 向上和向下箭头可以然后使用智能标记的项之间移动。 展开后，Enter 或空间密钥执行智能标记项 （以及折叠智能标记） 的相关的的操作。|  
|DELETE|删除当前所选内容。|  
|CTRL + 向上 MWHEEL|显示扩展设计图面的视图的缩放。|  
|CTRL + 向下 MWHEEL|显示缩放根据到目前设计图面。|
