# <a name="running-sso-projects"></a>运行 SSO 项目
您可以从 Internet Explorer 运行示例单一登录 (SSO) 项目。  
  
## <a name="run-a-sample-from-a-web-browser"></a>从 web 浏览器中运行示例
  
1.  打开浏览器。  
  
2.  请转到以下 URL：

     在这种情况下，不需要提供凭据。
