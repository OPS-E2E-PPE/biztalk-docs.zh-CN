# <a name="building-and-running-orchestrations"></a>生成和运行业务流程
设计您的业务流程后，可以编译并运行它，以对其进行测试的完整性和正确性。  
  
 您的业务流程具有编译后未出现错误，您可以将其部署在测试环境中之前进行任何必要的修改并最终将其在生产环境中运行部署。
