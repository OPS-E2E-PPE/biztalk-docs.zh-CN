# <a name="building-and-running-orchestrations"></a>生成并运行业务流程
设计业务流程之后，您可以对其进行编译和运行，以测试该业务流程是否完整和正确。  
  
 如果在编译该业务流程时没有出现错误，您就可以在测试环境中对其进行部署，然后进行任何必要的修改，最终将其部署为在生产环境中运行。
