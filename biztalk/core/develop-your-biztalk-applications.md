# <a name="develop-your-biztalk-applications"></a>开发 BizTalk 应用程序
本部分包含的开发人员负责创建 BizTalk 应用程序的信息。 一旦创建了这些应用程序，它们通常由管理员使用的操作和管理任务中找到的信息管理。  
  
 主题的开发 BizTalk Server 应用程序介绍了与开发 BizTalk Server 应用程序，其中包括此类内容架构定义、 消息映射、 接收和发送管道、 消息订阅相关联的活动和业务流程。  
  
 主题创建自定义组件讨论用于修改或扩展 BizTalk Server 基础结构的某些元素，如发送或接收适配器组件、 发送或接收管道组件和 functoid 组件的开发组件。 这些基础结构元素由 BizTalk Server 应用程序，但本身不是 BizTalk Server 应用程序的部分。
