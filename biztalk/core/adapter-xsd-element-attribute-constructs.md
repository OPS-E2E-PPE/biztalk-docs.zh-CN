# <a name="adapter-xsd-element-attribute-constructs"></a>适配器 XSD 元素属性构造

> [!NOTE]

>  不支持具有“枚举”facet 的固定属性。

>  配置 XSD 应具有名为的元素**uri**的 xsd: string 类型。
