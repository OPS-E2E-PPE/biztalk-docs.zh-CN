# <a name="single-sign-on-event-10774"></a>单一登录：事件 10774
## <a name="details"></a>详细信息  
  
|                 |                                                                                      |
|-----------------|--------------------------------------------------------------------------------------|
|  产品名称   |                              企业单一登录                               |
| 产品版本 |              [!INCLUDE[btsSSOVersion](../includes/btsssoversion-md.md)]              |
|    事件 ID     |                                        10774                                         |
|  事件源   |                                        ENTSSO                                        |
|    组件    |                                         不可用                                          |
|  符号名称  |                          ENTSSO_E_SSO_NOT_IN_NEW_SSO_ADMIN                           |
|  消息正文   | SSO 服务帐户不是指定的 SSO Administrators 帐户的成员。 |
  
## <a name="explanation"></a>解释  
 为了对 SSO 管理员帐户进行更改，您必须是该帐户的成员。  
  
## <a name="user-action"></a>用户操作  
 具有系统管理员帮助您联系 SSO 管理员帐户的成员。
