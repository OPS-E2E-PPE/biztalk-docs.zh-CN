# <a name="btadhostclass"></a>BTAD_HostClass
BTAD_HostClass 指示操作在 BizTalk 管理数据库中执行还是在本地计算机中执行。  
  
## <a name="remarks"></a>注释  
 下表介绍 BTAD_HostClass 的可能值。  
  
|值|Description|  
|-----------|-----------------|  
|ConfigurationDb|指定导入操作是否正在该组的 BizTalk 管理数据库中执行|  
|BizTalkHostInstance|指定本地计算机中正在执行的是安装操作还是卸载操作|
