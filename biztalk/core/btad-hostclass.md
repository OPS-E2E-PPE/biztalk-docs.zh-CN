# <a name="btadhostclass"></a>BTAD_HostClass
BTAD_HostClass 指示操作是否正在执行的 BizTalk 管理数据库或本地计算机上。  
  
## <a name="remarks"></a>备注  
 下表介绍 BTAD_HostClass 的可能值。  
  
|ReplTest1|Description|  
|-----------|-----------------|  
|ConfigurationDb|指定组的 BizTalk 管理数据库，正在执行导入操作|  
|BizTalkHostInstance|指定在安装或卸载本地计算机上正在执行操作|
