# <a name="how-to-configure-the-loop-shape"></a>如何配置循环形状
![](../core/media/ebiz-orch-loop.gif "ebiz_orch_loop")  
循环形状  
  
### <a name="to-configure-a-loop-shape"></a>配置循环形状  
  
1.  如果 BizTalk 表达式编辑器不可见，请右键单击**循环**的形状，然后单击**编辑布尔表达式**，或在属性窗口中，单击**省略号**(**...**) 按钮**表达式**属性。  
  
2.  在 BizTalk 表达式编辑器中，为迭代条件创建一个布尔表达式。 有关详细信息，请参阅[表达式的要求和限制](../core/requirements-and-limitations-for-expressions.md)。
