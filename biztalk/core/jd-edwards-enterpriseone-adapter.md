# <a name="jd-edwards-enterpriseone-adapter"></a>JD Edwards EnterpriseOne Adapter

[开始](../core/getting-started-with-biztalk-adapter-for-jd-edwards-enterpriseone.md)列出了安装适配器，并单步执行一些教程的步骤。
