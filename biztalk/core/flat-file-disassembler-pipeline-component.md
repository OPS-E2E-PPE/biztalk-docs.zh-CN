# <a name="flat-file-disassembler-pipeline-component"></a>平面文件拆装器管道组件
平面文件拆装器组件可解析分隔和位置平面文件格式的消息并将其转换为 XML 表示形式。 平面文件拆装器还从平面文件消息中，删除标头和尾部结构，并将消息中的交换分成各个文档。 它还可从文档和标头升级属性。  
  
 有关平面文件的详细信息，请参阅[带有位置记录的平面文件消息](../core/flat-file-messages-with-positional-records.md)。 另请参阅[带有分隔记录的平面文件消息](../core/flat-file-messages-with-delimited-records.md)。  
  
 有关配置平面文件拆装器管道组件的信息，请参阅[如何配置平面文件拆装器管道组件](../core/how-to-configure-the-flat-file-disassembler-pipeline-component.md)。
