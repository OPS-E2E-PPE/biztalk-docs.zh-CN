# <a name="error---input-validation"></a>错误-输入验证
**错误代码**  
  
 btm1044  
  
 **说明**  
  
 所指示的原因，无法验证根据源架构为测试映射操作指定的输入的实例消息文件。  
  
 **用户执行任何操作**  
  
 输入的实例消息文件或源架构中，或者对二者同时，根据所指示的原因，进行合适的更正。 可能会有帮助，BizTalk 编辑器中打开源架构和使用**验证架构**并**生成实例**右键单击解决方案资源管理器中的架构时可用的命令。
