# <a name="biztalk-server-as2-functionality"></a>BizTalk Server AS2 功能
[!INCLUDE[btsBizTalkServerNoVersion](../includes/btsbiztalkservernoversion-md.md)] 核心 BizTalk Server 功能和特定于 AS2 的 BizTalk Server 功能来处理 AS2 消息使用的组合。 这使得 BizTalk Server 可以执行 AS2 消息传送，同时利用其核心消息传送功能独有的处理。 如果通过一些特定于 EDI 的功能中的 AS2 传输 EDI 文档[!INCLUDE[btsBizTalkServerNoVersion](../includes/btsbiztalkservernoversion-md.md)]也将使用。  
  
 本部分介绍了基本 AS2 消息以及如何[!INCLUDE[btsBizTalkServerNoVersion](../includes/btsbiztalkservernoversion-md.md)]实现它。 它还介绍了如何利用 AS2 处理、 接收方 AS2 处理和发送端 AS2 处理的贸易合作伙伴协议定义。
