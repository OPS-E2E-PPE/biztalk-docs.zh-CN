# <a name="viewing-variables"></a>查看变量
你可以在两种模式中查看变量：  
  
- 当您正在跟踪存档的数据-脱机  
  
- 当执行操作上的实时数据进行交互式调试  
  
  当处理存档数据，您可以看到某些变量的状态是：  
  
- 尚不支持范围内的变量  
  
- 当前处于活动状态  
  
- 可以单击了解详细信息的值  
  
- 值已不在范围内;你不再能够访问任何详细信息
