# <a name="viewing-variables"></a>查看变量
可以采用两种模式查看变量：  
  
- 在跟踪存档数据时 - 脱机  
  
- 在对实时数据进行交互调试时  
  
  在处理存档数据时，可以看到某些变量的状态为：  
  
- 尚未在作用域中的变量  
  
- 当前处于活动状态  
  
- 单击该值可获取详细信息  
  
- 值已超出作用域；不能再访问任何详细信息
