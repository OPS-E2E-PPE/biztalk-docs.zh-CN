# <a name="how-to-open-the-functoid-toolbox"></a>如何打开 Functoid 工具箱
创建映射并选择源架构和目标架构之后，您可以在网格上放置 functoid。 Functoid 出现在[!INCLUDE[btsVStudioNoVersion](../includes/btsvstudionoversion-md.md)]工具箱中，每个 functoid 类别的某个工具箱选项卡。  
  
### <a name="to-open-the-functoid-toolbox"></a>打开 functoid 工具箱  
  
-   在[!INCLUDE[btsVStudioNoVersion](../includes/btsvstudionoversion-md.md)]BizTalk 映射打开和编辑器窗口中的处于活动状态的、 采用上**视图**菜单上，单击**工具箱**。  
  
    > [!NOTE]
    >  你还可以使用**Ctrl + Alt + X**或**工具箱**工具上**标准**显示 functoid 工具箱的工具栏。  
  
    > [!IMPORTANT]
    >  如果 functoid 选项卡在工具箱中不可见，请确保已通过在主编辑窗口中选择要编辑的 BizTalk 映射打开了 BizTalk 映射器。 如果工具箱是仍不可见，请参阅[疑难解答 BizTalk Server Tools](../core/troubleshooting-biztalk-server-tools.md)。  
  
    > [!IMPORTANT]
    >  有时，当您修改工具箱的默认设置时，可能会看不到工具箱中某些/所有 functoid。 然后，在[!INCLUDE[btsVStudioNoVersion](../includes/btsvstudionoversion-md.md)]，单击**工具**菜单，，然后单击**选择工具箱项**。 在对话框中，转到**BizTalk 映射程序 Functoid**选项卡。确保所有 functoid 是否选择，然后单击**确定**。  
    >   
    >  如果你仍未看到所有 functoid，请都参阅[疑难解答 BizTalk Server Tools](../core/troubleshooting-biztalk-server-tools.md)。
