# <a name="how-to-open-the-functoid-toolbox"></a>如何打开 Functoid 工具箱
在创建映射并选择源和目标架构之后，您可以在网格上放置 functoid。 Functoid 显示在[!INCLUDE[btsVStudioNoVersion](../includes/btsvstudionoversion-md.md)]工具箱中，每个 functoid 类别的一个工具箱选项卡。  
  
### <a name="to-open-the-functoid-toolbox"></a>若要打开 functoid 工具箱  
  
- 在[!INCLUDE[btsVStudioNoVersion](../includes/btsvstudionoversion-md.md)]、 使用 BizTalk 映射打开和在编辑器窗口中，活动上**视图**菜单中，单击**工具箱**。  
  
  > [!NOTE]
  >  此外可以使用**Ctrl + Alt + X**或**工具箱**上**标准**显示 functoid 工具箱的工具栏。  
  > 
  > [!IMPORTANT]
  >  如果 functoid 选项卡未显示在工具箱中，请确保已打开，请选择在主编辑窗口中编辑的 BizTalk 映射的 BizTalk 映射器。 如果工具箱中仍不可见，请参阅[故障排除 BizTalk Server 工具](../core/troubleshooting-biztalk-server-tools.md)。  
  > 
  > [!IMPORTANT]
  >  有时，当您修改工具箱的默认设置，你看不部分/所有工具箱中的 functoid。 然后，在[!INCLUDE[btsVStudioNoVersion](../includes/btsvstudionoversion-md.md)]，单击**工具**菜单，并单击**选择工具箱项**。 在对话框中，转到**BizTalk 映射器 Functoid**选项卡。确保所有 functoid 都选中，然后单击**确定**。  
  > 
  >  如果您仍然没有看到所有 functoid，请都参阅[故障排除 BizTalk Server 工具](../core/troubleshooting-biztalk-server-tools.md)。
