# <a name="how-to-open-a-saved-tracking-query"></a>如何打开已保存的跟踪查询
您可以打开针对实时跟踪数据 （目前运行服务实例和消息） 的消息或服务实例查询。 这是通过使用查询选项卡组中心页上从 BizTalk Server 管理控制台中。  
  
## <a name="prerequisites"></a>先决条件  
 您必须为要执行此过程的 BizTalk Server Administrators 组的成员身份登录。  
  
### <a name="to-open-a-saved-query"></a>若要打开已保存的查询  
  
1. 单击**启动**，单击**所有程序**，单击[!INCLUDE[btsBizTalkServerStartMenuItemui](../includes/btsbiztalkserverstartmenuitemui-md.md)]，然后单击**BizTalk Server 管理**。  
  
2. 上**组中心**页面上，在**新查询**选项卡上，单击**打开查询**。  
  
3. 单击查询，并依次**打开**。  
  
4. 如果第一次打开查询，将显示安全警告，或者选中**不再显示此消息**以防止其再次发生。 单击 **“是”** 继续操作。
