# <a name="access-control-to-business-information"></a>对业务信息的访问控制
只有 BizTalk Server 管理员能够访问 BizTalk 跟踪数据库中的跟踪数据（如跟踪消息属性和消息正文数据）。 BizTalk Server 操作员对 BizTalk 跟踪数据库中的消息流数据具有访问权限。
