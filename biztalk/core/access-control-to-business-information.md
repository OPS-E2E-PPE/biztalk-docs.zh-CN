# <a name="access-control-to-business-information"></a>对业务信息的访问控制
只有 BizTalk Server 管理员可以访问跟踪数据，如跟踪消息属性和 BizTalk 跟踪数据库中的消息正文数据。 BizTalk Server Operators BizTalk 跟踪数据库中有权访问消息流数据。
