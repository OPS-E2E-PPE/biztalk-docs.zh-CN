# <a name="xml-validator-pipeline-component"></a>XML 验证器管道组件
XML 验证器管道组件可在同时发送和接收管道中除拆装或组装之外的任何阶段。 XML 验证器组件验证指定的架构或架构，将根据对消息，如果消息不符合这些架构，则组件将产生错误并消息引擎将消息放置在挂起队列中。  
  
 有关配置 XML 验证器管道组件的信息，请参阅[如何配置 XML 验证器管道组件](../core/how-to-configure-the-xml-validator-pipeline-component.md)。
