# <a name="xml-validator-pipeline-component"></a>XML 验证器管道组件
XML 验证器管道组件可以用在除拆装或组装之外的任何阶段的发送管道和接收管道中。 XML 验证器组件根据指定的一个或多个架构验证消息，如果消息不符合这些架构，则该组件将产生一个错误，并且消息引擎也会将该消息放入挂起队列中。  
  
 有关配置 XML 验证器管道组件的信息，请参阅[如何将 XML 验证程序管道组件配置](../core/how-to-configure-the-xml-validator-pipeline-component.md)。
