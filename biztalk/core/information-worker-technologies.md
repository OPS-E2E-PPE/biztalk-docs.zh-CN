# <a name="information-worker-technologies"></a>信息工作者技术
Microsoft BizTalk Server 引擎为跨多个应用程序的自动化的业务流程提供支持。 但这些进程创建后，使用它们的信息工作者，业务人员，不是开发人员 — 具有其他的要求。 他们可能需要监视的过程中，各个业务相关的方面，例如，或与贸易合作伙伴以各种方式工作。  
  
 BizTalk Server 提供了多个组件，用于解决这些问题。 这些组件划归为业务活动监视，本部分中所述。
