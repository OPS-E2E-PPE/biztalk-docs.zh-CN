# <a name="information-worker-technologies"></a>信息工作者技术
Microsoft[!INCLUDE[btsBizTalkServer2006r3](../includes/btsbiztalkserver2006r3-md.md)]引擎提供支持跨多个应用程序的自动化的业务流程。 但是，在创建这些流程后，使用这些流程的信息工作者（业务人员，并非开发人员）还有其他要求。 例如，他们可能需要对流程中各种与业务相关的方面进行监视，或通过各种方式与贸易合作伙伴协同工作。  
  
 [!INCLUDE[btsBizTalkServer2006r3](../includes/btsbiztalkserver2006r3-md.md)]提供许多组件，解决这些问题。 这些组件划归为“业务活动监视”，本部分将对此进行介绍。
