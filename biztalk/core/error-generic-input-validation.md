# <a name="error---generic-input-validation"></a>错误-常规输入验证
**错误代码**  
  
 btm1045  
  
 **说明**  
  
 为测试映射操作指定的输入的实例消息文件不是根据源架构验证原因不明。  
  
 **用户执行任何操作**  
  
 在 BizTalk 编辑器中打开源架构并使用**验证架构**并**生成实例**命令中，右键单击解决方案资源管理器中，以找出问题的架构后即可。
