# <a name="error---generic-input-validation"></a>错误-泛型输入验证
**错误代码**  
  
 btm1045  
  
 **说明**  
  
 无法根据源架构来验证为测试映射操作指定的输入实例消息文件，具体原因并未指明。  
  
 **用户执行任何操作**  
  
 在 BizTalk 编辑器中打开源架构和使用**验证架构**和**生成实例**命令中，右键单击解决方案资源管理器，以找出问题中的架构时可用。
