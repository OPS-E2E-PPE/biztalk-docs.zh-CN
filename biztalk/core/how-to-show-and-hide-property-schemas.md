# <a name="how-to-show-and-hide-property-schemas"></a>如何显示和隐藏属性架构
本主题介绍如何使用 BizTalk Server 管理控制台在应用程序 Schemas 文件夹中显示和隐藏属性架构。 属性架构是 BizTalk 架构的简化版本，它在实例消息和消息上下文之间来回复制升级属性的过程中起到一定作用。 可能要隐藏属性架构以简化架构视图，从而只显示文档架构。  
  
## <a name="prerequisites"></a>必要條件  
 若要执行本主题中描述的过程，必须以 BizTalk Server Administrators 组或 BizTalk Server Operators 组成员的帐户身份登录。 有关详细的权限的信息，请参阅[用于部署和管理 BizTalk 应用程序所需权限](../core/permissions-required-for-deploying-and-managing-a-biztalk-application.md)。  
  
### <a name="to-show-or-hide-property-schemas-in-the-schemas-folder"></a>在 Schemas 文件夹中显示或隐藏属性架构  
  
1. 单击**启动**，单击**所有程序**，单击[!INCLUDE[btsBizTalkServerStartMenuItemui](../includes/btsbiztalkserverstartmenuitemui-md.md)]，然后单击**BizTalk Server 管理**。  
  
2. 在控制台树中，展开**BizTalk Server 管理**，展开包含想要显示或隐藏属性架构的架构文件夹的 BizTalk 组，然后展开包含此架构文件夹的应用程序。  
  
3. 右键单击**架构**文件夹，然后单击**隐藏属性架构**或**显示属性架构**。
