# <a name="btadchangerequestaction"></a>BTAD_ChangeRequestAction
BTAD_ChangeRequestAction 包含指示更改请求操作的值。  
  
## <a name="remarks"></a>备注  
 下表介绍 BTAD_ChangeRequestAction 的可能值。  
  
|ReplTest1|Description|  
|-----------|-----------------|  
|创建|指定导入而无需覆盖|  
|Update|导入或安装 （指定覆盖）|  
|DELETE|Uninstall|
