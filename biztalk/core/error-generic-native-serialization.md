# <a name="error---generic-native-serialization"></a>错误-泛型的本机序列化
**错误代码**  
  
 btm1049  
  
 **说明**  
  
 由映射指定的转换所生成的 XML 输出实例消息无法转换为目标架构所指定的本地格式，具体错误并未指明。  
  
 **用户执行任何操作**  
  
 在 BizTalk 编辑器中打开目标架构，并使用**验证架构**，**验证实例**，和**生成实例**命令中，右键单击时可用在解决方案资源管理器，以找出问题的架构。
