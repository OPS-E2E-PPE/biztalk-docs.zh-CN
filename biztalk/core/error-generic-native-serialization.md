# <a name="error---generic-native-serialization"></a>错误-常规本地序列化
**错误代码**  
  
 btm1049  
  
 **说明**  
  
 由于未指定错误的目标架构所指定的本机格式，无法转换由映射指定的转换生成的 XML 输出实例消息。  
  
 **用户执行任何操作**  
  
 在 BizTalk 编辑器中打开目标架构并使用**验证架构**，**验证实例**，并**生成实例**命令中，右键单击时可用在解决方案资源管理器，以找出问题的架构。
