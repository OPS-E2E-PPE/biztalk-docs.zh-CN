# <a name="error---property-schema-field-data-type-not-valid"></a>错误-属性架构字段数据类型无效
**错误代码**  
  
 BEC2010  
  
 **说明**  
  
 **数据类型**属性**Field 元素**属性架构中的节点必须设置为指定的数据类型之一。 由于属性升级仅限于这些类型的数据，因此，不能在属性架构中使用其他数据类型。  
  
 **用户执行任何操作**  
  
 选择指示**Field 元素**节点，然后在 Microsoft[!INCLUDE[btsVStudioNoVersion](../includes/btsvstudionoversion-md.md)]属性窗口中，更改**数据类型**属性设置为指定的数据类型之一。
