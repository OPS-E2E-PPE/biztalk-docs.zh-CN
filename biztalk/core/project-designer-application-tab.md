# <a name="project-designer-application-tab"></a>项目设计器：应用程序选项卡
使用应用程序选项卡提供程序集信息。  

|使用此选项|执行的操作|  
|--------------|----------------|  
|**程序集名称**|更新应用程序的程序集名称。|  
|**默认 Namespace**|更改应用程序的默认命名空间。|  

> [!CAUTION]
>  值不更改**目标框架**字段。 更改此属性的值会导致不可预知的行为。  

## <a name="assembly-information-dialog-box"></a>程序集信息对话框  
 当您单击时显示程序集信息对话框**程序集信息**按钮**应用程序**页。  


|           使用此选项            |                                                                       执行的操作                                                                        |
|-------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------|
|           **标题**           |                                                      更新应用程序的程序集名称。                                                      |
|        **说明**        |                                                          键入程序集的说明。                                                           |
|          **Company**          |                                                                 键入你的公司名称。                                                                 |
|          **Product**          |                                                                键入产品信息。                                                                |
|         **Copyright**         |                                                            键入你的版权信息。                                                             |
|         **Trademark**         |                                                               键入商标信息。                                                               |
|     **程序集版本**      |                                            键入按 major.minor.build.revision 格式的版本号。                                             |
|       **文件版本**        |                                          键入按 major.minor.build.revision 格式的文件版本号。                                          |
|           **GUID**            | 创建项目时[!INCLUDE[btsVStudioNoVersion](../includes/btsvstudionoversion-md.md)]生成标识的程序集的唯一 GUID。 |
|    **非特定语言**     |                                                      指定的区域性程序集支持。                                                       |
| **使程序集 COM 可见** |                               选中复选框以指定集中的类型是否可由 com 使用。                               |
