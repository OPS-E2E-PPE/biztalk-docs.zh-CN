# <a name="net-object-facts"></a>.NET 对象事实
在业务规则编辑器中，可以将 .NET 程序集指定为数据源。 随后可以从该程序集，选择一个类或类成员，并将其拖到词汇定义或规则。  
  
 添加的 .NET 对象流程是最简单的处理类型。 引擎将每个对象作为唯一的实例进行处理。 有关详细信息**断言**函数中，请参阅[引擎控制功能](../core/engine-control-functions.md)。
