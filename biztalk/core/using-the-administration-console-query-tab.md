# <a name="using-the-administration-console-query-tab"></a>使用管理控制台的查询选项卡
可以使用 BizTalk Server 管理控制台中的组中心页上的查询选项卡来搜索和查找特定的运行和挂起的服务实例、 消息或订阅。 使用管理控制台执行的查询找到存储在 MessageBox 数据库中的实时项。 新的查询选项卡将显示每次运行新查询。  
  
 若要定位跟踪或存档的消息或服务实例，使用消息事件和服务实例跟踪。 有关详细信息，请参阅[查看跟踪消息和实例数据](../core/viewing-tracked-message-and-instance-data.md)。  
  
> [!NOTE]
>  返回的结果集时执行的服务实例查询时，将显示的值**\<名称不可用\>** 有关**ServiceName**服务的字段实例的相应发送端口，如果接收位置，或业务流程已被删除。  **ServiceName**通过查找到的发送端口的友好名称的 BizTalk 管理数据库填充服务实例的字段、 接收位置或业务流程。  如果该发送端口，接收位置，或删除业务流程则的友好名称的查询失败并**\<名称不可用\>** 显示。
