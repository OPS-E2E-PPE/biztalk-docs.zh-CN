# <a name="business-activity-monitoring-biztalk-server-samples-folder"></a>业务活动监视 （BizTalk Server 示例文件夹）
Microsoft BizTalk Server 在其软件开发工具包 (SDK) 包含几个业务活动监视 (BAM) 示例。 本部分提供有关每个 BAM 示例，说明用于构建和运行示例，以及可以预期的结果所演示的功能的详细的信息。  
  
## <a name="in-this-section"></a>本节内容  
  
-   [BAM API （BizTalk Server 示例）](../core/bam-api-biztalk-server-sample.md)。 演示如何：  
  
    -   业务分析员定义感兴趣的数据使用向导，考察聚合和查询活动数据。  
  
    -   信息技术 (IT) 管理员创建 SQL、 数据转换服务 (DTS) 和联机分析处理 (OLAP) 基础结构。  
  
    -   开发员编写业务实现。  
  
-   [BAM 端对端 （BizTalk Server 示例）](../core/bam-end-to-end-biztalk-server-sample.md)。 演示如何将来自多个组件使用的 BAM 事件相关联。  
  
-   [业务流程表达式 （BizTalk Server 示例） 中的 BAM API](../core/bam-api-from-an-orchestration-expression-biztalk-server-sample.md)。 演示如何：  
  
    -   使用 BizTalk Server 业务流程中的 BAM API。  
  
    -   跟踪作为单独活动实例内部的一条消息重复项。  
  
    -   建立从使用跟踪配置文件和跟踪从 API 的数据业务流程跟踪的 BAM 数据之间的关系。
