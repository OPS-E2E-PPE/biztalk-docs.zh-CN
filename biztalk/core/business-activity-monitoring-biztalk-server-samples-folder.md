# <a name="business-activity-monitoring-biztalk-server-samples-folder"></a>业务活动监视 （BizTalk Server 示例文件夹中）
Microsoft [!INCLUDE[btsBizTalkServer2006r3](../includes/btsbiztalkserver2006r3-md.md)] 在其软件开发工具包 (SDK) 中包括了几个业务活动监视 (BAM) 示例。 本节提供每个 BAM 示例所演示功能的详细信息、生成和运行示例的说明以及可以预期得到的结果。  
  
## <a name="in-this-section"></a>本节内容  
  
-   [BAM API （BizTalk Server 示例）](../core/bam-api-biztalk-server-sample.md)。 演示如何：  
  
    -   业务分析员使用向导定义感兴趣的数据，考察聚合和查询活动数据。  
  
    -   信息技术 (IT) 管理员创建 SQL、数据转换服务 (DTS) 和联机分析处理 (OLAP) 基础结构。  
  
    -   开发员编写业务实现。  
  
-   [BAM 端到端 （BizTalk Server 示例）](../core/bam-end-to-end-biztalk-server-sample.md)。 演示如何使用 BAM 关联来自多个组件的事件。  
  
-   [业务流程表达式 （BizTalk Server 示例） 中的 BAM API](../core/bam-api-from-an-orchestration-expression-biztalk-server-sample.md)。 演示如何完成以下任务：  
  
    -   从 BizTalk Server 业务流程中使用 BAM API。  
  
    -   跟踪作为单独活动实例的消息内部的重复项。  
  
    -   建立从使用跟踪配置文件的业务流程中进行跟踪的 BAM 数据和从 API 中进行跟踪的数据之间的联系。
