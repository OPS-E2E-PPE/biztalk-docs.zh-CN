# <a name="single-sign-on-event-10841"></a>单一登录：事件 10841
## <a name="details"></a>详细信息  
  
|                 |                                                                                |
|-----------------|--------------------------------------------------------------------------------|
|  产品名称   |                           企业单一登录                            |
| 产品版本 |           [!INCLUDE[btsSSOVersion](../includes/btsssoversion-md.md)]           |
|    事件 ID     |                                     10841                                      |
|  事件源   |                                     ENTSSO                                     |
|    组件    |                                      不可用                                       |
|  符号名称  |                   ENTSSO_E_MAPPINGS_ONLY_FOR_DOMAIN_ACCOUNTS                   |
|  消息正文   | 可以为仅限 Windows 域帐户 （而非本地帐户） 创建映射。 |
  
## <a name="explanation"></a>解释  
 可以为仅限 Windows 域帐户 （而非本地帐户） 创建映射。  
  
## <a name="user-action"></a>用户操作  
 有关详细信息，请参阅[SSO 映射](../core/sso-mappings.md)。
