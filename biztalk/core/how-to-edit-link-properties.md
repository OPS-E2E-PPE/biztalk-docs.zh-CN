# <a name="how-to-edit-link-properties"></a>如何编辑链接属性
链接具有多个属性中显示[!INCLUDE[btsVStudioNoVersion](../includes/btsvstudionoversion-md.md)]**属性**窗口中的，如下所示：  
  
- **标签**属性提供的链接的简短说明，并且在表驱动循环中尤其有用。 璝惠**标签**属性，请参阅[如何管理现有链接](../core/how-to-manage-existing-links.md)。  
  
- **源链接**属性用于确定链接表示的元素的文本值或元素的名称。  
  
- **目标链接**属性使您能够控制在映射中处理链接的顺序。  
  
  ![无标注的屏幕截图](../core/media/properties-grid.gif "Properties_Grid")  
  
  有关源和目标架构的常规信息，请参阅[编译器指令和链接](../core/compiler-directives-and-links.md)。 有关设置和使用信息**源链接**并**目标链接**属性，请参阅[配置匹配节点层次结构如何](../core/how-to-configure-node-hierarchy-matching.md)和[如何设置源链接编译器值](../core/how-to-set-the-source-links-compiler-value.md)。
