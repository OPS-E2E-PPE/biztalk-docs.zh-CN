# <a name="mappings-enlisted-party-node"></a>映射 （已登记参与方节点）
绑定文件的映射节点是所有映射节点描述了参与方端口和角色端口类型操作之间的映射的父节点。  
  
## <a name="nodes-in-the-mappings-node"></a>在映射节点中的节点  
 下表列出了可为绑定文件的此节点设置的属性：  
  
|**名称**|**节点类型**|**数据类型**|**说明**|**限制**|**注释**|  
|--------------|-------------------|-------------------|---------------------|----------------------|------------------|  
|[映射](../core/mapping-mappings-node.md)|录制|EnlistedPartyMapping (ComplexType)|指定参与方端口和角色端口类型操作之间的映射。|可选|默认值：无|
