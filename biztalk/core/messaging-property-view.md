# <a name="messaging-property-view"></a>消息传送属性视图
消息传送属性视图显示与属性关联的 XML 消息的架构。 该视图是可从某些业务流程调度视图中的形状的快捷菜单。  
  
## <a name="working-with-messaging-properties"></a>使用消息传送属性  
 通过单击选择消息传送属性视图**选择事件源**按钮，单击**选择消息属性**菜单项。 然后选择要从中选择架构的消息属性。 一旦选择架构，您可以将元素从此架构到活动的数据项文件夹以指示您希望此操作的消息内的特定 XPath 表达式提取这些特定的数据。  
  
 可以通过右键单击包含消息负载的形状从业务流程调度视图打开消息属性视图。 这将打开上下文菜单，单击要检索的负载与形状相关联的列表的消息负载菜单项。  
  
 可用的消息属性的列表可能非常大。 如果您知道要搜索的业务流程的名称的一部分，您可以键入**字符串中**文本框中，然后单击**搜索**按钮。 这将选择包含您输入的部分字符串这些消息属性。
