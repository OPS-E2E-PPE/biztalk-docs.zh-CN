# <a name="tpe-keyboard-shortcuts"></a>TPE 键盘快捷方式
在跟踪配置文件编辑器 (TPE)，可以使用键盘快捷方式来执行相同的导航和编辑使用鼠标执行的函数。 这些快捷键用于执行在 TPE 中使用的所有功能。 本部分描述的键盘快捷方式，并提供有关使用键盘在 TPE 中导航的一些提示。  
  
## <a name="tpe-tabs"></a>TPE 选项卡  
 如果首先决定要创建新的配置文件或打开一个现有从 TPE 内的，tab 键定焦是最初在右窗格在事件源视图。  
  
 当您 tab 键离开活动视图时，tab 键定焦转到右窗格中，事件源视图。 当你在事件源视图，向上键和向下箭头键可用于导航框架的内容，并突出显示特定的业务流程形状或架构项。  
  
## <a name="tpe-keyboard-shortcuts"></a>TPE 键盘快捷方式  
 TPE 不带有鼠标导航提供键盘快捷方式。 下表列出了支持 TPE 的键盘快捷键。  
  
|操作|键盘快捷键|  
|------------|-----------------------|  
|创建新配置文件|ALT + F + N|  
|打开现有跟踪配置文件|ALT + F + O|  
|保存跟踪配置文件|ALT+ F + S|  
|另存为|ALT+ F + A|  
|导入活动定义|ALT + F + I|  
|删除上下文菜单项|DELETE|  
|重命名的上下文菜单项|F2|  
|打开跟踪配置文件编辑器帮助|F1|  
|应用跟踪配置文件|ALT+ T + A|  
|删除跟踪配置文件|ALT + T + R|  
|设置管理数据库|ALT + T + M|  
|消息负载架构|Shift + F10 + M|  
|退出，请关闭 TPE|ALT + F + X|  
|选择业务流程计划|ALT + S + O|  
|选择消息传送负载|ALT + S + M|  
|选择上下文属性|ALT + S + C|  
|选择消息传送属性|ALT + S + P|  
|上下文属性架构|Shift + F10 + M|  
|消息属性架构|Shift + F10 + P|
