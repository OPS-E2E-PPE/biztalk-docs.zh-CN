# <a name="error---exception-caught"></a>错误-捕获到异常
**错误代码**  
  
 btm1023  
  
 **说明**  
  
 遇到错误消息中所指示的异常，说明在映射验证或测试时出现错误。  
  
 **用户执行任何操作**  
  
 根据所指示的异常的性质采取正确的措施。
