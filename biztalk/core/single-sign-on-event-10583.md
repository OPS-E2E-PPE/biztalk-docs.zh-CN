# <a name="single-sign-on-event-10583"></a>单一登录：事件 10583
## <a name="details"></a>详细信息  
  
|                 |                                                                                                                            |
|-----------------|----------------------------------------------------------------------------------------------------------------------------|
|  产品名称   |                                                 企业单一登录                                                  |
| 产品版本 |                                 [!INCLUDE[btsSSOVersion](../includes/btsssoversion-md.md)]                                 |
|    事件 ID     |                                                           10583                                                            |
|  事件源   |                                                           ENTSSO                                                           |
|    组件    |                                                            不可用                                                             |
|  符号名称  |                                                   SSO_INFO_SSO_DISABLED                                                    |
|  消息正文   | SSO 系统已 disabled.%r<br /><br /> 跟踪 ID: %1 %r<br /><br /> 客户端计算机: %2 %r<br /><br /> 客户端用户： %3 |
  
## <a name="explanation"></a>解释  
 这是一条信息性消息，也可用于跟踪 SSO 系统中的发生的与安全相关的重要事件。 此消息表明 SSO 系统已被禁用。  
  
## <a name="user-action"></a>用户操作  
 不需要任何用户操作。
