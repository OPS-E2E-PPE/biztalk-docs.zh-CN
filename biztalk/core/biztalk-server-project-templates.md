# <a name="biztalk-server-project-templates"></a>BizTalk Server 项目模板
当你安装[!INCLUDE[btsBizTalkServerNoVersion](../includes/btsbiztalkservernoversion-md.md)]，安装过程会将添加到 BizTalk 项目文件夹**新项目**对话框。 BizTalk 项目文件夹包含用于创建空 BizTalk Server 项目和 BizTalk Server BPEL 导入项目的模板。  
  
> [!NOTE]
>  若要将用于 Biztalk 项目迁移[!INCLUDE[btsBizTalkServer2006r3](../includes/btsbiztalkserver2006r3-md.md)]，使用[Visual Studio 转换向导](http://go.microsoft.com/fwlink/?LinkId=124086)。  
  
 下表对 BizTalk 项目模板进行了说明。  
  
|使用此选项|执行的操作|  
|--------------|----------------|  
|**空的 BizTalk Server 项目**|创建新的空白 BizTalk Server 项目。|  
|**BizTalk Server BPEL 导入项目**|将业务处理执行语言 (BPEL)、Web Services 描述语言 (WSDL) 或 XML 架构定义语言 (XSD) 导入到 BizTalk 项目中。|
