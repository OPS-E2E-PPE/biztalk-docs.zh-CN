# <a name="how-to-use-the-schema-collection-property-editor"></a>如何使用架构集合属性编辑器
您可以使用架构集合属性编辑器为管道组件选择架构。  
  
### <a name="to-use-the-schema-collection-property-editor"></a>若要使用架构集合属性编辑器  
  
1.  在架构集合属性编辑器中，在**可用架构**框中，选择应可供该管道组件的架构。 使用 CTRL 键可选择多个架构。  
  
2.  单击 **“添加”**。  
  
     所选的架构此时显示在**添加架构**框。  
  
3.  单击 **“确定”**。
