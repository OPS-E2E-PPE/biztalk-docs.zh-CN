# <a name="single-sign-on-event-10650"></a>单一登录：事件 10650
## <a name="details"></a>详细信息  

|                 |                                                            |
|-----------------|------------------------------------------------------------|
|  产品名称   |                 企业单一登录                  |
| 产品版本 | [!INCLUDE[btsSSOVersion](../includes/btsssoversion-md.md)] |
|    事件 ID     |                           10650                            |
|  事件源   |                           ENTSSO                           |
|    组件    |                            N\A                             |
|  符号名称  |              SSO_INFO_PASSWORD_SYNC_STARTING               |
|  消息正文   | 将可用于此 computer.%r 密码同步服务  |

## <a name="explanation"></a>解释  
 此信息事件表明密码同步正在启动。
