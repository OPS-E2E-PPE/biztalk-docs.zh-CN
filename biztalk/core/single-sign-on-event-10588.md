# <a name="single-sign-on-event-10588"></a>单一登录： 事件 10588
## <a name="details"></a>详细信息  
  
|||  
|-|-|  
|产品名称|企业单一登录|  
|产品版本|[!INCLUDE[btsSSOVersion](../includes/btsssoversion-md.md)]|  
|事件 ID|10588|  
|事件源|ENTSSO|  
|组件|N/A|  
|符号名称|SSO_WARN_SQL_NOT_FOUND|  
|消息正文|找不到指定的 SQL Server。 检查 SQL Server 名称是否正确并且 SQL Server 是否在该计算机上运行。%r<br /><br /> SQL Server 名称： %1|  
  
## <a name="explanation"></a>解释  
 找不到消息中指定的 SQL Server。  
  
## <a name="user-action"></a>用户操作  
 检查 SQL Server 名称是否正确以及 SQL Server 是否在该计算机上运行。
