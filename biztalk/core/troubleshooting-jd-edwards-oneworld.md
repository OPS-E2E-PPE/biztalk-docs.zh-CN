# <a name="troubleshooting-jd-edwards-oneworld"></a>JD Edwards OneWorld 疑难解答

## <a name="overview"></a>概述
本节描述您可能在用于 JD Edwards OneWorld 的 BizTalk 适配器中遇到的常见问题和错误消息，并提供可能的更正方法。 此外，还讨论 windows 事件跟踪的用法。  
  
 可以使用下列调试/跟踪工具来解决适配器问题：  
  
-   BizTalk Server 本机调试（例如，您端口扩业务流程调试器上的跟踪类型）。  
  
-   提交到事件日志的错误消息。  
  
-   捕获的发送器、 接收方和管理消息通过 BTAJDEOneWorldTrace.cmd 和将.etl 文件，如 tracerpt.exe 或 tracedmp.exe 转换的工具。
