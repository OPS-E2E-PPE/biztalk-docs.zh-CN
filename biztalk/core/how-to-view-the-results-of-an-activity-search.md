# <a name="how-to-view-the-results-of-an-activity-search"></a>如何查看活动搜索的结果
用户可以通过单击查看查询的结果**执行查询**的 BAM 门户内容框架的上半部分中的按钮。 查询的结果将显示在**结果**框架底部的部分。  
  
 活动搜索结果在显示时每行显示一条记录。 每个记录包含的数据和里程碑中列选择器选择和排列中的顺序显示**要显示的项**框。  
  
 若要获取有关返回记录的其他详细信息，可以单击该行的任何地方。  
  
## <a name="prerequisites"></a>必要條件  
 只有具有已部署的活动，才可以执行本主题中的过程。 有关如何执行此操作的信息，请参阅[如何应用和删除跟踪配置文件](../core/how-to-apply-and-remove-a-tracking-profile.md)。  
  
## <a name="return-results"></a>返回的结果  
  
#### <a name="to-execute-a-query-to-return-results"></a>执行查询以返回结果  
  
1. 单击**启动**，依次指向**所有程序**，单击[!INCLUDE[btsBizTalkServerStartMenuItemui](../includes/btsbiztalkserverstartmenuitemui-md.md)]，然后单击**BAM 门户网站**。  
  
2. 在中**我的视图**窗格中，单击要为其创建活动搜索的视图。  
  
3. 单击与该视图相关联的某个活动，将其加载到内容框架中。  
  
4. 创建一个活动搜索或打开现有的活动搜索。 有关创建或打开活动搜索的信息，请参阅[如何在活动搜索中创建一个查询](../core/how-to-create-a-query-in-activity-search.md)。  
  
5. 单击**执行查询**按钮位于门户内容框架顶部。  
  
6. 向下滚动以查看中的查询的输出**结果**内容框架的区域。  
  
## <a name="view-results-information"></a>查看结果信息  
 您可以查看查询返回的项的详细状态。 显示的其他详细信息包括：  
  
- 活动状态，由相关的里程碑和数据组成  
  
- 相关文档  
  
- 相关活动  
  
  此外，你可以请求协助您的管理员通过单击**协助**下面的按钮**相关活动**内容框架中的区域。  
  
#### <a name="to-view-detailed-status-information-on-results"></a>查看结果的详细状态信息  
  
1.  按照上述过程执行查询以返回结果。  
  
2.  单击某行以加载该结果项的详细状态信息。  
  
3.  如果含有您感兴趣的详细信息的区域是折叠的，请单击该区域标签旁边的加号 (+) 打开它。
