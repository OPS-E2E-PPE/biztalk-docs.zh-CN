# <a name="sql-adapter"></a>SQL 适配器
[!INCLUDE[btsBizTalkServerNoVersion](../includes/btsbiztalkservernoversion-md.md)] 不再支持 SQL 适配器。 我们建议你使用[!INCLUDE[adaptersql](../includes/adaptersql-md.md)]来实现相同的功能。  
  
 但是，[!INCLUDE[btsBizTalkServerNoVersion](../includes/btsbiztalkservernoversion-md.md)]附带组件支持 SQL 适配器[!INCLUDE[btsBizTalkServerNoVersion](../includes/btsbiztalkservernoversion-md.md)]，如 EDI 控制消息 SQL 接收管道或 AS2 重新发送控制消息接收管道。
