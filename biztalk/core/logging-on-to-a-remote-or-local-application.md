# <a name="logging-on-to-a-remote-or-local-application"></a>登录到远程或本地应用程序的日志记录
您设置完关联凭据为您的用户后，可以使用企业单一登录服务 (ENTSSO) 提供对应用程序访问。 如果用户是本地的可以使用 ENTSSO 以非 Windows 应用程序检索凭据。 与此相反，如果用户是远程的您可以使用 ENTSSO 来检索本地应用程序的凭据。
