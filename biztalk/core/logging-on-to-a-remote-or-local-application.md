# <a name="logging-on-to-a-remote-or-local-application"></a>登录到远程或本地应用程序
为用户设置完关联凭据之后，可以使用企业单一登录服务 (ENTSSO) 提供对应用程序的访问。 如果用户为本地用户，可以使用 ENTSSO 为非 Windows 应用程序检索凭据。 相反，如果用户为远程用户，可以使用 ENTSSO 为本地应用程序检索凭据。
