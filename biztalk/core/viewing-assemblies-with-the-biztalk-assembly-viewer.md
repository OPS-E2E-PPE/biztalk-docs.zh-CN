# <a name="viewing-assemblies-with-the-biztalk-assembly-viewer"></a>查看具有 BizTalk 程序集查看器的程序集
当开发应用程序或部署程序集时，需要查看部署了哪些程序集以及可以使用哪些程序集。 Microsoft [!INCLUDE[btsBizTalkServerNoVersion](../includes/btsbiztalkservernoversion-md.md)] 为此提供了一个方便的工具，即 BizTalk 程序集查看器。  
  
 BizTalk 程序集查看器是一个 Windows 外壳程序扩展，可用于检查已部署的 BizTalk 程序集和程序集类型。 使用 BizTalk 程序集查看器可以：  
  
-   查看安装在本地计算机上的全局程序集缓存 (GAC) 中的所有 BizTalk Server 程序集。  
  
-   查看 BizTalk Server 程序集中的类型。  
  
-   查看 BizTalk Server 程序集的属性，如名称、版本、区域性、代码位置和公钥标记。  
  
-   查看 BizTalk Server 程序集的引用的程序集。  
  
-   在 GAC 中添加程序集。  
  
-   从 GAC 中删除程序集。  
  
-   查看各种类型（包括架构、映射、业务流程和管道）的 XML 代码。  
  
-   在给定服务器上已部署的全部 BizTalk Server 程序集中搜索特定类型的程序集。  
  
-   跨 BizTalk Server 程序集搜索类型依存关系。
