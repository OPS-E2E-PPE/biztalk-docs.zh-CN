# <a name="viewing-assemblies-with-the-biztalk-assembly-viewer"></a>查看程序集的 BizTalk 程序集查看器
开发应用程序或部署程序集时需要查看已部署并可供您的程序集。 Microsoft[!INCLUDE[btsBizTalkServerNoVersion](../includes/btsbiztalkservernoversion-md.md)]实现此目的的 BizTalk 程序集查看器提供的便利工具。  
  
 BizTalk 程序集查看器是用于检查已部署的 BizTalk 程序集和程序集类型的 Windows shell 扩展。 BizTalk 程序集查看器，可以：  
  
-   查看安装在本地计算机上的全局程序集缓存 (GAC) 中的所有 BizTalk Server 程序集的列表。  
  
-   查看 BizTalk Server 程序集内的类型。  
  
-   查看 BizTalk Server 程序集，例如名称、 版本、 区域性、 基本代码和公钥标记的属性。  
  
-   查看 BizTalk Server 程序集引用的程序集。  
  
-   添加程序集位于 GAC 中。  
  
-   从 GAC 删除程序集。  
  
-   查看包括架构、 映射、 业务流程和管道的各种类型的 XML 代码。  
  
-   搜索特定类型的给定服务器上部署的所有 BizTalk Server 程序集。  
  
-   在 BizTalk Server 程序集中搜索类型依存关系。
