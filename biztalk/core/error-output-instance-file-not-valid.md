# <a name="error---output-instance-file-not-valid"></a>错误-输出实例文件无效
**说明**  
  
 为指定的值**输出实例文件名**架构文件的属性不是有效的文件名称。  
  
 **用户执行任何操作**  
  
 右键单击在 Microsoft 中的相关架构文件[!INCLUDE[btsVStudioNoVersion](../includes/btsvstudionoversion-md.md)]解决方案资源管理器，然后单击**属性**以打开**属性页**对话框中，此架构。 在此对话框中，在**常规**选项卡上，单击省略号 (**...**) 的值字段中的按钮**输出实例文件名**属性，，然后使用**选择输出文件**对话框以标识将包含输出实例的文件消息。 此外可以直接在的值字段中键入文件名**输出实例文件名**属性。
