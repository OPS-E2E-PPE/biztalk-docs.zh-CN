# <a name="error---output-instance-file-not-valid"></a>错误-输出实例文件无效
**说明**  
  
 为指定的值**输出实例文件名**属性架构文件不是有效的文件名。  
  
 **用户执行任何操作**  
  
 右键单击相关的架构文件在 Microsoft[!INCLUDE[btsVStudioNoVersion](../includes/btsvstudionoversion-md.md)]解决方案资源管理器，并单击**属性**以打开**属性页**此架构的对话框。 在此对话框中，在**常规**选项卡上，单击省略号 (**...**) 中的值字段按钮**输出实例文件名**属性，，然后使用**选择输出文件**对话框以标识要包含输出实例文件消息。 此外可以直接在值字段中键入文件名称**输出实例文件名**属性。
