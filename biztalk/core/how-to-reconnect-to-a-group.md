# <a name="how-to-reconnect-to-a-group"></a>如何重新连接组
以前连接到 BizTalk server 的组有时可能会为 BizTalk Server 管理控制台中不可用。 例如，组所在的计算机重新启动时，一组可能会向管理控制台不可用。 您可以重新连接到不可用的 BizTalk Server 组使用**刷新**如果组是不可用。  
  
## <a name="prerequisites"></a>先决条件  
 若要执行此过程，必须为 BizTalk Server Operators 组的成员或 BizTalk Server Administrators 组的成员身份登录。  
  
### <a name="to-reconnect-to-a-group"></a>若要重新连接组  
  
1. 单击**启动**，单击**所有程序**，单击[!INCLUDE[btsBizTalkServerStartMenuItemui](../includes/btsbiztalkserverstartmenuitemui-md.md)]，然后单击**BizTalk Server 管理**。  
  
2. 在控制台树中，双击[!INCLUDE[btsBizTalkServerAdminConsoleui](../includes/btsbiztalkserveradminconsoleui-md.md)]。  
  
3. 右键单击 **（不可用）** 组，然后依次**刷新**。
