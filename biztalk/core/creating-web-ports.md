# <a name="creating-web-ports"></a>创建 Web 端口
Web 端口是用于使用（调用）Web 服务的专门配置端口。 一个 Web 端口可以包含表示混合使用单向（仅限请求）和双向（请求-响应）Web 方法的多个操作。 Web 端口中的每个操作表示 Web 服务的一个方法。  
  
 在创建 Web 端口之前，必须添加 Web 引用。  
  
> [!NOTE]
>  你不能删除、剪切、复制、粘贴或修改 Web 端口类型和 Web 消息类型的配置。
