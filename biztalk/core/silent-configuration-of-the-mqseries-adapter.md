# <a name="silent-configuration-of-the-mqseries-adapter"></a>MQSeries 适配器的无提示配置
可以使用 MQSAgent COM + 配置向导从命令行执行无提示配置。 此外可以使用命令行版本的向导来删除 MQSAgent。 该向导从 XML 文件读取配置信息。
