# <a name="silent-configuration-of-the-mqseries-adapter"></a>无提示 MQSeries 适配器配置
可以在命令行中使用 MQSAgent COM+ 配置向导来执行无提示配置。 还可以使用该向导的命令行版本来删除 MQSAgent。 该向导从 XML 文件中读取配置信息。
