# <a name="how-to-modify-vocabulary-definitions"></a>如何修改词汇定义
可以通过使用词汇定义向导更改显示名称或更改与显示名称关联的绑定修改未发布的词汇版本中的词汇定义。  
  
> [!NOTE]
>  可以修改不是所有元素。  
  
 本主题包含以下任务的过程：  
  
-   若要修改词汇定义  
  
-   若要向词汇或词汇定义添加事实  
  
### <a name="to-modify-a-vocabulary-definition"></a>若要修改词汇定义  
  
1.  右键单击词汇定义，然后依次**修改**。  
  
2.  使用词汇定义向导根据需要创建新的词汇定义。  
  
### <a name="to-add-a-fact-to-a-vocabulary-or-vocabulary-definition"></a>若要向词汇或词汇定义添加事实  
  
1.  在事实浏览器窗口中，单击**词汇**选项卡，并选择你想要更新的未发布的词汇版本。  
  
2.  单击**数据库**， **XML 架构**，或 **.NET 类**。  
  
3.  导航到所需的事实的数据源层次结构。  
  
    > [!NOTE]
    >  这一事实可以是数据库表或数据库表列、 XML 文档元素或属性，或.NET 类或类成员。  
  
4.  单击该事实并按住鼠标按钮。  
  
5.  该事实拖**词汇**选项卡。  
  
     **词汇**选项卡随即打开。  
  
6.  将该事实拖到你想要修改的词汇版本。
