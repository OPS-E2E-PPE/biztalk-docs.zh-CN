# <a name="how-to-invoke-the-orchestration-debugger-and-the-message-flow-views"></a>如何调用业务流程调试器和消息流视图
在 BizTalk Server 管理控制台的“组中心”页中，“查询结果”窗格显示查询的输出。 从上下文菜单中，可以显示业务流程调试器或消息流视图。  
  
## <a name="prerequisites"></a>先决条件  
 必须以 BizTalk Server Administrators 组成员的身份登录，才能执行此过程。  
  
### <a name="to-display-the-orchestration-debugger-or-message-flow-views"></a>显示业务流程调试器或消息流视图  
  
1.  若要显示的消息流视图，右键单击中的某个业务流程实例相关联的单元**查询结果**列表并选择**消息流**。  
  
2.  若要显示业务流程调试器，右键单击中的某个业务流程实例相关联的单元**查询结果**列表并选择**业务流程调试器**。
