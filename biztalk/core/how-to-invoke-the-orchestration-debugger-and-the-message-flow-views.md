# <a name="how-to-invoke-the-orchestration-debugger-and-the-message-flow-views"></a>如何调用业务流程调试器和消息流视图
在 BizTalk Server 管理控制台组中心页中，查询结果窗格显示查询的输出。 从上下文菜单中，您可以显示业务流程调试器或消息流视图。  
  
## <a name="prerequisites"></a>先决条件  
 您必须为要执行此过程的 BizTalk Server Administrators 组的成员身份登录。  
  
### <a name="to-display-the-orchestration-debugger-or-message-flow-views"></a>若要显示业务流程调试器或消息流视图  
  
1.  若要显示的消息流视图，请右键单击与在业务流程实例相关联的单元格**查询结果**列表，然后选择**消息流**。  
  
2.  若要显示业务流程调试器，请右键单击与在业务流程实例相关联的单元格**查询结果**列表，然后选择**业务流程调试器**。
