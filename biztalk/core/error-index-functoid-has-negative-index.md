# <a name="error---index-functoid-has-negative-index"></a>错误-索引 Functoid 具有负值索引
**错误代码**  
  
 btm1017  
  
 **说明**  
  
 一个或多个索引输入参数提供给所指示**索引**functoid 不是有效。 索引输入的参数必须为正整数。
