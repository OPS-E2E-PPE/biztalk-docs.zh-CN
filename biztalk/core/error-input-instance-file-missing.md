# <a name="error---input-instance-file-missing"></a>错误-输入实例文件缺少
**说明**  
  
 没有输入的实例消息文件已指定，阻止**验证实例**运行命令。  
  
 **用户执行任何操作**  
  
 右键单击在 Microsoft 中的相关架构文件[!INCLUDE[btsVStudioNoVersion](../includes/btsvstudionoversion-md.md)]解决方案资源管理器，然后单击**属性**以打开**属性页**对话框中，此架构。 在此对话框中，在**常规**选项卡上，单击省略号 (**...**) 的值字段中的按钮**输入实例 Filename**属性，，然后使用**选择输入文件**对话框以标识包含输入的实例消息的文件. 此外可以直接在的值字段中键入文件名**输入实例 Filename**属性。
