# <a name="error---input-instance-file-missing"></a>错误-输入实例文件丢失
**说明**  
  
 已指定任何输入的实例消息文件，防止**验证实例**命令从正在运行。  
  
 **用户执行任何操作**  
  
 右键单击相关的架构文件在 Microsoft[!INCLUDE[btsVStudioNoVersion](../includes/btsvstudionoversion-md.md)]解决方案资源管理器，并单击**属性**以打开**属性页**此架构的对话框。 在此对话框中，在**常规**选项卡上，单击省略号 (**...**) 中的值字段按钮**输入实例文件名**属性，，然后使用**选择输入文件**对话框以标识包含输入的实例消息的文件. 此外可以直接在值字段中键入文件名称**输入实例文件名**属性。
