# <a name="how-to-list-subscribers-to-an-alert"></a>如何列出警报的订户
管理员使用**获取订阅**命令以列出所有订阅服务器对指定的警报。  
  
### <a name="to-list-subscribers-to-an-alert"></a>对列出警报的订户  
  
1. 打开命令提示符，如下所示：单击**启动**，单击**运行**，类型**cmd**，然后单击**确定**。  
  
2. 导航到[!INCLUDE[btsBiztalkServerPath](../includes/btsbiztalkserverpath-md.md)]跟踪。  
  
3. 类型**bm 获取订阅的视图：\<视图名称\>-警报：\<警报名称\>**。  
  
   > [!NOTE]
   >  在支持用户帐户控制 (UAC) 的系统上，可能需要具有管理权限才能运行该工具。  
  
4. 按 **Enter**。
