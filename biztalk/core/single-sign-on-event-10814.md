# <a name="single-sign-on-event-10814"></a>单一登录：事件 10814
## <a name="details"></a>详细信息  
  
|                 |                                                            |
|-----------------|------------------------------------------------------------|
|  产品名称   |                 企业单一登录                  |
| 产品版本 | [!INCLUDE[btsSSOVersion](../includes/btsssoversion-md.md)] |
|    事件 ID     |                           10814                            |
|  事件源   |                           ENTSSO                           |
|    组件    |                            不可用                             |
|  符号名称  |               ENTSSO_E_REENCRYPT_IN_PROGRESS               |
|  消息正文   |         SSO 数据库重新加密正在进行中。         |
  
## <a name="explanation"></a>解释  
 重新加密 SSO 数据库时，不能执行此操作。  
  
## <a name="user-action"></a>用户操作  
 重试该操作。
