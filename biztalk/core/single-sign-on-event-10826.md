# <a name="single-sign-on-event-10826"></a>单一登录：事件 10826
## <a name="details"></a>详细信息  
  
|                 |                                                            |
|-----------------|------------------------------------------------------------|
|  产品名称   |                 企业单一登录                  |
| 产品版本 | [!INCLUDE[btsSSOVersion](../includes/btsssoversion-md.md)] |
|    事件 ID     |                           10826                            |
|  事件源   |                           ENTSSO                           |
|    组件    |                            不可用                             |
|  符号名称  |                 ENTSSO_E_NO_GROUP_ADAPTER                  |
|  消息正文   |        指定的组适配器不存在。         |
  
## <a name="explanation"></a>解释  
 指定的组适配器不存在。  
  
## <a name="user-action"></a>用户操作
