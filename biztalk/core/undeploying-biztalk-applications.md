# <a name="undeploying-biztalk-applications"></a>正在取消部署 BizTalk 应用程序
必须取消部署 BizTalk 应用程序以将其从 BizTalk 组和主机计算机。 本部分介绍在取消部署应用程序所涉及的以下任务：  
  
-   **删除 BizTalk 应用程序。** 这将从 BizTalk 组的 BizTalk 数据库中删除其数据。 执行此操作时，应用程序将不再显示在 BizTalk 管理控制台中。  
  
-   **卸载 BizTalk 应用程序。**  此删除的文件和设置与应用程序和从本地文件系统及其项目相关联。  
  
-   **删除其他文件和设置。** 您可能还需要删除某些其他文件和应用程序相关的设置。
