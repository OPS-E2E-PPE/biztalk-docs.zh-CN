# <a name="undeploying-biztalk-applications"></a>正在取消部署的 BizTalk 应用程序
你必须取消部署 BizTalk 应用程序从你的 BizTalk 组和主机计算机删除它。 本部分介绍中取消部署应用程序涉及以下任务：  
  
-   **删除 BizTalk 应用程序。** 这会从该 BizTalk 组的 BizTalk 数据库中删除其数据。 执行此操作时，此应用程序不再显示在 BizTalk 管理控制台中。  
  
-   **卸载 BizTalk 应用程序。**  这将删除与该应用程序相关联的文件和设置，并且从本地文件系统中删除其项目。  
  
-   **删除其他文件和设置。** 还可能需要删除与应用程序相关联的某些其他文件和设置。
