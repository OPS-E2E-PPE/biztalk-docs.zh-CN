# <a name="error---root-node-duplicate-class-name"></a>错误-根节点重复类名称
**错误代码**  
  
 BEC2013  
  
 **说明**  
  
 **RootNode TypeName**架构中的根节点的所有属性都必须唯一。 已找到两个或多个此架构中的根节点具有相同的值，其**RootNode TypeName**属性。  
  
 **用户执行任何操作**  
  
 选择相关的根节点反过来，从而确保你为提供的值及其**RootNode TypeName**属性是有效且唯一。
