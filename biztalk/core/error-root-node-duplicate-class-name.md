# <a name="error---root-node-duplicate-class-name"></a>错误-根节点类名重复
**错误代码**  
  
 BEC2013  
  
 **说明**  
  
 **RootNode TypeName**属性的所有根节点在架构中都必须唯一。 已找到两个或多个根节点，此架构中具有相同的值及其**RootNode TypeName**属性。  
  
 **用户执行任何操作**  
  
 确保您为提供的值中依次选择相关的根节点及其**RootNode TypeName**属性是有效且唯一。
