# <a name="monitoring-edi-and-as2-solutions"></a>监视 EDI 和 AS2 解决方案
通过 EDI 和 AS2 状态报告 UI，您可以显示传入和传出 EDI 及 AS2 消息的状态。 可以从查看状态报告**组概述**窗格中单击 BizTalk 组节点时可访问，链接到 EDI、 AS2 和批处理状态报告是否都存在底部**组概述**窗格。
