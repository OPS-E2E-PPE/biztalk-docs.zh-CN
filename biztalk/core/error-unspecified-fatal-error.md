# <a name="error---unspecified-fatal-error"></a>错误-未指定的错误
**错误代码**  
  
 BEC1001、BEC2004、BEC3001  
  
 **说明**  
  
 在架构编译过程中发生未指明的错误。  
  
 **用户执行任何操作**  
  
 确保架构文件有效且未被破坏。
