# <a name="trackedpropertynames-schema-node"></a>TrackedPropertyNames （Schema 节点）
绑定文件的 TrackedPropertyNames 节点包含指定的父架构中的哪些属性是要跟踪的元素。  
  
## <a name="nodes-in-the-trackedpropertyname-node"></a>TrackedPropertyName 节点中的节点  
 下表列出了可为绑定文件的此节点设置的属性：  
  
|**名称**|**节点类型**|**数据类型**|**说明**|**限制**|**注释**|  
|--------------|-------------------|-------------------|---------------------|----------------------|------------------|  
|string|元素|xs:string|指定要跟踪的属性名称。|可选|默认值：空|
