# <a name="what-is-a-real-time-aggregation"></a>什么是实时聚合？
业务分析员可以将透视表定义为实时聚合 (RTA) 或计划聚合。 RTA 可为您提供最新的数据视图，例如，特定 PO 在发货流程中所处的位置。 您可以随时刷新屏幕以更新数据视图。
