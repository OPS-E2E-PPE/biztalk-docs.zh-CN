# <a name="components-of-the-bam-portal"></a>BAM 门户组件
BAM 门户是划分为三个功能区域对应于在门户的三个页面的 ASP.NET 网页。 在本部分将介绍这些页：活动搜索页、 聚合页和警报管理器页。 这些页可用于创建活动，搜索现有活动，并修改现有活动。
