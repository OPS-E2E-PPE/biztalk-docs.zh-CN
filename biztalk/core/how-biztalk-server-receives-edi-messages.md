# <a name="how-biztalk-server-receives-edi-messages"></a>BizTalk Server 如何接收 EDI 消息
当[!INCLUDE[btsBizTalkServerNoVersion](../includes/btsbiztalkservernoversion-md.md)]接收 EDI 消息时，它执行贸易合作伙伴协议查找和架构发现，验证消息、 发送确认 （如果适用），并解析 EDI 批处理。 此处理将由 EDI 接收管道中的 EDI 拆装器执行。
