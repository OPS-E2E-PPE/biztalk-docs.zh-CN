# <a name="configuring-ports-for-an-edi-solution"></a>为 EDI 解决方案配置端口
若要接收和发送 EDI 消息和确认，请创建下列接收端口和发送端口：  
  
-   接收 EDI 交换和确认的单向静态 FILE 接收端口或接收 EDI 交换和发送确认的双向请求-响应 FILE 接收端口。  
  
-   发送 EDI 交换或确认的单向 FILE 发送端口。
