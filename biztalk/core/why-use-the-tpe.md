# <a name="why-use-the-tpe"></a>为什么要使用 TPE？
业务流程像订货、 开具发票和库存跟踪是电子商务的基本操作。 Microsoft[!INCLUDE[btsBizTalkServerNoVersion](../includes/btsbiztalkservernoversion-md.md)]使知识工作者来收集有关业务流程的状态的特定或聚合实时信息。 例如，它们可以找出为什么特定客户的订单已处理但尚未发货，或产品的多少个单位已售出和发送在以前的 48 小时内。  
  
 TPE 是端到端业务活动监视，Microsoft 的解决方案中的关键组件[!INCLUDE[btsBizTalkServerNoVersion](../includes/btsbiztalkservernoversion-md.md)]提供。 BAM 框架提供了轻松、 实时、 事务一致的方式来监视各种业务应用程序，并提供通过使用联机分析处理 (OLAP) 技术的 SQL 查询和聚合的报告的数据。 知识工作者可以使用此信息来评估其业务的当前状态，以便它们可以快速响应更改。
