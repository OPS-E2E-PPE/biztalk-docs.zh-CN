# <a name="why-use-the-tpe"></a>为何使用键入？
像订货、开具发票和库存跟踪这样的业务流程是电子商务的基本操作。 Microsoft[!INCLUDE[btsBizTalkServerNoVersion](../includes/btsbiztalkservernoversion-md.md)]使知识工作者能够收集有关业务流程的状态的特定或聚合实时信息。 例如，他们可以找出为什么特定客户的订单已处理但尚未发货，或者在过去 48 个小时内已售出和发送了多少产品。  
  
 键入是端到端业务活动监视 Microsoft 的解决方案中的关键组成部分[!INCLUDE[btsBizTalkServerNoVersion](../includes/btsbiztalkservernoversion-md.md)]提供。 BAM 框架以轻松、实时、事务一致的方式来监视各种不同的业务应用程序，并通过联机分析处理 (OLAP) 技术为 SQL 查询和聚合报告提供数据。 知识工作者可以利用这些信息来评估其业务的当前状态，以便对各种变化迅速做出响应。
