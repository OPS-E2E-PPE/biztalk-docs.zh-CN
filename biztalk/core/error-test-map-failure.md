# <a name="error---test-map-failure"></a>错误-测试映射失败
**错误代码**  
  
 btm1054  
  
 **说明**  
  
 测试映射操作失败指示的映射。  
  
 **用户执行任何操作**  
  
 使用随附的错误消息，确定为测试映射操作失败的原因，对代码图或其关联的源和目标架构中，或同时进行必要的更改。
