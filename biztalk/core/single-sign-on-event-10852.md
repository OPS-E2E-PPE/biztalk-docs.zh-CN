# <a name="single-sign-on-event-10852"></a>单一登录：事件 10852
## <a name="details"></a>详细信息  
  
|                 |                                                                                               |
|-----------------|-----------------------------------------------------------------------------------------------|
|  产品名称   |                                   企业单一登录                                   |
| 产品版本 |                  [!INCLUDE[btsSSOVersion](../includes/btsssoversion-md.md)]                   |
|    事件 ID     |                                             10852                                             |
|  事件源   |                                            ENTSSO                                             |
|    组件    |                                              不可用                                              |
|  符号名称  |                         ENTSSO_E_CREATE_FIELD_NO_DIRECT_PASSWORD_SYNC                         |
|  消息正文   | 无法创建该字段，因为此应用程序设置了直接密码同步标志。 |
  
## <a name="explanation"></a>解释  
 当应用程序具有直接密码同步标志设置时，您不能创建任何字段。  
  
## <a name="user-action"></a>用户操作  
 没有任何用户操作。 不支持此选项。
