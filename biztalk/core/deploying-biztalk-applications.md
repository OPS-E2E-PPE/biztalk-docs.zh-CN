# <a name="deploying-biztalk-applications"></a>部署 BizTalk 应用程序
本部分提供有关部署 BizTalk 应用程序的以下信息：  
  
- 将应用程序及其项目导出到 Windows Installer (.msi) 文件  
  
- .Msi 文件导入到其他 BizTalk 组，以便在新组中创建应用程序及其项目  
  
- 将运行它的计算机上安装应用程序  
  
- BizTalk 程序集安装到全局程序集缓存 (GAC)  
  
- 监视部署进度和测试应用程序  
  
- 应用程序部署到的业务合作伙伴  
  
- 编辑绑定文件以自定义不同的部署环境的绑定  
  
  有关需要在不同的应用程序部署方案中执行的任务的清单，请参阅[BizTalk 应用程序部署和管理清单](../core/biztalk-application-deployment-and-management-checklists.md)。 BizTalk 应用程序和其部署的背景信息，请参阅[了解 BizTalk 应用程序部署和管理](../core/understanding-biztalk-application-deployment-and-management.md)。 有关创建应用程序和修改通过添加或删除项目或添加另一个应用程序的引用信息，请参阅[创建和修改 BizTalk 应用程序](../core/creating-and-modifying-biztalk-applications.md)。 有关部署 BizTalk 程序集的信息，请参阅[从到 BizTalk 应用程序的 Visual Studio 部署 BizTalk 程序集](../core/deploying-biztalk-assemblies-from-visual-studio-into-a-biztalk-application.md)。
