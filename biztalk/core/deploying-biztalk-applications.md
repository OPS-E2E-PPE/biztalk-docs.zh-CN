# <a name="deploying-biztalk-applications"></a>部署 BizTalk 应用程序
本部分提供有关部署 BizTalk 应用程序的下列信息：  
  
-   将应用程序及其项目导出到 Windows Installer (.msi) 文件中  
  
-   将此 .msi 文件导入到其他 BizTalk 组中，以在新组中创建该应用程序及其项目  
  
-   将应用程序安装在要运行该应用程序的计算机上  
  
-   将 BizTalk 程序集安装到全局程序集缓存 (GAC) 中  
  
-   监视部署的进度并测试应用程序  
  
-   将应用程序部署到业务合作伙伴  
  
-   编辑绑定文件以为不同的部署环境自定义绑定  
  
 你需要在不同的应用程序部署方案中执行的任务的清单，请参阅[BizTalk 应用程序部署和管理清单](../core/biztalk-application-deployment-and-management-checklists.md)。 在 BizTalk 应用程序和其部署的背景信息，请参阅[了解 BizTalk 应用程序部署和管理](../core/understanding-biztalk-application-deployment-and-management.md)。 有关创建应用程序和对其进行修改通过添加或删除项目或添加对另一个应用程序的引用的信息，请参阅[创建和修改 BizTalk 应用程序](../core/creating-and-modifying-biztalk-applications.md)。 有关部署 BizTalk 程序集的信息，请参阅[部署 BizTalk 中的程序集到 BizTalk 应用程序的 Visual Studio](../core/deploying-biztalk-assemblies-from-visual-studio-into-a-biztalk-application.md)。
