# <a name="wcf-custom-adapter"></a>WCF 自定义适配器
WCF 自定义适配器是进程内适配器以允许配置接收位置和发送端口的绑定和行为设置为 Windows Communication Foundation (WCF) 适用于[!INCLUDE[btsCoName](../includes/btsconame-md.md)] [!INCLUDE[btsBizTalkServerNoVersion](../includes/btsbiztalkservernoversion-md.md)]。 WCF 自定义适配器可用于标准 WCF 适配器不支持的方案。 WCF 自定义适配器还可以使用 WCF 扩展点在[!INCLUDE[btsBizTalkServerNoVersion](../includes/btsbiztalkservernoversion-md.md)]。
