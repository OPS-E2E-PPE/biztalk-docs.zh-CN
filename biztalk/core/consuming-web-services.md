# <a name="consuming-web-services"></a>使用 Web 服务
使用 Web 服务，可以将现有的 Web 服务添加到您的业务流程。 可以将若干 Web services 聚合成单个业务流程。  
  
 通过使用 Web 端口，可以从您的业务流程使用 （调用） Web 服务。 若要使用业务流程中的 Web 服务，创建一个 Web 端口并构造 Web 消息。  
  
 可以使用 SOAP 标头与已使用的 Web 服务、 更改使用的 Web 服务的 URI 和动态设置已使用的 Web 服务的 URI。  
  
 了解如何配置 SOAP 接收处理程序，请参阅[如何配置 SOAP 接收处理程序](../core/how-to-configure-a-soap-receive-handler.md)。 了解如何配置 SOAP 接收位置，请参阅[如何配置 SOAP 接收位置](../core/how-to-configure-a-soap-receive-location.md)。
