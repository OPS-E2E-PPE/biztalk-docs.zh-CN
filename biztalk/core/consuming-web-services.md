# <a name="consuming-web-services"></a>使用 Web Services
通过使用 Web Services，可以将现有的 Web Services 添加到业务流程中。 可以将若干 Web Services 聚合成单个业务流程。  
  
 可以借助 Web 端口来从业务流程中使用（调用）Web Services。 若要从业务流程中使用 Web Services，需创建一个 Web 端口并构造 Web 消息。  
  
 可以将 SOAP 标头与所使用的 Web Services 一起使用，更改所使用的 Web Services 的 URI，并动态设置所使用的 Web Services 的 URI。  
  
 璝惠砞 SOAP 接收处理程序，请参阅[如何配置一个 SOAP 接收处理程序](../core/how-to-configure-a-soap-receive-handler.md)。 璝惠砞 SOAP 接收位置，请参阅[如何配置 SOAP 接收位置](../core/how-to-configure-a-soap-receive-location.md)。
