# <a name="bam-wcf-interceptor"></a>BAM WCF 侦听器
BAM WCF 侦听器提供全面的支持，以便跟踪您的 WCF 服务中的数据。  
  
 你可以使用到 BAM WCF 侦听器：  
  
-   以透明方式将工作流应用程序数据传递到 BAM，而无需重新编译 WCF 应用程序。  
  
-   从参数和 WCF 服务调用中的消息跟踪信息。  
  
-   从 WCF 服务调用中的消息跟踪信息。  
  
-   参与事务从客户端流入或内部事务处理的服务调用 WCF 运行时自动启动的。
