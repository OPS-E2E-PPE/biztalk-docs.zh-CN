# <a name="bam-wcf-interceptor"></a>BAM WCF 侦听器
BAM WCF 侦听器提供跟踪您的 WCF 服务中的数据的全面支持。  
  
 可以使用 BAM WCF 侦听器到：  
  
-   以透明方式将工作流应用程序数据传送给 BAM，无需重新编译您的 WCF 应用程序。  
  
-   从参数和 WCF 服务调用中的消息跟踪信息。  
  
-   从 WCF 服务调用中的消息跟踪信息。  
  
-   参与事务从客户端流入或在内部由 WCF 运行时针对事务的服务调用启动。
