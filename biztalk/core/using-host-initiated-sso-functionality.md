# <a name="using-host-initiated-sso-functionality"></a>使用主机启动的 SSO 功能
主机启动的单一登录使用 Windows Server 2008 的协议转换功能来执行非 Windows 用户的单一登录。 此功能需要 Wiiinnndows Server 2008，并且必须在具有的域及其**域功能级别**设置为**Windows Server 2008**。
