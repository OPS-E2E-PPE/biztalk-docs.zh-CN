# <a name="as2-security"></a>AS2 安全性
Microsoft [!INCLUDE[btsBizTalkServerNoVersion](../includes/btsbiztalkservernoversion-md.md)] 的本机 AS2 功能通过使用加密密钥、数字签名和证书来保护 AS2 消息的安全。 这要求对 [!INCLUDE[btsBizTalkServerNoVersion](../includes/btsbiztalkservernoversion-md.md)] 以及证书进行正确配置。  
  
 有关为签名和/或加密的 AS2 消息的结构的信息，请参阅[AS2 消息](../core/as2-messages.md)。
