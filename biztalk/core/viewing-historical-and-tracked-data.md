# <a name="viewing-historical-and-tracked-data"></a>查看历史记录和跟踪数据
本部分提供有关如何在 BizTalk Server 管理控制台中查看历史记录和跟踪数据，从组中心页使用的特定于任务的信息。  
  
-   显示消息所采用的处理步骤。 有关消息的处理步骤的详细信息，请参阅[查看消息流](../core/viewing-message-flow.md)。  
  
-   显示当前在你的 BizTalk 跟踪数据库和数据从数据存档还原数据。 有关存档和跟踪数据的详细信息，请参阅[如何选择实时或存档的数据源](../core/how-to-select-a-live-or-archived-data-source.md)。  
  
-   限制对那些具有适当的权限的用户数据访问。 关于数据访问的详细信息，请参阅[访问控制和数据安全性](../core/access-control-and-data-security.md)。  
  
-   可以进行实时调试您的业务流程。 有关详细信息实时调试，请参阅[使用业务流程调试器视图](../core/working-with-the-orchestration-debugger-view.md)。
