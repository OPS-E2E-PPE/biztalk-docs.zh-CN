# <a name="developer-tools"></a>开发人员工具
Microsoft[!INCLUDE[btsBizTalkServerNoVersion](../includes/btsbiztalkservernoversion-md.md)]包含用于设计、 组织和构建 BizTalk 应用程序的各种元素的 BizTalk 项目系统设计环境。  
  
 BizTalk 项目系统托管在 Microsoft[!INCLUDE[btsVStudioNoVersion](../includes/btsvstudionoversion-md.md)]和您提供了集成化的设计体验。 它是提供适用于 Visual Studio 的多个项目系统之一，可以使用许多标准的 Visual Studio 工具、 编辑器和其他用户界面。 例如，可以使用解决方案资源管理器来管理业务解决方案和属性窗口来查看和编辑文件、 项目和解决方案的属性中的 BizTalk 项目。  
  
 在本部分中的内容假定您熟悉[!INCLUDE[btsVStudioNoVersion](../includes/btsvstudionoversion-md.md)]。 如果您不熟悉[!INCLUDE[btsVStudioNoVersion](../includes/btsvstudionoversion-md.md)]，查看[!INCLUDE[btsVStudioNoVersion](../includes/btsvstudionoversion-md.md)]文档。 在本部分中的内容重点介绍 BizTalk 项目系统。
