# <a name="developer-tools"></a>开发人员工具
Microsoft [!INCLUDE[btsBizTalkServerNoVersion](../includes/btsbiztalkservernoversion-md.md)] 包含用于设计、组织和生成各种 BizTalk 应用程序元素的 BizTalk 项目系统设计环境。  
  
 BizTalk 项目系统以 Microsoft [!INCLUDE[btsVStudioNoVersion](../includes/btsvstudionoversion-md.md)] 为宿主，为您提供了集成化的设计体验。 它是可用于 Visual Studio 的若干项目系统之一，您可以使用许多 Visual Studio 标准工具、编辑器和其他用户界面。 例如，您可以使用解决方案资源管理器来管理业务解决方案中的 BizTalk 项目，并使用“属性”窗口来查看和编辑文件、项目及解决方案的属性。  
  
 本部分的内容建立在您已经熟悉 [!INCLUDE[btsVStudioNoVersion](../includes/btsvstudionoversion-md.md)] 的基础之上。 如果不熟悉 [!INCLUDE[btsVStudioNoVersion](../includes/btsvstudionoversion-md.md)]，则请查看 [!INCLUDE[btsVStudioNoVersion](../includes/btsvstudionoversion-md.md)] 文档。 本部分中的内容重点介绍 BizTalk 项目系统。
