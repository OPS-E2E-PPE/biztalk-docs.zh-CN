# <a name="links-to-and-from-the-any-element-and-anyattribute-nodes"></a>与“Any Element”和“anyAttribute”节点相关的链接
必须使用**大容量复制**functoid 的地图元素包含**任何**或**anyAttribute**节点。 BizTalk 映射程序不支持之间进行直接链接**任何**或**anyAttribute**节点。 **任何**和**anyAttribute**节点，结合**大容量复制**functoid，通常用于将映射的 HTML 文档部分。
