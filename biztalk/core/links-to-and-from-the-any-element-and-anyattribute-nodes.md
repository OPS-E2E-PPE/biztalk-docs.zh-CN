# <a name="links-to-and-from-the-any-element-and-anyattribute-nodes"></a>链接到和从任何元素和 anyAttribute 节点
必须使用**批量复制**functoid 的地图元素包含**任何**或**anyAttribute**节点。 BizTalk 映射器不支持之间的直接链接**任何**或**anyAttribute**节点。 **任何**并**anyAttribute**节点，结合**批量复制**functoid，最常用于映射 HTML 文档的部分。
