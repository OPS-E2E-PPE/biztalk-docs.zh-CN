# <a name="communication-pattern"></a>通信模式
每个*端口类型*具有通信模式。 通信模式决定了在给定类型的端口上是采用单向传输还是双向（请求响应）传输。 有关详细信息，请参阅**端口配置向导** [!INCLUDE[ui-guidance-developers-reference](../includes/ui-guidance-developers-reference.md)]。
  
> [!NOTE]
>  同一类型的不同端口将始终具有相同的通信模式，但不必具有相同的通信方向。
