# <a name="warning---envelope-schema-needs-root-record-node"></a>警告-信封架构需要根记录节点
**错误代码**  
  
 BEC1010  
  
 **说明**  
  
 信封架构都需要有至少一个根**记录**节点的子级作为其**架构**节点。  
  
 **用户执行任何操作**  
  
 右键单击**架构**节点，单击**插入子记录**上为快捷菜单，然后按插入的描述性名称**记录**节点。 一般情况下，信封架构也将具有插入根目录内的其他节点**记录**节点。
