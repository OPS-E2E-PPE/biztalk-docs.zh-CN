# <a name="implementing-enterprise-single-sign-on"></a>实现企业单一登录
企业单一登录 (SSO) 提供服务，以启用单一登录到企业应用程序集成 (EAI) 解决方案的最终用户。 SSO 系统将 Microsoft Windows 帐户映射到后端凭据。 SSO 简化了管理用户 Id 和密码，同时为用户和管理员。 它使用户能够登录到 Windows 网络仅一次访问后端系统和应用程序。
