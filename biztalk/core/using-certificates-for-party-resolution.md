# <a name="using-certificates-for-party-resolution"></a>将证书用于参与方解析
A*方*是外部 BizTalk Server 业务流程交互的实体。 在 BizTalk Server 接收某一消息时，它使用公钥证书确定该消息的发件人并且将该发件人解析为 BizTalk Server 环境中的已知参与方。  
  
 本部分提供有关如何配置信息[!INCLUDE[btsBizTalkServerNoVersion](../includes/btsbiztalkservernoversion-md.md)]管道，接收位置、 端口和参与方解析的 BizTalk Server 环境。  
  
 有关消息的身份验证的详细信息，请参阅[进行身份验证消息的发件人](../core/authenticating-the-sender-of-a-message.md)。
