# <a name="using-certificates-for-party-resolution"></a>为参与方解析使用证书
一个*参与方*是与业务流程交互的 BizTalk Server 外部的实体。 在 BizTalk Server 接收一条消息，它使用公钥证书，以确定谁发送的消息，并将发件人解析为 BizTalk Server 环境中的已知参与方。  
  
 本部分提供有关如何配置信息[!INCLUDE[btsBizTalkServerNoVersion](../includes/btsbiztalkservernoversion-md.md)]管道、 接收位置、 端口和参与方解析为 BizTalk Server 环境。  
  
 有关消息的身份验证的详细信息，请参阅[进行身份验证消息的发件人](../core/authenticating-the-sender-of-a-message.md)。
