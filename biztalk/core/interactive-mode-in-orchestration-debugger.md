# <a name="interactive-mode-in-orchestration-debugger"></a>业务流程调试器中的交互模式
在交互式-调试-模式，业务流程调试器视图包含三个方面： 服务窗格中、 跟踪事件窗格中和业务流程窗格。 此外，在交互模式下，在视图底部会依次显示“变量列表”和“变量属性”。  
  
> [!NOTE]
>  除非在“断点”模式下显示服务并将该服务附加到实例中，否则业务流程调试器将无法显示该服务的真实状态。
