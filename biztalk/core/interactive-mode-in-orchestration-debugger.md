# <a name="interactive-mode-in-orchestration-debugger"></a>业务流程调试器中的交互模式
在交互-调试-模式下，业务流程调试器视图包含三个区域： 服务窗格中，跟踪的事件窗格和业务流程窗格。 此外，在交互模式下，在视图底部会依次显示“变量列表”和“变量属性”。  
  
> [!NOTE]
>  业务流程调试器无法显示该服务的真实状态，除非它将出现在断点模式并已附加到的实例。
