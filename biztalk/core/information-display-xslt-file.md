# <a name="information---display-xslt-file"></a>信息-显示 XSLT 文件
**错误代码**  
  
 btm1055  
  
 **说明**  
  
 为执行指定映射而生成的 XSLT 已写入所指示的文件，该映射将与源架构一致的实例消息转换为与目标架构一致的实例消息。  
  
 **用户执行任何操作**  
  
 可以使用提供的链接访问生成的 XSLT，然后可在其他工具（例如 XSLT 调试器）中使用它来执行各种任务。
