# <a name="how-to-work-with-multiple-nodes-simultaneously"></a>如何同时处理多个节点
某些操作可以同时，在多个节点上执行，但相关的节点必须所有可选定。 本主题提供有关如何同时使用多个节点的说明。  
  
### <a name="to-select-multiple-nodes-that-are-not-adjacent"></a>若要选择不相邻的多个节点  
  
1.  选择要在同一时间选择不相邻节点集中的第一个节点。  
  
2.  按和按住 ctrl 键，然后选择在同一时间要选择不相邻节点集中的剩余的节点。  
  
     在同一时间均被选中不相邻节点。  
  
### <a name="to-select-a-range-of-adjacent-nodes"></a>选择相邻的节点的范围  
  
1.  选择在极端情况的一套相邻的节点在同一时间要选择的节点。  
  
2.  按住 SHIFT，然后选择要在同一时间选择的节点集的另一种极端节点。  
  
     在同一时间均被选中的范围由步骤 1 和 2 中选定的节点定义的相邻节点。  
  
### <a name="to-modify-property-values-of-simultaneously-selected-nodes"></a>若要修改的同时所选节点的属性值  
  
1.  选择多个节点，"选择不相邻的多个节点"的过程中所述。 本文前面。  
  
2.  修改中的相关属性值**属性**窗口。  
  
> [!NOTE]
>  可以同时更改仅对所选的所有节点通用的属性。
