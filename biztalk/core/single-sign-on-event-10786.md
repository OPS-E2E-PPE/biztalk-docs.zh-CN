# <a name="single-sign-on-event-10786"></a>单一登录：事件 10786
## <a name="details"></a>详细信息  
  
|                 |                                                                                                         |
|-----------------|---------------------------------------------------------------------------------------------------------|
|  产品名称   |                                        企业单一登录                                        |
| 产品版本 |                       [!INCLUDE[btsSSOVersion](../includes/btsssoversion-md.md)]                        |
|    事件 ID     |                                                  10786                                                  |
|  事件源   |                                                 ENTSSO                                                  |
|    组件    |                                                   不可用                                                   |
|  符号名称  |                                       ENTSSO_E_ACCOUNT_NOT_VALID                                        |
|  消息正文   | 帐户名称无效或不存在。 查看更多详细信息的日志 （在计算机"%1"）。 |
  
## <a name="explanation"></a>解释  
 帐户名称无效或不存在。  
  
## <a name="user-action"></a>用户操作  
 检查名称和拼写的帐户，然后重试。
