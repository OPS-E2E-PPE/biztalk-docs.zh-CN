# <a name="managing-bam-definitions"></a>管理 BAM 定义
BAM 定义是 BAM 基础结构的一部分。 它定义要跟踪和聚合的数据，以及所跟踪数据的业务最终用户视图。 本部分的主题介绍管理 BAM 定义的各个元素（包括活动、视图、项目和警报）的过程。
