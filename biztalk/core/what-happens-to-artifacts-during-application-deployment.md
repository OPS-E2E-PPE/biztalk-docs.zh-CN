# <a name="what-happens-to-artifacts-during-application-deployment"></a>对项目采取的应用程序部署过程
在本部分中的主题介绍添加和删除项目、 导出和导入应用程序和项目，并安装和卸载应用程序时，与应用程序关联的项目会发生什么情况。 它涵盖主题如下所示：  
  
-   如何添加到和从 BizTalk Server 数据库中删除项目  
  
-   如何复制到和从本地文件系统中删除项目  
  
-   如何添加或从 Windows 注册表中删除项目 （.NET 程序集和 COM 组件）  
  
-   如何安装或在全局程序集缓存中卸载项目 （BizTalk 程序集和.NET 程序集）  
  
-   如何将证书添加到本地证书存储区
