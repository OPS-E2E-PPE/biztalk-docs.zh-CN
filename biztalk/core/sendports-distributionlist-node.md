# <a name="sendports-distributionlist-node"></a>发送端口 （DistributionList 节点）
绑定文件的 DistributionList 节点的发送端口节点是通讯组列表中的发送端口引用的容器节点。  
  
> [!NOTE]
>  通讯组列表称为发送端口组在 BizTalk 管理器。  
  
## <a name="nodes-in-the-sendports-node"></a>节点中的发送端口节点  
 下表列出了可为绑定文件的此节点设置的属性：  
  
|**名称**|**节点类型**|**数据类型**|**说明**|**限制**|**注释**|  
|--------------|-------------------|-------------------|---------------------|----------------------|------------------|  
|[SendPortRef](../core/sendportref-sendports-node.md)|录制|SendPortRef (ComplexType)|对所做的通讯组列表的发送端口的引用的容器节点。|可选|默认值：无|
