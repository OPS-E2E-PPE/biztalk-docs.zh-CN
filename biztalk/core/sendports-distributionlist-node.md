# <a name="sendports-distributionlist-node"></a>发送端口 （DistributionList 节点）
绑定文件 DistributionList 节点发送端口节点是通讯组列表中的发送端口引用的容器节点。  
  
> [!NOTE]
>  分发列表指 BizTalk 管理员中的发送端口组。  
  
## <a name="nodes-in-the-sendports-node"></a>发送端口节点中的节点  
 下表列出了可为绑定文件的此节点设置的属性：  
  
|**名称**|**节点类型**|**数据类型**|**Description**|**限制**|**注释**|  
|--------------|-------------------|-------------------|---------------------|----------------------|------------------|  
|[SendPortRef](../core/sendportref-sendports-node.md)|录制|SendPortRef (ComplexType)|容器的引用所做的通讯组列表发送端口的节点。|可选|默认值：无|
