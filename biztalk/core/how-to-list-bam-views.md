# <a name="how-to-list-bam-views"></a>如何列出 BAM 视图
管理员使用**get 视图**命令以列出在其运行该命令的计算机上的 BAM 主导入数据库中部署的 BAM 视图。  
  
### <a name="to-list-bam-views"></a>列出 BAM 视图  
  
1. 打开命令提示符，如下所示：单击**启动**，单击**运行**，类型**cmd**，然后单击**确定**。  
  
2. 导航到[!INCLUDE[btsBiztalkServerPath](../includes/btsbiztalkserverpath-md.md)]。  
  
3. 类型**bm get 视图**。  
  
4. 按 **Enter**。
